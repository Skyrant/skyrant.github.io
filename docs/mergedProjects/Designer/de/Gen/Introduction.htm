Einführung

Der GefasoftDesigner stellt ein Werkzeug dar, um schnell und komfortabel Bilder zur Prozessvisualisierung des Legato Systems erstellen zu können. Als Grundlage dient das OpenSource Grafikformat SVG (Scalable Vector Graphics). Jedoch wird im Designer nicht der komplette Umfang des Dateiformates abgebildet. Er beschränkt sich auf die Funktionen, die für eine technisch orientierte Visualisierung notwendig sind. Neben dem Zeichnen von grafischen Elementen bietet das Leistungsspektrum dieses Produktes die Nutzung von Symbolbibliotheken, Einbinden von JavaScript und Dynamisierungen für das Legato System.

Nach der Erstellung der SVGs werden diese auf den WebServer hochgeladen. Es wird empfohlen, die Verzeichnisstruktur auf dem WebServer inklusive Unterverzeichnisse zum Zeichnen lokal nachzuziehen (z.B. Ordner "SvgSrc" für SVGs und "images" für Bilder, die in SVGs verwendet werden). So wird sichergestellt, dass in der WebApp auch alles korrekt dargestellt werden kann.

Für erweiterte Parametrierung und Beispiele zur Anwendung der einzelnen Funktionen, Dynamisierungen etc., siehe bitte auch die Schulungs-Unterlagen sowie das Workshop-Dokument GefasoftDesigner.

:

Es kann zu Darstellungsfehlern innerhalb des Gefasoft Designers kommen, wenn die Anzeige-Optionen von Windows geändert werden, um die Lesbarkeit zu erleichtern (über Systemsteuerung – Anzeige). Ist hier ein anderer Wert als 100% eingestellt, kann dies Probleme verursachen. Insbesondere werden dann Schriften falsch dargestellt.
