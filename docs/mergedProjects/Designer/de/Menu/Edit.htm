Klicken Sie hier, um diese Seite mit vollem Kontext anzuzeigen.

Edit

Undo

Letzten Schritt rückgängig machen.

Redo

Rückgängig gemachte Schritte wiederherstellen.

Cut

Schneidet Elemente aus dem Designer oder den Code-Editoren aus und kopiert sie in die Zwischenablage.

Copy

Kopiert Elemente aus dem Designer oder den Code-Editoren und kopiert sie in die Zwischenablage.

Paste

Fügt Elemente aus der Zwischenablage ein. Die Elemente werden immer in X- und Y-Richtung um einen Offset von 4 versetzt.

Paste at

Vor dem Einfügen eines Elements wird ein Dialog geöffnet zur Eingabe von beliebigen X- und Y-Werten (absolut oder Delta). Dementsprechend ergibt sich beim Einfügen ein beliebiger Versatz oder eine exakte Position.

Delete

Löscht ein Element oder einen selektierten Text.

Send to Back

Versetzt ein selektiertes Element im Designer in den Hintergrund. Im DOM wird es an oberste Stelle gesetzt.

Bring to Front

Versetzt ein selektiertes Element im Designer in den Vordergrund. Im DOM wird es an unterste Stelle gesetzt.

Send to Back by Step

Setzt ein Element eine Ebene nach hinten. Im DOM rückt es eine Position nach oben.

Bring to Front by Step

Setzt ein Element eine Ebene nach vorne. Im DOM rückt es eine Position nach unten.

Align Left

Richtet die selektierten Elemente an der linken Kante aus. Als Bezugspunkt wird das Element genommen, das den Punkt mit der kleinsten X-Koordinate hat.

Align Right

Richtet die selektierten Elemente an der rechten Kante aus. Als Bezugspunkt wird das Element genommen, das den Punkt mit der größten X-Koordinate hat.

Align Top

Richtet die selektierten Elemente an der oberen Kante aus. Als Bezugspunkt wird das Element genommen, das den Punkt mit der kleinsten Y-Koordinate hat.

Align Bottom

Richtet die selektierten Elemente an der unteren Kante aus. Als Bezugspunkt wird das Element genommen, das den Punkt mit der größten Y-Koordinate hat.

Align Horizontal Center

Richtet die selektierten Elemente an der gemeinsamen horizontalen Mittelachse aus.

Align Vertical Center

Richtet die selektierten Elemente an der gemeinsamen vertikalen Mittelachse aus.

Move to

Ermöglicht das Verschieben eines oder mehrerer selektierter Elemente / Gruppen. Es öffnet sich ein Dialog zur Eingabe von X-/Y-Koordinaten oder eines Deltas, um die selektierten Elemente entweder entsprechend zu verschieben oder an der gewählten festen Koordinate zu positionieren. Die Angabe von X- und Y-Wert ist optional, um beispielsweise nur eine Verschiebung in horizontaler Richtung zu ermöglichen unter Beibehaltung der X-Position.

Rotate by 90°

Rotiert die selektierten Elemente um genau 90 Grad im Uhrzeigersinn.
