Layer Manager

Dieses Fenster dient zum Verwalten der Layer / Ebenen.

Prinzipiell hat jede Grafik immer einen default Layer. Dieser wird aber zunächst nicht im DOM angezeigt. Wird ein weiterer Layer angelegt, werden alle vorhandenen Layer im DOM angezeigt. Dem default Layer werden automatisch alle bisher gezeichneten Elemente untergeordnet.

Jeder Layer wird in der Liste mit einer Checkbox und seiner ID angezeigt. Wird die Checkbox deselektiert, werden alle in dem entsprechenden Layer enthaltenen Elemente ausgeblendet. Auf die spätere SVG Datei hat dies keine Auswirkung.

Neue Elemente werden automatisch in den im DOM selektierten Layer gezeichnet.

Im Preferences Dialog (Option: "Operate on all layers") kann definiert werden, ob unabhängig von der momentanen Layerselektion gearbeitet werden kann oder ob nur mit denjenigen Elementen gearbeitet werden kann, die sich im momentan selektierten Layer befinden.

Im unteren Bereich sind einige Menüpunkte zur Verwaltung erhältlich.

 

Option

Beschreibung

Refresh

Zurzeit nicht implementiert.

New Layer

Erstellt einen neuen Layer.

Delete Layer

Löscht einen Layer und alle enthaltenen Elemente.

Move Up

Verschiebt einen Layer eine Position nach oben.

Move Down

Verschiebt einen Layer eine Position nach unten.

 

Die Reihenfolge der Layer wird ebenfalls im DOM widergespiegelt.
