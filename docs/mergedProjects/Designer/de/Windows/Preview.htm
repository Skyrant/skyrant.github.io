Preview

Das Preview Fenster bietet eine Vorschau, wie das Bild später im Browser aussehen wird. Dazu wird meist der interne SVG Viewer des Designers genutzt. Demnach richtet sich die Bedienung in diesem Fenster nach der allgemeinen GUI des Viewers. Zusätzlich wird neben der grafischen Ansicht eine Ansicht des zugehörigen SVG-Sourcecodes angeboten.

Über den Button "Refresh Preview" können bei geöffnetem Fenster zwischenzeitlich vorgenommene Änderungen sichtbar gemacht werden.

Über die Preferences kann neben der GefasoftDesigner eigenen Preview auch das Öffnen der verknüpften SVG Standardapplikation (in der Regel IE) eingestellt werden.
