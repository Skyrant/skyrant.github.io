Funktion

Den Zeitschemata können Schichtmodelle zugewiesen werden. In einem Schichtmodell werden die möglichen Tagesabläufe erfasst. Jedes Schichtmodell beschreibt also einen vollständigen Arbeitstag, der später im Kalender einem Tag zugeordnet werden kann.

Ein typisches Schichtmodell ist beispielsweise "3 Schichten". Außerdem sollte zu jedem Zeitschema ein Schichtmodell "keine Produktion" erstellt werden, um auch diesen Zustand sauber abbilden zu können.

Gesperrt

Checkbox. Wenn aktiv, ist das Zeitmodell gesperrt. Es erscheint dann nicht im Schicht-Kalender in der WebApp.

 

Kontextmenü - Schichtmodell kopieren

Das Kontextmenü eines Schichtmodells bietet die Option, das Schichtmodell zu kopieren. Es kann durch rechten Mausklick beim zu kopierenden Schichtmodell im Navigationsbaum oder in der Tabelle aufgerufen werden. Bei Aufruf aus der Tabelle können sogar mehrere Schichtmodelle gleichzeitig kopiert werden (Markierung mehrerer Einträge und dann Aufruf des Kontextmenüs). Es werden alle untergeordneten Daten wie Schichtzeiten, Pausenzeiten und Arbeitszeitprofile mit kopiert.

Anschließend können die in der Zwischenablage gespeicherten Schichtmodelle an anderer Stelle wieder eingefügt werden. Dies erfolgt ebenfalls über das Kontextmenü im Navigationsbaum beim Eintrag "Schichtmodell" oder in der Tabelle der Schichtmodelle (Menüpunkt "Schichtmodell einfügen").

Wird das Schichtmodell innerhalb des selben Zeitschemas eingefügt, werden auch die Sollwertvorlagen beibehalten. Wird das Schichtmodell in ein anderes Zeitschema eingefügt, so werden die Sollwertvorlagen auf NULL gesetzt und müssen ggf. händisch nachgepflegt werden.
