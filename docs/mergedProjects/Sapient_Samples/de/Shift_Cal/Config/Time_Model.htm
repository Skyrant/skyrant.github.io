Zeitmodell

Die folgenden Daten müssen zur Verwendung der Zeitmodelle parametriert werden:

Schicht- und Pausentypen

Zusätzlich zu den Systemdaten können beliebige eigene Schicht- und Pausentypen angelegt werden. Siehe Schichttypen und Pausentypen.

Feiertage

Es können Feiertage geplant werden, die dann bei schichtabhängigen Berechnungen entsprechend berücksichtigt werden. Siehe Feiertage.

Zeitschemata

Erstellen Sie nun die gewünschten Zeitschemata, siehe Zeitschemata. Diese können dann einem Knoten zugewiesen werden; siehe Fertigungsknoten.

Schichtmodelle

Für jedes Zeitschema können nun beliebig viele Schichtmodelle erstellt werden, z.B. keine Produktion, 2-Schicht und 3-Schicht. Diese können dann in der WebApp auf die einzelnen Tage geplant werden. Siehe Schichtmodelle.

Schichten und Pausen

Ein Schichtmodell besteht aus einer beliebigen Anzahl Schichten und Pausen. Siehe Schichten und Pausen.

Schicht- und Produktionsgruppen

Optional können zusätzlich noch Schicht- und Produktionsgruppen angelegt werden. Siehe Schichtgruppen und Produktionsgruppen.
