Systemparameter

Folgende Systemparameter müssen gesetzt oder deren Werte überprüft werden. Die Beschreibung der einzelnen Parameter finden Sie in den folgenden Kapiteln.

Systemparameter -260:

Um sich zu Sapient verbinden zu können, benötigt der LC2 die Verbindungsdaten der Sapient Engine aus dem Systemparameter -260. Um diesen Wert setzen zu können, muss der LC2 mit Emergency Access gestartet werden. Alternativ kann der Parameter auch direkt in der Datenbank gesetzt werden.

Systemparameter -201, -202, -203

Überprüfen Sie die Werte für den DB-Benutzer, das zugehörige Passwort und das Schema.

Systemparameter -13

Ändern Sie gegebenenfalls den Titel der Applikation im Browser
