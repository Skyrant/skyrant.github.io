Meldungsanzahl Boardlet

Allgemein

Dieses Boardlet zeigt zu einem selektierten Knoten die Anzahl oder die summierte Nettodauer aller Meldungen (archiviert und aktuell) im selektierten Zeitraum.

Bei Mouseover über einen Balken werden die Meldeklasse / Meldeart und die Anzahl aufgetretener Meldungen eingeblendet.

Durch Klick auf den Namen einer Meldeklasse / Meldeart unten in der Legende werden Meldungen dieser Klasse / Art aus- bzw. wieder eingeblendet werden. Ausgeblendete Werte werden in der Legende  hellgrau dargestellt.

Nach Auswahl eines oder mehrerer Balken kann über das Quick-Edit-Menü ein Drill Down ausgeführt werden; wahlweise ins Meldungsarchiv oder den Meldungsverlauf. Die gewünschte Ansicht wird dann für die selektierte Meldeklasse und den gesetzten Zeitraum rechts in der Seitenleiste angezeigt.

Reiter Filter

Gruppieren nach:

Hier kann über eine Drop-Down-Liste ausgewählt werden, wonach gruppiert werden soll: Meldeklasse, Meldeart oder beides.

Zählen nach:

Hier kann über eine Drop-Down-Liste ausgewählt werden, was gezählt werden soll: Anzahl der Meldungen oder Summe der Nettodauern.

Meldeklassen:

Filter auf Meldeklassen. Es werden alle Meldungen der hier eingegebenen Klassen für die selektierten Knoten angezeigt. Die Meldeklassen können aus einer Dropdown-Liste gewählt werden. Leer = kein Filter, alle Meldeklassen werden angezeigt.

Meldearten:

Filter auf Meldearten. Es werden alle Meldungen der hier eingegebenen Arten für die selektierten Knoten angezeigt. Die Meldearten können aus einer Dropdown-Liste gewählt werden. Leer = kein Filter, alle Meldearten werden angezeigt.

Benötigte Parametrierung im LC2

Es wird keine gesonderte Parametrierung im LC2 benötigt.
