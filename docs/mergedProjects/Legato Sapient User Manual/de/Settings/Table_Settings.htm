Reiter Tabelleneinstellungen

Für jedes Boardlet, das eine Tabelle anzeigt, steht ein weiterer Reiter mit Tabelleneinstellungen zur Verfügung. Hierüber können Spalten ausgeblendet, gelöscht und hinzugefügt werden. Außerdem können für jede Spalte Einstellungen getroffen werden.

Alternativer Spaltenname:

Hier können Sie die Spaltenüberschrift definieren.

Sortierbar:

Über den Schieberegler können Sie festlegen, ob die Spalte sortiert werden kann.

Skalierbar:

Über den Schieberegler können Sie festlegen, ob die Spaltenbreite verändert werden kann.

Verschiebbar:

Über den Schieberegler können Sie festlegen, ob diese Spalte per Drag & Drop woanders hin gezogen werden kann oder nicht.

Breite:

Über den Schieberegler können Sie festlegen, ob die Spaltenbreite voreingestellt werden soll. Falls aktiv, wird eine neue Zeile eingeblendet, in der Sie die Spaltenbreite in Pixeln (px) oder Prozent (%) eingeben können. Achten Sie darauf, die Maßangabe direkt hinter dem Wert anzugeben, z.B. 80px oder 15%

Header Component:

Der Komponentenname einer Kopfzeilenzelle bestimmt die Formatierung einer Kopfzeilenzelle. Die Kopfzeilenzellen und deren Eigenschaften werden von Ihrem GEFASOFT Consultant konfiguriert.

Header Component Props:

Weitere Eigenschaften einer Kopfzeilenzelle. Die Eigenschaften werden von Ihrem GEFASOFT Consultant konfiguriert.

Cell Component:

Der Name einer Zellenkomponente bestimmt die Formatierung der Zellen in einer Tabellenspalte. Die Zellenkomponenten und deren Eigenschaften werden von Ihrem GEFASOFT Consultant konfiguriert.

  

Beispiele:

Namen von Zellenkomponenten

Beschreibungen

date-table-cell-component

stellt den Inhalt im lokalen Datumsformat (Datum, Uhrzeit) dar. Der Inhalt der Zelle muss vom Typ Date sein.

icon-table-cell-component

stellt den Inhalt als Icon dar. Der Inhalt der Zelle muss der Name des Symbols als String sein.

colored-ratio-cell-component

stellt den Inhalt als Stapelbalkengrafik dar. Ein Beispiel finden Sie weiter unten.

line-break-cell-component

stellt den Inhalt in mehreren Zeilen dar.

etc.

etc.

Cell Component Props:

Weitere Eigenschaften von Tabellenzellen in einer Spalte. Die Eigenschaften einer Zellenkomponente werden von Ihrem GEFASOFT Consultant konfiguriert.

Beispiel für die Konfiguration der Eigenschaften der Zellenkomponente colored-ratio-cell-component:

  

 

{"object":"state_ratio","colorClass":{"cnt_inactive":"light-gray-fill","cnt_never_connected":"warning-color-fill","cnt_ok":"success-color-fill","cnt_err":"error-color-fill"}}
A "cnt_inactive":"light-gray-fill"
B "object":"state_ratio"
C "cnt_never_connected":"warning-color-fill"
D "cnt_err":"error-color-fill"
E "cnt_ok":"success-color-fill"
 

Sichtbar:

Über den Schieberegler können Sie festlegen, ob diese Spalte ausgeblendet oder angezeigt wird.
