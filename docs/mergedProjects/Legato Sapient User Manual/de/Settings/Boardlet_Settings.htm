Boardlet-Einstellungen

In diesem Hilfe-Thema werden die häufigsten Einstellungen beschrieben, die allen/den meisten Boardlets auf folgenden Reitern zur Verfügung stehen:

  • Reiter Einstellungen

  • Reiter Layout

  • Reiter Filter

Wird etwas an den Boardlet-Einstellungen geändert, ändern sich die Reiter-Symbole in ein Annehmen und ein Abbrechen Symbol.

Erst nach Bestätigung oder Ablehnen der Änderung kann auf einen anderen Reiter gewechselt werden.

Klick auf Annehmen übernimmt die im jeweiligen Reiter eingegebenen Werte und überträgt sie an das Dashboard. Einige Einstellungen werden direkt übertragen; bei diesen können Sie also jederzeit prüfen, welche Auswirkung ihre Änderung hat. Andere Änderungen erfordern ein Neuladen der Seite, um wirksam zu werden.

Klick auf Boardlet löschen löscht das Boardlet bzw. den Container mit allen Inhalten.

Reiter Einstellungen

Titel:

Es wird der aktuelle Titel des Elements angezeigt. Durch Eingabe eines Texts in das Eingabefeld kann der Titel geändert werden.

Titel anzeigen:

Über den Schieberegler kann festgelegt werden, ob der Titel über dem Element angezeigt werden soll oder nicht. Nur verfügbar, wenn der Schieberegler Menüleiste anzeigen aktiv ist.

Menüleiste anzeigen:

Über den Schieberegler kann festgelegt werden, ob die Menüleiste des Boardlets (Titel und Menü Quick-Edit []) angezeigt werden soll oder nicht. Wird der Regler deaktiviert, wird auch der Titel deaktiviert.

Drucken:

Über den Schieberegler kann festgelegt werden, ob dieses Boardlet beim Drucken des Dashboards mit ausgegeben werden soll oder nicht.

Verwandte Themen:

    
  • Druck-Zeitpläne für Dashboards
    

 

  

Reiter Layout

Menüleiste links:

Über diesen Schieberegler kann definiert werden, ob die Menüleiste oben im Boardlet angezeigt werden soll (aus) oder ob sie links angezeigt werden soll (an).

Position X: / Position Y:

Hier werden die X- und Y-Koordinaten des Boardlets angezeigt und können geändert werden (dadurch ändert sich die Position des Boardlets im Dashboard). Das Boardlet kann auch per Drag & Drop im Dashboard verschoben werden, dann aktualisieren sich die Koordinaten entsprechend. Die Werte beziehen sich auf die obere linke Ecke.

Breite: / Höhe:

Hier werden die Breite und Höhe des Boardlets angezeigt und können geändert werden (dadurch ändert sich die Größe des Boardlets).

Schriftgröße:

  • Wert in grau: Er zeigt die Standardschriftgröße an, die für alle Boardlets verwendet wird. Die Standardschriftgröße können Sie einstellen unter Menü Benutzer Einstellungen Benutzereinstellungen Boardlet Eigenschaften.

  • Wert in schwarz: Alternativ können Sie einen Wert eingeben, mit dem die Schriftgröße für dieses spezielles Boardlet festgelegt wird.

Scroll-Position halten:

Über diesen Schieberegler kann definiert werden, ob die aktuelle Scroll-Position im Boardlet bei Aktualisierung beibehalten werden soll. Wenn nicht aktiv, springt die Position beim Aktualisieren nach oben.

 

Reiter Filter

Über das Symbol (, , oder ) rechts neben einem Wert kann festgelegt werden, ob / wie dieser Wert als Parameter in Zusammenhang mit anderen Boardlets verwendet werden soll. Für Details dazu, siehe bitte Parameter.

Selektierte Knoten: / Ausgewählte Knoten:

IDs der Knoten, für die Daten angezeigt werden. Kann entweder hier geändert werden oder durch Auswahl eines anderen Knotens im Layout-Boardlet (außerhalb des Bearbeitungsmodus).

Zeitbereich:

Hier kann der Zeitbereich festgelegt werden. Dafür stehen mehrere Möglichkeiten zur Verfügung, die durch Klick auf < eingeblendet werden. Das gewünschte Datum kann über eine Kalender ausgewählt werden, oder es kann ein Start- und Endezeitpunkt mit Uhrzeit eingegeben werden. Außerdem stehen relative Zeitfilter zur Verfügung wie heute, gestern, letzter Monat, etc. zur Verfügung.

 

INFO:

Die anderen Felder sind Boardlet-spezifische Filterfelder.

 

INFO:

Im Bearbeitungsmodus eines Dashboards lassen sich die Parametertypen der Boardlet-Filter leicht ermitteln.
Die Parametertypen werden in runden Klammern neben den Feldnamen angezeigt.
