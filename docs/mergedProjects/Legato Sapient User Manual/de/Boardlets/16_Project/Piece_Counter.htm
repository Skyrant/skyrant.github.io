Stückzähler (kompakt)

Allgemein

Das Boardlet Stückzähler (kompakt) zeigt zu einem selektierten Knoten den Stückzahl-Verlauf der aktuellen Schicht mit Sollwerten, zwei beliebige weitere Variablen und, sofern parametriert, einen berechneten Wert mit dem Trend der letzten 15 Minuten als Symbol. Grundsätzlich kann jede Variablenart visualisiert werden. Damit überhaupt etwas angezeigt wird, müssen in der aktuell laufenden Schicht bereits Werte gebucht worden sein.

Welcher Wert wo visualisiert wird, legt die erweiterte Eigenschaft "Position in Kompakt-Ansicht" in der Tabelle p_value_relations fest. Das Konzept ist dabei wie folgt:

Benötigte Parametrierung im LC2

Knoten mit aktivem Zeitschema (Produktion), Schichtkalender muss in Sapient gepflegt sein.

Unter Anwendungsstruktur Fertigungsstruktur [Knoten] Variablenzuordnungen werden die Variablen dem Knoten zugeordnet, für den sie angezeigt werden sollen. Hier muss das Flag "Anzeigen" auf 1 gesetzt sein, damit die Variable angezeigt wird. Die Sortiernummer legt fest, in welcher Reihenfolge die Variablen angezeigt werden.

Hier muss die Spalte "Position in Kompakt-Ansicht" wie im Bild oben befüllt werden.

Für die Variable in Position 4 wird der Wert der angegebenen Variablen im Verhältnis zur Variable in Position 1 als Symbol dargestellt. Daher empfiehlt es sich, für diese Variable einen berechneten Wert Short Term Trend zu konfigurieren.
