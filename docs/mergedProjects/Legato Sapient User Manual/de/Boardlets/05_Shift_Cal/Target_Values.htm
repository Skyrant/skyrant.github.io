Sollwerte

Allgemein

Jeder Schicht kann eine Sollwertvorlage zugewiesen werden, über die dann zeitgesteuert die zu erbringenden Sollwerte vorgegeben werden.

Zum Planen der Sollwerte werden zwei Boardlets benötigt:

  • Eines zur Auswahl der gewünschten Sollwertvorlage und

  • eines zur Anzeige/Änderung der Sollwerte.

Sollwertvorlagen

In diesem Boardlet werden alle Zeitschemata angezeigt, für die der angemeldete Benutzer Rechte hat.

Durch Klick auf eine Zeile klappen die diesem Zeitschema zugeordneten Sollwertvorlagen auf. Die gewünschte kann durch Klick selektiert werden.

Sollwerte planen

Dieses Boardlet dient dazu, die Sollwerte für eine bestimmte Variable der Sollwertvorlage zu einem bestimmten Zeitpunkt zu planen. Zum gegebenen Zeitpunkt werden die Werte dann auf die entsprechende Variable geschrieben.

Auf der linken Seite werden die Variablen angezeigt, die der Sollwertvorlage zugeordnet sind. Durch Klick auf einen Eintrag werden auf der rechten Seite die Werte zu den geplanten Zeitpunkten angezeigt. Nach Auswahl eines Wertes können die Daten editiert werden oder ein neuer Wert hinzugefügt werden.

Dazu öffnet sich ein Dialog in der Seitenleiste.

Durch Klick auf das Kalender-Symbol kann das Datum ausgewählt werden, ab dem der Sollwert gelten soll. Im Eingabefeld darunter kann der Wert eingegeben werden, der gesetzt werden soll.

Benötigte Parametrierung im LC2

Sollwertvorlagen werden im LC2 unter Werksverwaltung Arbeitszeit-Einstellungen Zeitschemata parametriert.

Die Variablen werden im LC2 unter Anwendungsstruktur Datenaustausch [Modul] [Verbindung] Variablen gepflegt. Hier wird auch die Einheit festgelegt.

Unter Werksverwaltung Arbeitszeit-Einstellungen Zeitschemata Schichtmodelle Schichten wird einer Schicht eine Sollwertvorlage zugewiesen.
