Schichtausnahmen

Allgemein

Im Boardlet Schichtausnahmen kann dem gewählten Knoten und allen Unterknoten mit demselben Zeitschema für einen bestimmten Zieltag ein abweichendes Schichtmodell zugewiesen werden. Das Standard-Schichtmodell der restlichen Knoten im selben Zeitschema bleibt davon unberührt.

Außerdem ist es möglich, die Start- oder Endzeit von Schichten oder Pausen zu ändern. Siehe auch Systemparameter -31.

Es gelten folgende grundlegenden Bestimmungen:

  • Schichtausnahmen können nur auf Grund eines Standardschichtmodells für ein Zeitschema (und einen Tag) eingegeben werden, wenn die Schichten/Pausen bereits ausgerollt wurden.

  • Eine nachträgliche Änderung des Standardschichtmodells eines Zeitschemas (und Tages) kann getätigt werden auch wenn:

ab einem Unterknoten ein abweichendes (Knoten-)Schichtmodell parametriert wurde (dieses wird nicht verändert, eventuell eingegebene Schicht- und/oder Pausenänderungen bleiben erhalten),

an Knoten Schicht- und/oder Pausenänderungen eingegeben wurden (diese werden ohne Rückfrage überschrieben).

  • Die Eingabe eines Knotenschichtmodells (Abweichendes Modell für Knoten und dessen Unterknoten) ist zulässig wenn:

übergeordnet oder untergeordnet kein Knotenschichtmodell existiert.

Schichten/Pausen von Knoten, welche nicht von der Eingabe eines Knotenschichtmodells betroffen sind, werden nicht verändert (eventuell eingegebene Schicht- und/oder Pausenänderungen bleiben erhalten).

  • Wurde an einem Knoten ein Knotenschichtmodell eingegeben, können auch hier noch Schicht- und/oder Pausenänderungen eingegeben werden, allerdings nur auf der Knotenebene, in der auch das Knotenschichtmodell eingegeben wurde.

  • Eine nachträgliche Änderung eines Knotenschichtmodells kann getätigt werden auch wenn:

am (identischen) Knoten Schicht- und/oder Pausenänderungen eingegeben wurden (diese werden ohne Rückfrage überschrieben).

 

Zunächst muss ein Knoten selektiert sein sowie ein Datum. Dann wird im Boardlet angezeigt, welches Zeitschema dem Knoten zugewiesen ist und welches Schichtmodell aktuell eingeplant ist. Darunter werden alle aktuell geplanten Schichten und Pausen mit Start- und Endzeit aufgelistet.

Nun kann entweder das Schichtmodell für den selektierten Tag geändert werden (durch Auswahl in der Dropdown-Liste). Durch Klick auf "Anwenden" wird die Änderung eingetragen und bei der nächsten Schichtplanung entsprechend verarbeitet. Danach ist es nicht mehr möglich, Änderungen an den Schicht- und Pausenzeiten vorzunehmen (da die neuen Schichten ja noch nicht geplant sind).

Alternativ kann das Schichtmodell beibehalten werden und nur einzelne Zeiten von Schichten oder Pausen geändert werden. Durch Klick auf das Pfeilsymbol (>) zu Beginn der gewünschten Zeile öffnet sich ein Dialog, in dem die Zeiten entsprechend geändert werden können.

:

Dies ist nur für die Zukunft möglich. Wurde als Datum der aktuelle Tag selektiert, werden die Pfeilsymbole (>) nicht eingeblendet.

Beim Speichern wird überprüft, dass es durch die Änderung nicht zu Überschneidungen mit den restlichen Schichten kommt und dass zugehörige Pausen immer noch innerhalb des Schichtzeitraums liegen. Ist dies nicht der Fall, ist Speichern nicht möglich und es wird eine entsprechende Fehlermeldung ausgegeben.

Durch Klick auf das + rechts von jeder Schicht kann eine zusätzliche Pause für diese Schicht erfasst werden.

Wurden Änderungen vorgenommen, erscheint ein zusätzliches Symbol vor den geänderten Einträgen:

Durch Klick darauf, kann entweder die zusätzliche Pause wieder gelöscht werden oder die Änderung wieder auf den ursprünglich geplanten Wert zurückgesetzt werden.
