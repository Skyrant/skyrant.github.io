Chart - Balken-/Liniendiagramm

Allgemein

Das Boardlet Balken-/Liniendiagramm zeigt definierte Messwerte für den im Layout selektierten Knoten in einem über den Zeitfilter definierten Zeitraum an. Pro Einheit wird eine eigene Achse angezeigt; für Werte ohne Einheit wird eine weitere Achse angezeigt.

Durch Klick auf kann die Legende ein- oder ausgeblendet werden. Durch Klick auf wird die Anzeige der Legende zwischen unten und rechts umgeschaltet.

Zusätzlich können die definierten Warn- und Alarmgrenzen sowie die Sollwert-Vorgabe für die Messwerte eingeblendet werden. Das Diagramm sieht dann so aus:

Die Warngrenzen werden als gelbes Band dargestellt, die Alarmgrenzen als rotes Band. Gibt es für mehr als einen Wert Einträge, so werden die Bänder gegebenenfalls überlagernd angezeigt. Die Sollwert-Vorgabe wird in der selben Farbe wie der zugehörige Messwert eingeblendet.

Bei Mouseover über eine Linie oder einen Balken werden der Zeitstempel und der Wert zu diesem Zeitpunkt eingeblendet.

Bei Mouseover über einen gelben oder roten Punkt links oder rechts am Rand werden Details zu den Zusatz-Linien (Warn-, Alarmgrenzen, Sollwert-Vorgabe) eingeblendet.

Durch Klick auf den Namen eines Wertes unten in der Legende kann dieser aus- bzw. wieder eingeblendet werden. Ausgeblendete Werte werden in der Legende hellgrau dargestellt.

Durch Doppelklick auf einen Eintrag in der Legende werden alle Werte außer diesem ausgeblendet. Ist nur ein Eintrag selektiert, werden bei Doppelklick alle Werte außer diesem sichtbar geschaltet. Zugehörige Warn- und Alarmgrenzen sowie Sollwert-Vorgaben werden zusammen mit dem Wert aus- bzw. eingeblendet.

Hinter dem jeweiligen Wert wird dessen Einheit in eckigen Klammern eingeblendet. Sofern mehrere Knoten selektiert sind, wird zusätzlich der Name des Knotens, dem die Variable zugeordnet ist, in runden Klammern dahinter angezeigt.

Durch Klicken und Ziehen mit gedrückter linker Maustaste kann in den Zeitbereich hineingezoomt werden. Es wird dann ein Button eingeblendet, über den der Zoom wieder zurückgesetzt werden kann.

:

Es werden nur die ersten 5000 Werte geladen. Das kann dazu führen, dass Variablen gar nicht angezeigt werden, z.B. wenn die erste geladene Variable mehr als 5000 Werte im eingegebenen Zeitraum hat. In diesem Fall wird eine entsprechende Meldung angezeigt. Verwenden Sie gegebenenfalls die Einstellung "Punktereduktion" oder passen Sie den Zeitraum an.

Unteransicht Drill-Down Teilwerte

Sind für den selektierten Knoten Teilwerte parametriert (Spalte Parent != Null), können diese durch das Drilldown-Menü in einer Unteransicht als gestapeltes Balkendiagramm visualisiert werden. Dazu muss zuerst der Parent-Wert im Bar Chart durch Klick auf den Namen in der Legende selektiert werden. Danach kann über das Quick-Edit Menü [] unter Drilldown der Punkt "Teilwerte" ausgewählt werden. In der Seitenleiste öffnet sich folgende Unteransicht:

Hier werden die einzelnen Teilwerte des selektieren Parents als gestapeltes Balkendiagramm im selektierten Zeitraum angezeigt.

:

Dafür müssen die Teilwerte alle denselben Zeitstempel haben. Daher muss die Buchung mittels der Prozedur Subvalues_in ausgeführt werden.

Einstellungen

Die häufigsten Einstellungen auf den Reitern Einstellungen, Layout und Filter sind unter Boardlet-Einstellungen beschrieben:

  • Reiter Einstellungen

  • Reiter Layout

  • Reiter Filter

Für ein Balken-/Liniendiagramm stehen außerdem folgende Einstellungen zur Verfügung.

Reiter Settings

Aktualisierungs-Intervall (Sek.):

Intervall in Sekunden, in denen die Ansicht automatisch aktualisiert wird. Minimum ist 9 Sekunden.

:

Ein zu niedriger Wert kann die Performance negativ beeinflussen.

Hilfslinie:

Über diesen Schieberegler kann eine vertikale Hilfslinie eingeblendet werden. Diese zeigt dann den exakten Zeitpunkt und alle zugehörigen Werte an.

Aggregierte Werte verwenden:

Über diesen Schieberegler wird festgelegt, ob aggregierte Werte oder Rohdaten im Bar Chart angezeigt werden sollen. Wird der Regler aktiviert, kann im Reiter Filter der Aggregationstyp eingestellt werden. Standard ist -1 = Stunde.

Punktereduktion nutzen:

Über diesen Schieberegler wird definiert, ob die dargestellten Punkte reduziert werden sollen. Falls aktiv, werden alle Punkte einer Geraden ausgeblendet.

In Abhängigkeit von der Anzahl der Ausgangspunkte wird die folgende Punktemenge zurückgeliefert:

  • wenn es weniger als 1.000 Punkte in der Ausgangsmenge gibt: alle Punkte der Ausgangsmenge

  • wenn das Ergebnis der Punkteelemination mit dem RDP-Algorithmus etwa 1.000 Punkte hat, dieses Ergebnis

  • sonst nur die Punktemenge aus den lokalen Maxima und Minima, wenn es insgesamt etwa 1.000 Punkte oder weniger sind

  • in allen anderen Fällen: für 1.000 Zeitabschnitte jeweils der Durchschnittswert in x- und y-Richtung

Reiter Layout

Diagrammtyp:

Aus der Dropdown-Liste kann der Diagrammtyp gewählt werden. Zur Verfügung stehen Liniendiagramm, Balkendiagramm, Flächendiagramm und gestapeltes Balkendiagramm.

Sollwert anzeigen:

Über den Schieberegler wird definiert, ob die Sollwerte als Linie eingeblendet werden sollen. Im Reiter Filter kann dann festgelegt werden, welche Variablenart als Sollwert verwendet werden soll.

Verfügbarkeitsabschnitte einblenden:

Über den Schieberegler wird definiert, ob zusätzlich Verfügbarkeitsabschnitte wie beispielsweise "geplante Pausen" im Diagramm eingeblendet werden sollen. Die Darstellungsfarbe wird im LC2 definiert. Welche Verfügbarkeitsabschnitte angezeigt werden sollen, kann im Tab Filter festgelegt werden.

Warngrenzen anzeigen:

Über diesen Schieberegler wird definiert, ob Warngrenzen visualisiert werden sollen.

Alarmgrenzen anzeigen:

Über diesen Schieberegler wird definiert, ob Alarmgrenzen visualisiert werden sollen.

Reiter Filter

Variablenarten:

Filter auf Variablenarten. Für alle Variablen der hier definierten Variablenarten, die dem oben selektierten Knoten zugeordnet sind, werden die Werte im Diagramm angezeigt. Die Variablenarten können aus einer Dropdown-Liste ausgewählt werden. Leer = kein Filter, alle Variablenarten werden angezeigt.

Variablenart Sollwert:

Variablenart, die als Sollwert-Linie angezeigt werden soll.

Wird nur eingeblendet, wenn im Reiter Layout der Regler "Sollwert anzeigen" aktiviert ist.

Variablen-ID:

Standardmäßig -1 = deaktiviert. Dieser Parameter ist relevant, wenn in diesem Balken-/Liniendiagramm die Historie einer Variablen via das Aktuelle Werte Boardlet angezeigt werden soll. ID der Variable, deren Historie angezeigt wird. Der Parametertyp muss auf "In" oder "InOut" stehen, um den Wert vom Aktuelle Werte Boardlet zu empfangen.

Verfügbarkeits-Arten:

IDs der Verfügbarkeitsabschnittsarten, die im Diagramm eingeblendet werden sollen.

Wird nur eingeblendet, wenn im Reiter Layout der Regler "Verfügbarkeitsabschnitte einblenden" aktiviert ist.

Aggregationsart:

Dropdown-Liste zur Auswahl der Aggregationsart. Wird nur eingeblendet, wenn im Reiter Settings der Regler "Aggregierte Werte verwenden" aktiviert ist.

Teilwerte:

Über diesen Schieberegler wird definiert, ob Parent-Werte oder Teilwerte visualisiert werden sollen.

Benötigte Parametrierung im LC2

Die Variablenarten werden im LC2 unter Grundkonfiguration Allgemeine Anwendungen Variablenarten parametriert.

Die Verfügbarkeitsabschnittsarten (Time slices) werden im LC2 unter Grundkonfiguration Verfügbarkeitsabschnittsarten parametriert. Hier wird auch die Farbe festgelegt, in der die Balken eingefärbt werden sollen.

Die Variablen werden im LC2 unter Anwendungsstruktur Datenaustausch [Modul] [Verbindung] Variablen gepflegt. Hier wird auch die Einheit festgelegt.

Hier muss das Flag "Archivieren" auf 1 gesetzt sein, damit die Werte der Variable im Archiv gespeichert werden. Hier können auch die Warn- und Alarmgrenzen sowie die Sollwert-Vorgabe für die Variablen definiert werden.

Unter Anwendungsstruktur Fertigungsstruktur [Knoten] Variablenzuordnungen werden die Variablen dem Knoten zugeordnet, für den sie angezeigt werden sollen. Hier muss das Flag "Anzeigen" auf 1 gesetzt sein, damit die Variable angezeigt wird. Die Sortiernummer legt fest, in welcher Reihenfolge die Variablen angezeigt werden.
