Monitoring - Knoteninfo

Allgemein

Das Boardlet Knoteninfo zeigt zu einem selektierten Knoten alle Variablen mit folgenden Informationen an:

  • Name der Variablen

  • aktueller Wert

  • parametrierter Sollwert

  • Abweichung

Außerdem wird der Vergleich zwischen Ist und Soll als Balken und Prozentzahl visualisiert. Ab wieviel Prozent die Balken in welcher Farbe eingefärbt werden, kann über die Einstellungen definiert werden. Dort wird auch festgelegt, wieviel Prozent ein Balken maximal lang sein soll (im Beispiel 120%).

Es kann als eigenständiges Boardlet angezeigt werden oder als Detail-Info ins Layout-Boardlet eingebettet werden.

Einstellungen

Die häufigsten Einstellungen auf den Reitern Einstellungen, Layout und Filter sind unter Boardlet-Einstellungen beschrieben:

  • Reiter Einstellungen

  • Reiter Layout

  • Reiter Filter

Für das Boardlet Knoteninfo stehen außerdem folgende Einstellungen zur Verfügung.

Reiter Layout

IO-Grenze:

Grenzwert in %, bis zu welchem Wert der Balken in grün dargestellt wird (Iststück/Sollstück).

Warn-Grenze:

Grenzwert in %, bis zu welchem Wert der Balken in der Warn-Farbe dargestellt wird (Iststück/Sollstück).

Max. Prozent:

Wert, wie viel Prozent maximal angezeigt werden. Diese Einstellung beeinflusst die Balkenlänge. Ist hier z.B. 120% eingestellt, so sind 100% nicht am Ende des Balkens erreicht, sondern schon vorher.

Störungsfreie Zeit anzeigen:

Über diesen Schieberegler wird definiert, ob fehlerfreie Zeit angezeigt werden soll.

Reiter Filter

Meldeklassen für Stillstand:

Dropdown-Liste zur Auswahl einer oder mehrere Meldeklassen, deren Auftreten als Stillstand betrachtet werden.

Benötigte Parametrierung im LC2

Die Variablenarten werden im LC2 unter Grundkonfiguration Allgemeine Anwendungen Variablenarten parametriert.

Die Variablen werden im LC2 unter Anwendungsstruktur Datenaustausch [Modul] [Verbindung] Variablen gepflegt. Hier wird auch die Einheit festgelegt.

Unter Anwendungsstruktur Fertigungsstruktur [Knoten] Variablenzuordnungen werden die Variablen dem Knoten zugeordnet, für den sie angezeigt werden sollen. Hier muss das Flag "Anzeigen" auf 1 gesetzt sein, damit die Variable angezeigt wird. Die Sortiernummer legt fest, in welcher Reihenfolge die Variablen angezeigt werden.

Dem selektierten Knoten müssen Variablen zugewiesen sein. Außerdem sollte den Variablen ein Sollwert zugeordnet sein (Spalte Sollwert).
