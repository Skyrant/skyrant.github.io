Monitoring - Multi-Knoten

Allgemein

Das Boardlet Multi-Knoten kann verwendet werden, um für mehrere im Layout selektierte Knoten Daten anzuzeigen. Dazu wird ein Detail-Boardlet eingebettet und mit verschiedenen Parametern versehen. Die Werte aller selektierten Knoten werden dann untereinander dargestellt.

Es ist auch möglich, mehrere dieser Boardlets im Dashboard zu verwenden, um z.B. Ist-, Soll- und Ausschussstückzähler der selektierten Knoten nebeneinander darzustellen.

Im Beispiel sind im Layout die Knoten "Schweißen 1", "Schweißen 2" und "Schweißen 3" selektiert. Daneben werden zunächst die Iststückzähler, dann die Sollstückzähler und schließlich die Ausschussstückzähler für jeden der selektierten Knoten angezeigt.

Für das Boardlet Multi-Knoten stehen außerdem folgende Einstellungen zur Verfügung.

Reiter Layout

Detail-Boardlet:

Über eine Dropdown-Liste kann ein weiteres Boardlet gewählt werden, dessen Werte als Detail-Informationen zu den Knoten angezeigt werden, beispielsweise die Werte bestimmter Variablen. Hier ist das node-values-boardlet (entspricht dem Aktuelle Werte Boardlet) ausgewählt. Außerdem stehen das Knoteninfo Boardlet, das Tortendiagramm Boardlet, das Balkendiagramm Boardlet und das Knotenstatus Boardlet zur Verfügung.

Detail-Boardlet-Parameters:

Hier können Parameter für das Detail-Boardlet eingetragen werden, z.B. die zu zeigenden Variablenarten

Beispiel: { "valueTypes": [2,40] } zeigt die Variablenarten mit der ID 2 und 40 an.

Detail-Boardlet-Höhe:

Höhe des jeweiligen Detail-Boardlets.
