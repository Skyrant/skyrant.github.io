Bearbeiten

Änderungsmodus

Der Änderungsmodus Layout bearbeiten kann über das Menü Quick-Edit [] des Boardlets Knotenlayout aufgerufen werden:

Benötigte Berechtigung

Damit diese Funktion zur Verfügung steht, muss der angemeldete Benutzer das Recht "Editieren" für die Datenquelle "L_NODES" haben.

Allgemein

Grundsätzlich lässt sich das Layout-Boardlet überwiegend per Drag & Drop editieren. Um schnell Höhe und Breite eines Bereichs zu ändern, packen Sie die schwarze Ecke des jeweiligen Elements unten rechts und ziehen Sie sie mit der Maus.

Die Knoten können entweder durch Klick im Knotenbaum selektiert werden oder durch Klick auf den Knoten direkt im Layout.

TIPP:

Falls der überlagernde Bereich selektiert ist, können Sie diesen durch Klick auf ESC deselektieren, um auf die unteren Bereiche zuzugreifen.

Sind mehrere Knoten selektiert, so wird für alle die Größe geändert.

Klickt man in einen bestimmten Bereich hinein und hält die linke Maustaste gedrückt, kann der Bereich per Drag & Drop nach Belieben verschoben werden.

TIPP:

Kleinere Anpassungen können bequem hier vorgenommen werden. Für das Erstellen komplexer Strukturen und aufwändiger Layouts empfiehlt sich die Verwendung eines SVG-Tools, z.B. Inkscape. Lesen Sie hierzu auch die separate Dokumentation "Erweitere Bedienung" - "Knotenlayout in Inkscape erstellen".

Je nachdem, ob Knoten oder der Detailbereich bearbeitet werden, sind ggf. nicht alle Symbole eingeblendet bzw. unterschiedliche Aktionen möglich.

Layout bearbeiten

1 = Client-Bereich oder Zoom-Bereich. Je nach Größe dieses Rahmens werden die innerhalb dargestellten Daten skaliert. Ist dieser Bereich kleiner als die enthaltenen Knoten, so wird beim Zoomen alles außerhalb dieser Linie ausgeblendet.

2 & 3 = Pfad oder "Shape". Es ist möglich, über das Bearbeitungsmenü Punkte hinzuzufügen und so die unterschiedlichsten Formen zu erstellen.

4 = Detail-Bereich. Innerhalb der strichlierten Linie werden je nach Definition in den Settings die Daten des eingebundenen Detail-Boardlet auf der entsprechenden Ebene angezeigt.

5 = Ecke des jeweiligen Elements packen und mit der Maus ziehen, um schnell Höhe und Breite zu ändern.

Client-Bereich bearbeiten

Um den Client-Bereich zu bearbeiten, müssen in der Bearbeitungsleiste die Symbole Knoten bearbeiten und Verschieben und vergrößern aktiviert werden.

Eine dünne graue Linie mit Punkten wird angezeigt. Durch Klick auf einen dieser Punkte kann die Größe des Client-Bereichs mittels Drag & Drop verändert werden.

Beim Zoomen in diesen Bereich wird dann nur der Inhalt angezeigt, der sich innerhalb des Bereichs befindet. Ist der Client-Bereich kleiner gewählt als der Pfad (entspricht der dicken grauen Linie), so wird beim Zoomen die Knotenform an dieser Stelle abgeschnitten.

Pfad bearbeiten

Der Pfad bezeichnet den Umriss eines Knotens. Um den Pfad zu bearbeiten, müssen in der Bearbeitungsleiste die Symbole "Knoten bearbeiten" und "Pfad editieren" aktiviert werden.

Eine fette Linie mit initial vier Punkten wird angezeigt, da die Defaultform eines Knotens ein Viereck ist. Durch Klick auf einen der Punkte wird dieser Punkt selektiert. Markierte Punkte werden ausgefüllt dargestellt, nicht markierte haben nur einen Rahmen. Mittels Strg + Klick können auch mehrere Punkte gleichzeitig selektiert und geändert werden. Nun kann der Pfad mittels Drag & Drop verändert werden. Außerdem wird rechts unten ein dreieckiges Symbol angeboten. Wird dieses selektiert, kann der Knotenpfad skaliert werden.

Durch Klick auf können weitere Punkte zum Pfad hinzugefügt werden. Es wird nur ein Punkt hinzugefügt, wenn einer der Punkte markiert ist. Der neue Punkt wird links zwischen dem markierten und dem nächsten Punkt hinzugefügt. Auf diese Weise ist es möglich, komplexe Figuren und Layouts abzubilden.

Gleichermaßen können durch Klick auf Punkte gelöscht werden.

Detail-Bereich bearbeiten

Um den Detail-Bereich zu bearbeiten, muss in der Bearbeitungsleiste das Icon "Detailbereich bearbeiten" aktiviert werden.

Eine fette strichlierte Linie mit drei Punkten wird angezeigt. Der Bereich, zu dem der Detail-Bereich gehört, wird ebenfalls fett dargestellt (im Beispiel gehört der Detail-Bereich also zu Werk 1). Durch Klick auf einen der Punkte kann die Größe des Detail-Bereichs mittels Drag & Drop verändert werden. Innerhalb des Detail-Bereichs werden parametrierbare Informationen für den zugehörigen Knoten angezeigt, z.B. eine Statusanzeige. Was dort angezeigt wird, wird über die Einstellung "Detail Boardlet" in den Boardlet Layout Settings des Layout-Boardlets festgelegt.

Bearbeiten-Dialog in der Seitenleiste

Zum weiterführenden Editieren des Layouts steht in der Seitenleiste ein umfangreicher Dialog zur Verfügung. Er ist in die verschiedenen Bereiche des Layout-Boardlet unterteilt und bietet die Möglichkeit, diverse Parameter wie zum Beispiel Höhe und Breite eines Bereichs direkt einzugeben.

Die einzelnen Parameterwerte sind dabei in sinnvolle Schrittweiten gestaffelt, um eine einfachere Bearbeitung zu ermöglichen. Befindet sich ein Wert außerhalb einer solchen Schrittweite, wird er rot umrandet. Ein Klick auf den Spinner rechts neben dem Wert setzt den Wert auf die nächsthöhere bzw. nächstniedrigere Schrittweite.

Outline

Diese Parameter gelten für die im Knotenlayout Boardlet aktuell selektierte Outline.

Offset X: / Offset Y:

Mit diesen Werten kann der Bereich im Boardlet verschoben werden.

Width: / Height:

Mit diesen Parametern kann die Breite und die Höhe des Bereichs geändert werden. Für den Fall, dass eine Outline unten im Boardlet im unsichtbaren Bereich verschwindet, kann hier beispielsweise die Größe deutlich reduziert werden und der Pfad anschließend wieder auf ein geeignetes Maß gezogen werden.

Path:

Kompletter Pfad des Bereichs (ein Array der obigen Parameter). Wird dieser Eintrag gelöscht, wird der Pfad auf den Defaultwert zurückgesetzt.

Der Pfad kann auch aus einem externen Zeichentool wie zum Beispiel Inkscape eingefügt werden. Dies ist vor allem bei komplexen Formen zu empfehlen.

0/0:

Durch Klick auf diese Schaltfläche wird die Outline obersten Ecke links außen ausgerichtet. Dabei wird jeweils vom höchsten/äußersten Punkt ausgegangen.

   

 

Text

Defaultmäßig ist der Text mit dem Detail-Bereich verknüpft, d.h., wenn man den Text verschiebt, verschiebt man den Detail-Bereich ebenfalls. Wird der Detail-Bereich einmal alleine verschoben, so ist die Verbindung zum Text unterbrochen und der Text kann ebenfalls allein verschoben werden.

X: / Y:

X- und Y-Koordinate des Textes in Relation zum zugehörigen Bereich.

Rotate:

Ermöglicht es, den Text schräg darzustellen.

Size:

Größe des Texts in Relation zum Rahmen.

   

 

Details

X: / Y:

X- und Y-Koordinate des Detail-Bereichs.

Breite: / Höhe:

Mit diesen Parametern kann die Breite und die Höhe des Bereichs geändert werden. Für den Fall, dass ein Bereich unten im Boardlet im unsichtbaren Bereich verschwindet, kann hier beispielsweise die Größe deutlich reduziert werden und der Bereich anschließend wieder auf ein geeignetes Maß gezogen werden.

   

 

Etc

Platziere Element in Layer:

Hier kann eingegeben werden, in welcher Ebene sich das selektierte Element befindet. Wird nichts eingetragen, befindet es sich in allen Ebenen. Hier darf nur ein Eintrag vorgenommen werden.

Layers:

Hier kann eingegeben werden, aus welchen Ebenen das selektierte Element besteht, z.B. Erdgeschoss, 1. Stock, 2. Stock.

  

 

Unterelemente

Unterelemente:

Sofern es für das selektierte Element Unterelemente gibt, können diese hier aus der Liste ausgewählt werden, z.B., um sie einer bestimmten Ebene zuzuordnen.

Pfad / Text hinzufügen:

Durch Klick auf diese Schaltflächen kann ein Pfad oder ein Text hinzugefügt werden, der keinen unmittelbaren Knotenbezug hat, z.B. eine Fahrstraße für Halle 1. Diese Elemente können dann aus der Liste der Unterelemente selektiert werden.

   

 

Ebenen visualisieren und Elemente hinzufügen

Im Bearbeiten-Dialog in der Seitenleiste können durch Eingabe in das Feld "Layers" verschiedene Ebenen des selektierten Elements visualisiert werden. Diese Ebenen werden dann in der Kopfzeile neben dem Knotennamen eingeblendet. Durch Klick darauf kann zwischen den Ebenen umgeschaltet werden. Sind für einen Knoten Ebenen definiert, kann anschließend immer nur die selektierte Ebene bearbeitet werden.

Durch Klick auf "Pfad hinzufügen" wird ein neues Element in die selektierte Ebene eingefügt. Im Bearbeiten-Dialog in der Seitenleiste werden weitere Elemente angezeigt. Diese können nun bearbeitet werden.

Unterelement:

Dropdown-Liste zur Auswahl des Unterelements, das bearbeitet werden soll.

X: / Y:

X- und Y-Koordinate des Unterelements.

Platziere Element in Layer:

Hier kann eingegeben werden, in welcher Ebene sich das selektierte Element befindet. Hier darf nur ein Eintrag vorgenommen werden. Wird nichts eingegeben, erbt das Element die Layer-Einstellungen des übergeordneten Elements. Wird "*" eingegeben, ist das Element immer sichtbar. Wird "-" eingegeben, ist das Element nie sichtbar. Die "*" und "-" Optionen werden auch an die Unterelemente vererbt.

Pfad:

Nur bei Pfaden, nicht für Texte. Kompletter Pfad des Elements (ein Array der obigen Parameter). Wird dieser Eintrag gelöscht, wird der Pfad auf den Defaultwert zurückgesetzt.

0/0:

Nur bei Pfaden, nicht für Texte. Durch Klick auf diese Schaltfläche wird die Outline obersten Ecke links außen ausgerichtet. Dabei wird jeweils vom höchsten/äußersten Punkt ausgegangen.

Statischer Text:

Über diesen Schieberegler wird definiert, ob es sich um ein statisches Element handelt. Falls aktiv, werden für dieses Element nicht die Daten des Knotens übernommen, auf dem das Element basiert. So kann beispielsweise ein fester Text vorgegeben werden statt des Knotennamens.

Text:

Nur bei Texten, nicht für Pfade. Hier kann für statische Elemente der Text vorgegeben werden.

Drehen:

Nur bei Texten, nicht für Pfade. Ermöglicht es, den Text schräg darzustellen.

Größe:

Nur bei Texten, nicht für Pfade. Größe des Texts in Relation zum Rahmen.
