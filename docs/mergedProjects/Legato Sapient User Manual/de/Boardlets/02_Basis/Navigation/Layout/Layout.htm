Knotenlayout

Allgemein

Das Boardlet Knotenlayout bietet eine Übersicht über alle vorhandenen Knoten und dient der Navigation. Es zeigt die Fertigungsstruktur an. Über dieses Boardlet kann gesteuert werden, auf welchen Knoten sich die in den anderen Boardlets angezeigten Daten beziehen.

Durch Doppelklick kann in den jeweiligen Bereich hineingezoomt werden, es öffnet sich dann eine Unteransicht mit den untergeordneten Knoten.

Klick in einen Bereich selektiert den entsprechenden Knoten und zeigt in den anderen Boardlets die Werte für diesen Knoten an. Durch Strg + Klick können auch mehrere Knoten selektiert werden, z.B. für die Anzeige im Multi-Knoten Boardlet.

Der Hintergrund der Knoten wird eingefärbt, z.B. wenn eine Meldung auf diesem Knoten ansteht. Stehen mehrere Meldungen an, wird die Meldung mit der Klasse der höchsten Priorität visualisiert. Die Farbe wird dabei über den CSS-Style der Meldeklasse definiert. Welche Information zum Einfärben des Hintergrunds verwendet wird, kann über die Boardlet-Einstellungen festgelegt werden.

Bedienelemente

Folgende Bedienelemente stehen im Boardlet Knotenlayout zur Verfügung:

Springt zurück zum Root-Knoten, der über die Einstellungen definiert wurde.

Springt eine Ebene zurück, maximal bis zum Root-Knoten.

Über den Quick-Edit Button (...) oben rechts kann der Änderungsmodus geöffnet werden, um das Knotenlayout zu editieren.
