Legato Grafik Designer

Allgemein

Das Boardlet Legato Grafik Designer zeigt mit draw.io erstellte Visualisierungen an. Dabei kann zwischen fester Visualisierung und Knoten-spezifischer Anzeige umgeschaltet werden.

Einstellungen

Die häufigsten Einstellungen auf den Reitern Einstellungen, Layout und Filter sind unter Boardlet-Einstellungen beschrieben:

  • Reiter Einstellungen

  • Reiter Layout

Für das Boardlet Legato Grafik Designer stehen folgende weitere Einstellungen zur Verfügung.

Reiter Einstellungen

Dateiname:

Hier kann der Name der Datei angegeben werden, die angezeigt werden soll, z.B. design_new.xml. Die Datei muss unter folgendem Pfad gespeichert sein:

[Pfad zur SE]\web\app\custom\detail-layout

Knotenlayout laden:

Mit diesem Schieberegler kann auf knoten-spezifische Anzeige umgeschaltet werden. Dann wird je nach selektiertem Knoten ein anderes Bild angezeigt. Wenn aktiv, versucht das Boardlet zunächst das für den selektierten Knoten definierte Layout zu laden (Spalte "LGD Dateipfad" in l_nodes im LC2). Gibt es keine explizite Zuordnung einer Layout-Datei zu dem gewählten Knoten, wird die Datei des nächsthöheren Parent-Knotens geladen ("Vererbung von Layout-Dateien"). Diese Dateien müssen unter folgendem Pfad abgelegt sein:

[Pfad zur SE]\web\app\custom\detail-layout

INFO:

Ist der Filter "Knotenlayout laden" gesetzt, wird die Datei unter Boardlet-Settings ignoriert und immer die Datei node_layout_[Knoten-ID].xml geladen.

Reiter Layout

Hintergrundfarbe:

Wendet eine statische Hintergrundfarbe auf das Boardlet an. Die Hintergrundfarbe ist unabhängig vom aktuell gewählten Theme. Um eine Hintergrundfarbe anzugeben, haben Sie die folgenden Möglichkeiten:

  • Sie können die CSS Schlüsselwörter für Farben verwenden, z.B. red, blue, forestgreen, slateblue etc.

  • Alternativ können Sie die hexadezimalen Farbcodes verwenden, z.B. #CC00FF, #0040FF, #B30000 etc.
