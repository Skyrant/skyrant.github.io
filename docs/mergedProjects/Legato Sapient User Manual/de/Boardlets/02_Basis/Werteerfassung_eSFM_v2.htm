Werteerfassung

Allgemein

Das Boardlet Werterfassung wird für manuelles Pflegen von KPIs verwendet.  Das Boardlet Werteerfassung verfügt über ähnliche Funktionen wie das Boardlet Werteerfassung periodenbezogen. Zusätzlich kann durch das Boardlet Werterfassung Chart können die Werte dieses Boardlets visualisiert werden.

Das Boardlet kann Tages- und  schicht-basiert gepflegt werden. Es ist nicht vorgesehen, dass mit dem Boardlet Werteerfassung alle Daten manuell gepflegt werden. Vielmehr ist der Vorteil des Boardlets das manuelle Anreichern der Daten.

Das manuelle Eintragen der Werte ist nicht während eines Shopfloor-Meetings angedacht, sondern zeitlich entweder davor oder danach.

 

Das Boardlet ist nur für manuell gebuchte Variablen innerhalb von eSFM verwendbar.

 

 

Werte buchen

Durch Klick auf das Stift-Symbol wird der Dialog zum Buchen von Werten geöffnet.

Die Zeit oben zeigt das aktuelle Intervall an, für das die Werte erfasst werden. Diese Angabe kann nicht geändert werden.

Darunter werden alle Variablen aufgelistet, die für diesen Knoten in diesem Intervall bebucht werden sollen.

Wurde zu einer Variable ein Sollwert festgelegt (Spalte Zielwert in "Variablen"), so wird diese Variable ebenfalls zum Ändern angeboten, auch wenn kein Intervall für sie eingetragen wurde. Sofern vorhanden, wird der aktuelle Wert der Variablen angezeigt. Nun können die Werte für das Intervall erfasst werden.

Es können die Istwerte und die Sollwerte eingetragen werden.

  

 

Weitere Informationen

  • Das Boardlet ist ein vom bestehenden Werteerfassung periodenbezogen Boardlet getrenntes Boardlet. Hierdurch werden bestehende Installationen, die das alte Boardlet verwenden, nicht beeinträchtigt.

  • In den Optionen wird festgelegt, welche Variablen angezeigt werden. Dies geschieht durch den Variablentyp (wie in Sapient üblich) und das Zeitintervall der Variablen. Es kann nur ein Zeitintervall ausgewählt werden. Wählbare Zeitintervalle sind Schicht oder Tag.

  • Die Standardansicht innerhalb des Boardlets von Schicht- oder Tages-eSFM-Variablen ist immer eine Woche, beginnend mit Montag.

  • Wenn der in den Sapient-Filtern ausgewählte Zeitraum zwei verschiedene Wochen umfasst, wird die frühere Woche angezeigt. Wenn der Sapient-Filter in eine Woche fällt, dann wird diese Woche ab Montag angezeigt.

  • Es ist jederzeit möglich, Werte zu bearbeiten, einzugeben und zu löschen.

  • Die Einstellung 'editierbar' auf der Variablenrelation auf dem Knoten in LC2 gilt auch hier, so dass Werte, die nicht als editierbar eingestellt sind, nicht aus der ICB überschrieben werden können

  • Es werden die vorhandenen knotenbasierten Benutzerrechte verwendet, um die Erlaubnis zum Bearbeiten von Werten zu erteilen. Hiermit wird sichergestellt, dass immer eine feste Anzahl von sieben Spalten zulässig ist.

 

LC2

  • Im LC2 können zugewiesene eSFM (Intervall)-Variablen eingestellt und (Intervall)-Typen für die Variablen festlegt werden. Die Variablen werden im LC2 in der Spalte Variable angezeigt, die Intervalltypen werden im LC2 in der Spalte Intervalltyp angezeigt.

  • Diese Variablen werden im Boardlet angezeigt.

  • Für jede Variable können Sie Werte für jede Schicht festlegen, die sich auf denselben Knoten beziehen (auf den die Variable zugeordnet ist).

  • Sie können im LC2 fünf Intervalltypen einstellen: Tag, Woche, Monat, Jahr und Schicht, aber es werden im Boardlet nur Tag und Schicht angezeigt. Das Boardlet zeigt die Zeilen, die ihrer Variablen und dem entsprechenden Typ darstellen, an:
