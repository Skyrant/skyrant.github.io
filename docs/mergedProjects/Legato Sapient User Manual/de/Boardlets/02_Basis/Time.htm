Uhrzeit

Allgemein

Das Boardlet Uhrzeit zeigt die aktuelle Uhrzeit an. Über die Filtereinstellungen kann ein Zeitlimit und ein Schwellwert in % vorgegeben werden. Durch Klick auf "Start" wird die Stoppuhr dann gestartet; das gesamte Zeitlimit wird grün eingefärbt, ab dem Schwellwert wird gelb eingefärbt.

Einstellungen

Die häufigsten Einstellungen auf den Reitern Einstellungen, Layout und Filter sind unter Boardlet-Einstellungen beschrieben:

  • Reiter Einstellungen

  • Reiter Layout

  • Reiter Filter

Für das Boardlet Uhrzeit stehen folgende weitere Einstellungen zur Verfügung.

Reiter Filter

Zeitlimit (Minuten):

Hier kann das Zeitlimit für die Stoppuhr in Minuten eingetragen werden.

Warnung Schwellwert (Prozent):

Hier kann der Schwellwert für Warnung in Prozent eingegeben werden. Die Restzeit ab dem Schwellwert wird gelb eingefärbt dargestellt.

Benötigte Parametrierung im LC2

Es wird keine Parametrierung benötigt.
