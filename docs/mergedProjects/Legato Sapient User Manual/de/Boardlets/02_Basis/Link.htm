Boardletvorschau mit Link

Allgemein

Das Boardlet Boardletvorschau mit Link kann verwendet werden, um eine Vorschau von einem beliebigen Dashboard einzubetten. Diese Vorschau ist mit einem Link hinterlegt, der beim Anklicken entweder das eingetragene Dashboard öffnet oder eine beliebige URL aufruft. Je nach Einstellung erfolgt die Anzeige im selben Browserfenster oder in einem neuen Tab.

 

Varianten

Es wird zwischen drei Varianten unterschieden:

  1. Link-Boardlets mit ausschließlich Text

  2. Link-Boardlets mit ausschließlich Icons

  3. Link-Boardlets mit Kombination aus Icons und Text

Link-Boardlets mit ausschließlich Text

Bei Link-Boardlets mit ausschließlich Text wird empfohlen, die Schriftgröße an die restlichen Schriftgrößen anzugleichen. Hierdurch wird ein harmonisches Gesamtbild erzeugt. Im besten Fall gibt es auf einem Dashboard nicht mehr als zwei Schriftgrößen (Überschriften und Fließtext).

Link-Boardlets mit ausschließlich Icons

Bei Link-Boardlets mit ausschließlich Icons darf die Größe des Icons abweichen von Icons, die zusammen mit einem Fließtext verwendet werden oder zum Beispiel in der Fußzeile des Dashboards oder in Kopfzeilen der Boardlets enthalten sind.  Link-Boardlets mit Icons haben die Funktion von Buttons auf dem Dashboard und dürfen prominent gestaltet sein.

Beachten Sie, dass die Größe des Icons in einem adäquaten Verhältnis zu den restlichen Elementen sein muss. Es ist zu empfehlen, dass nicht mehr als die doppelte Größe der im Fließtext verwendeten Icons verwendet werden sollte.

 

Link-Boardlets mit Kombination aus Icons und Text

Bei der Verwendung von Link-Boardlets mit der Kombination aus Icon und Text sollte sich die Schriftgröße an die restlichen Schriftgrößen angleichen (z.B. der Fließtext in der Sidebar). Für die Icons sollte die gleiche Schriftgröße verwendet werden.

Beim Verwenden von mehreren Link-Boardlets auf einer Seite sollte die verwendete Größe der Icons bzw. der Schrift einheitlich sein.

Für das Link Boardlet stehen außerdem folgende Einstellungen zur Verfügung.

Reiter Layout

Detail Boardlet:

Über eine Dropdown-Liste kann ein weiteres Boardlet gewählt werden, dessen Werte als Detail-Informationen zu den Knoten angezeigt werden, beispielsweise die Werte bestimmter Variablen. Zur Verfügung stehen die Boardlets Aktuelle Werte, Knoteninfo, Balkendiagramm, Knotenstatus und Tortendiagramm.

Detail-Boardlet-Parameter:

Hier können Parameter für das Detail-Boardlet eingetragen werden, z.B. die zu zeigenden Variablenarten

Beispiel: Die Variablenarten mit der ID 1 und 3 werden angezeigt.

{ "valueTypes": [1,3] }

Die Parameter können vom Boardlet Knoteninfo und vom Boardlet Knotenstatus nicht interpretiert werden, dort sind sie fest vorgegeben.

Link-Typ:

Über eine Dropdown-Liste kann die Art des Links ausgewählt werden. Zur Verfügung stehen:

  • Dashboard - springt zum hinterlegten Dashboard

  • URL - öffnet eine beliebige hinterlegte URL im selben Tab

  • URL in neuem Tab - öffnet eine beliebige hinterlegte URL in einem neuen Tab

Dashboard:

Ist als Link-Typ Dashboard ausgewählt, so kann hier das aufzurufende Dashboard eingetragen werden. Es muss die ID des Dashboards eingegeben werden.

Profil:

Ist als Link-Typ Dashboard ausgewählt, so kann hier das gewünschte Profil des aufzurufenden Dashboards eingetragen werden. Es muss die ID des Profils eingegeben werden. Ist kein Profil angegeben, wird das Default-Profil geladen.

URL:

Ist als Link-Typ URL ausgewählt, so kann hier die aufzurufende URL eingetragen werden.
