KPI Chart

Allgemein

Das Boardlet KPI Chart zeigt definierte Schicht-KPIs für den im Layout selektierten Knoten in einem über den Zeitfilter definierten Zeitraum an.

Bei Mouseover über eine Linie oder einen Balken werden der Zeitstempel und der Wert zu diesem Zeitpunkt eingeblendet.

Durch Klick auf den Namen eines Wertes unten in der Legende kann dieser aus- bzw. wieder eingeblendet werden. Ausgeblendete Werte werden in der Legende hellgrau dargestellt.

Durch Klicken und Ziehen mit gedrückter linker Maustaste kann in den Zeitbereich hineingezoomt werden. Es wird dann ein Button eingeblendet, über den der Zoom wieder zurückgesetzt werden kann.

Die häufigsten Einstellungen auf den Reitern Einstellungen, Layout und Filter sind unter Boardlet-Einstellungen beschrieben:

Für das Boardlet  KPI Chart stehen außerdem folgende Einstellungen zur Verfügung.

Reiter Layout

Diagrammtyp:

Aus der Dropdown-Liste kann der Diagrammtyp gewählt werden. Zur Verfügung stehen Liniendiagramm, Balkendiagramm, Flächendiagramm und gestapeltes Balkendiagramm.

Anzeigevariante Legende:

Aus der Dropdown-Liste kann gewählt werden, ob und wie Beschriftungen angezeigt werden sollen. Zur Verfügung stehen: Legende (unter dem Diagramm), keine Legende / Beschriftungen, keine Legende / keine Beschriftung (Beschriftung nur bei Mouseover).

Reiter Filter

KPI-Typen:

Dropdown-Liste zur Auswahl eines oder mehrerer KPI-Typen. Keine Auswahl = alle KPI-Typen werden ausgewertet.

Benötigte Parametrierung im LC2

Knoten mit aktivem Zeitschema (Produktion), Schichtkalender muss in Sapient gepflegt sein.

KPIs werden dem Knoten im LC2 unter Anwendungsstruktur Fertigungsstruktur [Knoten] KPIs zugewiesen.

Dem Knoten müssen Variablen mit dem Flag "Statistik-relevant" = 1 zugeordnet sein.

Die folgenden SE-Jobs müssen laufen:

  • JOB_POLLER (allgemeiner Job)

  • HANDLE_EVENTS (zum Berechnen der Schichten)

  • REFRESH_EVENT_LIST (zum Nachberechnen von Änderungen an den Schichten)

  • SHIFT_KPI (für die Schicht-KPIs)
