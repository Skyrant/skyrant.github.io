KPI Daten

Allgemein

Das Boardlet KPI Daten zeigt zu einem selektierten Knoten eine statistische Auswertung von Produktionskennzahlen basierend auf den KPI-Typen. Beim Mouseover über eine Spaltenüberschrift wird ein Tooltipp mit dem vollständigen Namen des KPIs angezeigt.

Es werden in jeder Zeile der Tabelle die KPI-Werte von jeweils einer Schicht im eingestellten Zeitraum angezeigt. Ein Klick auf > öffnet die zugehörigen Schichtinformationen.

Mittels Drag & Drop lassen sich die Breite oder Position der Spalten einfach verändern. Durch Klick auf eine Zeilenüberschrift wird nach dieser Spalte sortiert. Nach welcher Spalte sortiert ist und ob auf- oder absteigend sortiert ist, wird durch die dreieckigen Symbole angezeigt.

Für das Boardlet KPI Daten stehen außerdem folgende Einstellungen zur Verfügung.

Reiter Filter

Einbezogene Unterknotenlevel bei Einzelknotenselektion:

Auswahl, wieviele Knotenlevel mit in die Auswertung einbezogen werden sollen, wenn ein einzelner Knoten selektiert ist. Relevant für Modus 4.

Akt. Knoten einbeziehen bei Unterknotenselektion:

Über diesen Schieberegler wird definiert, ob der selektierte Knoten mit angezeigt werden soll, wenn Unterknotenlevel angezeigt werden. Wenn aktiviert, wird der selektierte Knoten inklusive seiner Unterelemente bis zum definierten Level angezeigt. Wenn nicht aktiv, werden nur die Unterknoten des selektierten Knotens bis zum definierten Level angezeigt, er selbst nicht.

Produktionsdatum:

Zeitfilter, hier jedoch auf Produktionstage, nicht auf Kalendertage.

Schichttyp:

Dropdown-Liste zur Auswahl eines oder mehrerer Schichttypen. Keine Auswahl = alle Schichttypen werden ausgewertet.

KPI-Typen:

Dropdown-Liste zur Auswahl eines oder mehrerer KPI-Typen. Keine Auswahl = alle KPI-Typen werden ausgewertet.

Produktionstagsspalte anzeigen:

Über diesen Schieberegler wird definiert, ob die Spalte mit dem jeweiligen Produktionstag angezeigt werden soll.

Schichtspalte anzeigen:

Über diesen Schieberegler wird definiert, ob die Spalte mit der jeweiligen Schicht angezeigt werden soll.

Pfeil anzeigen:

Über diesen Schieberegler wird definiert, ob der Pfeil > zum Öffnen der Schichtdetails angezeigt werden soll.

Benötigte Parametrierung im LC2

Knoten mit aktivem Zeitschema (Produktion), Schichtkalender muss in Sapient gepflegt sein.

KPIs werden dem Knoten im LC2 unter Anwendungsstruktur Fertigungsstruktur [Knoten] KPIs zugewiesen.

Die folgenden SE-Jobs müssen laufen:

  • JOB_POLLER (allgemeiner Job)

  • HANDLE_EVENTS (zum Berechnen der Schichten)

  • REFRESH_EVENT_LIST (zum Nachberechnen von Änderungen an den Schichten)

  • SHIFT_KPI (für die Schicht-KPIs)

  • TIMESLICES (für Online-KPIs)
