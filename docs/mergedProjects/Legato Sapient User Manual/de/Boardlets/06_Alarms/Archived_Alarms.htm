Archivierte Meldungen

Allgemein

Das Boardlet Archivierte Meldungen zeigt zu einem selektierten Knoten alle archivierten Meldungen.

Die Anzeige erfordert einen Zeitfilter. Dieser wird defaultmäßig auf "heute", 00:00:00 - 23:59:59 gesetzt und kann geändert werden.

Die Anzeige kann außerdem nach Meldeklassen oder Meldearten gefiltert werden.

:

Es werden nur Meldungen angezeigt, die sich komplett (also Kommend- und Gehend-Zeitstempel) innerhalb des im Filter eingestellten Zeitbereichs befinden.

 

Die Texte können je nach Meldeklasse anders eingefärbt sein, sofern die Farben konfiguriert sind (neben rot und schwarz, orange und grün). Dies ist jedoch nicht per default verfügbar.

Mittels Drag & Drop lassen sich die Breite oder Position der Spalten einfach verändern. Durch Klick auf eine Zeilenüberschrift wird nach dieser Spalte sortiert. Nach welcher Spalte sortiert ist und ob auf- oder absteigend sortiert ist, wird durch die dreieckigen Symbole angezeigt.

 

Mehrere Funkrufempfänger

Im Boardlet Archivierte Meldungen ist es möglich, nach mehreren Alarm-Funkrufempfängern zu filtern, die mit dem Filter "Alarm Recipients" ausgewählt werden können. Diese Mehrfachauswahl ist via Dropdown-Liste möglich.

 

 

Detailliertere Informationen über die gefilterten Alarme sind in der Seitenleiste "Paging History" abrufbar und werden dort angezeigt.

 

 

Meldung anlegen

Durch Klick auf in der Menüleiste des Boardlets Archivierte Meldungen kann eine neue Meldung angelegt werden.

Es öffnet sich folgende Seitenleiste: --->>

Als Erstes können Sie sich entscheiden, ob eine bestehende Vorlage (Template) verwendet werden soll. Wenn Sie eine Vorlage aus der Dropdown-Liste Vorlage auswählen, werden alle Felder entsprechend vorbefüllt und können anschließend geändert werden.

Am Ende können Sie sich entscheiden, ob eine neue Vorlage mit den Änderungen angelegt werden soll oder ob die ausgewählte Vorlage aktualisiert werden soll. Standardmäßig erfolgt nichts davon.

Außerdem können die Daten für die Meldung eingetragen werden. Die Meldung wird automatisch dem aktuell selektierten Knoten zugewiesen.

Es ist außerdem möglich, direkt beim Anlegen etliche Aktionen für die Meldung durchzuführen. Sie können:

  • eine Begründung und einen Kommentar eingeben,

  • einen Stückzahlverlust erfassen,

  • die Meldung auf Erkannt oder auf Quittiert setzen.

Wurde der selektierten Meldungsbegründung eine Default-Meldeklasse und/oder -art zugewiesen, werden diese Felder automatisch aktualisiert. Zur Visualisierung dass die eingegebenen Werte vom System vergeben wurden, werden die Felder ausgegraut. Es ist jedoch weiterhin möglich, eine andere Klasse oder Art auszuwählen.

 

Vorlage:

Enthält eine Liste der Vorlagen, die erstellt und aktualisiert wurden, wie im Abschnitt als Template benutzen beschrieben. Wählen Sie die Vorlage, mit der Sie die Felder einer neuen Meldung auf der Seitenleiste Meldungen vorbefüllen wollen.

Farbkonvention für Einträge in der Dropdown-Liste Vorlage:

  • Der Name der zuletzt verwendeten Vorlage ist blau markiert.

  • Die Daten, die (a) zuletzt auf der Seitenleiste Meldungen eingegeben wurden und (b) nicht als Vorlage gespeichert wurden, sind ebenfalls blau markiert. Jedoch ist dieser Listeneintrag - vergleichbar mit einem Eintrag in der Zwischenablage Ihres Computers - nur vorübergehend verfügbar, bis der Listeneintrag von einem neuen überschrieben wird.

als Vorlage benutzen:

Wenn Sie (a) keine Vorlage aus der Dropdown-Liste Vorlage gewählt und (b) den Schieberegler als Template benutzen aktiviert haben, dann wird eine neue Vorage angelegt. Alle Daten, die Sie in die Felder auf der Seitenleiste Meldungen eingeben, werden in die neue Vorlage übernommen. Ihre Eingabe in das Feld Meldungstext wird als Name für die Vorlage verwendet.

Meldung bearbeiten - Einzelne Meldung

Wird eine Meldung selektiert, kann sie durch Doppelklick zum Bearbeiten geöffnet werden.

Es öffnet sich folgende Seitenleiste: --->>

Folgende Aktionen sind möglich:

  • Begründen der Meldung (mit zusätzlicher Kommentareingabe).

Wurde der selektierten Meldungsbegründung eine Default-Meldeklasse und/oder -art zugewiesen, werden diese Felder automatisch aktualisiert. Zur Visualisierung dass die eingegebenen Werte vom System vergeben wurden, werden die Felder ausgegraut. Es ist jedoch weiterhin möglich, eine andere Klasse oder Art auszuwählen.

  • Stückzahlverlust erfassen.

  • Ändern des Meldetexts, des Zeitstempels, der Meldeklasse und der Meldeart.

Ist die Meldung bereits zu alt, um noch editiert werden zu können (gemäß Systemparameter -170), wird eine entsprechende Fehlermeldung angezeigt.

 

Meldung bearbeiten - Mehrere Meldungen

Es können auch mehrere Meldungen im Boardlet Archivierte Meldungen werden. Durch Klick auf in der Menüleiste des Boardlet können diese dann gleichzeitig editiert werden.

Folgende Aktionen sind möglich:

  • Erkanntsetzen der Meldungen.

  • Quittieren der Meldungen.

  • Begründen der Meldung (mit zusätzlicher Kommentareingabe).

Wurde der selektierten Meldungsbegründung eine Default-Meldeklasse und/oder -art zugewiesen, werden diese Felder automatisch aktualisiert. Zur Visualisierung dass die eingegebenen Werte vom System vergeben wurden, werden die Felder ausgegraut. Es ist jedoch weiterhin möglich, eine andere Klasse oder Art auszuwählen.

 

 

  • Ändern des Meldetexts, des Zeitstempels, der Meldeklasse und der Meldeart. Enthalten die selektierten Meldungen unterschiedliche Werte für diese Felder, wird ein angezeigt.

 

Hinter der jeweiligen Aktion wird angegeben, für wie viele der selektierten Meldungen die jeweilige Aktion möglich ist. Außerdem kann jede Auswahl über den Schieberegler Nicht ändern zurückgesetzt werden.

 

Beispiel:

Meldung aufteilen

Ist eine einzelne Meldung im Boardlet Archivierte Meldungen markiert, wird in der Menüleiste des Boardlets ein zusätzliches Symbol angezeigt: .

Durch Klick darauf kann die Meldung aufgeteilt werden.

Es öffnet sich folgende Seitenleiste: --->>

Zunächst wird festgelegt, in wieviele Meldungen die markierte Meldung aufgeteilt werden soll. Darunter werden automatisch die benötigte Anzahl Meldungen angezeigt. Hier können die Details wie Klasse und Begründung geändert werden.

Außerdem kann für alle Meldungen außer der letzten der Zeitstempel des Zurücksetzens und damit die Dauer der Meldung geändert werden. Die Zeitstempel der letzten neuen Meldung werden automatisch berechnet. Desweiteren kann der Stückzahlverlust neu verteilt werden. Diese beiden Aktionen können über die Buttons unten auch automatisiert durchgeführt werden:

  • Zeit basierend auf Stückzahlverlusten ausfüllen

  • Stückzahlverlust basierend auf Zeit ausfüllen

 

Außerdem können weitere Meldungen angelegt werden. Über neben jeder Meldung kann diese wieder gelöscht werden.

 

Meldungen zusammenführen

Sind mehrere Meldungen im Boardlet Archivierte Meldungen markiert, wird in der Menüleiste des Boardlets ein zusätzliches Symbol angezeigt: .

Durch Klick darauf können die Meldungen zu einer Meldung zusammengefasst werden.

Es öffnet sich folgende Seitenleiste: --->>

Die Zeitstempel werden automatisch auf den Ausgelöst-Zeitstempel der früheren und den Zurückgesetzt-Zeitstempel der späteren Meldung gesetzt. Sie können für die neue Meldung auch angepasst werden. Außerdem müssen Meldetext, Meldeklasse, Meldeart und ggf. Begründung und Stückzahlverlust eingegeben werden.

 

Zusammenführen/Aufteilen rückgängig machen

Wurde eine Meldung selektiert, die zuvor entweder aufgeteilt oder zusammengeführt wurde, wird in der Menüleiste des Boardlets ein zusätzliches Symbol angezeigt: .

Durch Klick darauf kann die Aktion rückgängig gemacht werden und die originale Meldung/die originalen Meldungen wiederhergestellt werden.

 

 
Einstellungen

Die häufigsten Einstellungen auf den Reitern Einstellungen, Layout und Filter sind unter Boardlet-Einstellungen beschrieben:

  • Reiter Einstellungen

  • Reiter Layout

  • Reiter Filter

Für das Boardlet Archivierte Meldungen stehen außerdem folgende Einstellungen zur Verfügung.

Reiter Filter

Meldeklassen:

Filter auf Meldeklassen. Es werden alle Meldungen der hier eingegebenen Klassen für die selektierten Knoten angezeigt. Die Meldeklassen können aus einer Dropdown-Liste gewählt werden. Leer = kein Filter, alle Meldeklassen werden angezeigt.

Meldearten:

Filter auf Meldearten. Es werden alle Meldungen der hier eingegebenen Arten für die selektierten Knoten angezeigt. Die Meldearten können aus einer Dropdown-Liste gewählt werden. Leer = kein Filter, alle Meldearten werden angezeigt.

Meldungstext:

Hier kann ein Meldungstext eingegeben werden, auf den gefiltert werden soll. Als Platzhalter können % für beliebig viele Zeichen oder _ für genau ein Zeichen verwendet werden. Der Filter berücksichtigt Groß- und Kleinschreibung. Mehrere Filterkriterien können durch Komma verbunden werden (ODER-Filter).

Benötigte Parametrierung im LC2

Die Meldeklassen werden im LC2 unter Werksverwaltung Meldungsmanagement Meldeklassen gepflegt.

Die Meldearten werden im LC2 unter Werksverwaltung Meldungsmanagement Meldearten gepflegt.

Meldungsdefinitionen werden im LC2 an zwei Stellen gepflegt werden:

  • unter Anwendungsstruktur Datenaustausch [Modul] [Verbindung] Meldungsdefinition gepflegt

- oder -

  • unter Anwendungsstruktur Fertigungsstruktur [Knoten] Meldungszuordnung gepflegt.

Die Meldungsbegründungen werden im LC2 unter Werksverwaltung Meldungsmanagement Meldungsbegründungen gepflegt.
