Bearbeitungs-Shortcuts

Allgemein

Dieses Boardlet bietet Buttons zum Ändern der Meldeklasse als Shortcut. Welche Buttons zur Verfügung stehen, kann über den Filter festgelegt werden. Die Meldungen selbst werden in einem anderen Boardlet visualisiert, z.B. dem Meldungs-Gantt Boardlet. Solange keine Meldung dort selektiert ist, werden die Shortcuts ausgegraut dargestellt. Ist eine Meldung zur Bearbeitung selektiert, werden die Shortcuts farbig angezeigt. Die Buttons sind je nach Meldeklasse anders eingefärbt und zeigen zudem das Icon der Meldeklasse.

Für ein Bearbeitungs-Shortcut Boardlet stehen außerdem folgende Einstellungen zur Verfügung.

Reiter Filter

Meldeklassen:

Filter auf Meldeklassen. Es werden alle Meldungen der hier eingegebenen Klassen für die selektierten Knoten angezeigt. Die Meldeklassen können aus einer Dropdown-Liste gewählt werden. Leer = kein Filter, alle Meldeklassen werden angezeigt.

Benötigte Parametrierung im LC2

Die Meldeklassen werden im LC2 unter Werksverwaltung Meldungsmanagement Meldeklassen gepflegt.
