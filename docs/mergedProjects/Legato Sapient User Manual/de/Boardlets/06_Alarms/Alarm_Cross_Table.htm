Kreuztabelle

Allgemein

Das Boardlet Kreuztabelle zeigt die Häufigkeit und Dauer von Meldungen pro Schicht in Tabellenform. Jede Reihe steht für einen Eintrag pro Schicht. In Reihen, in denen die Aggregierung der Schichtdaten noch läuft, wird das Symbol Uhr neben dem Schichtnamen angezeigt. Das Symbol Uhr ist ausgeblendet, wenn die Daten aktuell sind.

Mittels Drag & Drop lassen sich die Breite oder Position der Spalten einfach verändern. Durch Klick auf eine Zeilenüberschrift wird nach dieser Spalte sortiert.

Für das Boardlet Kreuztabelle stehen außerdem folgende Einstellungen zur Verfügung.

Reiter Layout

Fußzeilenkomponente:

Die Werte der einzelnen Spalten werden in der Tabellenfußzeile aufsummiert. Die Tabellenfußzeile wird mit dem Komponentennamen alarm-cross-table-footer-component angezeigt.

Reiter Filter

Zugriff

Zugriff auf den Reiter Filter eines Boardlets erhalten Sie über das Menü Quick-Edit [] Filter.

Knoten-ID:

ID des Knotens, für den Werte angezeigt werden. Kann entweder hier geändert werden oder durch Auswahl eines anderen Knotens im Boardlet Knotenlayout oder im Boardlet Knotenbaum (außerhalb des Bearbeitungsmodus).

Meldeklassen:

Filter auf Meldeklassen. Es werden alle Meldungen der hier eingegebenen Klassen für die selektierten Knoten angezeigt. Die Meldeklassen können aus einer Dropdown-Liste gewählt werden. Leer = kein Filter, alle Meldeklassen werden angezeigt.

Meldearten:

Filter auf Meldearten. Es werden alle Meldungen der hier eingegebenen Arten für die selektierten Knoten angezeigt. Die Meldearten können aus einer Dropdown-Liste gewählt werden. Leer = kein Filter, alle Meldearten werden angezeigt.

Meldungstext:

Hier kann ein Meldungstext eingegeben werden, auf den gefiltert werden soll. Als Platzhalter können % für beliebig viele Zeichen oder _ für genau ein Zeichen verwendet werden. Der Filter berücksichtigt Groß- und Kleinschreibung. Mehrere Filterkriterien können durch Komma verbunden werden (ODER-Filter).

Meldesymbolik:

Filter auf Symbole in der Datenbank.

Knoten:

Filter auf Knotennamen. Leer = kein Filter, alle Knoten werden angezeigt.

 

:

Mit einer neuen Funktion können Sie beim Anklicken eines Eintrags in einem Boardlet für Archivierte Meldungen die zugehörigen Meldungen anzeigen lassen.

Die Kreuztabelle und das Boardlet für Archivierte Meldungen haben unterschiedliche Filtereinstellungen. Dadurch kann es vorkommen, dass nicht die richtigen Meldungen angezeigt werden können. Dies ist jedoch abhängig von den gewählten Filtern in der Kreuztabelle.

 

Benötigte Parametrierung im LC2

Die Meldeklassen werden im LC2 unter Werksverwaltung Meldungsmanagement Meldeklassen gepflegt.

Die Meldearten werden im LC2 unter Werksverwaltung Meldungsmanagement Meldearten gepflegt.

Meldungsdefinitionen werden im LC2 an zwei Stellen gepflegt werden:

  • unter Anwendungsstruktur Datenaustausch [Modul] [Verbindung] Meldungsdefinition gepflegt

- oder -

  • unter Anwendungsstruktur Fertigungsstruktur [Knoten] Meldungszuordnung gepflegt.

Damit die Berechnungen durchgeführt werden, müssen die zwei SE Jobs CROSS_TABLE_EXCEPTIONS und CROSS_TABLE_RUN_SHIFTS im LC2 aktiviert werden unter System-Administration SE Jobs SE Jobs siehe die Checkboxen in der Spalte Aktiv?.
