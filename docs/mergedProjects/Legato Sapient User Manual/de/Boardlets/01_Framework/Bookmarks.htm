Lesezeichen

Allgemein

Das Boardlet Lesezeichen kann verwendet werden, um alle für den angemeldeten Benutzer gespeicherten Lesezeichen anzuzeigen.

In der Spalte Aktionen werden die möglichen Aktionen angezeigt:

Öffnet das Dashboard mit dem Boardlet, für das das Lesezeichen erstellt wurde. Das entsprechende Boardlet ist im Dashboard grün umrandet.

Löscht das Lesezeichen.

 

Einstellungen

Die häufigsten Einstellungen auf den Reitern Einstellungen, Layout und Filter sind unter Boardlet-Einstellungen beschrieben:

  • Reiter Einstellungen

  • Reiter Layout

  • Reiter Filter

Für das Boardlet Lesezeichen stehen keine weiteren Einstellungen zur Verfügung.

Benötigte Parametrierung im LC2

Es wird keine Parametrierung benötigt.

Zu Informationszwecken kann die Liste der Lesezeichen für den Benutzer im LC2 unter System-Administration Benutzer Lesezeichen eingesehen werden.
