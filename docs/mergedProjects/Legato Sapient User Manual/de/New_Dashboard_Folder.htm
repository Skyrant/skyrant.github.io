Dashboards in Ordnern gruppieren

Sie können Dashboards in Ordnern und Unterordnern gruppieren. Die Ordner werden im LC2 (Legato Configuration Center) angelegt und können beliebig ineinander verschachtelt werden.

So legen Sie Ordner für Dashboards an:

1

Starten Sie LC2 und loggen Sie sich ein.

2

Wählen Sie Frontend Dashboard-Ordner.

3

Klicken Sie mit der rechten Maustaste auf Dashboard-Ordner und wählen Sie dann Dashboard-Ordner hinzufügen.

Der Dialog Hinzufügen Dashboard-Ordner erscheint.

4

Geben Sie einen Ordnernamen ins Feld Name (de) ein und klicken Sie OK.

  • Um einen Unterordner anzulegen, klicken Sie mit der rechten Maustaste auf den gewünschten Ordner und wählen Sie dann Dashboard-Ordner hinzufügen.

  • Sie können nun über die Legato Sapient WebApp die gewünschten Dashboards in den neuen Ordner bzw. Unterordner verschieben.

So verschieben Sie ein Dashboard in einen anderen Ordner:

1

Wechseln Sie in die Legato Sapient WebApp.

2

Rufen Sie den Bearbeitungsmodus für das gewünschte Dashboard auf. [so geht's]

3

Klicken Sie in der Seitenleiste rechts auf den Reiter Einstellungen

4

Geben Sie im Such-Feld der Dropdown-Liste Ordner den gewünschten Ordnernamen ein:

5

Bestätigen Sie die Ordnerauswahl mit einem Mausklick.

6

Speichern Sie Ihre Dashboard-Änderung.
