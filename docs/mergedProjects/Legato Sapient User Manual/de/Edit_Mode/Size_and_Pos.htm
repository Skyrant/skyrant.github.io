Größe und Position eines Boardlets

Durch Klicken und Ziehen auf ein Boardlet kann dieses an eine andere Position gezogen werden. Die anderen Boardlets werden dann ggf. auch verschoben, so dass es keine Überlappungen gibt. Die Boardlets werden dabei automatisch ausgerichtet.
