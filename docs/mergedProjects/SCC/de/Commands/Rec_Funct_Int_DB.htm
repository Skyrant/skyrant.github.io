Recordfunktionen für die interne Datenbank

Für die Recordfunktionen kann in einem Script eine Variable vom Typ 'DBREC' deklariert werden. Alle Recordfunktionen werden dann mit dem Namen der Variablen, einem Punkt und dem Namen der Funktion gemeinsam angesprochen.

Beispiel:

dbtbl    tbl

dbrec    rec

tbl.open('c:\\files\\data.dbf')

tbl.bufopen(rec)

rec.close

 

ASCII

Deklaration: string dbrec.Ascii(long field)

Holt oder schreibt das Feld field aus dem Recordpuffer bzw. in den Recordpuffer. Unabhängig vom Feldtyp wird das Feld in ASCII bearbeitet.

Beispiel:

dbtbl    tbl

dbrec    rec

string   s

tbl.open('c:\\files\\data.dbf')

tbl.bufopen(rec)

tbl.get(rec)

s := rec.ascii(1)

rec.ascii(1) := '123'

rec.close

tbl.close

 

Alpha

Deklaration: string dbrec.Alpha(long field)

Holt oder schreibt das Feld field aus dem Recordpuffer bzw. in den Recordpuffer. Das Feld muss den Typ 'C' (Character, alphanumerisch) haben.

Beispiel:

dbtbl    tbl

dbrec    rec

string   s     

tbl.open('c:\\files\\data.dbf')

tbl.bufopen(rec)

tbl.get(rec)

s := rec.alpha(1)

rec.alpha(1) := 'writing'

rec.close

tbl.close

Close

Deklaration: dbrec.Close

Schließt den Recordpuffer und gibt den belegten Speicher frei.

Beispiel:

string   s

tbl.bufopen(rec)

tbl.get(rec)

s := rec.ascii(3)

rec.close

tbl.close

Empty

Deklaration: dbrec.Empty

Löscht den Inhalt des Recordpuffers.

Beispiel:

dbrec    rec

string   s

tbl.open('c:\\files\\data.dbf')

tbl.bufopen(rec)

rec.empty

tbl.append(rec)

rec.close

tbl.close
