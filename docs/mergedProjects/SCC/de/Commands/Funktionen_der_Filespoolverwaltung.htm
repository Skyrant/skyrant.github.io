Funktionen der Filespoolverwaltung

 

FileSpoolInit

Deklaration: int FileSpoolInit(string node, long id, bool spool_in, bool spool_out)

Öffnet einen Spool für den Knoten node, sofern dieser die Spoolfunktionalität unterstützt. Die id kann frei vergeben werden und dient der Zuordnung für die folgenden Funktionen. Die Spools können verwendet werden um Aufgaben zwischen zwei ITC's zu verteilen. Die Richtung wird über die Parameter spool_in (Einbuchen in den Spool) und spool_out (Ausbuchen aus dem Spool) festgelegt. Wird der Spool von einem einzelnen ITC verwendet, sind beide Parameter auf true zu setzen.

Die Funktion liefert folgende Rückgabewerte:

0                 kein Fehler, Spool wurde angelegt

1                 Spool mit der id ist bereits vorhanden

-1        Fehler, Spool kann nicht angelegt werden           

 

Beispiel:

int rv

rv := FileSpoolInit('PubSub.Mosquitto', 2, true, true)

 

FileSpoolExit

Deklaration: int FileSpoolExit(string node, long id)

Der Spool mit der id des Knotens node wird geschlossen. Es erfolgt keine Abarbeitung des Spools mehr und es ist auch nicht mehr möglich etwas einzubuchen.


Die Funktion liefert folgende Rückgabewerte:

0                 kein Fehler, Spool wurde geschlossen

-1        Spool oder Knoten nicht vorhanden       

 

Beispiel:

int rv

rv := FileSpoolExit('PubSub.Mosquitto', 2)

 

FileSpoolPut

Deklaration: int FileSpoolPut(string node, long id, string data, long hold)

Bucht einen Datensatz data in den Spool mit der id des Knotens node. Der Spool wird nicht angelegt oder geöffnet, dies muss zuvor einmalig mit FileSpoolInit gemacht worden sein.
Über den Parameter hold kann festgelegt werden, wie lange der Datensatz verarbeitet werden darf. Ist der Datensatz älter als hold, wird er beim Ausbuchen verworfen. Die Angabe erfolgt in ms. Soll der Datensatz in jedem Fall verarbeitet werden, ist -1 anzugeben.

Die Funktion liefert folgende Rückgabewerte:

0                 kein Fehler, Buchung erfolgt

-1        Spool oder Knoten nicht vorhanden

 

Beispiel:

long rv
string stmt, js

 

Value.Index := Value.Index + 1
js := '{\"value\":'+ltostr(Value.Index,10)+'}'

stmt := publishfromstr('value/json', js, 0, false, false)

 

rv := filespoolput('PubSub.Mosquitto', 2, stmt, -1)

 

FileSpoolSize

Deklaration: int FileSpoolSize(string node, long id)

Liefert die Anzahl der Datensätze, die im Spool mit der id des Knotens node vorhanden sind.

Die Funktion liefert folgende Rückgabewerte:

 

-1           Spool oder Knoten nicht vorhanden

>=0        Anzahl der Datensätze

 

Beispiel:

int size

size := FileSpoolSize('PubSub.Mosquitto', 2)

 

PublishFromStr

Deklaration: string PublishFromStr(string topic, string message, int qos, bool dup, bool retain)

Erzeugt einen spoolfähigen String aus den übergebenen Publish-Angaben. message ist die Nachricht zum Thema topic. qos definiert die Qualität des Services.

qos:

0 höchstens einmal

1 mindestens einmal

2 genau einmal

 

dup gibt an, ob es sich um eine Wiederholung der Nachricht handelt (dup = true). Bei qos = 0 muss dup false sein!

retain = true veranlasst den Server, die Nachricht zu speichern, um den Wert auch zukünftigen Clients senden zu können.

Beispiel:

long rv
string stmt, js

 

Value.Index := Value.Index + 1

js := '{\"value\":'+ltostr(Value.Index,10)+'}'

 

stmt := PublishFromStr('value/json', js, 0, false, false)

 

PublishFromBlk

Deklaration: string PublishFromBlk(string topic, block message, int qos, bool dup, bool retain)

Erzeugt einen spoolfähigen String aus den übergebenen Publish-Angaben. message ist die Nachricht zum Thema topic. qos definiert die Qualität des Services.

qos:

0 höchstens einmal

1 mindestens einmal

2 genau einmal

 

dup gibt an, ob es sich um eine Wiederholung der Nachricht handelt (dup = true). Bei qos = 0 muss dup false sein!

retain = true veranlasst den Server, die Nachricht zu speichern, um den Wert auch zukünftigen Clients senden zu können.

Beispiel:

long rv

string stmt

block b

int i, j

 

j := Value.Index + 2

Value.Index := Value.Index + 1

 

b := newblock(j)

while i < j do

setblocknum(b, i, 1, false, i)

               i := i + 1

enddo

 

stmt := publishfromblk('value/datablock', b, 0, false, false)
