Menüleiste

Datei-Menü

Datei | Mit Service verbinden

Verwenden Sie diesen Befehl, um sich mit einem Service zu verbinden. Der gewünschte Service kann über eine Drop-Down Liste ausgewählt werden.

Datei | Verbindung beenden

Beendet eine bestehende Verbindung zu einem Service.

Datei | Services verwalten

Hier können Sie Services verwalten. Dies beinhaltet das Anlegen neuer Services, das Bearbeiten bestehender Services sowie die Bearbeitung der Logeinstellungen für die Services.

Datei | Editor beenden

Mit diesem Eintrag wird der Editor beendet.

Projekt-Menü

Projekt | Neu

Öffnet den Dateibrowser, um den Namen und Speicherort einer neuen Projektdatei festzulegen.

Projekt | Öffnen

Öffnet den Dateibrowser zur Auswahl eines Cloud Connector Projekts, das geladen werden soll.

Projekt | Schließen

Schließt das aktuell geöffnete Projekt.

Projekt | Aktualisieren

Aktualisiert das aktuell geöffnete Projekt.

Projekt | Speichern

Speichert die Änderungen am aktuell geöffneten Projekt.

Projekt | Speichern unter...

Öffnet den Dateibrowser, um den Speicherort der Projektdatei festzulegen.

Projekt | Ablauf Start

Alle Elemente des Projekts werden gestartet. Soll ein Element nicht automatisch gestartet werden, so kann dies durch Anwahl der Menüoption "Nicht mit Projekt starten" im Elementmenü verhindert werden. Hat ein Element noch Unterelemente, so werden diese dann ebenfalls nicht gestartet. Wurden Eventelemente vom Typ "Ablaufstart" angelegt, so werden diese im Anschluss einmalig ausgeführt.

Projekt | Ablauf Stop

Alle Elemente des Projekts werden gestoppt. Wurden Eventelemente vom Typ "Ablaufende" angelegt, so werden diese zuvor einmalig ausgeführt.

Diagnose-Menü

Diagnose | Beobachten Neu...

Öffnet ein neues Fenster zum Beobachten der Variablenwerte.

Diagnose | Beobachten Öffnen...

Öffnet eine vorhandene Datei zum Beobachten der Variablenwerte (Format .DBG).

Diagnose | Performance Monitor

Blendet das Fenster zum Beobachten der Performance ein. Für verschiedene Prozesse können die minimalen, maximalen und Durchschnittswerte als Wert oder als Zeit in Millisekunden geprüft werden. Für jeden Wert wird zusätzlich angezeigt, wann dieser aufgetreten ist.

Logging-Menü

Logging | Logeinstellungen Editor...

Hier kann ein Name und der Speicherort für die Logdatei des Editors angegeben werden. Außerdem kann über die Logmaske festgelegt werden, welche Meldeklassen geloggt werden sollen. Siehe auch Editor-Log.

Logging | Logeinstellungen Service...

Hier können die Logklassen für das Log des Services an- bzw. abgewählt werden, ohne den Service neu starten zu müssen. Die allgemeinen Logeinstelllungen für den Service werden unter Datei | Serviceadministration getroffen. Siehe auch Service-Log.

Logging | Verwaltung Logfilezipper Service...

Hier können Einstellungen für den Logfilezipper Service getroffen werden. Siehe auch Logfilezipper.

Fenster-Menü

Fenster | Nebeneinander

Ordnet alle Fenster, die sich im Multidokumentfenster befinden, nebeneinander an, wobei der zur Verfügung stehende Platz voll genutzt wird.

Fenster | Übereinander

Ordnet alle Fenster, die sich im Multidokumentfenster befinden, kaskadenförmig hintereinander an.

Fenster | Symbole Anordnen

Ordnet alle Fenster, die als Symbol im Multidokumentfenster angezeigt werden, nebeneinander am unteren Rand des Multidokumentfensters an.

Fenster | Alle schließen

Schließt alle Fenster, die sich im Multidokumentfenster befinden.

Fenster| Im Vordergrund halten

Hält den Cloud Connector immer im Vordergrund, auch wenn ein anderes Programm gerade aktiv ist.

Extras-Menü

Extras | DBF-Editor

Es wird ein einfacher Editor aufgerufen, mit dem dBase-Dateien angesehen werden können, die auf der Festplatte gespeichert sind.

Extras | DBF-Editor Memorydateien

Es wird ein einfacher Editor aufgerufen, mit dem dBase-Dateien angesehen werden können, die im Hauptspeicher gehalten werden.

Extras | Editoreinstellungen

Projektpfad: Der Projektpfad gibt den Standardpfad an, der beim Dialog "Datei Öffnen" bzw. "Speichern" verwendet wird.

Sprache: Hier kann die Sprache für die Benutzeroberfläche gewählt werden.

Scripteditor: Ermöglicht die Einstellung der Farben sowie der Tab-Länge im Scripteditor.

 

Extras | Editor SCC-OPC

Der Editor für die Konfiguration des GEFASOFT OPC-Servers wird aufgerufen.

Hilfe-Menü

Hilfe | Inhalt

Ruft die HTML-Hilfe auf und zeigt den Inhalt der Hilfe an.

Hilfe | Info

Zeigt den Info-Dialog mit der Version des Cloud Connectors.
