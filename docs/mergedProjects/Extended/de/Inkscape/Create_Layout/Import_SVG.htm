SVG in Inkscape importieren

SVG-Pfad in Sapient auslesen

Um den Pfad einer SVG auszulesen, wird das Boardlet "Einstellungen zum Knotenlayout" benötigt. Wenn man nun auf eine SVG klickt erscheint im Einstellungsboardlet der dazugehörige Pfad.

SVG in Inkscape erstellen

Als erstes muss ein beliebiges SVG Objekt erstellt werden, um einen Eintrag in der XML-Datei zu generieren. Dazu kann beispielsweise das Bézier-Tool verwendet werden.

Der XML-Editor

Der XML-Editor wird nun unter Bearbeiten -> XML-Editor oder mit Umschalt+Strg+X geöffnet. Es erscheint folgendes Fenster:

Das Element hat ein Attribut d, in dem der Pfad hinterlegt ist. Dieser Pfad kann jetzt durch den SVG Pfad aus Sapient ersetzt werden. Der Pfad ist jetzt importiert und das Objekt kann bearbeitet werden.
