Neues Boardlet basierend auf Datasource-Model

Es ist möglich, ein neues Boardlet basierend auf einem Datasource-Model zu erstellen. Dies kann folgendermaßen geschehen:

Im Client:

neues Dashboard anlegen

Boardlet "Generische Tabelle" reinziehen

das gewünschte Datasource-Model auswählen, das als Basis dienen soll

die anzuzeigenden Spalten und Parameter konfigurieren

In der Datenbank:

neuen Eintrag in gp_r_boardlets erstellen (Flags published = 1 und allow_system_update = 0 setzen)

aus u_dashboards das neue Dashboard auswählen und den Eintrag in der Spalte "configuration" herauskopieren

die Zwischenablage in einem Texteditor öffnen

alles bis zum Eintrag "tableColumns":{"value":{            "datasourceModel":" (ohne diesen Teil) löschen

vorne "{" ergänzen

den Text in die Spalte gp_r_boardlets.params einfügen

Nun steht das neue Boardlet in der WebApp zur Verfügung.
