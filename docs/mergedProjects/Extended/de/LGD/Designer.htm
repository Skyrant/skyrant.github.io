Visualisierung - Designer

Der Visualisierungsdesigner ermöglicht es, solche Zeichnungen zu erstellen, die mithilfe des Graphic Designers oder Graphic Production Flow Boardlets in jedem beliebigen Dashboard angezeigt werden können. Weitere Anwendungsfälle sind individuelle Maschinenklienten (HMI) und Andon-Bildschirme..

Der Designer ist vollkommen webbasiert und kann über jeden Standardbrowser geöffnet werden. Er basiert auf der Open-Source Software draw.io, die eine Vielzahl von Anleitungen und Webartikeln beinhaltet.

Bisher kann der Designer statische, schematische Zeichnungen erstellen, wobei er auf eine außerordentlich umfangreiche Bibliothek von Standardelementen zurückgreift.

Gefasoft hat den Designer in Legato Sapient integriert und ihn um die Möglichkeit erweitert, diese Zeichnungen zu animieren, indem Elementeigenschaften/-stile mit Prozessdaten über Bindings verbunden werden – die Animationen werden während der Laufzeit befüllt.

Normalerweise werden Prozessdaten als Text angezeigt oder dazu verwendet, die Farbe, Position, Größe oder Sichtbarkeit eines Elements festzulegen. Bindings können außerdem verwendet werden, um Aktionen für Events auszuführen, wie Anklicken eines Elements, Überfahren eines Elements mit der Maus, etc.
