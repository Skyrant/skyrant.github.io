LC2 Versions

Typ:

Systemparameter

Name:

LC2 Versions

ID:

-200

Default:

-

Format:

String

Beschreibung:

Freigegebene Versionsnummern für das Legato Configuration Center LC2. Diese müssen im Format Vx.x.x.x und durch Komma getrennt angegeben werden. Beispiel: V1.0.0.7,V1.0.1.0, etc. Bleibt das Feld leer, so gibt es auch keine Einschränkungen und man kann mit jeder Version des LC2 arbeiten.
