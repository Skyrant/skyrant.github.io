Wiederholen der Ausführung

Die Haupt-Task (und nur diese) kann, bevor sie "fertig" zurückmeldet (indem sie aus execute() null zurückgibt), eine weitere "Runde" der Ausführung verlangen, indem sie der Eigenschaft solver.again den Wert true zuweist. Anmerkung: in allen anderen Fällen (nicht Haupt-Task oder bei Ergebnis ungleich null) werden Zuweisungen an solver.again ignoriert.
