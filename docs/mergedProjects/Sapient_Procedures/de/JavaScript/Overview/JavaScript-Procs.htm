Besonderheiten für JavaScript-Prozeduren

Über JSON-RPC aufgerufene JS-Funktionen haben ebenfalls die beiden Parameter "SapientContext" und "params". Letzteres enthält die vom Remote-Aufrufer angegebenen Parameter entweder als Array oder als JS-Objekt.

:

Beide Fälle müssen also in den jeweiligen Funktionen berücksichtigt werden!

Um als Array angegebene Parameter in das Objekt-Format zu wandeln, kann die Funktion "jsonRpcArgsHash()" aus "scripts/common/jsutils.js" verwendet werden.

/** Helper for JSON-RPC functions. Those may either be called with their

 * parameters passed by position, if the caller passes them as an JSON array

 * or by name,  if the caller passes them as a JSON object.<p>

 * A JSON-RPC function may call this function to ...

 *

 * @param args The arguments object of the JSON-RPC function.  The 1st argument

 * is the Sapient context,  the 2nd is either a Object containing parameters

 * passed by name or the 1st parameter passed by position

 * @param names An array containing the names of the parameters expected by the

 * RPC function

 * @returns An Object with properties for

 */
