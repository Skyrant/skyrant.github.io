Besonderheiten für JavaScript-Jobs

Ein JavaScript-Job bekommt SapientContext übergeben. Außerdem werden folgende Informationen / Parameter übergeben (aus se_jobs und se_job_triggers):

scheduler

job_symbol

function_path

config_status (kann folgende Strings haben):

initial

unchanged

updated

 

job_params

trigger_symbol (1 String oder Array von Strings, z.B. "5 min" oder ["5 min", "10 min", "1 day"])

pro auslösendem Trigger Parameter aus se_job_triggers.params
