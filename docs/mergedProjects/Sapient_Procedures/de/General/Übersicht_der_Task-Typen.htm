Übersicht der Task-Typen

Die Aufgaben einer Sapient Engine sind in Tasks aufgeteilt. Jeder Task hat einen Typ und kann nur einmal pro SE konfiguriert werden.

Hierbei hat einer der Tasks den Typ "Jobs", der dafür zuständig ist, die Jobs auszuführen.

 

Die Sapient Engine kann folgende Task-Typen ausführen:
