TARGETCHAINS_ONLINE

Beschreibung

Dieser Job dient zum Ausrollen von ad-hoc Änderungen der Sollwerte für Schichten. Es werden nur Vererbungsketten behandelt.

Dieser Job beruht auf Ereignissen des Eventbus der Sapient Engine. Nachdem diese Ereignisse nicht über verschiedene Instanzen der SE verteilt werden können, kann der Job nur auf Ereignisse reagieren, der in der selben Instanz der Sapient Engine auftragen, in der der Job läuft. Das heißt, der Job muss für jede Instanz implementiert werden, die Werterfassungsdienste anbietet (SCC, WebApp, etc.)

Trigger

Standard-Trigger: Wert (-6)

Standard-Parameter:

{

"evcheck_func":"de/gefasoft/sapient/sapieng/jobs/targetvals/TargetValues/isTargetChainParent"

}

Damit wird überprüft, ob es sich beim übergebenen Wert um den Parent einer Vererbungskette handelt.

Dem Trigger kann ein zusätzlicher Parameter "headsOnly" übergeben werden. Existiert dieser Parameter und ist er auf "false" gesetzt, wird der Job ausgelöst, auch wenn der geänderte Wert nicht der Parent einer Vererbungskette ist, solange es in der Kette vom übergebenen Wert abhängige Werte gibt. Defaultmäßig existiert dieser Parameter nicht, so dass nur "komplette" Vererbungsketten ausgerollt werden.
