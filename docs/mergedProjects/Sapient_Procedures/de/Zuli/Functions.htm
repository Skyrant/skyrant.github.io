Verfügbare (Hilfs-)Funktionen

Da der Job komplett in JavaScript implementiert wurde, können (theoretisch) alle Funktionen direkt aufgerufen werden. Beschrieben sind hier nur die Funktionen, deren Verwendung als sinnvoll erachtet wird.

Alle im Standard verwendeten Hilfsfunktionen beginnen mit einem '_'.

(Sinnvolle) Hilfsfunktionen (alle diese Hilfsfunktionen sind in der Datei 'ZuliImp_Utils.js' angesiedelt):

_trim(s)

Rückgabewert String, entfernt alle Leerzeichen vor dem ersten und nach dem letzten Zeichen von String 's'

_S5OpToOpCode(operand)

Rückgabewert Long, wandelt S5-Operanden wie z.B. M 127.0 in eine Zahl

_S7OpToOpCode(operand)

Rückgabewert Long, wandelt S7-Operanden wie z.B. M 127.0 in eine Zahl

_getImportLine(headLine, importFile_Line, vh)

Rückgabewert Map, kombiniert die übergebene Kopfzeile mit den übergebenen Werten der Importdatei-Zeile

_getNullLine()

Rückgabewert Map, liefert eine Map zur Übergabe an _saveLine(...), als Wert wird immer = null übergeben

_logLine(vh, line)

loggt die übergebene Zeile mit dem Aufbau von _getNullLine()

_saveLine(storeLine)

speichert (inserted oder updated) die übergebenen Map-Werte in der Datenbank

_saveListLine(storeLine)

speichert (inserted oder updated) die übergebenen Werte der Zeile in der Datenbank
