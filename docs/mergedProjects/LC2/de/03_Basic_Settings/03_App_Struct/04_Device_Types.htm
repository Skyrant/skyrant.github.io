Funktion

Hierarchische Definition und Benennung von verschiedenen Typen von Verbindungen / Koppelpartnern. Die Gerätetypen werden bei der Parametrierung der Datenquellen (Verbindungen) verwendet. Jeder externen Verbindung, z.B. zu einem Gateway und dessen SPS-Verbindungen, muss ein eindeutiger Gerätetyp zugeordnet werden.

ID

Eindeutige Identifikationsnummer zur Darstellung der hierarchischen Ordnung. Wird automatisch vom System vergeben. Schreibgeschützt.

Name

Frei wählbarer Name des Gerätetyps.

Beschreibung (optional)

Zusätzliche Beschreibung des Gerätetyps.

 

Beispiel

Verbindung des Gateways zu einem SPS als eine physikalische Verbindung über das Netzwerk. Innerhalb der SPS-Struktur existieren mehrere Untergruppen, die unabhängige Anlagenbereiche steuern, und deren Werte und Störmeldungen unabhängig voneinander ausgewertet werden sollen. Die Gerätetypen könnten dann SPS, Anlagenbereich, Funktionsgruppe und Startbereich heißen.
