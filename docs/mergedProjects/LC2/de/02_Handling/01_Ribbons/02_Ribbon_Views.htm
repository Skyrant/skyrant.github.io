Ribbon Ansichten

Im Reiter "Ansichten" können alle verfügbaren Ansichten aktiviert oder deaktiviert werden. Aktivierte Ansichten sind dunkelgrau hinterlegt.

Die Ansichten können beliebig im Screen platziert werden. Wenn eine Ansicht aktiviert wird, wird sie am selben Ort hinzugefügt, wo sie das letzte Mal angehängt war.

Standardmäßig sind die folgenden Ansichten aktiviert: Navigationsbaum, Dokumente, Details, Beschreibung, Favoriten, Log.

 Blendet den Navigationsbaum ein oder aus. Der Navigationsbaum ist standardmäßig auf der linken Seite dargestellt. Beim Ein- bzw. Ausblenden werden die Einstellungen für den Navigationsbaum (z.B. "Ansicht autom. öffnen") beibehalten.

 Aktiviert bzw. deaktiviert die Anzeige von Dokumenten / Tabellen. Tabellen werden standardmäßig in der Mitte dargestellt.

 Aktiviert bzw. deaktiviert die Anzeige der Details zu den Tabellen. Die Details werden standardmäßig auf der rechten Seite dargestellt.

 Aktiviert bzw. deaktiviert die Beschreibung der aktuell selektierten Tabelle. Die Beschreibung wird standardmäßig auf der rechten Seite dargestellt (Reiter neben "Details").

 Aktiviert bzw. deaktiviert die Favoriten-Leiste. Die Favoriten werden standardmäßig in der Zeile zwischen dem Menü und den anderen Ansichten dargestellt.

 Blendet die Ansicht zum Auto-Vervollständigen ein oder aus. Diese Ansicht wird standardmäßig auf der rechten Seite dargestellt.

 Blendet den Verlauf ein oder aus. Der Verlauf wird standardmäßig am unteren Rand dargestellt.

 Nur im Ssytem Developer Mode verfügbar. Diese Ansicht zeigt die verfügbaren Plugins an.

 Nur im System Developer Mode verfügbar. Blendet eine Liste aller verfügbaren Screens ein.

 Blendet das Log ein oder aus. Das Log wird standardmäßig am unteren Rand dargestellt.

 Nur für Administratoren verfügbar.
