Favoriten bearbeiten

Aktive Tabellen können durch Klick auf den gelben Stern neben dem Tabellennamen als Favoriten gespeichert werden. Der Name für den Favoriten kann dabei frei vergeben werden. Durch einen Rechtsklick - Umbenennen können die Favoriten nachträglich umbenannt werden.

Die Favoriten werden in der Reihenfolge in der Favoritenleiste angezeigt, in der sie gespeichert wurden. Soll diese Reihenfolge nachträglich geändert werden, kann dies wie folgt geschehen:

Öffnen Sie die Tabelle GP_R_FAVORITES unter System-Administration - Repository - Alle Tabellen. Filtern Sie in der Spalte "Besitzer" auf Ihren Benutzernamen. Ändern Sie nun über die Spalte "Sortiernummer" die Reihenfolge.
