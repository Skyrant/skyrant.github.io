Bedienung im Navigationsbaum

Durch die Aufwärts- und Abwärts-Pfeiltasten kann im Navigationsbaum nach oben bzw. unten navigiert werden. Die Links- bzw. Rechts-Pfeiltasten klappen untergeordnete Einträge zu oder auf. Alternativ kann dazu der Pfeil vor dem jeweiligen Eintrag mit der Maus angeklickt werden.

Zum Öffnen von Elementen aus dem Navigationsbaum gibt es verschiedene Möglichkeiten. Das Element wird im aktuellen Fenster geöffnet durch:

linker Mausklick

Drücken des Mausrads

Enter

 

Zum Öffnen des Elements in einem neuen Tab können die selben Aktionen mit gedrückter STRG-Taste verwendet werden.
