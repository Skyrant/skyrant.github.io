Datentypen

Im Navigationsbaum werden verschiedene Datentypen angezeigt (z.B. hierarchische Tabellen, Tabellenzeilen, etc.). Diese sind durch unterschiedliche Symbole gekennzeichnet.

 Kennzeichnet einen Ordner. Kann nicht angezeigt werden.

 Kennzeichnet eine Tabelle. Kann in den Dokumenten dargestellt werden.

 Kennzeichnet eine flache Zeile / einen flachen Tabelleneintrag. Kann in den Details angezeigt werden.

 Kennzeichnet eine hierarchische Zeile / einen hierarchischen Tabelleneintrag. Kann in Dokumenten und Details angezeigt werden.

 Kennzeichnet eine referenzierte Tabelle. Kann in den Dokumenten dargestellt werden.

 Kennzeichnet eine Sondermaske für bestimmte Einstellungen (z.B. Export-Einstellungen oder Aggregierungs-Einstellungen). Kann in den Dokumenten angezeigt werden.
