Navigationsbaum

Der Navigationsbaum auf der linken Seite bietet alle im LC2 verfügbaren Tabellen in einer Baumstruktur. Über den Navigationsbaum erfolgt abhängig von der Berechtigungsstufe der Zugriff auf die Konfigurationsdaten der Legato-Datenbasis. Über die Einstellungen kann definiert werden, ob die Tabellen beim Anklicken in der Dokumenten-Ansicht angezeigt werden sollen oder ob der Aufruf über rechte Maus – Anzeigen erfolgen soll.

Um die Navigation zu erleichtern, bietet der Navigationsbaum Such- und Filteroptionen sowie Einstellungen, wie die Daten im Baum und in den Dokumenten dargestellt werden sollen.

Über rechte Maus auf einen Eintrag im Navigationsbaum kann darüber hinaus ein Kontextmenü mit weiteren Funktionen geöffnet werden.
