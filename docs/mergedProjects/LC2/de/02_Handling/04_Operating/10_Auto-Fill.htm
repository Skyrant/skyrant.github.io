Autofill

Der Autofill-View dient zum einfacheren Befüllen von Tabellen. Er besteht aus mehreren Teilen.

Reiter "Lookup"

Der Reiter "Lookup" dient zum Öffnen der Nachschlagetabelle für Lookup-Felder. Dadurch können einfach weitere Daten in die Tabelle übertragen werden.

Initial sind keine Daten geladen.

 

Der Inhalt des Autofill-Fensters wird aus der in "Dokumente" geöffneten Tabelle geladen. Klicken Sie dort in ein Lookup-Feld und wählen Sie im Kontextmenü "Lookup-Elemente im Autofill anzeigen". Nun wird die entsprechende Lookup-Tabelle geladen.

 

Nun können beliebig viele Einträge der Lookup-Tabelle selektiert werden. Mit dem Button "Ausgewählte Elemente an aktuelle Tabelle anhängen" werden in der aktiven Tabelle in der Dokumenten-Ansicht die entsprechende Anzahl Zeilen angelegt und die Lookup-Felder mit den selektierten Einträgen vorbelegt. Nun können die restlichen Daten befüllt werden.

Außerdem kann die Lookup-Tabelle direkt im Autofill-Fenster editiert werden, z.B. wenn die benötigten Daten dort noch nicht enthalten sind.

In einigen Fällen kann nicht direkt die Lookup-Tabelle geladen werden (z.B. bei hierarchischen Tabellen). In diesem Fall wird eine Spiegelung des Navigationsbaums im Autofill Fenster angezeigt, wo die Gruppe der möglichen Einträge vorselektiert ist.

 

Ein Klick auf "Details" rechts blendet die Details zum im Navigationsbaum selektierten Eintrag ein.

 

Die Unteransicht "Details" besteht aus zwei Teilen. Der Reiter "Details" zeigt die Detail-Ansicht des selektierten Eintrags an, sofern es sich hier um einen Datensatz handelt; der Reiter "Items" zeigt die Elemente der selektierten Tabelle an.

Wird die Checkbox "Wert setzen durch Klick" aktiviert, kann der Eintrag in der darunterliegenden Zeile per Mausklick in die bearbeitete Tabelle übernommen werden. Diese Zeile kann entweder mit einem Eintrag aus der Lookup-Tabelle gefüllt werden, oder es kann manuell etwas eingetragen werden.

 

In diesem Beispiel wird jede angeklickte Zelle der bearbeiteten Tabelle mit dem Eintrag "group" gefüllt (sofern dieser Werttyp dort zulässig ist).

Die Daten im Autofill View werden nicht automatisch aktualisiert. Beim Wechsel der Tabelle müssen die Lookup-Daten ggf. neu angefordert werden.

Reiter "Inkrementieren"

Zusätzlich gibt es die Funktion "Inkrementieren", die insbesondere für Zahlwerte und oder Texte mit Zahl-Elementen (z.B. Location 1, Location 2, Location 3, etc.) hilfreich ist.

 

In das Texteingabefeld "Format" kann ein beliebiger Text eingegeben werden. Der vorhandene Eintrag {0} ist ein Platzhalter für das Inkrementieren. Z.B. Location {0}.

Die Eingabe im Feld "Inkrementieren" legt fest, um wieviel der Platzhalter jeweils inkrementiert werden soll. Im obigen Beispiel ist der Inkrement-Wert 2; es wird also in das erste Feld, das durch Mausklick befüllt wird, "Location 1" eingetragen, in das zweite "Location 3", in das dritte "Location 5" usw.

Ein Klick auf "Zurücksetzen" setzt die Eingaben in diesem Dialog zurück.

Reiter "Aus LPR-Datei importieren"

Der Reiter "Aus LPR-Datei importieren" dient dem Import von Variablen aus einem Gateway-Projekt in die Variablenliste einer Verbindung des Datenaustauschs. Siehe auch Datenaustausch | Module | Verbindungen | Variablen.
