Formatvorlage für ID (optional)

Formatvorlage für den Identifier. Eine Formatvorlage kann aus verschiedenen Teilen bestehen, die durch "-" getrennt werden. Die folgenden Zeichen dürfen in der Syntax verwendet werden:

#... Zahlen rechts ausgerichtet; aufgefüllt mit "0"

~… Zahlen links ausgerichtet

0-9 links ausgerichtet; aufgefüllt mit "0"

@... beliebige Buchstaben links ausgerichtet

Beispiel: @@-### g  IT-0112

Prüf-Ausdruck (optional)

Regulärer Ausdruck um das ID Format zu überprüfen. Beispiel: ^.{0,5}$ bedeutet max. 5 Zeichen und keinen Zeilenumbruch. Siehe auch das Kapitel "Reguläre Ausdrücke".
