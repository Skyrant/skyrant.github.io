Funktion

Diese Tabelle kann nur im Developer Mode geändert werden. Unter dem jeweiligen Repository werden die darin enthaltenen Datenquellen aufgeführt.

Feldbeschreibung

 

Feld

Beschreibung

ID

Vom System vergebene, eindeutige ID. Schreibgeschützt.

Objektname

Name der Datenquelle in der Datenbank.

Verbindung (optional)

Verbindung, der diese Datenquelle zugeordnet ist.

Angezeigter Name (optional)

Name der Datenquelle, der im LC2 angezeigt wird.

Angezeigter Instanz-Name (optional)

Name, der verwendet wird, wenn über das Kontextmenü ein neuer Eintrag erstellt wird (... hinzufügen)
Tipp (optional) Enthält den Tooltip, der beim Mouseover über der Spalte angezeigt wird.
Hilfetext (optional) Enthält den Text, der bim Hilfefenster für die jeweilige Sprache angezeigt wird oder den Link zu einer Datei.
Schreibgeschützt Ist diese Checkbox aktiviert, so kann die Datenquelle nicht geändert werden, sondern nur angeschaut.
Angezeigtes Feld (optional) Enthält den Text, der links in der Navigation angezeigt wird.
Default-Navig-Element (optional) Default-Navigationselement.
Instanz-Navig-Element (optional) Navigationselement der Instanz.
Baum (optional) Checkbox. 1 = Tabelle ist hierarchisch und soll als Baum dargestellt werden; 0 = kein Baum.
Default-Sortierung (optional)

Ein oder mehrere DB Spaltennamen, nach denen im Navigationsbaum sortiert werden soll, z.B. NAME_0. Default ist ID.
Anzeige-Recht (optional)

Recht, das zur Anzeige benötigt wird.
Ändern-Recht (optional)

Recht, das zum Ändern benötigt wird.
Eigenschaften (optional)

Eigenschaften der Datenquelle.
Ansichts-Eigenschaften (optional)

Definiert die Quelltabelle und mögliche Filter für CVs (custom views)
Inaktives Löschen & Update (optional)

 
Veröffentlicht (optional)

Checkbox. 1 = veröffentlicht; 0 = nicht veröffentlicht

System-Updates (optional)

Checkbox. Sind für diese Spalte System-Updates erlaubt? 0 = nein, 1 = ja

Besitzer (optional)

Name des Besitzers.

Domäne (optional)

Name der übergeordneten Domäne.

Datenquellen-Typ (optional)

FK - Verweis auf einen Datenquellentyp, z.B. Config-Tabelle

Proxy (optional)

Proxy des Datensatzes, für den eine Berechtigung vergeben werden kann.
