Funktion

In dieser Tabelle werden alle durchgeführten Evaluierungen und Aktualisierungen aufgeführt. Die Tabelle dient nur der Information und ist schreibgeschützt.

Feldbeschreibung

 

Feld

Beschreibung

ID

Eindeutige Identifikationsnummer.
Vorlage Lookup-Liste zur Auswahl der Projektierungsvorlage, für die ein Befehl durchgeführt wurde.
Instanz Lookup-Liste zur Auswahl der Instanz, für die ein Befehl durchgeführt wurde.
Befehl Ausgeführte Funktion (update, delete, evaluate)
Vorlagen-Eigenschaften Eigenschaften der Vorlage zum Zeitpunkt der Ausführung.
Parameterwerte Parameterwerte zum Zeitpunkt der Ausführung.
Start

Zeitstempel, wann die Ausführung begonnen wurde.

Ende

Zeitstempel, wann die Ausführung abgeschlossen war.

Info

Kurze Information über die Ausführung (z.B. n Änderungen durchgeführt; 1 Fehler).

Änderungen

Details zu den durchgeführten Änderungen im JSON-Format.

  "action" : "delete",

  "tableName" : "p_values_config",

  "instanceKey" : "450"

}, {

  "action" : "delete",

  "tableName" : "l_nodes",
