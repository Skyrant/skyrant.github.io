Funktion

Direkt unter der jeweiligen Vorlagen-Instanz werden die Datensätze angezeigt, die durch eine Instanziierung entstanden sind. Die Tabelle dient nur der Information und ist schreibgeschützt.

Feldbeschreibung

 

Feld

Beschreibung

ID

Eindeutige Identifikationsnummer.
Vorlagen-Instanz Lookup-Liste zur Auswahl der Instanz, zu der diese Zeile angelegt wurde.
Aktions-Schlüssel Aktions-Schlüssel (üblicherweise im Template-Skript registriert), z.B. node1 für einen  Knoten, var1 für eine Variable.
Zieltabelle Name der Zieltabelle für diese Aktion (also die Tabelle, in die der Datensatz eingefügt wurde).
Datensatz-ID ID (üblicherweise numerisch) des über die Aktion adressierten Datensatzes.
Gedroppt?

Checkbox. 1 = das Vorlagen-Element wurde von der Vorlage entfernt und ist als gedroppt markiert. Das tatsächliche Löschen muss vom Anwender ausgeführt werden (z.B. über Referenzen löschen).

Ausführungsreihenfolge

Reihenfolge, in der die Aktionen ausgeführt werden (normalerweise die Reihenfolge, in der die Datensätze angelegt wurden).
