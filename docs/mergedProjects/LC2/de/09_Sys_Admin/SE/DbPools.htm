Funktion

Unter den einzelnen Task-Instanzen kann nun festgelegt werden, wieviele DbPools ihr zur Verfügung stehen sollen und ggf. individuelle Eigenschaften des DbPools festgelegt werden. Wird nichts eingetragen, werden die Standard-Eigenschaften des DbPools verwendet.
