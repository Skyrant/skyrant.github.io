CurrTargetPartsShift

Beschreibung

Für einen parametrierten Knoten wird der Kennwert Soll aktuell für die laufende Schicht auf Basis einer Soll-Stückzahl berechnet. Dazu muss in der Fertigungsstruktur der Knoten mit einem Zeitschema verknüpft sein. Die Soll-Stückzahl wird der Funktion als Parameter übergeben.

Mit Zeitschema und dem aktuellen Schicht-/Pausenmodell aus dem Kalender kann das Verhältnis vergangene Schichtzeit zu gesamter Schichtzeit (jeweils als Netto-Zeiten) berechnet werden. Im gleichen Verhältnis stehen Soll aktuell und Soll-Stückzahl gesamt zueinander:

Die Zeiten werden in [sec] berechnet.

Parameterliste

ParamNo 0 – RESULT

Typ: Variable, Datentyp numerisch

InOut: Out

Multi: 0

Beschreibung: Sollstückzahl aktuell für laufende Schicht

ParamNo 1 – PARTS

Typ: Variable oder Konstante, Datentyp Numerisch

InOut: In

Multi: 0

Beschreibung: Sollstückzahl für die aktuelle Schicht

ParamNo 2 – NODE

Typ: Knoten

InOut: In

Multi: 0

Beschreibung: Über den Knoten wird die Verbindung zum Schicht- Pausenmodell hergestellt.

ParamNo 3 – BREAK_TYPE_LIST

Typ: Variable oder Konstante, Datentyp String

InOut: In

Multi: 0

Beschreibung: Liste der Pausentypen, die bei der Berechnung berücksichtigt werden sollen. Die einzelnen Pausentypen müssen voneinander anhand von Semikolons getrennt werden, z.B. "1;2".
