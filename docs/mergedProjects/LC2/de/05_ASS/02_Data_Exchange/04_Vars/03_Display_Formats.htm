Darstellungsformate

Zu jeder numerischen Variable kann im Feld Darstellungsformat (Display format) eine Formatierungsanweisung angegeben werden, um beispielsweise die Anzahl der Nachkommastellen eines Wertes in der Client-Aktuellansicht zu bestimmen.

:

Geben Sie bei numerischen Variablen immer ein Format vor, damit auch andere Anwendungen, die mit den Variablen arbeiten (z.B. der GefasoftDesigner) korrekt funktionieren.

Die Formatanweisung kann sich aus verschiedenen Bestandteilen zusammensetzen: <Zahlenbasis><Optionen>

An erster Stelle steht die Zahlenbasis, keine Angabe führt zu dezimaler Darstellung:

d

Dezimal

I

Integer

h

Hexadezimal

b

Binär

o

Oktal

t

Dauer (hh:mm:ss)

                

Sind keine Optionen angegeben erfolgt eine Standarddarstellung aufgrund der gewählten Zahlenbasis. Es werden dabei so viele Stellen (auch Nachkommastellen!) wie notwendig dargestellt.

Die zusätzlichen Optionen hängen teilweise von der gewählten Zahlenbasis ab:

Platzhalter für Ziffernstellen, die generell angezeigt werden, wenn nötig mit führenden Nullen oder Nachkommastellen mit angegebener Anzahl Nullen aufgefüllt.

Beispiele:

keine Formatangabe

Wert 128,345

Anzeige 128.345

Format d 

Wert 128,345

Anzeige 128.345

Format d0000

Wert 128,345

Anzeige 0128

Format h 

Wert 128,345

Anzeige 0x0080

Format h0000

Wert 128,345

Anzeige 0x0080

 

Weitere Optionen für dezimale Werte:

#

Platzhalter für Ziffern, nur notwendige Stellen werden angezeigt

.

landesspezifisches Trennzeichen zwischen Vor- und Nachkommastellen verwenden

,

landesspezifisches Tausendertrennzeichen

E

Exponentialdarstellung

%

Darstellung als Prozentzahl (der Wert wird zuvor mit 100 multipliziert). Das %-Zeichen wird dort angezeigt, wo es eingefügt wird

 

Beispiele:

Format d 

Wert 128,345

Anzeige 128.345

 

Wert 8,345

Anzeige 8.345

 

Wert 0,345

Anzeige .345

Format d#0.##

Wert 128,345

Anzeige 128.34

 

Wert 128

Anzeige 128

 

(mindestens eine Stelle vor dem Komma, nach dem Komma max. zwei Stellen)

Format d%#0.##

Wert 0,7

Anzeige %70

Format d#0.##%

Wert 0,7

Anzeige 70%

Format d#,###,##0.0

Wert 1289,345

Anzeige 1,289.3

 

Wert 128

Anzeige 128

 

 (bei hohen Zahlen Tausendertrennzeichen einblenden)

Format d0.00E00

Wert 1289,345 

Anzeige 1.29E03
