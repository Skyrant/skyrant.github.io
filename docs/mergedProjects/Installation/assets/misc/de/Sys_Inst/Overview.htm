Über dieses Dokument

Dieses Dokument beschreibt die Installation eines Legato-Basissystems in den am häufigsten anzutreffenden Varianten.

Im Dokument werden folgende Symbole und Formatierungen verwendet:

:

Stellt einen Sicherheitshinweis dar. Die folgenden Informationen warnen vor möglichem Sachschaden, z.B. Datenverlust oder mangelnder Funktionalität bei unsachgemäßer Handhabung. Beachten Sie diese Hinweise unbedingt!

INFO:

Hier finden Sie wichtige Informationen über die Funktionsweise des Systems. Diese Informationen können Sie dabei unterstützen, die Konfiguration korrekt vorzunehmen.

Übersicht über die Komponenten

Die Installationen der einzelnen Komponenten werden teilweise über Setup-Programme und teilweise manuell durchgeführt.

Für ein komplettes Legato System müssen folgende Hauptkomponenten eingerichtet werden:

Legato Datenbankgateway (bis Version 4.7.2.4)

Gateway mit Send / Receive und OPC-Clientschnittstelle

GEFASOFT OPC-Server GOPS (bei Bedarf)

Treiber zur Anbindung der SPS-Ebene (in Verbindung mit dem OPC-Server)

Sapient Cloud Connector (als Nachfolge-Tool des Datenbank-Gateways)

Gateway mit Send / Receive und OPC_UA-Clientschnittstelle

SQL-Datenbank

SQL-Datenbank (Oracle)

Datenmodell mit Tabellen und Prozeduren

Application Engine-Instanz(en) für verschiedene ergänzende, zyklische Datenbankjobs

Legato Configuration Center (LC2) auf einer oder mehreren Workstations mit Datenbankverbindung

Webapplikation

Web-Application-Server (z.B. Tomcat)

Servlet-basierende Webanwendung

Inhalt des Produkt-Verzeichnisses

Folgende Basisdaten für die Installation sind im Auslieferungsverzeichnis vorhanden:

SQL-Skripte für die Installation der Legato-Datenbank

WAR-File für die Legato-Webapplikation

LeAppEng (Legato ApplicationEngine) mit allen benötigten Komponenten (es existiert kein automatisches Setup)

Treiber zur Anbindung der SPS-Ebene (in Verbindung mit dem OPC-Server)

 

Folgende Setup-Programme und Installer sind im Auslieferungsverzeichnis vorhanden:

Legato Datenbank-Installer (LegatoDB) für die Ausführung der Datenbank-Installationsskripte

Konfigurationstool für die Webapplikation

GatewaySetup für das Legato-Datenbankgateway

setupscc.exe für den Sapient Cloud Connector

GopsSetup für den Gefasoft OPC-Server (nur in Verbindung mit dem Datenbankgateway)

LC2Setup für das Legato Configuration Center (LC2)
