Abschließende Einstellungen

Die soeben erstellte Datenbank-Instanz LEGATO ist in der Datei /etc/oratab eingetragen worden. Der User oracle kann diese Datei zum Editieren öffnen:

$ vi /etc/oratab

 

Wie an dem abschließenden N (für "no") in der Zeile, die mit "LEGATO" beginnt (hier: die vorletzte Zeile), erkennbar, wird die Instanz per Default nicht zusammen mit dem Datenbanksystem gestartet. Dieses Verhalten kann geändert werden, indem das N durch ein Y (für "yes") ersetzt wird.

Nach dieser letzten Änderung sind Datenbank und Instanz voll funktionsfähig. Allerdings müssten bei einem Neustart des Servers sowohl das Datenbank-System als auch der Listener "von Hand" neu gestartet werden. Nicht zwingend notwendig, aber äußerst sinnvoll ist deshalb die Erstellung einer Autostart-Funktionalität für Datenbank. Diese wird im folgenden Abschnitt beschrieben.
