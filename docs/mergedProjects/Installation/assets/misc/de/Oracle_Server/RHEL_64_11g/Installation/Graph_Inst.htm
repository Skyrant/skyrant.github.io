Installation mit dem grafischen Installer

Dann wird (vorübergehend) der X Window-Zugang für alle Hosts geöffnet:

# xhost +

 

Für die eigentliche Installation wird zum User oracle gewechselt. Dieser wurde durch das Pre-Install Script angelegt. Der User oracle setzt die Variable DISPLAY und startet runInstaller:

# su oracle

$ export DISPLAY=:0.0

$ cd /tmp/OracleInstaller/database/

$ ./runInstaller

 

Die Installation startet mit der Anzeige des Oracle-Installer-Startdialogs und dort mit der Nachfrage nach der Installation von Sicherheitsupdates:

Die Checkbox vor "Ich möchte Sicherheitsupdates über My Oracle Support erhalten" wird weggeklickt und auch die Nachfrage mit "Ja" bestätigt:

Auch in der nachfolgenden Dialogmaske wird das Herunterladen von Softwareupdates an dieser Stelle der Installation abgelehnt:

Damit entfällt auch der Installationspunkt "Softwareupdates einspielen". Als Installationsoption wird "Nur Datenbanksoftware installieren" ausgewählt:

Bei den "Grid-Installationsoptionen" wird der Unterpunkt "Datenbankinstallation mit nur einer Instanz" gewählt:

Auf der nächsten Maske werden direkt vor der eigentlichen Installation die Voraussetzungen überprüft:

Die als "fehlend" angezeigten Packages sind (als Ergebnis der vorhergehenden Installationen) sehr wohl auf dem Rechner vorhanden; die Warnungen sind das Ergebnis eines Bugs im Installer-Programm (Oracle-Versionsstand 11.2.0.3). Durch Anklicken des Kästchens "Alles ignorieren" kann die Installation fortgesetzt werden. Die folgende Sicherheitsabfrage ist mit "Ja" zu bestätigen:

Es folgt der "Überblick" über die auszuführenden Aktionen:

Ein Klick auf "Installieren" startet die Installation.

Kurz vor dem Abschluss der Installation erscheint ein Pop-up Dialog mit der Aufforderung, zwei Skripte (in der angegebenen Reihenfolge) mit root-Rechten auszuführen:

Folgen Sie dazu der Anleitung im Dialog.

Beim zweiten Skript wird ein Pfad abgefragt; überprüfen Sie diesen und bestätigen oder ändern Sie die Vorauswahl.

Sobald die Skripte durchgelaufen sind, klickt man auf "OK", und das Ende der Installation wird angezeigt:

Ein Klick auf "Schließen" beendet den Installations-Dialog.
