Installation der LEGATO Tabellen

Nachdem die Datenbank-Instanz soweit eingerichtet und getestet ist, kann nun der eigentliche Inhalt der LEGATO-Datenbank installiert werden. Dazu dient eine Windows-Applikation, die auf allen gängigen Windows-Betriebssystemen mit Zugang zur Datenbank-Instanz LEGATO gestartet werden kann. Diese Applikation ist im Standard-Lieferumfang enthalten oder kann über den Projektleiter oder den Vertrieb bezogen werden.

Weitere Infos hierzu finden Sie in der LEGATO Systeminstallationsdokumentation.
