Eintragen der neuen Instanz in den Listener

Damit der Listener ordnungsgemäß arbeitet, sollte man diesem nun die neue Instanz bekanntgeben. Es gibt zwar auch die Möglichkeit, dass sich die Instanz automatisch beim Listener anmeldet, aber es ist trotzdem sicherer die Instanz in der Listener-Konfiguration manuell einzutragen.

# su oracle

$ cd /u01/app/oracle/product/12.1.0/dbhome_1/bin

$ ./netmgr

 

Es öffnet sich der Oracle Net Manager.

Klicken Sie auf "Listener".

Wählen Sie in der Drop-Down-Liste "Datenbank-Services" aus und klicken Sie auf "Datenbank hinzufügen".

Geben Sie beim "Globalen Datenbanknamen" und bei der SID den vorher gewählten Datenbanknamen ein; im Beispiel "LEGATO". Zur Speicherung muss man unter dem Menüpunkt "Datei" die Netzwerkkonfiguration sichern. Das Oracle-Standardverzeichnis ist ORACLE_HOME (unter Linux üblicherweise "/u01/app/oracle/product/12.1.0/dbhome_1").

Nun kann der Net Manager geschlossen werden.
