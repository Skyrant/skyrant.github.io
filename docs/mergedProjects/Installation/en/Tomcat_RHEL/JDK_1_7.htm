Installation of JDK 1.7

Create a New Directory /usr/java

Open the Linux Terminal and login as root using the command "su -", if you are not already logged-in as root and then create a new directory called "java" under the directory "usr".

[gefasoft@localhost ~]$ su -

[root@localhost ~]# mkdir /usr/java

  

Download and Unpack JDK

Change to the /usr/java directory we created.

[root@localhost ~]#  cd /usr/java

[root@localhost java ]#

 

Download the appropriate JDK and save it to /usr/java directory we created above.

Unpack jdk-7u5-linux-x64.tar.gz in the /usr/java directory using tar -xzf:

[root@localhost java]# tar -xzf jdk-7u45-linux-x64.tar.gz

 

This will create the directory /usr/java/jdk1.7.0_45.
