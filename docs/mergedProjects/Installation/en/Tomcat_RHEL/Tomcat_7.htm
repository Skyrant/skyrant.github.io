Download and Unpack Tomcat 7

Download Tomcat 7

We will install Tomcat 7 under /usr/share.

Switch to the /usr/share directory:

[root@localhost ~]# cd /usr/share  

[root@localhost share ]#

 

Browse http://tomcat.apache.org/ in your internet browser. Click on Tomcat 7.0 from the left sidebar to locate the download version of Apache Tomcat 7.0.

 Download the latest release of Apache Tomcat that is found reliable and stable. In this tutorial we will be using the Tomcat version 7.0.42. Download apache-tomcat-7.0.42.tar.gz from the link above and save it to /usr/share.

Verify MD5 Checksum

Once downloaded, you should verify the MD5 Checksum for your Tomcat download using the md5sum command.

[root@localhost share ]# md5sum apache-tomcat-7.0.42.tar.gz 

1dd9abf3ae5154677e55a5d4e1fae5ed *apache-tomcat-7.0.42.tar.gz

 

Compare the output above to the MD5 Checksum provided next to the download link and you used above and check that it matches.

Unpack tar file

[root@localhost share ]# tar -xzf apache-tomcat-7.0.42.tar.gz

 

This will create the directory /usr/share/apache-tomcat-7.0.42
