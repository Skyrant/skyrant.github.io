Overview

Introduction

CentOS is an open source OS delivered by worldwide community of developers. It’s completely upstream release of RHEL & it’s free to use and share. This tutorial will cover step by step guide to install CentOS 6.5 and Java OpenJDK 1.7 from scratch on a new 64 bit machine. The install type is minimal which is perfect for servers, no GUI will be installed and the installation will be as lean as possible.

Prerequisites

To begin with visit http://wiki.centos.org/Download and download the install ISO image(s) for your servers architecture (i686, x86_64 etc). In this tutorial we are going to use the 64 bit (x86_64) version for the installation.

:

If you are running a 32-bit version machine, you MUST use a 32-bit distribution of CentOS-6.
