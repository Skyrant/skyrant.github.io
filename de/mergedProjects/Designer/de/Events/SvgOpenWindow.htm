SvgOpenWindow

Mithilfe dieser Funktion kann bei einem Event eine angegebene URL in einem eigenen Browser-Fenster geöffnet werden. Beim jeweiligen Event wird beispielsweise Folgendes eingetragen:

SvgOpenWindow('Intranet', 'http ://intranet.firma.com/index.html', 'dependent=yes,resizable=yes')

Alternativ kann der Code auch in die SVG-Source eingetragen werden:

<a xlink:href="SvgOpenWindow('Intranet', 'http ://intranet.firma.com/index.html', 'dependent=yes,resizable=yes')"> ... </a>

Die Parameter haben folgende Bedeutung:

Parameter

Beschreibung

name

Der Name des angesprochenen Fensters. Die Namen "_blank", "_parent", "_self" und "_top" haben spezielle Bedeutungen, eigene Namen bieten die Möglichkeit, bestehende Fenster wiederzuverwenden statt bei jedem Klick ein neues Fenster zu öffnen.

url

Die Adresse der Seite, die geladen werden soll.

opts

Optionale Angaben zum Aussehen des Fensters (Rahmen, Position, Größe, u.v.m.). Kann weggelassen werden.
