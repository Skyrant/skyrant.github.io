SvgShowDialog

Öffnet einen Dojo-Dialog über dem SVG-Bild und zeigt darin den per URL referenzierten oder im Aufruf direkt angegebenen HTML-Inhalt an. Es kann immer nur ein Dialog angezeigt werden. Wird diese Funktion aufgerufen, während bereits ein Dialog sichtbar ist, wird dieser durch den neuen Dialog ersetzt.

Die Angaben zu Breite und Höhe sind optional. Fehlen sie, wird der Dialog so groß dargestellt, wie sein Inhalt es erfordert. Ist die Größe des Inhalts unbekannt, sollten beide Parameter angegeben werden, damit der Dialog nicht zu groß und somit unbedienbar wird. Dann gibt es gegebenenfalls im Dialog Bildlaufleisten für den dargestellten Inhalt (zwar nicht unbedingt schön, aber immerhin bedienbar). Größenangaben in Prozent beziehen sich auf die Größe der Visu-Ansicht (Browserfenster minus Baum links und Menü oben).

Externe Seiten können hier nur aufgerufen werden, wenn die Systemeinstellungen (z.B. X-Frame Options) dies erlauben!

Beim jeweiligen Event wird beispielsweise Folgendes eingetragen:

SvgShowDialog("URL","Title","iframe","width","height")

Alternativ kann der Code auch in die SVG Source eingetragen werden. Hierbei kann beispielsweise auch ein Array für den Titel eingegeben werden, um je Systemsprache einen anderen Text anzuzeigen:

<a><?gefasoft

<attrrepl attr = "xlink:href">

        <jsfncall name = "SvgShowDialog">

                <jsexpr quotes = "1">/help/de/CntHelp.htm</jsexpr>

                <jsnewarr nelems = "2">

                        <jsexpr quotes = "1">Titel</jsexpr>

                        <jsexpr quotes = "1">Title</jsexpr>

                </jsnewarr>

                <jsexpr quotes = "0">true</jsexpr>

                <jsexpr quotes = "0">320</jsexpr>

                <jsexpr quotes = "0">240</jsexpr>

        </jsfncall>

</attrrepl>?>

Die Parameter haben folgende Bedeutung:

Parameter

Beschreibung

url

String. Gibt an, was angezeigt werden soll.

1. Falls sie mit http:!!!!!!!// oder https:!!!!!!!// beginnt, wird sie als absolute URL (z.B. für externe Inhalte) angesehen. Beachten Sie hier die Sicherheitseinstellungen Ihres Systems. Ggf. verwenden Sie für externe Seiten die Funktion "SvgOpenWindow".

2. Falls sie mit einem Schrägstrich (/) beginnt wird sie als relative URL bezüglich der Legato-Web-Applikation ausgewertet.

3. Falls sie mit einem oder zwei Punkten und einem Schrägstrich beginnt (./ oder ../), wird sie als relative URL bezogen auf die Visu-Ansicht ausgewertet.

4. Falls sie mit einem Kleiner-Zeichen (<) beginnt, wird angenommen, dass es keine URL, sondern der fertige HTML-Code ist, der im Dialog angezeigt werden soll.

title

String. Titel des Dialogs. In zweisprachigen Installationen sollte statt eines Titels besser ein Array angegeben werden, das den Titel in beiden Sprachen enthält.

iframe

Boolean. Ob der im Dialog anzuzeigende Inhalt eine komplette HTML-Seite ist (true), oder nur ein HTML-Fragment (false). Ein Fragment hat weder ein <html> noch ein <body> Tag.

width

Any, optional. Gewünschte Breite des Dialogs. Ist dies nur eine Zahl, wird automatisch px als Maßeinheit angenommen. Es kann auch die gewünschte Einheit mit übergeben werden.

height

Any, optional. Gewünschte Höhe des Dialogs. Ist dies nur eine Zahl, wird automatisch px als Maßeinheit angenommen. Es kann auch die gewünschte Einheit mit übergeben werden.
