DOM

Grafische Baumdarstellung des DOM (Document Object Models) der SVG Datei.

Die Elemente werden entsprechend ihrer Reihenfolge im Dokument und eingerückt nach Hierarchieebene dargestellt. Jedes Element wird dabei durch ein Symbol, der ID und den Tag-Namen dargestellt. Gruppen, Layers oder Links können durch das Plus bzw. Minus auf- und zugeklappt werden.

Der DOM erlaubt es, durch Drag&Drop die Elemente zu verschieben. Dabei wird die Restriktion bezüglich Layers nicht beachtet. Zu beachten ist hierbei, dass entstehende leere Gruppen nach Rückfrage gelöscht werden.

Wird ein Element im DOM selektiert, wird es auch in der GUI markiert. Auf diese Weise können auch Elemente in Gruppen einzelnen verschoben werden.

Durch STRG können mehrere einzelne Elemente bzw. durch SHIFT ein Bereich im DOM selektiert werden.

Die Interaktivität des DOMs steht nur im Designer zur Verfügung. Code-Ansichten unterstützen dies nicht.
