Bibliothek(en) importieren

Zum Importieren von Bibliotheken gibt es zwei Möglichkeiten (über die Checkbox "Open all libraries in path" wählbar): es kann genau eine Bibliothek importiert werden (Checkbox nicht aktiviert) oder alle Bibliotheken aus einem Ordner (Checkbox aktiviert). Ist die Checkbox aktiviert, werden alle im Pfad befindlichen Bibliotheksdateien geöffnet und im Symbolbrowser als Ordner angezeigt.

Bibliotheks-Browser und Symbolliste

Die geladene Bibliothek wird als Verzeichnisstruktur dargestellt. Durch Klick auf den Button "Show Symbol List" kann zusätzlich eine Symbolliste angezeigt werden. In diesem Fall wird die Ansicht zweigeteilt (ähnlich einem File-Browser). Der Baum auf der linken Seite zeigt dann nur noch die Ordnerstruktur an. Abhängig vom ausgewählten Ordner werden die Unterelemente in der Symbolliste eingeblendet. Die Symbole sind alphabetisch nach Symbol-Dateinamen sortiert.

Ein Doppelklick auf einen Ordner im Baum öffnet seinen Inhalt in der Symbolliste. Ein Doppelklick auf einen Ordner in der Symbolliste wechselt zum entsprechenden Ordner im Baum und zeigt in der Symbolliste dessen Inhalt an.

Kontextmenü

Über die rechte Maustaste wahlweise auf einen Ordner oder ein Symbol wird ein Kontext-Menü geöffnet, das die Verwaltung der Bibliothek ermöglicht.

Beschreibung

Insert Selection as new Symbol

Fügt die derzeitige Selektion im Designer als neues Symbol in die Bibliothek ein. Anschließend kann ein Name für das Symbol gewählt werden.

Mehrere Elemente werden automatisch als Gruppe in die Bibliothek eingetragen.

Preview

Öffnet eine Vorschau des selektierten Symbols. Nur für Symboldateien verfügbar, nicht für Ordner. Wurde für ein Symbol ein Bild hinterlegt, wird dieses angezeigt, ansonsten wird versucht, das Symbol-SVG in der Vorschau anzuzeigen. Dabei werden keine Parameterwerte ersetzt, dementsprechend kann es abhängig vom Aufbau des Symbols zu Darstellungsfehlern kommen. Im Reiter "Notes" werden, sofern vorhanden, die weiteren Angaben zum selektierten Symbol angezeigt.

New Folder

Erzeugt ein neues Verzeichnis.

Rename

Erlaubt es, ein Element oder einen Ordner umzubenennen.

Delete

Löscht ein Element oder einen Ordner nach Rückfrage.

Edit Symbol

Öffnet ein Element zum Bearbeiten. Dabei wird das aktuelle Bild geschlossen (Abfrage zum Speichern). Das Symbol wird dann als neues Bild geladen. Der Button Speichern speichert die Änderungen im Bibliothekssymbol. Die Bibliothek als Ganzes muss anschließend noch mal gespeichert werden.

Edit Symbol Infos...

Öffnet einen Dialog zum Editieren der Symbol-Informationen. Im Reiter "Preview" kann ein Bild für das Symbol eingefügt werden, das dann in der Vorschau angezeigt wird. Das Bild kann über einen File-Browser hochgeladen oder aus der Zwischenablage eingefügt werden. Wird die Checkbox "Downscale to Client Area" aktiviert, wird die Darstellung des Bildes auf die Größe des Dialog-Fensters angepasst.

Im Reiter "Notes" kann ein Kommentar für das Symbol eingegeben werden, der dann als Tooltip über dem Symbol in der Bibliothek angezeigt wird. Nur für selbst erstellte Symbole verfügbar.

Save Library

Speichert die Änderungen an der Bibliothek.

Open Library

Lädt eine Bibliothek oder alle Bibliothekselemente aus einem Pfad.

Extract Symbol Files from Library

Extrahiert alle Symbole der selektierten Bibliothek als einzelne Symbol-Dateien (.sym) in einen neuen Ordner.

Arbeiten mit Bibliothekssymbolen

Elemente können mit Drag&Drop innerhalb der Bibliothek verschoben werden.

Mit Drag&Drop können Symbole aus der Baumstruktur der Bibliothek in die Zeichenebene geholt werden. Diese werden mit ihrer linken oberen Ecke an der Position des Mauszeigers eingefügt.

Via rechte Maus – Edit Symbol ist es möglich, Symbole aus der Baumstruktur der Bibliothek in die Zeichenebene zu holen, um sie zu editieren. Über "Save" werden die Änderungen am Symbol in der Bibliothek gespeichert. Die Bibliothek muss separat gespeichert werden. Über das Menü "File – Save as …" kann das editierte Symbol auch als ein neues Symbol gespeichert werden. Als Speicherort ist der Ort, an dem sich das ursprüngliche Symbol befunden hat, als Default hinterlegt.

Erstellen eigener Symbole

SVG-Dateien können als eigene Symbole gespeichert werden. Dazu markiert man das gewünschte Element in der Zeichenfläche. Im Zielordner der Bibliothek kann man dieses Element dann via rechte Maus – Insert Selection as New Symbol einfügen. Für diese Dateien wird automatisch im Bibliotheksordner eine Datei mit der Endung .SYM erstellt.

Außerdem kann man Symbole, die sich nicht innerhalb eines XML-Ordners befinden, im Editier-Modus öffnen und als neue Symbole speichern.

Diese eigenen Symbole können zusätzlich mit weiteren Informationen versehen werden. Den entsprechenden Dialog erreicht man durch einen Rechtsklick auf das Symbol – "Edit Symbol Infos …" Im Reiter "Preview" kann ein Bild für das Symbol eingefügt werden, das dann anstelle des Symbols in der Vorschau angezeigt wird. Das Bild kann über einen File-Browser hochgeladen oder aus der Zwischenablage eingefügt werden. Die folgenden Formate sind zulässig: *.JPG, *.JPEG, *.BMP, *.ICO, *.EMF, *.WMF. Wird die Checkbox "Downscale to Client Area" aktiviert, wird die Darstellung des Bildes auf die Größe des Dialog-Fensters angepasst. Dies kann insbesondere bei großen Vorschau-Bildern hilfreich sein.

Im Reiter "Notes" kann ein Kommentar für das Symbol eingegeben werden, der dann als Tooltip über dem Symbol in der Bibliothek angezeigt wird. Dazu muss die Bibliothek neu geladen werden (Refresh-Button).

Bibliothekssymbole mit Parametern

Bibliothekssymbole können mit Parametern versehen werden, die bei der Instanziierung abgefragt werden. Dazu wird das gewünschte Symbol mit Rechtsklick zum Bearbeiten geöffnet. Im Quelltext des Symbols kann dann der Textteil um einen Parameter erweitert werden. Diese Parameter sind für jedes Textfeld verfügbar. Interessant sind sie in dieser Hinsicht für Dynamiken, genauer gesagt für die Variablen.

Es gibt zwei Versionen der Parameter:

#!<parameter>#

Dies erzeugt einen Parameter, der als Freitext abgefragt wird.

#!v!<parameter>#

Dies erzeugt einen Parameter, der über den Variablendialog abgefragt wird.

Die einzelnen Parameter sind untereinander bzw. mit anderen Freitexten kombinierbar. #!test#anhang beispielsweise fragt im Dialog eine Angabe für "test" ab und hängt anschließend "anhang" an den eingegebenen Text an.

Wird ein Symbol mit Parameter in den Canvas eingefügt, so öffnet sich ein Dialog, der die Parameterwerte abfragt. Ist unter "Options – Settings" die Checkbox "Suggest Last Symbol Parameter Values" aktiviert, sind die Parameter bereits mit dem zuletzt verwendeten Wert vorbelegt. Diese Werte können geändert werden.
