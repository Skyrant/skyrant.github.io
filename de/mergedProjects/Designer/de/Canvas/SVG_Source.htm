SVG Source

Dieser Text-Editor bietet eine Möglichkeit, den SVG-Quelltext direkt zu ändern. Es können Änderungen an bereits bestehenden Elementen vorgenommen bzw. Elemente gelöscht oder neu hinzugefügt werden.

Änderungen in diesem Fenster werden bei einem Wechsel in einen anderen Bereich übernommen. Bevor die Ansicht verlassen werden kann, wird eine automatische Überprüfung des Quellcodes veranlasst. Ist dieser gültig, werden die Änderungen übernommen. Ist der Quellcode fehlerhaft, erscheint eine Fehlermeldung mit Angabe der fehlerhaften Zeile. Eine genauere Überprüfung kann auch mit F8 aufgerufen werden (siehe Seite 15).

Bei Wechsel in den Source wird das gerade selektierte Element automatisch in der textuellen Darstellung markiert. Darüber hinaus bietet das DOM-Fenster die Möglichkeit, durch Selektion ebenfalls die entsprechende Zeile im Source zu markieren.

Änderungen wie Löschen oder Hinzufügen von Elementen bzw. Änderungen der Struktur oder der IDs werden im DOM nicht automatisch übernommen. Dieser muss über den angebotenen Button" refresh DOM" (F5) aktualisiert werden, um die Änderungen darzustellen.

:

Ein Undo nach einem Wechsel von Source zu Designer-Tab macht alle getroffenen Änderungen im Sourcecode auf einmal rückgängig.

Neben den üblichen Tastaturabkürzungen gibt es einige Abweichungen / Zusätze:
