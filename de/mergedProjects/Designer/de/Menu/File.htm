File

Preview SVG

Öffnet die Preview. Je nach Einstellung in den Preferences wird entweder die GefasoftDesigner interne Preview geöffnet oder die externe. Auch für den Designer nicht darstellbare Elemente werden dargestellt und die Funktionalität der JavaSkripten kann getestet werden. Einzig die Dynamisierungen für Legato sind nicht verfügbar.

Update Preview

Eine bereits mittels "Preview SVG" geöffnete Vorschaudatei wird aktualisiert, ohne ein neues Browserfenster zu öffnen. Nach Wechsel in den Browser und F5 wird die aktualisierte Version angezeigt.

New

Öffnet eine neue Datei. Die derzeitige Grafik wird geschlossen, weil nur an einer Datei gleichzeitig gearbeitet werden kann. Wurden Änderungen noch nicht gespeichert, erscheint ein entsprechender Dialog mit der Option zum Speichern.

Load default.svg

Erzeugt eine neue leere Datei aus der default.svg.

Open…

Öffnet den Windows-Dialog zum Öffnen von SVG-Dateien.

Reopen

Listet die zuletzt geöffneten Dateien auf, die durch Auswahl wieder geöffnet werden.

Save

Speichert die Grafik unter dem angegebenen Dateinamen (Titelleiste).

Save as…

Öffnet den Windows-Dialog zum Speichern einer Datei. Es kann nur im Format *.svg gespeichert werden.

Open Library…

Öffnet den Dialog zum Öffnen einer Bibliotheksdatei. Um genau eine Bibliotheksdatei zu importieren, darf die Checkbox "Open all libraries in path" nicht aktiviert sein. Über die drei Punkte neben dem Pfad-Textfeld öffnet sich ein Dialog zur Auswahl der gewünschten Bibliotheksdatei.
Um mehrere Bibliotheksdateien zu importieren, muss die Checkbox "Open all libraries in path" aktiviert werden. Durch Klick auf die drei Punkte neben dem Pfad-Textfeld öffnet sich ein Dialog zur Auswahl des gewünschten Ordners. Es werden dann alle XML-Dateien importiert, die sich in diesem Ordner befinden. Eine Einzelselektion ist in diesem Fall nicht möglich.

Exit

Verlässt das Programm. Falls Änderungen an einer Grafik noch nicht gespeichert wurden, wird ein Bestätigungsdialog geöffnet.
