Draw

Select

Ruft das Selektionstool auf. Hiermit können Elemente selektiert, verschoben oder skaliert werden. Auch können bei Polylinien und Polygonen die einzelnen Punkte individuell verschoben werden.

Mehrere Elemente können durch Drücken von SHIFT während der Selektion zusammengefasst werden.

Durch Halten der linken Maustaste und gleichzeitiges Ziehen kann ein Selektionsrahmen erstellt werden. Alle darin komplett enthaltenen Elemente werden selektiert.

Rotate

Ruft das Rotationswerkzeug auf. Die Umrandungsrechtecke werden durch Rauten ausgetauscht. Damit lässt sich das selektierte Element drehen.

Line

Zeichnet eine Linie bestehend aus Start- und Endpunkt.

Über die Eigenschaft MarkerEnd oder MarkerStart kann ein Punkt oder Pfeil-Element ausgewählt werden, das als End- oder Startpunkt der Linie angezeigt wird. Dieser Marker wird in der Zeichenoberfläche nicht angezeigt, sondern nur in der Vorschau und natürlich im Browser.

Polyline

Werkzeug zum Zeichnen einer Linie, die aus mehreren Punkten besteht. Der erste Klick bestimmt den Startpunkt. Jeder weitere Punkt wird durch einen Klick auf die entsprechende Position angefügt. Ein zweiter Klick auf den zuletzt angefügten Punkt beendet das Erzeugen.

Weitere Punkte bzw. das Löschen von Punkten können nur manuell bei den Properties oder im Quelltext erzeugt werden.

Über die Eigenschaft MarkerEnd oder MarkerStart kann ein Punkt oder Pfeil-Element ausgewählt werden, das als End- oder Startpunkt der Linie angezeigt wird. Dieser Marker wird in der Zeichenoberfläche nicht angezeigt, sondern nur in der Vorschau und natürlich im Browser.

Polygon

Werkzeug zum Zeichnen eines N-Ecks, das aus beliebig vielen Ecken besteht. Die Vorgehensweise entspricht dem bei der Polyline. Ein Polygon wird automatisch vom letzten Punkt zum ersten Punkt geschlossen, wenn der erste Punkt erneut angeklickt wird.

Rectangle

Zeichnet ein Rechteck. Je nach Einstellung der Zeichenoptionen unter Preferences bestimmt bei "Double-Click" Drawing der erste Klick die linke obere Ecke, der zweite Klick die letztendliche Größe des Rechtecks. Bei Click&Drag bestimmt der erste Klick eine beliebige Ecke. Durch Gedrückthalten der Maustaste und Ziehen wird das Rechteck gebildet. Loslassen der Maustaste setzt den Endpunkt des Rechtecks.

Ellipse

Zeichnet eine Ellipse.

Text

Fügt einen Text an der angegebenen Position ein. Der anschließende Dialog ermöglicht die Eingabe des auszugebenden Textes. Mit OK werden die Eingaben umgesetzt, Cancel bricht die Aktion ab.

Image

Werkzeug zum Einfügen eines externen Bildes. Zunächst muss der Größenrahmen gezeichnet werden. Das externe Bild wird auf die angegebene Größe skaliert. Im Properties-Fenster kann im Attribut "Other – Href" die URL des Bildes angegeben werden. Zulässig sind die Formate JPEG, GIF, PNG und SVG. Das Bild wird automatisch an den zuvor gezeichneten Rahmen angepasst.

Im Attribut "Other - PreserveAspectRatio" kann die Art der Skalierung des Bildes bestimmt werden. Die gängigsten Einstellungen für PreserveAspectRatio werden vom Gefasoft Designer unterstützt und über eine Drop-Down-Liste zur Auswahl angeboten. Diese sind:

 

Option

Beschreibung

xMidYMid meet

Proportionale Skalierung, möglichst groß, in x- und y-Richtung mittig zentriert (es ist das komplette Bild sichtbar).

xMidYMid slice

Proportionale Skalierung, die Größe des Image-Elements wird komplett ausgefüllt, in x- und y-Richtung mittig zentriert (das Bild wird ggf. abgeschnitten).

none

Das Bild wird in x- und y-Richtung skaliert ohne Berücksichtigung der Proportionen.

xMin/MaxYMin/Max meet/slice

Entspricht den o.g. Skalierungsvarianten, nur abweichende Ausrichtung im image-Element - diese Varianten können im Designer nicht mit Drehungen kombiniert werden (bei mittiger Darstellung können beliebige Drehwinkel verwendet werden).

 

Wurde keine Auswahl getroffen, wird standardmäßig die Skalierung PreserveAspectRatio="xMidYMid meet" verwendet.

Der Designer unterstützt jedoch nur das Format JPG, welches auch zu Anzeige gebracht wird.

Damit das Bild nach dem Hochladen der Grafik in der WebApp angezeigt werden kann, sollte lokal dieselbe Ordnerstruktur wie in der WebApp angelegt werden (z.B. "SvgSrc" für die SVGs und "images" für die Bilder). Die Bilder sollten bereits lokal im entsprechenden Ordner gespeichert werden.

Pie

Erlaubt es, einen Kreis- oder Ellipsenbogen zu zeichnen. Die entsprechenden Parameter werden im Properties Dialog beschrieben.

Symbol

Zeichnet ein spezielles, textähnelndes  Platzhalter-Objekt, das im DOM-Fenster selektiert werden kann. Der Text bzw. Inhalt des Symbol-Objektes kann jederzeit im SVG-Quelltext bearbeitet werden.

Link

Gruppiert die selektierten Elemente als Link. Erlaubt es, externe Dokumente zu verlinken. Wird für eine Dynamisierung mittels SetLink benötigt.

Group

Gruppiert die selektierten Elemente. Diese werden in einen g-Tag eingeschlossen. Änderungen (Verschieben, Drehen) werden auf die gesamte Gruppe angewendet.

Ungroup

Entfernt eine Gruppierung bzw. einen Link. Änderungen an der Gruppe werden bei den enthaltenen Elementen beibehalten. Ungroup kann auch für mehrere selektierte Gruppen gleichzeitig ausgeführt werden.
