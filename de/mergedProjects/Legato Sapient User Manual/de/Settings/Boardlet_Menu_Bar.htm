Boardlet-Menüleiste

Filter-Info

Bei Mouseover über das Icon werden die gesetzten Filter eingeblendet. Die einzelnen Kriterien sind durch Strichpunkt getrennt.

Ob die Filterinfo angezeigt wird oder nicht, kann durch den Entwickler für jedes Boardlet einzeln festgelegt werden.

 

Info und Warnungen

Für manche Boardlets wird zusätzlich eine Information oder eine Warnung als Tooltipp ausgegeben.

 

Menü Quick-Edit []

Über die drei Punkte oben rechts bei jedem Boardlet können bestimmte Dinge auch ohne den Bearbeitungsmodus angepasst werden.

Filters

Klick auf diesen Eintrag öffnet die für das Boardlet zur Verfügung stehenden Filterkriterien, ohne in den Bearbeitungsmodus umzuschalten. Die Filter können somit zur Laufzeit umgeschaltet werden.

:

Die Filtereinstellungen werden hier nur für die aktuelle Session gespeichert. Bei einem Neuladen des Dashboards werden wieder die für das Dashboard gespeicherten Einstellungen geladen.

Lesezeichen:

Durch Auswahl eines Lesezeichen-Typs (kundenspezifisch; im Beispiel Go and see, Rückmeldung, Unterstützung) kann das Boardlet als Lesezeichen markiert werden. Es wird dann mit einem Direkt-Link im Boardlet Lesezeichen aufgeführt. Im aktuellen Dashboard wird ein Boardlet, das als Lesezeichen markiert ist, mit einem blauen Rahmen dargestellt.

Maximieren:

Stellt das Boardlet in der gesamten Dashboard-Größe dar, die Navigationsleisten bleiben bestehen. Nach Schließen der maximalen Ansicht (Wiederherstellen) wird wieder die ursprüngliche Anordnung des Dashboards angezeigt.

CSV Export:

Ermöglicht es, die Daten als CSV zu exportieren.

 

:

Beim Verwenden eines Endgeräts mit iOS oder iPadOS (iPhone und iPad) funktioniert ein CSV-Export ausschließlich mit dem Safari-Browser. Downloads werden per default im Downloads-Ordner von iCloud Drive abgelegt.

Beim Verwenden eines Endgeräts mit dem Android-Betriebssystem ist ein CSV-Export im Chrome Browser möglich. Der CSV-Export wurde nicht mit anderen Browsern getestet.

 

Aufgabe hinzufügen:

Öffnet einen Dialog zum Erfassen einer Aufgabe direkt zu diesem Boardlet. Siehe auch Aufgabenverwaltung - Neue Aufgabe anlegen.

Hilfe:

Öffnet die Hilfe für das selektierte Boardlet in einem neuen Tab im Browser.

Drill down:

Für manche Boardlets steht zusätzlich eine Drill-down-Funktion zur Verfügung, z.B. für die Boardlets Bar Chart und Meldungsdiagramm. Über diese Funktion kann in der Seitenleiste ein anderes Boardlet eingeblendet werden, auf welches die selben Filterkriterien (z.B. Knoten und Zeitraum) angewendet werden wie im Ausgangs-Boardlet eingestellt. Dadurch kann beispielsweise analysiert werden, welche Meldungen im selben Zeitraum auftraten wie zu niedrige Stückzahlen und so eine Ursachenermittlung durchgeführt werden. Für weitere Details zur Funktionsweise, siehe bitte Detail Info.
