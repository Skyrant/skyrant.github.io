Dashboard-Einstellungen / Container-Einstellungen

Ist das gesamte Dashboard oder ein Container selektiert, stehen im Bearbeitungsmodus folgende Optionen zur Verfügung. Wird etwas geändert, ändern sich die Reiter-Symbole in ein Annehmen und ein Abbrechen Symbol.

Erst nach Bestätigung oder Ablehnen der Änderung kann auf einen anderen Reiter gewechselt werden.

Manche Änderungen wie z.B. Berechtigungen werden direkt in die Datenbank geschrieben. In diesem Fall erfolgt kein Wechsel der Symbole.

:

Diese direkten Änderungen werden auch durch Klick auf Abbrechen in der Kopfzeile nicht rückgängig gemacht.

Klick auf Gruppe löschen unten in den Reitern löscht alle Elemente des Dashboards oder Containers.

Reiter Einstellungen

Favorit?

Über den Schieberegler kann festgelegt werden, ob das Dashboard im Menü Favoriten angezeigt werden soll oder nicht. Blau hinterlegt bedeutet, dieses Dashboard ist ein Favorit. Nur für Dashboards verfügbar.

Verwandte Themen:

    
  • Menü Favoriten
    

Titel:

Es wird der aktuelle Titel des Elements angezeigt. Durch Eingabe eines Texts in das Eingabefeld kann der Titel geändert werden.

Max. Länge für Dashboard-Name: 80 Zeichen

Rechte für Dashboard:

Hier können Berechtigungen für das Dashboard vergeben werden. Die Gruppen, in denen der Owner des Dashboards Mitglied ist, sind blau eingefärbt. Diese Einstellung wird nur angezeigt, nachdem das neue Dashboard gespeichert wurde (also beim Editieren, nicht beim Anlegen).

Ordner:

Dropdown-Liste zur Auswahl eines Ordners, in dem das Dashboard gespeichert werden soll. Die Ordner werden im LC2 unter Frontend Dashboard-Ordner angelegt.

Reiter Layout

Darstellungsmodus:

Hier kann der Darstellungsmodus gewählt werden. Die Boardlets im Dashboard können wie folgt dargestellt werden:

  • als Tabs nebeneinander oder

  • als Gridstack oder

  • als Slideshow (Karussell)

Weitere Informationen finden Sie unter Darstellungsmodi.

Reiter Parameter

Hier werden bestimmte Filter-Kriterien der im Dashboard / Container enthaltenen Boardlets aufgeführt. Welche genau das sind, hängt davon ab, welche Boardlets in das Dashboard / den Container eingebettet sind. Für jeden dieser Parameter kann festgelegt werden, ob und wie er nach außen (also an andere Elemente im Dashboard oder in einem Container) weitergegeben werden soll.

Weitere Informationen zur Funktionsweise und Verwendung der Parameter finden Sie unter Parameter.
