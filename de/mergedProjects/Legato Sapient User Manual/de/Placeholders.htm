Platzhalter

Allgemein

Die Parameter "Knoten", "Detail-Info", "Startzeit" und "Endzeit" können als Platzhalter in Filterstrings und URLs verwendet werden. Dann werden diese Platzhalter durch änderbare Werte aus anderen Boardlets befüllt. Grundsätzlich gilt hierfür die Beschreibung der Parametertypen.

$node$ Knoten

Dieser Platzhalter wird immer entweder aus dem Layout Boardlet oder aus dem Knotenbaum Boardlet befüllt. Er kann in diversen anderen Boardlets als Filter eingesetzt werden, um beispielsweise abhängig vom gewählten Knoten andere Daten zu erhalten.

Bei vielen Boardlets wird dieser Platzhalter automatisch verwendet, z.B. werden üblicherweise Meldungen oder KPIs immer mit Knotenbezug angezeigt. Dies erfolgt natürlich nur, wenn die Knoten-ID auf "In" oder "InOut" gesetzt ist (siehe hierzu "Parameter").

Die Knoten-ID kann aber als Variable in den Filterstring oder die URL eingesetzt werden, z.B. beim iFrame Boardlet oder beim Bild Boardlet. Dadurch kann dann abhängig von der Knoten-ID ein anderes Bild aus Legato Classic geladen werden. Die URL könnte dann beispielsweise so aussehen: http://legato.classic.de/Legato/index.html?nodeId=$node$&viewId=1001

Der Platzhalter $node$ wird dann jeweils bei Klick im Knotenbaum Boardlet mit dem entsprechenden Wert befüllt und anschließend die Ansicht aus Legato Classic für diesen Knoten geladen.

$detailinfo$ Detail-Info

Dieses Feld wird überall automatisch mit den Informationen aus der Generischen Tabelle befüllt. Gibt es mehrere Generische Tabellen in einem Dashboard, so werden die Werte der aktuell selektierten Generischen Tabelle eingetragen oder die derjenigen, bei der dieser Parameter auf den Parametertyp "Out" oder "InOut" gesetzt ist.

$timeStart$ und $timeEnd$ Start-Datum und End-Datum

Diese Platzhalter werden mit dem jeweiligen Datum aus dem Zeitfilter des sendenden Boardlets befüllt.
