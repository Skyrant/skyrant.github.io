Darstellungsmodi

Es gibt drei Möglichkeiten, wie Boardlets im Dashboard oder einem Container dargestellt werden können:

  • als Tabs

  • als Gridstack oder

  • im Karussel-Modus (Slideshow)

Der gewünschte Darstellungsmodus kann in den Einstellungen für das ganze Dashboard sowie für jede Boardlet-Gruppe / jeden Container gesondert eingestellt werden. Die Darstellungsmodi können dabei auch beliebig miteinander kombiniert werden.

 

Darstellungsmodus "Tab"

Hier werden alle Boardlets als Tabs dargestellt. Durch Klick auf den gewünschten Tab wird dessen Inhalt angezeigt. Im Beispiel ist der Reiter Bar Chart aktiv.

Darstellungsmodus "Grid Stack"

Hier werden alle Boardlets als Grid Stack dargestellt.

Darstellungsmodus "Karussell"

Hier werden alle Boardlets hintereinander angezeigt. Sie können zwischen den Boardlets wechseln, indem Sie auf die Pfeile klicken oder die Maus mit gedrückter linker Taste darüber  ziehen.

Darstellungsmodus "Tabs" innerhalb von "Grid Stack"

Hier ist das gesamte Dashboard als "Grid Stack" dargestellt, der Inhalt des Containers jedoch als "Tabs".

Darstellungsmodus "Tabs" innerhalb von "Tabs"

Hier sehen wir die 3 Boardlets des Containers als Tabs innerhalb der übergeordneten Tab-Darstellung.

Darstellungsmodus "Grid Stack" innerhalb von "Tabs"

Hier ist der Inhalt des letzten Tabs als Grid Stack dargestellt: die Boardlets "Nodes" und "Messages" untereinander, ein Chart rechts daneben.

Darstellungsmodus "Grid Stack" innerhalb von "Grid Stack"

Hier wird der Inhalt des Containers oben rechts als Grid Stack innerhalb des übergreifenden Grid Stacks angezeigt.

Darstellungsmodus "Karussell" innerhalb von "Grid Stack"

Hier wird der Inhalt des Containers unten links als Slideshow innerhalb eines übergreifenden Grid Stack Containers angezeigt.
