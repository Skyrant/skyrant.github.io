Allgemein

INFO:

Je nach gewähltem Theme kann die Darstellung von Legato Sapient zur Dokumentation in Farbe und Form abweichen.

Im Bereich Einstellungen unter Allgemein können Sie z.B. das neue Layout ("New layout") aktivieren. Hierdurch verändert sich die Darstellung von Legato Sapient maßgeblich.

 

 

 

 

Nach dem Öffnen von Legato Sapient gelangt man auf die Hauptseite. Diese ist in verschiedene Bereiche unterteilt.

  1. Menü Dashboards und Menü Favoriten mit Dashboard-Profil und integrierter Boardlet-Suche

  2. Menüs zum Bearbeiten und Drucken des aktuell gewählten Dashboards und zum Hinzufügen neuer Dashboards

  3. Menü Benutzer

  4. Top Custom Bar

  5. Arbeitsbereich und Dashboarddarstellung

Menü Dashboards und Menü Favoriten [1]

Über dieses Menü kann das gewünschte Dashboard ausgewählt werden. Das Menü ist hierarchisch aufgebaut. Dadurch können die Dashboards in Ordner gruppiert werden. Für weitere Details siehe Menü Dashboards.

Im Favoritenordner finden sich als Favoriten markierte Dashboards. Für weitere Details siehe Menü Favoriten.

Menü Profile [1]

Durch Klick auf können die Profile des angemeldeten Benutzers für das selektierte Dashboard geöffnet werden. Siehe Profile.

Suche [1]

Hier kann nach einem Dashboard gesucht werden. Siehe Suche.

Bearbeiten und Drucken [2]

Dashboard bearbeiten. Öffnet das Menü zur Bearbeitung des Dashboards. Welche Einstellungen vorgenommen werden können, hängt vom selektierten Dashboard ab. Für weitere Details siehe Bearbeitungsmodus.

Dashboard anlegen. Fügt ein neues Dashboard hinzu. Dieser Button wird nur angezeigt, wenn der Benutzer das benötigte Recht hat.

Drucken. Öffnet die Druckeinstellungen. Für weitere Details siehe Menü Drucken.

 

Benötigte Berechtigung

Zum Neuanlegen von Dashboards benötigt der angemeldete Benutzer das Recht "Editieren" auf die Datenquelle "GP_R_DASHBOARDS".

Zum Ändern eines Dashboards benötigt der angemeldete Benutzer das Recht "Editieren" auf dieses Dashboard oder er ist der Owner des Dashboards, d.h. er hat es selbst erstellt.

Menü Benutzer [3]

Im Menü Benutzer finden sich Links zu den Einstellungen, die Hilfe und "Über". Hier findet sich auch der Logout Button. Für weitere Details siehe Menü Benutzer.

Custom Bar [4]

In Sapient stehen eine Top Custom Bar [4] und eine Side Custom Bar zur Verfügung, in welche nach Belieben Dashboards zum Schnellzugriff gezogen werden können. Beispielsweise kann hier eine Zeitauswahl eingebunden werden, oder der Knotenbaum. Welche Custom Bars angezeigt werden, kann über die Benutzereinstellungen gesteuert werden. Für weitere Details siehe auch Custom Bar.

Arbeitsbereich und Dashboarddarstellung [5]

Der Arbeitsbereich ist die Hauptfläche und stellt alle Inhalte und Informationen dar. Hier können Boardlets platziert werden.

Feedback / Meldungen

Über ausgeführte Aktionen oder ggf. aufgetretene Fehler wird der Benutzer durch kurze Pop-up-Meldungen ("Toasts") informiert. Meldungen werden beispielsweise beim Laden oder Speichern eines Dashboards ausgegeben oder beim Ändern des Passworts.
