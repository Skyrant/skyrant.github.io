Menü Bearbeitung

Bei aktivem Bearbeitungsmodus öffnet sich das zusätzliche Menü Bearbeitung, das standardmäßig überlagernd (show as overlay) angezeigt wird. Durch Klick auf kann das Menü Bearbeitung auch schwebend (Float) eingeblendet oder als Seitenleiste an der rechten Seite des Bildschirms angedockt werden (show as sidebar).

:

Ist das Menü Bearbeitung als Seitenleiste angedockt (show as sidebar), werden ggf. die Container und Boardlets verkleinert dargestellt. Die Darstellung (Größe, Scrollbars, etc.) im Bearbeitungsmodus kann somit von der tatsächlichen Darstellung abweichen.

Das Menü Bearbeitung ist in folgende Reiter unterteilt:

Boardlet-Bibliothek
Boardlet-Einstellungen / Dashboard-Einstellungen
Boardlet-Layout / Dashboard-Layout
Boardlet-Filter / Dashboard-Filter
Boardlet-Container-Parameter / Dashboard-Parameter
Boardlet-Tabelleneinstellungen
Boardlet-Datenquellen

Die verfügbaren Optionen in den einzelnen Reitern sind abhängig davon, was im Dashboard ausgewählt ist:

  • das Dashboard selbst

  • ein bestimmtes Boardlet auf dem Dashboard

  • ein Container mit mehreren Boardlets

Die häufigsten Einstellungen werden unter Boardlet Einstellungen beschrieben; spezielle Optionen gegebenenfalls beim jeweiligen Boardlet.
