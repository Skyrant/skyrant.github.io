Boardlet-Bibliothek

Die Boardlet-Bibliothek dient dazu, neue Boardlets in ein Dashboard einzufügen. Außerdem kann nach einem bestimmten Boardlet gesucht werden. Siehe hierzu auch Suche.

Es wird zwischen Container-Boardlets und Content-Boardlets unterschieden. Container sind leere Behälter, in die dann Content-Boardlets eingefügt werden können. Sie dienen zur Gruppierung von Boardlets als Grid oder als Tabs. Boardlets können aber auch ohne Container direkt in das Dashboard eingefügt werden.

Außerdem gibt es noch Boardlets, die der Navigation dienen (z.B. das Boardlet Knotenlayout), sowie weitere Design-Einstellungen.

In der Boardlet-Bibliothek haben Sie die folgenden Möglichkeiten:

  • Suche nach Boardlets

  • Durch Klick auf den Namen kann das Boardlet ins Dashboard eingefügt werden.

  • Durch Klick auf wird eine Beschreibung für das Boardlet eingeblendet und ein Link zur Webhilfe für dieses Boardlet. Außerdem kann eine Preview des Boardlets angezeigt werden.

  • Klick aufs Bild vergrößert die Vorschau, ESC schließt die Vorschau.

  • Erneuter Klick auf schließt die Information.

  • Klick auf öffnet die Kontexthilfe für das selektierte Boardlet in einem neuen Tab.
