Dashboard bearbeiten

Sie können ein bereits bestehendes Dashboard und die Boardlets auf dem Dashboard bearbeiten, indem Sie den Bearbeitungsmodus des Dashboards aufrufen.

So rufen Sie den Bearbeitungsmodus für ein Dashboard auf:

1

Suchen Sie das Dashboard, das Sie bearbeiten wollen.

2

Klicken Sie auf das Symbol Dashboard bearbeiten, um den Bearbeitungsmodus aufzurufen.

Verwandte Themen:

      
  • Menü Dashboards
  • Wissenswertes über den Bearbeitungsmodus
  • Dashboard-Einstellungen
  
  • Boardlet-Einstellungen
  • Boardlet-Bibliothek
