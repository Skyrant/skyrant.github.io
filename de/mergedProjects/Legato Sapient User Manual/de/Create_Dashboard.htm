Neues Dashboard anlegen

Wenn der angemeldete Benutzer das entsprechende Recht hat, kann er über den Button ein neues Dashboard anlegen.

Zu den möglichen Einstellungen, siehe bitte Dashboard-Einstellungen.

Alle für die eingefügten Boardlets festgelegten Filter, Layout-Einstellungen etc. werden für das Dashboard gespeichert. Beim Öffnen wird das Dashboard dann immer mit diesen Einstellungen geladen.

Der Benutzer, der das Dashboard erstellt hat, wird für dieses als Owner eingetragen und hat somit immer Änderungsrechte für dieses Dashboard. Weitere Berechtigungen können über die Dashboard-Einstellungen oder den LC2 vergeben werden (nachdem das Dashboard gespeichert wurde).
