Maßnahmenübersicht

Allgemein

Das Boardlet Maßnahmenübersicht zeigt definierte Maßnahmen an. Die Kopfzeile bietet zusammengefasste Informationen über alle enthaltenen Arbeitsschritte, z.B. wieviele Arbeitsschritte sich aktuell in welchem Status befinden.

Über den Filter kann definiert werden, ob man nur Maßnahmen anzeigen möchte, die man selbst erstellt hat oder alle. Außerdem kann festgelegt werden, ob die Anzeige knotenbezogen erfolgen soll. Dann werden alle Maßnahmen angezeigt, die dem selektierten Knoten oder einem seiner Unterknoten zugeordnet sind.

Die einzelnen Symbole haben folgende Bedeutung:

Anzahl der Arbeitsschritte zu dieser Maßnahme in diesem Status.

Fälligkeitsdatum der Maßnahme. Wird rot eingefärbt, wenn überfällig.

Tag / Schlagwort für diese Maßnahme.

Zu Boardlet springen. Springt zu dem Dashboard, von dem aus über ein Boardlet diese Maßnahme erstellt wurde.

Maßnahme editieren. Siehe Edit-Modus.

Maßnahme löschen.

Edit-Modus

Durch Klick auf das Stift-Icon bei einer Maßnahme kann diese bearbeitet werden. Es öffnet sich folgender Dialog:

Die folgenden Daten können geändert werden: Titel der Maßnahme, Tags, Beschreibung der Maßnahme.

Bei den Tags kann nach Eingabe eines oder mehrere Buchstaben ein bereits existierender Tag aus einer Liste selektiert werden. Benutzer mit Administratorenrechten können auch neue Tags anlegen.

 

Detail-Ansicht

Durch Klick auf eine Maßnahme wird die Detail-Ansicht zu dieser Maßnahme geöffnet.

Hier werden alle zu dieser Maßnahme erfassten Arbeitsschritte angezeigt (siehe auch Maßnahmen Arbeitsschritte).

Bei Mouseover über den Namen des Verantwortlichen werden dessen Kontaktdaten eingeblendet, z.B. Mail, Skype etc. Durch Klick auf einen Kontakt-Eintrag wird direkt das entsprechende Programm geöffnet, sofern es installiert ist.

Durch Klick auf das Plus-Symbol kann ein neuer Arbeitsschritt für diese Maßnahme erfasst werden (siehe auch Edit-Modus).

Durch Klick auf das Mülltonnen-Icon werden alle Arbeitsschritte dieser Maßnahme gelöscht.

Für das Boardlet Maßnahmenübersicht stehen folgende weitere Einstellungen zur Verfügung.

Reiter Filter

Knoten-ID:

Über diesen Schieberegler kann definiert werden, ob für die Anzeige der selektierte Knoten berücksichtigt werden soll oder ob die Daten aller Knoten angezeigt werden sollen.

Wenn aktiv, werden alle Maßnahmen angezeigt, die dem selektierten Knoten oder einem seiner Unterknoten zugeordnet sind.

Eigentümer:

Dropdown-Liste zur Auswahl, ob nur vom angemeldeten Benutzer erstellte Maßnahmen angezeigt werden sollen oder "alle".

Tags:

Dropdown-Liste zur Auswahl eines oder mehrerer Tags, die den angezeigten Maßnahmen zugewiesen sein müssen.

Auch Arbeitsschritte nach Tags filtern:

Über diesen Schieberegler kann definiert werden, ob auch die angezeigten Arbeitsschritte der Maßnahmen nach den oben eingegebenen Tags gefiltert werden sollen.
