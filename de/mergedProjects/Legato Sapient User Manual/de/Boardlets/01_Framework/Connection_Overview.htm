Verbindungsübersicht

Allgemein

Das Boardlet Verbindungsübersicht zeigt für jedes Modul/Gateway (aus p_modules) und jede untergeordnete Verbindung (aus p_connections) den Status an. Die Status untergeordneter Datensätze werden zusätzlich zusammengefasst.

Die Spalte Count visualisiert den Status des Moduls und all seine Verbindungen. Daneben wird die Anzahl der Verbindungen je Status angezeigt.
In der Spalte State ist der Status des Elements auf oberster Ebene ersichtlich.

Durch Klick auf > kann ein Element aufgeklappt werden. Darunter werden dann alle Unterelemente angezeigt.

Durch Klick auf einen bestimmten Status wird eine Unteransicht geöffnet, in der nur die Elemente angezeigt werden, für die entweder selbst oder eins ihrer untergeordneten Elemente der selektierten Status gesetzt ist.
