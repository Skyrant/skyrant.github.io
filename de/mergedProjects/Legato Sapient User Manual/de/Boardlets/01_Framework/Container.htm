Allgemein

Ein Container-Boardlet kann dazu benutzt werden, mehrere Boardlets zu gruppieren. Die Boardlets können als Grid nebeneinander oder als Tabs angeordnet werden. Außerdem kann eine Slideshow erstellt werden (Karussell):

Gruppeneinstellungen

Die häufigsten Einstellungen auf den Reitern Einstellungen, Layout und Filter sind unter Boardlet-Einstellungen beschrieben:

  • Reiter Einstellungen

  • Reiter Layout

  • Reiter Filter

Da ein Container außerdem ähnliche Eigenschaften wie das Dashboard hat, sind weitere mögliche Einstellungen unter Dashboard Einstellungen beschrieben.

Slider automatisch rotieren (in Sekunden)

Diese Einstellung steht nur für den Container vom Typ Slider zur Verfügung. Wenn aktiv, kann in der Zeile darunter eine Anzahl Sekunden eingegeben werden, nach der die Tabs automatisch weitergeblättert werden.
