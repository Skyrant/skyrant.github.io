Attribut-Statistik

Allgemein

Das Boardlet Attribut-Statistik zeigt alle gebuchten Attributwerte zu den selektierten Produktionseinheiten an.

Bei Mouseover über eine Linie oder einen Balken werden der Zeitstempel und der Wert zu diesem Zeitpunkt eingeblendet.

Durch Klick auf den Namen eines Wertes unten in der Legende kann dieser aus- bzw. wieder eingeblendet werden. Ausgeblendete Werte werden in der Legende hellgrau dargestellt.

Einstellungen

Die häufigsten Einstellungen auf den Reitern Einstellungen, Layout und Filter sind unter Boardlet-Einstellungen beschrieben:

  • Reiter Einstellungen

  • Reiter Layout

  • Reiter Filter

Für das Boardlet Attribut-Statistik stehen folgende weitere Einstellungen zur Verfügung.

Reiter Layout

Diagrammtyp:

Aus der Dropdown-Liste kann der Diagrammtyp gewählt werden. Zur Verfügung stehen Liniendiagramm oder Histogramm.

Reiter Filter

Produktionseinheit:

Hier kann die ID einer Produktionseinheit eingegeben werden, für die die Attributwerte angezeigt werden sollen.

Produkte:

Aus der Dropdown-Liste können Produkte ausgewählt werden, für die PU-Attributwerte angezeigt werden sollen.

Benötigte Parametrierung im LC2

Attributbeschreibungen für PUs werden im LC2 unter Produktionsmanagement Tracking & Tracing Attributbeschreibungen für PUs angelegt.
