Produktionseinheiten

Allgemein

Das Boardlet Produktionseinheiten zeigt alle verfügbaren Produktionseinheiten (Production Units bzw. PUs) an. Es werden verschiedene Informationen angeboten, beispielsweise den Zeitpunkt und Ort der letzten Identifikation, den aktuellen Zustand etc.

Die Spalte "Letzte Identifizierung" zeigt den Zeitstempel der letzten Identifizierung der PU an, unabhängig vom Identifikationspunkt. Die Spalte "Ident-Zeit" zeigt den Zeitstempel der Identifizierung an einem bestimmten, über den Filter selektierten Identifikationspunkt an.

Mittels Drag & Drop lassen sich die Breite oder Position der Spalten einfach verändern. Durch Klick auf eine Zeilenüberschrift wird nach dieser Spalte sortiert.
Nach welcher Spalte sortiert ist, und ob auf- oder absteigend sortiert ist, wird durch die dreieckigen Symbole angezeigt:

Defaultmäßig ist die Ansicht nach PU-ID sortiert.

Über das Suchfeld in der Menüleiste des Boardlets kann nach einer PU-ID gesucht werden. Eine Dropdown-Liste zeigt alle Einträge, die zu den eingegebenen Buchstaben passen.

PU-Details

Durch Klick auf eine Zeile öffnet sich eine Unteransicht mit Details zu dieser PU:

In einer neuen Tabelle werden alle Identifikationsereignisse und Statusänderungen mit Zeitstempel aufgelistet. Ist die Einstellung "Detailansicht gruppieren" gesetzt, so werden Ereignisse am selben Identifikationspunkt in einer Zeile zusammengefasst, sonst wird pro Ereignis eine eigene Zeile angezeigt.

Bei nicht-gruppierter Anzeige öffnet sich durch Klick auf einen Eintrag darunter ein Dialog zum Ansehen der Daten bzw. zum Erfassen / Ändern von Zustand und Q-Zustand.

Hat man einen Eintrag ausgewählt, bei dem für den Q-Zustand ein Dokument hinzugefügt wurde, so erscheint in der Menüleiste des Boardlets ein Symbol zum Öffnen dieses Dokuments .

Klick auf öffnet die Split-Historie der selektierten PU in dieser Ansicht. Für weitere Details, siehe Split-Historie.

Durch Klick auf kann man zur Gesamtübersicht zurückkehren.

Durch Klick auf kann ein neues Identifikationsereignis für diese PU erfasst werden. Es öffnet sich ein entsprechender Dialog in der rechten Seitenleiste: --->>

Über das Kalendersymbol kann das Datum des Ereignisses ausgewählt werden. Mithilfe der Dropdown-Listen kann ein Zustand, ein Q-Zustand sowie ein Identifikationspunkt selektiert werden. Mindestens der Identifikationspunkt muss eingegeben werden.

 

Durch Klick auf kann ein neues Attribut hinzugefügt werden. Es öffnet sich ein entsprechender Dialog in der rechten Seitenleiste:

Mithilfe der Dropdown-Listen kann ein Attribut, ein Attributwert sowie ein Identifikationspunkt selektiert werden. Darunter kann der Zeitpunkt eingegeben werden.

Die häufigsten Einstellungen auf den Reitern Einstellungen, Layout und Filter sind unter Boardlet-Einstellungen beschrieben:

Für das Boardlet Produktionseinheiten stehen folgende weitere Einstellungen zur Verfügung.

Reiter Einstellungen

Detailansicht gruppieren:

Über diesen Schieberegler wird definiert, ob die Detailansicht gruppiert angezeigt werden soll oder nicht. Gruppiert bedeutet, es werden alle Ereignisse am selben Identifikationspunkt zusammengefasst und es werden keine weiteren Details zur Ansicht/Bearbeitung mehr angezeigt. Nicht gruppiert bedeutet, dass pro Ereignis eine eigene Zeile angezeigt wird und durch Klick darauf weitere Details zur Ansicht/Bearbeitung angeboten werden.

Filter:

Schieberegler für jedes einzelne Filterkriterium. Nur die, die hier aktiviert sind, stehen nach dem Speichern als Filter zur Verfügung.

Archivierte Identifikation (Equipment):

Ist dieser Schieberegler aktiv, so wird der Filter durch Auswahl eines Knotens automatisch gesetzt.

Reiter Filter

Zeitauswahl vom Dashboard übernehmen (letzte Identifikation):

Über diesen Schieberegler wird definiert, ob die Zeitauswahl des Dashboards für das Boardlet übernommen werden soll (Filter auf letztes Identifikationsereignis) oder ob alle PUs angezeigt werden sollen (ohne Zeitfilter).

Zeitauswahl vom Dashboard übernehmen (archivierte Identifikation):

Über diesen Schieberegler wird definiert, ob die Zeitauswahl des Dashboards für das Boardlet übernommen werden soll (Filter auf archivierte Identifikationsereignisse) oder ob alle PUs angezeigt werden sollen (ohne Zeitfilter).

Produktionseinheiten:

Hier kann nach den PU-IDs gefiltert werden. Nach Eingabe mindestens eines Buchstabens in das Feld wird eine Drop Down Liste mit allen PUs geladen, die diesen Buchstaben enthalten. Durch Klick auf eine PU wird diese als Auswahl eingetragen. Mehrfachauswahl ist möglich.

Produktgruppe:

Über die Dropdown-Liste kann nach Produktgruppen gefiltert werden, denen das Produkt, das der PU zugeordnet ist, angehört.

Produkte:

Über die Dropdown-Liste kann nach Produkten gefiltert werden, die den angezeigten PUs zugeordnet sind.

Varianten:

Über die Dropdown-Liste kann nach Varianten gefiltert werden, die den angezeigten PUs zugeordnet sind.

Letzte Identifikation (Equipment):

Über die Dropdown-Liste kann der Identifikationspunkt ausgewählt werden, an dem die PU zuletzt identifiziert wurde.

Archivierter Knoten - mit Unterknoten:

Hier kann auf die ID eines Knotens gefiltert werden, an dessen Identifikationspunkten die PU irgendwann identifiziert wurde.

Archivierte Identifikation (Equipment):

Über die Dropdown-Liste kann der Identifikationspunkt ausgewählt werden, an dem die PU in der Vergangenheit identifiziert wurde. Diese Auswahl kann auch durch Selektion eines Identifikationspunkts im Lesestellen-Boardlet erfolgen.

Wird hier etwas ausgewählt, so wird in der PU-Ansicht in der Zeile "Ident-Zeit" der Zeitstempel angezeigt, an dem die PU an diesem Equipment identifiziert wurde.

Aktueller Zustand:

Über die Dropdown-Liste kann auf aktuelle Zustände gefiltert werden.

Aktueller Qualitätszustand:

Über die Dropdown-Liste kann auf aktuelle Qualitätszustände gefiltert werden.

Archivierter Qualitätszustand:

Über die Dropdown-Liste kann auf archivierte Qualitätszustände gefiltert werden.

Aktueller Arbeitsgang:

Über die Dropdown-Liste kann nach Arbeitsgängen gefiltert werden.

Aktuelles Transportmittel:

Über die Dropdown-Liste kann nach Transportmitteln gefiltert werden.

Benötigte Parametrierung im LC2

  • Identifikationspunkte werden im LC2 unter dem zugehörigen Knoten der Fertigungsstruktur unter Anwendungsstruktur Fertigungsstruktur Knoten Identifikationspunkt angelegt.

  • PU-Zustände werden im LC2 unter Produktionsmanagement Tracking & Tracing PU-Zustände gepflegt.

  • PU-Qualitätszustände werden im LC2 unter Produktionsmanagement Tracking & Tracing PU Qualitätszustände gepflegt.

  • Attributbeschreibung für PUs werden im LC2 unter Produktionsmanagement Tracking & Tracing Attributbeschreibung für PUs gepflegt.
