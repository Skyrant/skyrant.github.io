PU Qualitätsüberwachung

Allgemein

Das Boardlet PU Qualitätsüberwachung zeigt für alle Schichten des selektierten Knotens für einen Zeitraum von maximal einer Woche die gebuchten Q-Zustände zu einem Arbeitsplan an. Über die Einstellungen des Boardlets kann festgelegt werden, welche Q-Zustände rot (Fehler) oder orange (Warnung) angezeigt werden sollen. Wurde kein Q-Zustand erfasst, wird der Eintrag für die Schicht schwarz angezeigt, wurde ein Q-Zustand erfasst, der nicht als Fehler oder Warnung definiert ist, wird der Eintrag grün angezeigt.

So lässt sich auf einen Blick erkennen, in welcher Schicht es bei welchem Arbeitsgang zu Qualitätsproblemen kam. Hat man im Dashboard zusätzlich das Produktionseinheiten-Boardlet eingefügt, so filtert ein Klick auf einen Eintrag auf die betroffene PU und färbt diese rot ein.

Für das Boardlet PU Qualitätsüberwachung stehen folgende weitere Einstellungen zur Verfügung.

Reiter Einstellungen

Nur Arbeitspläne von letzter Woche:

Ist dieser Schieberegler aktiv, werden im Reiter Filter nur die Arbeitspläne in der Dropdown-Liste angezeigt, die letzte Woche kativ waren.

Fehler:

Aus der Dropdown-Liste können ein oder mehrere Q-Zustände selektiert werden, die als Fehler (rot) visualisiert werden sollen.

Warnung:

Aus der Dropdown-Liste können ein oder mehrere Q-Zustände selektiert werden, die als Warnung (orange) visualisiert werden sollen.

Zeitschema:

Ist dieser Schieberegler aktiv, wird nicht automatisch das Zeitschema des selektierten Knotens verwendet, sondern kann individuell im Reiter Filter ausgewählt werden.

Maximale Tage zur Anzeige:

Angabe, wieviele Tage unabhängig vom gewählten Zeitfilter des Dashboards maximal angezeigt werden sollen.

Reiter Filter

Arbeitsplan auswählen:

Aus der Dropdown-Liste kann ein Arbeitsplan ausgewählt werden, für den die Auswertung erfolgen soll. Je nach Einstellung werden hier alle Arbeitspläne angezeigt oder nur solche, die letzte Woche aktiv waren.

Zeitschema:

Ist die Einstellung "Zeitschema" = true, kann das gewünschte Zeitschema hier ausgewählt werden. Andernfalls wird das Zeitschema des selektierten Knotens verwendet.

Benötigte Parametrierung im LC2

Für den selektierten Knoten müssen Schichten parametriert und über den Schichtkalender geplant sein.

Außerdem wird ein Arbeitsplan mit Arbeitsgängen benötigt. Diese Daten können im LC2 unter Produktionsmanagement Produktionsplanung Arbeitspläne Arbeitsgänge parametriert werden.

Schließlich muss für eine bestehende PU ein Q-Zustand mit Bezug zu einem Arbeitsgang gebucht werden. Dies erfolgt über die Prozedur "setQStateId".
