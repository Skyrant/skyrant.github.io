Lesestellen

Allgemein

Das Boardlet Lesestellen zeigt alle verfügbaren Lesestellen für den selektierten Knoten mit dem jeweils letzten Identifikationsereignis an.

Mittels Drag & Drop lassen sich die Breite oder Position der Spalten einfach verändern. Durch Klick auf eine Zeilenüberschrift wird nach dieser Spalte sortiert.
Nach welcher Spalte sortiert ist, und ob auf- oder absteigend sortiert ist, wird durch die dreieckigen Symbole angezeigt:

Durch Klick auf > zu Beginn einer Zeile werden weitere Identifikationsereignisse an diesem Identifikationspunkt mit Details geladen. Wieviele Zeilen geladen werden, kann über den Filter festgelegt werden.

Selektiert man den Identifikationspunkt durch Klick, wird die Auswahl an das Produktionseinheiten Boardlet weitergegeben und dort alle PUs angezeigt, die irgendwann an diesem Identifikationspunkt identifiziert wurden. Dazu muss dort die Einstellung "Archivierte Identifikationen" aktiv sein. Siehe auch Produktionseinheiten.

Selektiert man eine PU, so wird die Auswahl ebenfalls an das Produktionseinheiten Boardlet weitergegeben und dort die PU-Details angezeigt. Siehe auch Produktionseinheiten. Außerdem wird die Auswahl auch an das Attribut-Statistik Boardlet weitergegeben. Alternativ kann man sich die Attribut-Statistik für die selektierte PU auch über das Menü Quick-Edit [] als Drilldown in der Seitenleiste öffnen.

Reiter Filter

Hier kann festgelegt werden, wieviele Identifikationsereignisse beim Aufklappen eines Identifikationspunkts angezeigt werden sollen.

Benötigte Parametrierung im LC2

Identifikationspunkte werden im LC2 unter dem zugehörigen Knoten der Fertigungsstruktur unter Anwendungsstruktur Fertigungsstruktur Knoten Identifikationspunkt angelegt.
