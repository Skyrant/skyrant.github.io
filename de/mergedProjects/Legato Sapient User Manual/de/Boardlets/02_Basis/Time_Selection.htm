Zeitauswahl

Allgemein

Die im Boardlet Zeitauswahl vorgenommenen Einstellungen werden in den anderen Boardlets eines Dashboards als Filter angewendet.

Je nach Einstellung kann im Zeitstrahl links durch Klick ein Bereich ausgewählt werden (z.B. eine Woche oder eine Schicht). Mithilfe des Mausrads kann der Zeitstrahl vergrößert oder verkleinert werden.

INFO:

Relative Zeitfilter wirken sich folgendermaßen aus: "Diese(r)" visualisiert die laufende Zeiteinheit (z.B. 1.1.-31.1.), d.h., die angezeigten Werte enden normalerweise irgendwo mitten im Diagramm. "Letzte(r)" visualisiert die letzte abgeschlossene Zeiteinheit. "In den letzten" visualisiert die gewählte Anzahl an bereits abgeschlossenen Zeiteinheiten, die aktuell laufende Einheit wird nicht mit angezeigt.

Umschalten

Ganz rechts wird der aktuelle Auswahlmodus angezeigt, standardmäßig das Kalender-Symbol. Beim Überfahren mit der Maus öffnet sich eine Dropdown-Liste zum Umschalten.

Schichtauswahl

Durch Klick auf schaltet die Ansicht um und der Zeitraum kann durch Schichtauswahl selektiert werden.

Über den Schieberegler wird definiert, ob der Schichtfilter absolut oder relativ sein soll. Bei absolutem Schichtfilter kann über eine Dropdown-Liste ein bestimmter Schichttyp oder "Alle" selektiert werden. Die Auswahl wird dann als blaue Balken im Zeitstrahl visualisiert (in Abhängigkeit des selektierten Knotens).

Bei relativem Schichtfilter können relative Schichten oder Produktionstage (diese, letzte, die letzten xxx) eingestellt werden. Werden dabei Einträge angefordert, die außerhalb den angezeigten fünf Tagen liegen, so muss aus dem Zeitbalken herausgezoomt werden, um diese zu visualisieren.

Kalender

Durch Klick auf schaltet die Ansicht um und der Zeitraum kann durch Kalenderfunktionen selektiert werden.

Links wird der aktuell eingestellte Zeitbereich angezeigt.

Klick auf das Kalender-Icon öffnet einen Kalender, über den ein bestimmtes Datum selektiert werden kann. Der aktuelle Tag ist farbig markiert.

Schnellauswahl

Im rechten Bereich kann über Schnellauswahl ein relativer Zeitfilter eingestellt werden (gestern, heute, letzter Monat, in den letzten xxx Tagen etc.). Zum Übernehmen eines Werts in "letzte xxx" muss der Wert eingegeben und dann auf "Letzte" geklickt werden.

Zeitbereich

Durch Klick auf schaltet die Ansicht um und der Zeitraum kann durch ein Start- und Enddatum mit Uhrzeit festgelegt werden.

Links wird der aktuell eingestellte Zeitbereich angezeigt.

Durch Klick auf wird die Eingabe übernommen.

Klicken und Wählen

Durch Klick auf schaltet die Ansicht um und der Zeitraum kann durch Fragen eingestellt werden. Je nach Auswahl werden dann die entsprechenden weiteren Möglichkeiten angezeigt.

Schichten

Wurden Schichten ausgewählt, kann zunächst auf einen Schichttyp eingeschränkt werden. Im Anschluss kann zusätzlich auf eine Schicht- oder Produktionsgruppe eingeschränkt werden. Als letztes kann auf eine bestimmte Schicht, einen Produktionstag, Schichten aus einem relativen Zeitraum oder Schichten aus einem absoluten Zeitraum eingeschränkt werden.

Relative Zeit

Hier kann ein relativer Zeitfilter eingestellt werden (gestern, heute, letzter Monat, in den letzten xxx Tagen etc.) Zum Übernehmen eines Werts in "letzte xxx" muss der Wert eingegeben und dann auf "Letzte" geklickt werden.

Absolute Zeit

Hier kann der Zeitraum kann durch ein Start- und Enddatum mit Uhrzeit festgelegt werden.
