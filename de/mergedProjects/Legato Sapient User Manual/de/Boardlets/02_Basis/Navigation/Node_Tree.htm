Knotenbaum

Allgemein

Das Boardlet Knotenbaum zeigt alle parametrierten Knoten als hierarchische Liste an. Durch Klick auf die Checkbox vor einem Knoten wird dieser Knoten selektiert. Mittels Strg + Klick können auch mehrere Knoten selektiert werden,  z.B. für die Anzeige im Boardlet Multi-Knoten .

Oberhalb des Knotenbaums wird eine Suchfunktion angeboten. Die folgenden Funktionen stehen zur Verfügung:

Alle Knoten ausklappen.

Alle Knoten einklappen.

Suchergebnisse auswählen. Bereits selektierte Knoten werden beibehalten.

Suchergebnisse abwählen.

Alle Knoten durchsuchen bzw. selektierte Knoten durchsuchen.

 

Nachdem Knoten über die Suche selektiert wurden, deselektiert ein Klick auf einen Knoten innerhalb der Suchergebnisse diesen einen Knoten. Ein Klick auf einen Knoten außerhalb der Suchergebnisse setzt die Auswahl zurück und selektiert nur diesen einen Knoten. Ein Klick auf einen Knoten außerhalb der Suchergebnisse mit gedrückter Strg-Taste fügt diesen Knoten der Auswahl hinzu.

In den Benutzereinstellungen kann dieses Boardlet auch Dashboard-übergreifend in die Custom Bar eingebunden werden.

Durch Doppelklick auf einen Knoten wird dieser als neuer Root-Knoten gesetzt und die Ebenen darüber werden ausgeblendet.

Für das Boardlet Knotenbaum stehen außerdem folgende Einstellungen zur Verfügung.

Reiter Filter

Root-Knoten:

ID des Root-Knotens, d.h. den zuoberst angezeigten Knoten.

Knotenlevel:

Eingabe der ID des Knotenlevels, für das Knoten angezeigt werden sollen.

Knotentypen:

Auswahl eines oder mehrere Knotentypen, auf die gefiltert wird. Auswahl erfolgt über Dropdown-Liste. Der Filter wird nur angezeigt, wenn zusätzlich der Schieberegler "Filterergebnisse als flache Liste anzeigen" aktiv ist.

Toolbar anzeigen:

Dieser Schieberegler definiert, ob in der Kopfzeile des Node Tree Boardlet eine Toolbar angezeigt werden soll, mit der eine Ebene nach oben oder zum Root-Knoten gesprungen werden kann und mit dem die Filter angewendet werden können.

Filterergebnisse als flache Liste anzeigen:

Dieser Schieberegler definiert, ob die Knoten zum selektierten Root-Knoten als flache Liste dargestellt werden sollen, oder als hierarchische Liste. Diese Funktion kann auch über das Listen-Symbol in der Toolbar umgeschaltet werden, sofern diese eingeblendet wird.

Automatisch aufklappen:

Dieser Schieberegler definiert, ob der Knotenbaum beim Dashboardwechsel direkt beim selektierten Knoten aufgeklappt werden soll. Wenn aktiv, werden die Einstellungen "Knotentypen" und "Filterergebnisse als flache Liste anzeigen" deaktiviert.

Benötigte Parametrierung im LC2

Die angezeigten Knoten und ihre Hierarchie (einschließlich Knotenlevel und Knotentyp) werden im LC2 unter Anwendungsstruktur Fertigungsstruktur Knoten gepflegt.
