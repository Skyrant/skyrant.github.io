Rüstzeitmatrix

Allgemein

Das Boardlet Rüstzeitmatrix zeigt eine Übersicht über alle Rüstzeiten mit Bezug zu Produktwechseln (Rüstzeit = die Zeit, die benötigt wird, um die Maschine für die Produktion eines neuen Produkttyps vorzubereiten). Es werden die durchschnittlichen Rüstzeiten (Wechsel von einem Produkttyp zu einem anderen) für einen selektierten Zeitraum angezeigt. Die Rüstzeit wird dabei über Meldungen der Meldeklasse -14 erfasst; die Produkte werden über erweiterte Eigenschaften angegeben.

Die Matrix hat folgende Achsen:

x-Achse: Ausgangsprodukt (von Produkt)

y-Achse: Zielprodukt (zum Produkt)

Die einzelnen Kacheln zeigen die durchschnittliche Rüstzeit von Produkt x zum Produkt y in Sekunden im gewählten Zeitraum. Ein Tooltip gibt außerdem an, wieviele Produktwechsel per "Box" durchgeführt wurden.

Wird eine Kachel angeklickt, werden die Produkte als Filter für die Rüstzeittabelle übernommen.

Für das Boardlet Rüstzeitmatrix stehen außerdem folgende Einstellungen zur Verfügung.

Reiter Filter

Meldeklasse:

Dropdown-Liste zur Auswahl der Meldeklasse. Standardmäßig auf -14  Rüstzeit gesetzt, kann bei Bedarf aber geändert werden.

von Produkt:

Dropdown-Liste zur Auswahl des Produkts, von dem gewechselt wurde (x-Achse).

nach Produkt:

Dropdown-Liste zur Auswahl des Produkts, zu dem gewechselt wurde (y-Achse).

Benötigte Parametrierung im LC2

Es müssen Meldungen der jeweiligen Klasse für den selektierten Knoten vorhanden sein. Außerdem müssen für diese Meldungen die erweiterten Eigenschaften "von Produkt" und "zum Produkt" gebucht worden sein.
