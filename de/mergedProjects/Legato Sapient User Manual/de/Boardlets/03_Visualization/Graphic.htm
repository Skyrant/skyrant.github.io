Extern - Grafik

Allgemein

Das Boardlet Grafik kann verwendet werden, um ein Bild per URL oder per DB-Abfrage (Base 64 codiert) zu laden.

Einstellungen

Die häufigsten Einstellungen auf den Reitern Einstellungen, Layout und Filter sind unter Boardlet-Einstellungen beschrieben:

  • Reiter Einstellungen

  • Reiter Layout

  • Reiter Filter

Für das Boardlet Grafik stehen außerdem folgende Einstellungen zur Verfügung.

Reiter Einstellungen

Quelle:

Dropdown-Liste zur Auswahl der Quelle des Bildes (URL oder DB).

Verbindung:

Nur bei Quelle DB verfügbar. Angabe der Datenbank-Verbindung, z.B. legato.

Modell:

Nur bei Quelle DB verfügbar. Modellname der DB-Tabelle, die verwendet werden soll, z.B. sys-unit.

Abgefragte Spalten:

Nur bei Quelle DB verfügbar. Eingabe einer oder mehrerer Spalten, die abgefragt werden sollen (meist ID und die Spalte, die das Bild enthält).

Rückgabespalte:

Nur bei Quelle DB verfügbar. Eingabe der Spalte, deren Wert zurückgegeben werden soll (also die, die das Bild enthält)

Reiter Filter

URL:

Eingabe der URL, von der das Bild geladen werden soll. Nur verfügbar, wenn unter Settings URL als Quelle selektiert wurde.

Detailinformation:

Dieses Feld wird automatisch mit allen Werten einer sendenden Generic Table befüllt.

Beispiel:

{"name":"Milliliter","unit_symbol":"ml"}

Es werden hier also in der Generischen Tabelle die Spalten Name und Symbol angezeigt. Dahinter findet sich der Wert des gerade selektierten Eintrags.

Feld der Detailinformation:

Soll das Boardlet Empfänger und nicht Sender sein, kann hier festgelegt werden, mit welchem Wert ein im Filter verwendeter Platzhalter $detailinfo$ aus der sendenden Tabelle ersetzt werden soll, z.B. name. Der Parametertyp sollte hier immer auf local gesetzt sein.

Für weitere Infos zu Filterstrings, siehe bitte Filterstrings.

Für Details zur Bedeutung der einzelnen Platzhalter, siehe bitte Platzhalter.

Filtern nach ...

Nur verfügbar, wenn als Quelle DB selektiert wurde.

Möglichkeit, die Ausgabe mittels eines Filterstrings mit Parametern zu filtern. Es können die Platzhalter $node$, $detailinfo$, $timeStart$ und $timeEnd$ verwendet werden.

Für Details zur Formatierung siehe bitte Filterstrings.

Für Details zur Bedeutung der einzelnen Platzhalter, siehe bitte Platzhalter.

Benötigte Parametrierung im LC2

Für dieses Boardlet müssen keine Daten im LC2 konfiguriert werden.
