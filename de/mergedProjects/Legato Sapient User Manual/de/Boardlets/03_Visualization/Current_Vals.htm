Monitoring - Aktuelle Werte

Allgemein

Das Boardlet Aktuelle Werte zeigt zu einem oder mehreren selektierten Knoten alle aktuellen Werte von Variablen, die mit "Anzeigen" = 1 parametriert sind und die keine Teilwerte sind (Parent = NULL). Die Anzeige der Teilwerte kann über den Filter umgeschaltet werden. Die Anzeige kann nach Variablenart gefiltert werden. Wurde für die Variable eine Einheit parametriert, so wird diese hinter dem Namen der Variablen in eckigen Klammern angezeigt. Wurden mehrere Knoten selektiert, so wird der Knotenname vor dem Variablennamen angezeigt.

Klick auf einen Wert öffnet eine Detailansicht mit weiteren Informationen zu dieser Variablen, z.B. die hinterlegten Warn- und Alarmgrenzen, den hinterlegten Sollwert, die letzte Änderung.

Die Farben können angepasst werden, indem im LC2 pro Variablenart oder pro Variable eine CSS-Farbe zugewiesen wird und im Boardlet Layout der Schieberegler "Konfigurierte Farben der Variablen benutzen" aktiviert ist. Außerdem kann im Boardlet Layout auch ein direktes Mapping von Variablenart zu CSS-Farbe vorgenommen werden.

 

Ist eine Zeile selektiert, wird im Menü Quick-Edit [] ein Drilldown auf die Teilwerte dieses Werts angeboten. Dies ist nur für eine einzelne ausgewählte Zeile möglich; Multi-Select wird nicht unterstützt.

Grenzwertüberwachung

Ist die Grenzwertüberwachung aktiv, so werden Werte mit Grenzwert-Abweichungen folgendermaßen dargestellt:

Normal:

Groß:

Beispiel: Für 3 Variablen sind Grenzwerte festgelegt.

  • Die Variable "Ist 1" hat als Warngrenzen 4 und 65, als Alarmgrenzen 1 und 70. Der aktuelle Wert von 50 ist also innerhalb aller parametrierten Grenzwerte. Daher wird die Zeile grün eingefärbt, wenn keine konfigurierten Farbwerte verwendet werden. Werden die konfigurierten Farben für die Variablen verwendet, entfällt eine Markierung.

  • Die Variable "Ist 2" hat als Warngrenzen 7 und 85, als Alarmgrenzen 5 und 95. Der aktuelle Wert von 3 ist also unterhalb der Alarmgrenze. Deshalb wird die Zeile rot eingefärbt, wenn keine konfigurierten Farbwerte verwendet werden. Werden die konfigurierten Farben für die Variablen verwendet, dann wird nur das Symbol für die Unterschreitung ergänzt.

  • Die Variable "Gateway Test Num" hat als obere Warngrenze den Wert 8. Der aktuelle Wert von 100 ist also über dem Grenzwert. Deshalb ist die Zeile gelb eingefärbt, wenn keine Farbwerte verwendet werden. Werden die konfigurierten Farben für die Variablen verwendet, dann wird nur das Symbol für die Überschreitung ergänzt.

Die Grenzwerte können im LC2 für jede Variable festgelegt werden.

Werte ändern

Wenn für eine Variable im LC2 die Einstellung "Änderbar" auf 1 gesetzt ist, so kann ihr Wert online geändert werden. Dazu muss diese Funktion in den Layout-Settings des Boardlets aktiviert werden und der angemeldete Benutzer braucht das entsprechende Recht.

Dann wird vor der Variable ein Stift-Symbol angezeigt. Klick auf das Stift-Symbol öffnet einen Dialog, in dem der Wert geändert werden.

:

Bei Kommazahlen ist das Dezimaltrennzeichen der gewählten Sprache zu verwenden, also z.B. bei DE das Komma, bei EN der Punkt.

Variable hinzufügen / Variablendefinition ändern

Es ist auch möglich, eine neue Variable anzulegen oder die Variablendefinition zu ändern.

Zum Anlegen, in der Menüleiste des Boardlets klicken, zum Ändern die gewünschte Variable in der Ansicht selektieren und klicken.

Es öffnet sich folgender Dialog: --->>

:

Name und Beschreibung werden immer in alle Sprachfelder gespeichert.

Die Auswahl eines Intervalls ist nur für das periodenbezogene Wertbuchen relevant.

 

Variable löschen

Durch Klick auf können Variablen gelöscht werden.

Es öffnet sich folgender Dialog in der Seitenleiste, über den die zu löschenden Variablen selektiert werden können: --->>

Durch Klick auf Anwenden (bzw. Submit) werden die rot markierten Variablen gelöscht.

 

Einstellungen

Die häufigsten Einstellungen auf den Reitern Einstellungen, Layout und Filter sind unter Boardlet-Einstellungen beschrieben:

  • Reiter Layout

Für das Boardlet Aktuelle Werte stehen außerdem folgende Einstellungen zur Verfügung.

Reiter Layout

Link zur Historie anzeigen:

Ist dieser Schieberegler aktiviert, wird bei archivierten Werten (Variablen mit Archiv = 1) ein Button angezeigt. Durch Klick auf diesen Button kann die Historie des Wertes in einem Balkendiagramm angezeigt werden. Dazu muss zum einen ein Zeitraum selektiert werden und zum anderen beim Balkendiagramm das Filterkriterium "Variablen ID" auf den Parametertyp "In" gesetzt werden.

Link zur Statistik anzeigen:

Ist dieser Schieberegler aktiviert, wird bei archivierten Werten (Variablen mit Archiv = 1) ein Button angezeigt. Durch Klick auf diesen Button kann die Historie des Wertes in einem Histogramm angezeigt werden. Dazu muss zum einen ein Zeitraum selektiert werden und zum anderen beim Balkendiagramm das Filterkriterium "Variablen ID" auf den Parametertyp "In" gesetzt werden.

Historie in Seitenleiste öffnen:

Alternativ kann die Historie oder die Statistik des Wertes in der Seitenleiste angezeigt werden, wenn dieser Schieberegler zusätzlich aktiviert ist.

Grenzwertüberwachung:

Drop-Downliste zur Auswahl der Darstellung der Grenzwerte. Deaktiviert, Normal oder Groß.

Messwerte änderbar:

Ist dieser Schieberegler aktiviert, so kann der Wert von Messwerten geändert werden, wenn bei der Konfiguration das Flag Änderbar = 1 gesetzt wurde und der angemeldete Benutzer das entsprechende Recht hat. Dazu wird ein Stiftsymbol vor der Variable angezeigt.

Konfigurierte Farben der Variablen benutzen:

Ist dieser Schieberegler aktiviert, werden die pro Variable im LC2 definierten Farben verwendet. Wurde keine Farbe für eine Variable definiert, wird die Farbe der Variablenart verwendet. Ist der Schieberegler nicht aktiv, werden keine Farben angezeigt.

Variablenbezeichnung:

Über eine Dropdown-Liste kann ausgewählt werden, ob der Name der Variablen angezeigt werden sollen oder deren Beschreibung.

Reiter Filter

Variablen-ID:

ID der Variable, deren Historie angezeigt wird. Default ist -1 = keine Auswahl. Der Parametertyp muss auf "Out" oder "InOut" stehen, um den Wert an ein Balkendiagramm Boardlet oder die Seitenleiste weiterzugeben.

Variablenarten:

Filter auf Variablenarten. Für alle Variablen der hier definierten Variablenarten, die dem oben selektierten Knoten zugeordnet sind, werden die Werte im Diagramm angezeigt. Die Variablenarten können aus einer Dropdown-Liste ausgewählt werden. Leer = kein Filter, alle Variablenarten werden angezeigt.

Teilwerte:

Über diesen Schieberegler wird definiert, ob Parent-Werte oder Teilwerte visualisiert werden sollen.

Benötigte Parametrierung im LC2

Die Variablenarten werden im LC2 unter Grundkonfiguration Allgemeine Anwendungen Variablenarten parametriert.

Die Variablen werden im LC2 unter Anwendungsstruktur Datenaustausch [Modul] [Verbindung] Variablen gepflegt. Hier wird auch die Einheit festgelegt.

Damit für Variablen die Historie aufgerufen kann, muss die Spalte "Archiv" auf 1 gesetzt sein.

Soll die konfigurierte Variablenfarbe verwendet werden, muss in der Spalte "Farbklasse" ein Eintrag vorhanden sein.

Unter Anwendungsstruktur Fertigungsstruktur [Knoten] Variablenzuordnungen werden die Variablen dem Knoten zugeordnet, für den sie angezeigt werden sollen. Hier muss das Flag "Anzeigen" auf 1 gesetzt sein, damit die Variable angezeigt wird. Die Sortiernummer legt fest, in welcher Reihenfolge die Variablen angezeigt werden.

Damit Variablen änderbar sind, muss die Spalte "Änderbar" auf 1 gesetzt sein.
