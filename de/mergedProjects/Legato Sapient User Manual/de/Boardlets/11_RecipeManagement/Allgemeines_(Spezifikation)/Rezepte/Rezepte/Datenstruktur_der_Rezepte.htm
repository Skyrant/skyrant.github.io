Rezeptstruktur

Folgende Daten werden je Rezept in Legato Sapient gespeichert:

  • Rezeptname

  • Beschreibung

  • Bezug des Fertigungsknotens

  • Masterflag
    Pro Rezept gibt es ein sog. Masterflag. Solang das Masterflag nicht gesetzt ist, werden die Parameterwerte aus der Steuerung in Sapient übernommen.
    Mit gesetztem Masterflag werden keine Änderungen aus der Steuerung mehr übernommen, Legato Sapient nimmt für dieses Rezept die Masterrolle ein.

  • Letzte erfolgreiche Übertragung

  • Verweis auf das Änderungsjournal für dieses Rezept

  • Auftrag zur Übertragung

  • Parameterstruktur

 

Rezeptstruktur bearbeiten

  • Es muss für jedes Rezept eine vorgegebene Struktur existieren.

  • Diese kann für jedes Rezept im LC2 angelegt und bearbeitet werden.

  • Für die definierten Parametertypen sind vordefinierte Felder zum Aufbau der Rezeptstruktur vorhanden, so dass eine Rezeptstruktur schnell und leicht erstellt werden kann. Hierfür dienen die Kopierfunktionen.

Beachten Sie weitere Informationen zum Bearbeiten der Rezeptstruktur im Kapitel Rezeptvorlagen.
