Rezeptliste

Allgemein

Für ausgewählte Knoten kann eine Rezeptliste abgerufen werden.

Die Rezeptliste ist eine Übersicht über alle konfigurierten Rezepte in den Anlagen (gefiltert nach Knoten). Wenn Sie einen Knoten auswählen, können Sie alle Rezepte sehen, die mit diesem Knoten verknüpft sind (ein Rezept ist mit einer SPS-Verbindung verbunden; der betreffende Knoten ist der "Standardknoten" dieser Verbindung). Außerdem können Sie auch die Rezepte sehen, die mit einem Elternteil dieses Knotens verbunden sind.

Die Rezepte sind Zeilen aus der Tabelle RM_RECIPES (siehe Datenmodell für weitere Informationen zur Tabelle RM_RECIPES).

Das Rezeptliste-Boardlet (Sprache: Englisch) sieht wie folgt aus:

 

 

Spalte

Beschreibung

(1)

name

Rezeptname (in der gewählten Sprache)

(2)

symbol

Rezeptsymbol (in der SPS verwendet)

(3)

connection

SPS-Verbindung

(4)

last_upload

Letzter Upload (Zeitstempel)

(5)

upload_status

Upload-Status (OK, / Upload läuft gerade / NOK [Fehlernummer])

(6)

last_download

Letzter Download (Zeitstempel, Fehler oder Download läuft gerade)

(7)

download_status

Download-Status (OK, / Download läuft gerade / NOK [Fehlernummer])

(8)

last_change

Letzte Änderung (Zeitstempel der letzten Änderung eines Wertes)

(9)

mode

• Upload von der SPS zum MES

• Download vom MES zur SPS

Modus (der Modus eines Rezepts kann automatisch von SCC oder manuell eingestellt werden)

 

(10)

plc_match

Zeigt an, ob die Rezeptwerte in Legato mit den Werten in der SPS übereinstimmen.

 

Über die Schaltfläche ... im Rezeptlisten-Boardlet können Sie die angezeigten Elemente der Liste ein- und einblenden. Klicken Sie hierzu im geöffneten Menü auf Sichtbare Spalten.
