Rezeptverlauf

Allgemein

Im Boardlet Rezeptverlauf werden Änderungen an Rezepten sichtbar. Durch die Rezept-ID können Änderungen eindeutig identifiziert werden.

Mittels Drag & Drop lassen sich die Breite oder Position der Spalten einfach verändern. Durch Klick auf eine Zeilenüberschrift wird nach dieser Spalte sortiert.
Nach welcher Spalte sortiert ist, und ob auf- oder absteigend sortiert ist, wird durch die dreieckigen Symbole angezeigt:

 
Einstellungen

Die häufigsten Einstellungen auf den Reitern Einstellungen, Layout und Filter sind unter Boardlet-Einstellungen beschrieben:

  • Reiter Einstellungen

  • Reiter Layout

  • Reiter Filter

 

 

Das Rezeptverlauf-Boardlet sieht wie folgt aus:

 

 

(1) Eindeutige Rezept-ID
(2) Parent bezeichnet den Ordner, in den das Rezept eingefügt ist.
(3)

Rezeptname

 

Hier ist der Name des Rezeptes aufgelistet

 
(4)

Benutzername des Benutzer, der die letzte Änderung durchgeführt hat

 

Hier ist festgehalten, welcher Benutzer Änderungen am Rezept vorgenommen hat.

Wenn der Benutzer anonymous angezeigt wird, ist der Benutzer deaktiviert und auch anonymisiert. Siehe Benutzerverwaltung für mehr Informationen zum Thema Anonymisierung von Benutzern.

 
(5)

Letzte Änderung (Zeitstempel der letzten Änderung eines Wertes)

 Wenn eine Änderung an einem Rezept vorgenommen wird, wird eine neue Version des Rezeptes angelegt

 
(6)

Kommentar zur letzte Änderung

Hier ist ein Kommentar für die jeweilige Änderung angegeben.

Für das Boardlet Rezeptverlauf stehen weitere Einstellungen zur Verfügung:

Reiter Filter

Der Reiter Filter betrifft an dieser Stelle nur das Boardlet Rezeptverlauf.

Selektierte Knoten:

Ausgewählter Knoten.

Verändert von:

Name der User, der eine Änderung durchgeführt hat.

Kommentar ändern:
Änderungskommentar (siehe Kontexthilfe).

Name:

Setzen eines Suchfilters nur nach Namen (siehe Kontexthilfe).

Historie:

Rezept-ID im Verlauf.

Rezept-ID:

Die gleiche ID wie im LC2.

 

Benötigte Parametrierung im LC2

Wie bei anderen Boardlets, gibt es auch beim Rezeptverlauf-Boardlet notwendige Parameter.

Die Berechtigungen für die Funktion Upload/Download kann im LC2 konfiguriert werden unter System-Administration Verbindungen <verbindung> Spalte Objektname Reihe rm_recipes.
