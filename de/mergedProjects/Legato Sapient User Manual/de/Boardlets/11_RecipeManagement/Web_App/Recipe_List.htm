Rezepte

Allgemein

Das Boardlet Rezepte bietet einen nach Knoten gefilterten Überblick über alle konfigurierten Rezepten in den Systemen. Wenn Sie einen Knoten wählen, dann können Sie alle Rezepte sehen, die mit diesem Knoten verknüpft sind. Außerdem sehen Sie die Rezepte, die mit dem Parent dieses Knotens verknüpft sind. Jede Reihe steht für ein Rezept. Ein Rezept ist mit einer SPS-Verbindung verbunden; der relevante Knoten ist der "Standardknoten" dieser Verbindung.

Mithilfe des Boardlets Rezepte können Sie folgende Aktionen ausführen:

  • Den Modus eines Rezepts ändern.

  • Ein Rezept von der SPS in die Datenbank hochladen.

  • Ein Rezept von der Datenbank in die SPS herunterladen.

Spalten ein- und ausblenden

Sie können festlegen, welche Spalten Sie im Boardlet Rezepte anzeigen wollen.

So blenden Sie Spalten ein und aus:

1

Wählen Sie im Menü Quick-Edit [] > Sichtbare Spalten.

2

Aktivieren Sie den Schieberegler neben den Spaltennamen, die Sie im Boardlet anzeigen wollen.

 

Beachten Sie auch die Kapitel Menü Profile und Reiter Tabelleneinstellungen.

 

 

Modus wechseln

Ist ein Rezept im Boardlet Rezepte ausgewählt, erscheinen zusätzliche Symbole in der Boardlet-Menüleiste.

Durch Klick auf in der Boardlet-Menüleiste kann der Modus eines Rezepts geändert werden. Es erscheint der Dialog Modus wechseln in der Seitenleiste: ----->>

Upload PLC zu MES:

Diese Einstellung weist den SCC an, das Rezept aus der SPS zu holen und in der Datenbank zu speichern. Verwenden Sie diese Einstellung, wenn Sie das Rezept im Boardlet Rezeptdetails ändern wollen.

Download von MES zu PLC:

Diese Einstellung weist den SCC an, das geänderte Rezept von der Datenbank zur SPS (engl. PLC) herunterzuladen.

Keine Daten von/nach PLC:

Diese Einstellung wird durch das System nach einem erfolgreichen Download vorgenommen.

inaktiv:

LC2 blendet das Rezept auf der Benutzeroberfläche der Legato Sapient WebApp aus.

 

 

Start Upload / Start Download

Ist ein Rezept im Boardlet Rezepte ausgewählt, erscheinen zusätzliche Symbole in der Boardlet-Menüleiste.

  • Durch Klick auf können Sie einen Upload des ausgewählen Rezepts von der SPS zur Datenbank starten (Modus: Upload PLC zu MES).

  • Durch Klick auf können Sie einen Download des ausgewählten Rezepts von der Datenbank zur SPS starten (Modus: Download von MES zu PLC).

Reiter Filter

Der Reiter Filter betrifft an dieser Stelle nur das Boardlet Rezepte.

 

Download status:

Filter auf die Download-Statusarten: OK, Download is currently running, NOK <meldungstext>

Upload status:

Filter auf die Upload-Statusarten: OK, Upload is currently running, NOK <meldungstext>

Modus:

Filter auf den Modus eines Rezepts. Der Modus kann automatisch durch den SCC oder manuell eingestellt sein.

Screenshot der Dropdown-Liste Modus

Name:

Filter auf den Rezeptnamen in der gewählten Sprache.

 

 

Benötigte Parametrierung im LC2

Die Berechtigungen für die Funktion Upload/Download kann im LC2 konfiguriert werden unter System-Administration Verbindungen <verbindung> Spalte Objektname Reihe rm_recipes.
