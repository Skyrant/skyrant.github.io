Aktuelles

Allgemein

Das Boardlet Aktuelles kann verwendet werden, um aktuelle Informationen / Nachrichten anzuzeigen. Diese Nachrichten können Text und / oder Bilder enthalten. Sie können auch mit Knotenbezug gespeichert und dann entsprechend ausgewertet werden. Einträge, die in der Zukunft liegen, werden zuoberst angezeigt und sind aufsteigend sortiert. Einträge, die in der Vergangenheit liegen, sind absteigend sortiert.

 

News hinzufügen

Für das Boardlet Aktuelles gibt es im Menü Quick-Edit [] die Möglichkeit, neue Einträge hinzuzufügen:

Es kann ein Titel und ein Text eingegeben werden. Durch Klick auf Dateien auswählen... wird der Dateibrowser geöffnet, über den ein Bild ausgewählt werden kann. Diese Bilder werden in folgenden Ordner geladen: SapiEng/web/app/assets/images/upload/

Ist der Filter "Knoten-ID" aktiv, wird der neue Eintrag automatisch dem gerade selektierten Knoten zugeordnet.

 

Toggle Edit Mode

Für das Boardlet Aktuelles gibt es im Menü Quick-Edit [] die Möglichkeit, die Ansicht in den Edit-Modus zu setzen, um bestehende Einträge zu löschen. Es erscheint dann vor jedem News-Element ein rotes Mülleimer-Symbol. Durch Klick darauf wird das entsprechende Element gelöscht.

Einstellungen

Die häufigsten Einstellungen auf den Reitern Einstellungen, Layout und Filter sind unter Boardlet-Einstellungen beschrieben:

  • Reiter Einstellungen

  • Reiter Layout

  • Reiter Filter

Für das Boardlet Aktuelles stehen außerdem folgende Einstellungen zur Verfügung.

Reiter Layout

Anzahl News:

Anzahl der maximal angezeigten Nachrichten.

Reiter Filter

Knoten-ID:

Ist dieser Schieberegler aktiv, so werden die Nachrichten nach Knoten-ID gefiltert. Nachrichten, die keinem Knoten zugeordnet sind, werden in diesem Fall gar nicht angezeigt.

News von übergeordneten Knoten anzeigen:

Ist dieser Schieberegler zusätzlich zur Knoten-ID aktiv, werden nicht nur die News für den selektierten Knoten, sondern auch die seiner übergeordneten Knoten angezeigt.

Benötigte Parametrierung im LC2

Nachrichten werden im LC2 unter Frontend News gepflegt. Hier können Nachrichten auch einem Knoten zugeordnet werden.
