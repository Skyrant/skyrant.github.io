Schichtkalender Übersicht

Allgemein

Im Boardlet Schichtkalender Übersicht werden je nach Einstellung für alle selektierten Knoten oder für alle selektierten Zeitschemata die geplanten Schichtmodelle angezeigt. Haben untergeordnete Knoten ein abweichendes Schichtmodell zugewiesen, wird dies beim entsprechenden Knoten durch ein indiziert.

Der selektierte Zeitraum wird blau umrandet. Das Datum des aktuellen Tages wird blau hinterlegt.

Durch Klick auf einen Tag werden alle selektierten Knoten im Schichtplan angezeigt. Dies funktioniert nur, wenn Schichtmodelle per Knoten angezeigt werden. Werden Schichtmodelle per Zeitschema angezeigt, funktioniert dies nicht.

Werden zusätzlich zur Schichtkalender-Übersicht die Schichtkalender Boardlets verwendet, so kann durch Klick auf eine Checkbox der jeweilige Tag als Zieldatum für die Schichtplanung eingetragen werden. Durch Klick auf die Checkboxen vor oder über dem Plan kann die ganze Reihe selektiert werden.

Reiter Einstellungen

Zeitschema verwenden (anstelle von Knoten):

Über diesen Schieberegler wird definiert, ob statt Knoten die Schichtmodelle für die selektierten Zeitschemata im Plan angezeigt werden sollen.

Benötigte Parametrierung im LC2

  • Schichttypen, Pausentypen und Feiertage werden im LC2 unter Werksverwaltung Arbeitszeit-Einstellungen Grundeinstellungen parametriert.

  • Zeitschemata werden im LC2 unter Werksverwaltung Arbeitszeit-Einstellungen parametriert.

  • Schichtmodelle werden unter dem jeweiligen Zeitschema zugewiesen.

  • Unter Anwendungsstruktur Fertigungsstruktur Fertigungsknoten wird dem Knoten ein Zeitschema zugewiesen (Spalte "Zeitschema").
