Verfügbarkeit (kompakt)

Allgemein

Das Boardlet Verfügbarkeit (kompakt) zeigt zu einem selektierten Knoten die Online-KPIs für die Verfügbarkeit (AVORG, AVTEC, AVSYS) als Balken an. Als zusätzliche Information werden rechts die jeweiligen Stördauern (FTORG, FTTEC, FTSYS und FTTOT) angezeigt.

Einstellungen

Die häufigsten Einstellungen auf den Reitern Einstellungen, Layout und Filter sind unter Boardlet-Einstellungen beschrieben:

  • Reiter Einstellungen

  • Reiter Layout

  • Reiter Filter

Für das Boardlet Verfügbarkeit (kompakt) stehen keine weiteren Einstellungen zur Verfügung.

Benötigte Parametrierung im LC2

Knoten mit aktivem Zeitschema (Produktion), Schichtkalender muss in Sapient gepflegt sein.

KPIs werden dem Knoten im LC2 unter Anwendungsstruktur Fertigungsstruktur [Knoten] KPIs zugewiesen.

Die folgenden SE-Jobs müssen laufen:

  • JOB_POLLER (allgemeiner Job)

  • HANDLE_EVENTS (zum Berechnen der Schichten)

  • REFRESH_EVENT_LIST (zum Nachberechnen von Änderungen an den Schichten)

  • TIMESLICES (für Online-KPIs)

 

Auf dem Knoten müssen außerdem verfügbarkeitsrelevante Meldungen gebucht werden (von einer Meldeklasse mit KPI-relevant = 1 und einer der Meldearten organisatorisch, technisch oder systembedingt).
