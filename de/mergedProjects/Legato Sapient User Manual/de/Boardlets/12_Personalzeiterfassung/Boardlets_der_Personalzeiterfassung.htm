Boardlets der Personalzeiterfassung

Allgemeine Informationen

Die Personalzeiterfassung (PZE) ermöglicht die Erfassung und Auswertung von Arbeitsabläufen und Arbeitsaktivitäten und umfasst in Legato Sapient mehrere Boardlets:

  • Aktuelles Profil
  • Arbeitsplatzkennzahlen
  • Bereich --> Mitarbeiter
  • Gruppe --> Mitarbeiter
  • Gruppenkennzahlen
  • Mitarbeiter
  • Mitarbeiterkennzahlen
  • Profilvorlagen
  • Time Recording Log
  • Zeitaufzeichnungen

 

Wählen Sie die einzelnen Boardlets der PZE aus der Legato Sapient Boardlet Bibliothek aus.

 

In den einzelnen Boardlets können Sie im Menü Quick-Edit [...] unterschiedliche Spalten ein- und ausblenden (Sichtbare Spalten):

 

Achten Sie bei der Verwendung der PZE-Boardlets immer darauf über die Filter die richtigen Zeitpunkte auszuwählen. Nur so können die gewünschten Daten abgerufen werden.

Die Anzeige der Boardlets wird regelmäßig aktualisiert. Sie können das Aktualisierungsintervall ("Refresh interval") für jedes Boardlet, das die Aktualisierung unterstützt, einstellen. Das Aktualisierungsintervall kann eingestellt werden, indem Sie es auf der Registerkarte "Datenquelle" im Bearbeitungsmodus aktivieren.

Als Standard des Aktualisierungsintervalls sind 60 Sekunden eingestellt. Durch Deaktivieren der Checkbox Aktualisieren ("Refresh") im rechten, oberen Bereich der Ansicht kann die Aktualisierung zeitweise ausgeschaltet werden. Zusätzlich werden die verbleibenden Sekunden bis zur nächsten Aktualisierung angezeigt. Eine sofortige Aktualisierung kann durch zweimaliges Anklicken der Checkbox erreicht werden.

 

Dashboard Parameter

Sie können im Dashboard, in dem Sie die PZE-Boardlets verwenden, vorab sehen, welche Filter es für das PZE gibt. Bitte verwenden Sie jedoch die einzelnen Filter der Dashboards, wenn Sie Daten filtern möchten. Die Darstellung der Dashboard Filter dient lediglich einer Aufsummierung der Filter.

Boardlet Filter

Neben den Dashboard Parametern können auch durch verschiedene Boardlet Filter Ergebnisse gefiltert werden. Sie können die Boardlet Filter über die Boardlet-Menüleiste im Menü Quick Edit [...] unter Filter aufrufen.

Gruppenkennzahlen

Das Boardlet Gruppenkennzahlen zeigt die vorhandenen Mitarbeitergruppen (PRE_EMP_GROUPS [vom Typ "Organisationsgruppe"]), die der ausgewählten Gruppe zugeordneten Bereiche sowie die dem ausgewählten Bereich zugeordneten, eingebuchten Mitarbeiter (PRE_WORKAREAS) an. Für jede ausgewählte Gruppe und jeden ausgewählten Arbeitsbereich werden anschließend in der unteren Hälfte die entsprechenden Variablen PRE_MES_VALUE_RELS angezeigt.

Die Anzeige im Gruppenkennzahlen-Boardlet ist teilweise knotenabhängig, d.h. die Anzeige des Arbeitsbereichs und der damit verknüpften Mitarbeiter ist abhängig vom gewählten Knoten. Wenn keine Arbeitsbereiche angezeigt werden, gibt es für den aktuellen Knoten und das aktive Profil in der aktuellen Schicht keine Belegung.

Profilvorlagen

Das Boardlet Profilvorlagen stellt Arbeitsprofile dar, die für kurzfristige Änderungen der üblichen Arbeitsverteilung erstellt werden. Das Boardlet zeigt die im LC2 angelegten Arbeitsprofilvorlagen.

Beachten Sie folgende Informationen:

  • Das Boardlet zeigt Namen und Beschreibungen von Arbeitsbereichen, Entitäten, Symbolen an.

  • Die anzuzeigende Profilvorlage wird in einer Drop−Down−Liste ausgewählt und danach in einer Baumdarstellung angezeigt. Ein Profil kann in der Auswahlliste ausgewählt werden und wird dann in einer hierarchischen Struktur angezeigt, beginnend mit Arbeitsbereichen, Arbeitsplätzen und schließlich Ausrüstungen und Stationen auf der gleichen Ebene. Die Anzeige der aktuell gültigen (für die aktuelle und nächste Schicht) Profile ist knotenabhängig, d. h. dass ausgehend vom aktuellen Knoten ein Profil in der Knotenstruktur aufwärts gesucht wird, bis ein Profil gefunden wird. Dieses Profil wird dann angezeigt. Diese Anzeige der Profilvorlagen ist knotenunabhängig, d.h. der aktuell selektierte Knoten hat keinen Einfluss auf die Darstellung. Die Anzeige des Arbeitsprofils ist jedoch knotenabhängig.

  • Eine Entität kann Workarea, Workplace oder Equipment sein.

 

 

Spalte

Beschreibung

(1)

 Name Name der Profilvorlage
Zu finden im LC2 unter "Arbeitsbereiche".

(2)

(3)

Zu finden im LC2 unter "Arbeitsbereiche".

(4)

Zeitaufzeichnungen

Das Boardlet Zeitaufzeichnungen ermöglich die Anzeige und das Ändern der eingebuchten Vorgänge der PZE.

 

Die Änderungsfunktion versteht sich als "Administratorfunktion" und sollte nur in Ausnahmefällen benutzt werden.
