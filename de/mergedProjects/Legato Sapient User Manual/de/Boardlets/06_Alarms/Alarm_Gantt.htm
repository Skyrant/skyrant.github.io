Meldungs-Gantt

Allgemein

Das Boardlet Meldungs-Gantt zeigt zu einem selektierten Knoten den Meldungsverlauf an. Die Störmeldungen werden in Form eines Gantt-Diagramms dargestellt. Rechts davon werden Detailinformationen zu jeder Meldung eingeblendet. Sind im selektierten Zeitraum mehr Meldungen aufgetreten, als angezeigt werden können, können über die Buttons unten andere Meldungen angezeigt werden. Initial werden immer die aktuellsten Daten geladen.

Die Balken sind je nach Meldeklasse anders eingefärbt.

Zum Ändern der Meldeklasse von selektierten Meldungen muss zusätzlich das Meldungs-Management Shortcut Boardlet eingeblendet werden.

Für das Boardlet Meldungs-Gantt stehen außerdem folgende Einstellungen zur Verfügung.

Info immer anzeigen:

Über diesen Schieberegler wird definiert, ob die Detailinfo zu den Meldungen standardmäßig ein- oder ausgeblendet ist.

Reiter Filter

Minimale Dauer:

Wird hier ein Wert eingegeben, werden nur Meldungen mit einer minimalen Dauern der hier festgelegten Sekunden angezeigt.

Meldeklassen:

Filter auf Meldeklassen. Es werden alle Meldungen der hier eingegebenen Klassen für die selektierten Knoten angezeigt. Die Meldeklassen können aus einer Dropdown-Liste gewählt werden. Leer = kein Filter, alle Meldeklassen werden angezeigt.

Meldearten:

Filter auf Meldearten. Es werden alle Meldungen der hier eingegebenen Arten für die selektierten Knoten angezeigt. Die Meldearten können aus einer Dropdown-Liste gewählt werden. Leer = kein Filter, alle Meldearten werden angezeigt.

Meldungstext:

Hier kann ein Meldungstext eingegeben werden, auf den gefiltert werden soll. Als Platzhalter können % für beliebig viele Zeichen oder _ für genau ein Zeichen verwendet werden. Der Filter berücksichtigt Groß- und Kleinschreibung. Mehrere Filterkriterien können durch Komma verbunden werden (ODER-Filter).

Meldungsdefinition:

Hier können eine oder mehrere Meldungsdefinitionen eingegeben werden, auf die gefiltert werden soll.

Benötigte Parametrierung im LC2

Die Meldeklassen werden im LC2 unter Werksverwaltung Meldungsmanagement Meldeklassen gepflegt.

Die Meldearten werden im LC2 unter Werksverwaltung Meldungsmanagement Meldearten gepflegt.

Meldungsdefinitionen werden im LC2 an zwei Stellen gepflegt werden:

  • unter Anwendungsstruktur Datenaustausch [Modul] [Verbindung] Meldungsdefinition gepflegt

- oder -

  • unter Anwendungsstruktur Fertigungsstruktur [Knoten] Meldungszuordnung gepflegt.
