Detail-Info

Die Detail-Info bezeichnet Informationen, die durch Auswahl in einem oder mehreren Boardlets als zusätzliche Daten in der Seitenleiste angezeigt werden. Aufgerufen wird die Detail-Info durch die Drill-down-Funktion im Menü Quick-Edit [].

Drill-Down bedeutet, dass dieselben Einstellungen und Filter, die im Boardlet getroffen wurden, auf eine andere Ansicht angewendet werden und somit weitere Analysemöglichkeiten entstehen. Beispielsweise kann man von den archivierten Messwerten aus ins Meldungsarchiv springen, um herauszufinden, ob zu niedrige Stückzahlen mit bestimmten Meldungen zusammenhängen. Es werden dann beide Boardlets mit demselben Knoten- und Zeitfilter angezeigt.

Bei manchen Boardlets muss zunächst mindestens ein Eintrag selektiert werden, ehe der Drill-Down ausgeführt werden kann. Mehrfachauswahl ist ebenfalls möglich (via Strg+Klick). Bei Mehrfachauswahl werden die selektierten Daten alle in einer Tabelle dargestellt. Werden Daten im Drill-down-Fenster geöffnet und anschließend für eine weitere Auswahl ein Drill-Down ausgeführt, so werden die Daten untereinander in zwei separaten Tabellen dargestellt, um sie miteinander vergleichen zu können.

Aus den Tabellen in der Detail-Info können gegebenenfalls weitere Drill-Downs durchgeführt werden.

Durch Klick auf das Mülleimer-Icon über einer Tabelle wird diese Drill-down-Auswahl gelöscht. Klick auf "Alle löschen" ganz oben entfernt alle Drill-Downs aus der Detail-Info.

Klick auf "Abbrechen" schließt die Detail-Information, behält aber alle bereits getroffenen Auswahlen inklusive deren Filter bei. So kann beispielsweise zu einem anderen Dashboard gewechselt werden, um von dort zusätzliche Daten zum Vergleich in die Detail-Info zu laden.
