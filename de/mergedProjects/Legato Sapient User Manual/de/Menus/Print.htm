Menü Drucken

Allgemein
Voraussetzungen:

  • Damit die Funktion verfügbar ist, muss ein Druckserver installiert sein (siehe Installation - System-Installation - Druckserver).

  • Es werden nur Boardlets des Dashboards gedruckt, bei denen die Einstellung Drucken aktiv ist (siehe Boardlet Settings).

Im Menü Drucken finden sich die Einstellungen zum Drucken des Dashboards. In das Menü Drucken gelangen Sie mit einem Klick auf das Symbol Dashboard drucken .

Das Menü Drucken erscheint an der rechten Seite des Bildschirms in einer Seitenleiste. Die Seitenleiste ist in folgende Reiter unterteilt:

Druckeinstellungen
Druck-Zeitplan

 

Druckeinstellungen

Optimiert für:

Hier können Sie die Seitenausrichtung für den Druck wählen: Hochformat oder Querformat.

  • Hochformat: Die im Dashboard vorhandenen Boardlets werden untereinander ausgedruckt.

  • Querformat: Die im Dashboard vorhandenen Boardlets werden so ausgedruckt, wie sie hier angeordnet sind. Wichtig hierfür ist ein sinnvoller Eintrag im Systemparameter -70 Auflösung für Druckvorschau.

Kopfzeile anzeigen:

  • an: Filterzeit, Knoten und Dashboard-Name werden in der Kopfzeile angezeigt.

  • aus: Die Kopfzeile wird nicht angezeigt.

Dashboard drucken:

Öffnet einen Browser-spezifischen Druck-Dialog:

  • Bei Firefox der Drucker-Explorer zur Auswahl eines Druckers

  • Bei Edge die Druckvorschau
