Menü Favoriten

Das Menü Favoriten ist im vorgegebenen Menü Dashboards zu finden:

Vom Menü Dashboards aus können Dashboards schnell zum Menü Favoriten hinzugefügt oder daraus entfernt werden.

So fügen Sie ein Dashboard zum Menü Favoriten hinzu:

1

Bewegen Sie im Menü Dashboards die Maus über den gewünschten Dashboard-Namen.

Rechts des Dashboard-Namens wird ein Stern-Symbol eingeblendet.

Schwarz-weiß: Dieses Dashboard können Sie zum Menü Favoriten hinzufügen.

Weiß: Dieses Dashboard ist bereits im Menü Favoriten enthalten.

2

Klicken Sie auf das schwarz-weiße Stern-Symbol.

Das Symbol wird durch ein weißes Stern-Symbol ersetzt. Dies signalisiert, dass das betreffende Dashboard ins Menü Favoriten übernommen wurde.

So entfernen Sie ein Dashboard aus dem Menü Favoriten:

1

Wählen Sie im Menü Dashboards > Favoriten.

2

Klicken Sie auf das weiße Stern-Symbol neben dem gewünschten Dashboard.

Das Dashboard wird aus den Favoriten entfernt.

3

Schließen Sie das Menü Favoriten.
