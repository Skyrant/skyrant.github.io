Menü Dashboards

Allgemein

Über das Menü Dashboards kann das gewünschte Dashboard zur Anzeige ausgewählt werden. Es werden nur Dashboards angezeigt, für die der angemeldete Benutzer Anzeige- oder Bearbeiten-Rechte hat.

Im Menü Dashboards können die Dashboards in Ordner gruppiert werden. Die Ordner werden im Legato Configuration Center (LC2) angelegt und können beliebig ineinander verschachtelt werden. Auf der ersten Ebene werden die Dashboards dieser Ebene angezeigt sowie die Ordner. Durch Klick auf einen Ordner öffnet sich dessen Inhalt rechts davon. Weitere Unterordner sind durch entsprechende Symbole symbolisiert.

Der Name des Dashboards, das aktuell im Arbeitsbereich angezeigt wird, ist im Menü Dashboards blau hervorgehoben.

Parametrierung

Die Ordner werden im LC2 unter Frontend Dashboard-Ordner angelegt und die Hierarchie über Parent festgelegt. Leere Ordner (die also noch kein Dashboard enthalten), werden im Menü Dashboards nicht angezeigt. Sie können aber natürlich in den Dashboard-Einstellungen zugewiesen werden.

Jedes Dashboard kann einem Ordner zugeordnet sein.

Verwandte Themen:

      
  • Dashboard anlegen
  • Dashboard bearbeiten
  • Dashboards in Ordnern gruppieren
  • Menü Favoriten
