Überblick

Der Designer verwendet die gleiche Basis wie die kostenlose Diagramm-Software draw.io / diagrams.net (https://app.diagrams.net/). Das Menü “Hilfe” führt zu einer Vielzahl an Tutorials und Hilfeartikeln zur Verwendung des Designers.

Zwischen der in Legato Sapient integrierten Version und der Version im Web kann es einzelne Unterschiede geben, das grundlegende Konzept stimmt aber trotzdem überein.

Da der Designer intuitiv nutzbar ist, ist es in diesem Kontext nicht nötig, alle Anwendungen im Detail zu erläutern. Daher gibt es eine Beschränkung auf einige Hinweise und Tipps:

·         Auf der linken Seite erscheint eine Formenbibliothek. Klicken Sie auf ein Element, um es in der Zeichnung an einer standardmäßigen Stelle hinzuzufügen oder ziehen Sie es an eine spezielle Position in der Zeichnung.

·         Um mehrere Elemente auszuwählen, halten Sie die STRG-Taste gedrückt und klicken oder halten Sie die Elemente mit der Maus.

·         Durch einen Rechtsklick auf ein Element werden einige der meistgenutzten Aktionen angezeigt, wie „in den Vordergrund bringen“ etc.

·         Sie können Elemente gruppieren (nützlicher Shortcut: STRG + G).

·         Der Stil eines Elements (funktioniert auch bei mehrfacher Auswahl) kann über die rechte Stil-Sidebar geändert werden. Sie können Stile mit den jeweiligen Buttons in der Stil-Sidebar kopieren und einfügen.

·         Jedes Element kann einen Text anzeigen – um einen Text einzugeben, führen Sie einen Doppelklick auf ein Element aus oder halten Sie die F2-Taste gedrückt.

·         Über den Reiter Text in der Sidebar können die Texteigenschaften geändert werden.

·         Der Reiter Arrange in der Sidebar enthält einige Tools, um z. B. absolute Dimensionen oder Positionen von Elementen festzulegen.

·         Während des Positionierens oder Vergrößern/Verkleinern von Elementen werden die Dimensionen am Raster ausgerichtet.[LG1] 

·         Mithilfe der Pfeiltasten kann ein Element in kleinen Schritten bewegt werden (unabhängig vom Raster). Wenn die Pfeiltasten in Kombination mit der STRG-Taste gedrückt werden, wird die Größe in kleineren Abständen angepasst. Durch das zusätzliche Drücken der Shift-Taste wird die Rastergröße als Schrittgröße verwendet.

·         Wenn ein Element einer Gruppe ausgewählt wird, wird zuerst die ganze Gruppe ausgewählt. Durch Anklicken eines Elements einer ausgewählten Gruppe wird das innere Element ausgewählt.

·         Tastatur-Shortcuts sind nützlich, um den Designprozess zu beschleunigen. Die Menüpunkte zeigen den jeweiligen Tastatur-Shortcut an, z. B. STRG+Shift+L zur Anzeige des Ebenenfensters.

 

Dokumenteneigenschaften

Klicken Sie auf eine leere Stelle in der Zeichnung um die Sidebar „Dokumenteinstellungen“ anzuzeigen. Einige Einstellungen beeinflussen die Designdauer:

 

• Rasterweite und Farbe des Zeichenrasters

• Wichtig! Verbindungspfeile und Verbindungspunkte – wenn aktiviert, können Elemente einfach mithilfe der jeweiligen Klickbereiche, die für die ausgewählten Elemente angezeigt werden, verbunden werden. Diese Anwendung kann vor allem bei der Bearbeitung komplexer, detaillierter Zeichnungen stören, da die Verbindungspunkte oft versehentlich gedrückt werden könnten. Um dies zu vermeiden, schalten Sie einfach beide Einstellungen aus.

Die folgenden wichtigen Einstellungen beziehen sich auf die Laufzeit:

• Hintergrundfarbe, die während des Designs und der Laufzeit angezeigt wird: Wenn keine Farbe festgelegt ist, wird die Zeichnung als transparent dargestellt (die Hintergrundfarbe der Anwendung wird angezeigt).

• Sapient Größeneinstellung: die Zeichnung wird standardmäßig so groß wie möglich dargestellt (mit einem optionalen Maximalfaktor). Mit der Einstellung 100% wird die Zeichnung in ihrer Originalgröße angezeigt.

 

Built-in Shapes

Die Sidebar auf der linken Seite zeigt standardmäßig viele Standardformen, wie einfache Geometrieformen (Rechteck, Kreis, Linien, etc.), an. Indem man das Ende einer Linie an ein Element zieht, können Linien an Elemente geknüpft werden. Die Linie bleibt mit dem Element auch dann verbunden, wenn das Element bewegt wird.

Unten in der Formenbibliothek kann man über den Button Mehr Formen auf zusätzliche Formen zugreifen. Für unser Gebiet ist Prozesstechnik eine der interessantesten Kategorien:

Scratchpad

Formen und Gruppen können einfach mit den üblichen copy&paste Funktionen in eine Zeichnung kopiert werden (die Shortcuts STRG+C, STRG+V sind ebenfalls verfügbar). Der Notizblock oben in der Formen-Sidebar kann als eine Art erweitertes Clipboard verwendet werden:

• Verwenden Sie die Maus, um beliebige Elemente oder eine Gruppe in den Notizblock hinzuzufügen. Alternativ können Sie die Elemente / Gruppen auswählen und das Plus „+“ Symbol über dem Notizblock drücken.

• Sie können Notizblock-Elemente zu einer Zeichnung hinzufügen wie jede andere Standardform.

• Verwenden Sie den Bearbeiten Button über dem Notizblock um Elemente des Notizblocks zu löschen oder umzubenennen.

 

Individuelle Formen

Einfache, individuelle Formen können über das Menü Extras|Form erstellen erstellt werden. Eine detaillierte Einführung, wie eine Form definiert wird, findet man online: Komplexe, benutzerdefinierte Formen erstellen und bearbeiten

Um eine bereits hinzugefügte, benutzerdefinierte Form zu bearbeiten, wählen Sie die Form aus und verwenden Sie den Button Form bearbeiten.

 

Bilder

Bilder können prinzipiell einfach hinzugefügt werden, indem man sie vom Dateimanager in die Zeichnung zieht.

:

Wichtig! In diesem Fall wird jedes Bild komplett in dem Dokument gespeichert (auch wenn das gleiche Bild mehrere Male in einer Zeichnung verwendet wird)! Das führt zu unnötig großen Zeichnungsordnern!

Um dies zu vermeiden, laden Sie jedes Bild, das verwendet werden soll, in den Server hoch, indem Sie WinSCP verwenden. Am besten speichern Sie Bilder für eine Zeichnung einzeln im Verzeichnis Bild im Projektordner. Ein guter Speicherort für wiederverwendbare, gemeinsame Bilder ist der Ordner Detail-Layout, z. B. Detail-Layout/Bilder.

Einfügen eines der Bilder auf dem Server in die Zeichnung…

• Wählen Sie Bild ordnen/einfügen im Hauptmenü.

• Mit Aus Sapient öffnen kann jedes Bild, das auf dem Server gespeichert ist, geöffnet werden.

• Nachdem ein Bild ausgewählt wurde, wird eine URL kopiert, die mit dem Schlagtwort Daten anfängt.

• Um eine Bildreferenz in einer Zeichnung später zu ändern, wählen Sie das Bild aus und verwenden Sie die Funktion Bild bearbeiten in der Sidebar.

 

Ebenen

Ebenen stellen eine nützliche Funktion dar, um komplexe Zeichnungen zu organisieren. Um sie zu verwalten, wählen Sie Ansicht|Ebenen im Anwendungsmenü aus. Das Ebenenfenster ermöglicht es, Ebenen hinzuzufügen, zu entfernen, umzubenennen und zu sperren sowie die z Reihenfolge und Sichtbarkeit zu ändern. Wenn eine neue Form hinzugefügt wird, wird sie zur aktuellen Ebene hinzugefügt.

Tipp:

Um ein versehentliches Hinzufügen von Elementen zu Ebenen (z. B. Hintergrund) zu vermeiden, sperren Sie diese Ebene einfach.

Gruppen

Elemente können gruppiert werden, indem die einzelnen Elemente ausgewählt werden und der zugehörige Menüpunkt genutzt wird (verfügbar im Kontext und im Hauptmenü). Wenn die Gruppe oder Gruppenelemente nicht ausgewählt sind, wird durch den ersten Klick auf ein Element innerhalb der Gruppe zuerst die ganze Gruppe ausgewählt, durch den zweiten Klick das Element innerhalb der Gruppe. Beim Ändern der Größe einer Gruppe werden alle Gruppenelemente an die neue Größe angepasst. Durch Drücken der Shift-Taste während der Skalierung wird das Seitenverhältnis beibehalten.

Wenn ein neues Element zur Gruppe hinzugefügt werden soll, kann das Element an einer freien Stelle innerhalb der Gruppe platziert werden (das Platzieren auf einem anderen Element würde dieses Element durch das neue ersetzt werden). Wenn die Gruppengröße geändert werden soll, ohne dass das Element innerhalb der Gruppe skaliert wird, um z. B. mehr Platz für ein neues Element zu kreieren, halten Sie während der Skalierung die STRG-Taste gedrückt.

Das DOM-Fenster

Das DOM-Fenster ist ein nützliches Tool, um die Struktur einer komplexen Zeichnung zu erkunden. Es kann über das Menü Sapient|Dokumentstruktur geöffnet werden. Durch Auswählen eines Elements in der Zeichnung wird auch die Position des DOM-Fenster geändert und andersrum. Auf das Kontextmenü für Elemente kann auch durch einen Rechtsklick auf ein Element in der DOM zugegriffen werden.
