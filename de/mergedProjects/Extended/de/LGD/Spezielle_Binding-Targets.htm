Spezielle Binding-Targets

Die meisten Binding-Targets können einheitlich verwendet werden – ihnen wird einfach über die Werteigenschaft ein Wert zugeordnet. Die meisten Targets können leicht festgelegt werden, indem der Stil manuell über die Stil-Seitenleiste geändert wird (oder über den Designer, z. B. rotieren) und indem der jeweilige Zugang über den Edit Style Dialog gefunden wird. All diese Stile können 1:1 als Binding-Targets verwendet werden. Einige Eigenschaften (z. B. Text, Tooltip) sind nicht darin enthalten. Die folgende Referenz listet diese speziellen Bindings, aber nicht die standardmäßigen Stil-Bindings.

Es gibt einige besondere Bindings mit individuellen Eigenschaften, wie Binding-Targets zum Festlegen der x-/y-Position, die es ermöglichen, einen Eingabebereich einem Positionsbereich zuzuordnen.

 

Text

Die meisten Elemente können einen Text anzeigen – entweder statisch gesetzt über den Designer oder gebunden an Daten, indem die Texteigenschaft als Binding-Target verwendet wird:

{

    "datasources": {

        "varId": {

            "source": "var",

            "params": {

                "id": 7537

            }

        }

    },

    "bindings": {

        "text": {

            "value": {

                "source": "dataref",

                "params": {

                    "ref": "varId"

                },

                "format": "d0.####"

            }

        }

    }

}

 

Das Werteobjekt ist normalerweise ein Datenobjekt, wie ein dataref, das ein anderes Datenobjekt referenziert.

 

Verwenden Sie datarefs Formateigenschaft, um ein Displayformat zu spezifizieren, z. B. Definieren eines Maximums an Dezimalstellen beim Anzeigen eines Fließkommawertes.

 

Tooltip

Mithilfe des Tooltip-Bindings kann definiert werden, welcher Text angezeigt wird, wenn mit der Maus über ein Element bewegt wird.

{

    "bindings": {

        "tooltip": {

            "value": "I'm the tooltip!"

        }

    }

}

 

Der statische Wert dieses Beispiels kann durch ein Datenobjekt ersetzt werden, z. B. eine Datenreferenz, Formel, Berechnung oder ein Attribut.

Position

Die Binding-Targets x und y werden verwendet, um die Positionen von Elementen festzulegen – entweder absolut oder relativ zur ursprünglichen Position in der Zeichnung. Die Eigenschaften min/max und minPos/maxPos ermöglichen es, einen Eingabebereich einem Positionsbereich zuzuordnen. Die minPos und maxPos warden nie überschritten.

Beispiel:

{

    "datasources": {

        "posx": {

            "source": "localvar",

            "params": {

                "sym": "posx"

            }

        },

        "posy": {

            "source": "localvar",

            "params": {

                "sym": "posy"

            }

        }

    },

    "bindings": {

        "x": {

            "value": {

                "min": 0,

                "max": 100,

                "minPos": 0,

                "maxPos": 200,

                "input": {

                    "source": "dataref",

                    "params": {

                        "ref": "posx"

                    },

                    "defaultValue": 0

                },

                "absolute": false

            }

        },

        "y": {

            "value": {

                "min": 0,

                "max": 100,

                "minPos": 0,

                "maxPos": 200,

                "input": {

                    "source": "dataref",

                    "params": {

                        "ref": "posy"

                    },

                    "defaultValue": 0

                },

                "absolute": false

            }

        }

    }

}

 

Weite/Höhe

Mithilfe der Weite- und Höhe-Bindings kann die Größe eines Elements geändert werden. Der Bereich, der von min und max spezifiziert wird, wird einem 0…100% Bereich der Originalgröße zugeordnet.

  "bindings": {

    "width": {

      "value": {

        "min": 0,

        "max": 50,

        "input": {

          "source": "dataref",

          "params": {

            "ref": "someVar"

          }

        }

      }

    }

  }

  "bindings": {

    "height": {

      "value": {

        "min": 0,

        "max": 50,

        "input": {

          "source": "dataref",

          "params": {

            "ref": "someVar"

          }

        }

      }

    }

  }
