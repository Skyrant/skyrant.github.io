Berechnungen

Überblick

Berechnungen sind spezielle Datenobjekte für Vorgänge wie Werte hinzufügen oder Input-Werte mit besonderen Output-Werten abzubilden. Formeln sind vor allem für mathematische Vorgänge oder String-Vorgänge relativ leicht zu lesen und zu verwenden. Berechnungen führen manchmal zu besser lesbaren Ausdrücken (z. B. Abbildung von Werten).

Wenn das Rechenergebnis für verschiedene Bindings benötigt wird, können Sie im Datenquellenbereich des Bindings festgelegt werden. Es ist aber auch möglich, eine Rechnung einzufügen, wo auch immer ein Wert einer Eigenschaft zugeschrieben ist (kann wie eine Datenreferenz verwendet werden).

{

    "datasources": {

        "n1": {

            "source": "var",

            "params": {

                "sym": "examples.n1"

            }

        },

        "n2": {

            "source": "var",

            "params": {

                "sym": "examples.n2"

            }

        },

        "sum": {

            "source": "function",

            "params": {

                "name": "sum",

                "params": {

                    "values": [

                        {

                            "source": "dataref",

                            "params": {

                                "ref": "n1"

                            }

                        },

                        {

                            "source": "dataref",

                            "params": {

                                "ref": "n2"

                            }

                        }

                    ]

                }

            }

        }

    },

    "bindings": {

        "text": {

            "value": {

                "source": "dataref",

                "params": {

                    "ref": "sum"

                }

            }

        }
