Zeichnungen erstellen, öffnen und speichern

Erstellen und Speichern neuer Zeichnungen

• Öffnen Sie den Designer in einem neuen Browsertab, indem Sie die Route designer für die URL der Sapient-Anwendung aufrufen, z. B.

http://<serverurl>/designer

• Geben Sie Ihre Anmeldedaten ein um sich einzuloggen.

• Versichern Sie sich, dass das Kästchen Diese Einstellung speichern ausgewählt ist und wählen Sie Sapient als Speicherort für Ihre neue Zeichnung.



• Wählen Sie Neues Diagramm erstellen.

• Geben Sie einen sinnvollen Namen ein.

• Klicken Sie auf Erstellen  Sie können nun beginnen zu zeichnen!

 

• Wichtig! Alle Ordner sind auf dem Server gespeichert – meistens unter Nutzung eines Unix/Linux Ordnersystems. Denken Sie also an die Einschränkungen für Unix Ordernamen. Verwenden Sie keine besonderen Zeichen wie deutsche Umlaute (ä, ö, ü, ß). Ordnernamen berücksichtigen Groß- und Kleinschreibung!

• Der Designer speichert die Zeichnung automatisch in dem vorhergesehenen Speicherort auf dem Server. Alternativ kann ein spezielles Verzeichnis innerhalb des Stammverzeichnisses ausgewählt werden, indem man auf Verzeichnis ändern klickt.

• Der Designer kann Ordner auf einem lokalen Laufwerk und im Ordnersystem des Browsers speichern. Das ist nur in seltenen Sonderfällen nötig. Sapient wird während der Laufzeit keinen Zugriff auf diese Ordner haben.

• Jede Zeichnung, die in einem spezifischen HMI- oder Andon-Projekt verwendet wird, sollte in einem entsprechenden Projektordner gespeichert werden. Falls gemeinsame Zeichnungen für mehrere Projekte erstellt werden, wählen Sie einen Speicherort außerhalb des Projektordners, vorzugsweise in einer der Stammdestinationen detail-layout oder designer.

 

Zeichnungen öffnen

Um eine Zeichnung zu öffnen, starten Sie den Designer, wählen Sie Sapient als Speicherort und scrollen Sie, um die gewünschte Zeichnung zu finden.

 

• In längeren Ordnerlisten können Sie die Suchfunktion des Browsers nutzen, um die Liste von Ordnern zu durchzusuchen. Drücken Sie STRG+F und geben Sie ein Suchschema ein – je nach Browser, werden die jeweiligen Treffer in der Liste hervorgehoben werden.

• Der einfachste Weg, um eine Zeichnung im Designer zu öffnen, ist wie folgt: Nutzen Sie die Debug-Seitenleiste um eine Zeichnung während der Laufzeit anzuzeigen, drücken Sie STRG und klicken Sie auf ein Zeichenelement, um die Debug-Seitenleiste zu öffnen. Dann wählen Sie Im Designer öffnen. (Die Debug-Tools / Seitenleiste müssen über die Boardlet-Parameter oder den HMI-Projektordner aktiviert werden.)

 

Zeichnungen im Dashboard anzeigen

• Fügen Sie das Boardlet „Legato Graphic Designer“ zum Dashboard hinzu.

• Legen Sie den Parameter-Ordnernamen für die neu erstellte Zeichnung fest.

• Speichern Sie die Einstellungen und das Dashboard  die Zeichnung wird nun im Boardlet angezeigt.
