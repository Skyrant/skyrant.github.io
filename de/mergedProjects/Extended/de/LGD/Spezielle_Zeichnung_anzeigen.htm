Spezifische Zeichnung anzeigen

Gehen Sie zum Boardlet und legen Sie den Ordnernamen der Zeichnung fest, der angezeigt werden soll. Der Pfad ist relativ zu dem Detail-Layout Verzeichnis in dem gemeinsamen Ordner. Das bedeutet, dass, wenn die Zeichnung direkt im Detail-Layout Verzeichnis (dem standardmäßigen Speicherort des Designers) gespeichert wird, muss kein Pfad festgelegt werden – geben Sie einfach den Ordnernamen einschließlich der Dateiendung xml ein. Falls Ordner im Detail-Layout Verzeichnis verwendet werden, muss der relative Pfad spezifiziert werden. Verwenden Sie einen Schrägstrich (keinen Gegenschrägstrich!) als Pfadtrenner, aber verwenden Sie keinen Schrägstrich am Pfadanfang. Stellen Sie sicher, dass Load Node Layout nicht aktiviert ist, z. B.:

Spezifische Parameter für dieses Boardlet:

• Load node layout: Deaktivieren, um eine spezielle Zeichnung anzuzeigen (weitere Informationen im Kapitel „Knotenspezifische Zeichnungen anzeigen“)

• Enable compact view mode: Wird nur verwendet, wenn knotenspezifische Zeichnungen geladen werden

• Animate changes: Die Laufzeit versucht automatisch reibungslöse Übergänge durchzuführen, wenn sich ein Style wie Sichtbarkeit während der Laufzeit ändert. Deaktivieren Sie diesen Parameter, um dieses Verhalten auszuschalten. Da der Systemparameter -360 gesetzt ist, werden Animationen komplett deaktiviert – der Boardlet-Parameter wird in diesem Fall übergangen.

• Last update time from server: Zeigt die Zeit des letzten Werteupdates in der linken, oberen Ecke des Boardlets an.

• Debug mode: Ermöglicht es, die Debug-Seitenleiste für die Boardlet-Instanz anzuzeigen.

 

Knotenspezifische Zeichnungen anzeigen

Wenn der Logical Structure Tree mit LC2 erstellt wird, können jedem Knoten zwei Zeichnungen zugewiesen werden – der LGD file path und der LGD compact view file path:

Wenn der Boardlet-Parameter Load node layout aktiviert ist, lädt das Boardlet die Zeichnung, die dem momentan gewählten Knoten zugeordnet ist. Zuerst wird die Zeichnung, die im LGD file path Bereich spezifiziert ist, angezeigt. Wenn dem ausgewählten Knoten keine spezielle Zeichnung zugeordnet ist, sucht das Boardlet den Baum (alle Parents) ab und verwendet die Zeichnung, die dem nächsten Parent zugeordnet ist.

Wenn der Boardlet-Parameter Enable compact view mode aktiviert ist, wird der compact view file path verwendet. In diesem Fall – wenn hier keine Eingabe gemacht wird, wird der reguläre Dateipfad genutzt – wenn nichts Weiteres festgelegt ist, sucht das Boardlet den Baum erneut in Root-Richtung ab, um eine Zeichnung zu finden.
