Zahlenformate

Der Formatparameter eines Textbindings definiert, wie ein numerischer Wert formatiert wird. Die Formatausdrücke sind die gleichen, die optional für die Konfiguration der Variablen genutzt werden. Der LC2 Hilfsbereich enthält weitere Details zu diesen Formaten.

Beispiel:

"bindings": {

    "text": {

        "value": {

            "source": "dataref",

            "format": "d0.####",

            "params": {

                "ref": "someVar"

            }

        }

    }

}
