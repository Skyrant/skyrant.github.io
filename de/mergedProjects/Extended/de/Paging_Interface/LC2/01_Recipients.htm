Funktion

Meldungen können automatisch an ein Funkrufsystem (z.B. PageControl der Fa. informel) weitergegeben werden. Legato verfügt über eine offene Schnittstelle, über die ein beliebiges System zur Meldungsweitergabe versorgt werden kann. Nachdem die Parametrierung von Empfängern stark vom verwendeten Rufsystem und dem Übertragungsmedium abhängt, wird in Legato lediglich eine Liste von Weiterleitungszielen (Empfängern) gehalten, die eigentliche Parametrierung der Empfänger erfolgt im Rufsystem. Dort werden Empfängereinstellungen und Funktionen wie Gruppenbildung, Folgerufe oder Eskalation parametriert.

Die Anbindung eines Funkrufservers erfolgt in der Regel im Rahmen der Projekterstellung bzw. des Customizings. Detaillierte Informationen zur Verwendung der allgemeinen Legato-Funkrufschnittstelle erhalten Sie von unserem Legato-Service.

Beschreibung

ID

Eindeutige Identifikationsnummer. Kann frei vergeben werden, muss aber eindeutig sein.

Code (Nummer)

Code: Symbolische Empfängerbezeichnung (z.B. Empfänger- oder Gruppenname bei PageControl), bei anderen Servern kann hier beispielsweise auch direkt die Rufnummer hinterlegt sein.

Empfängername

Empfängername: Klartextbezeichnung für den Empfänger.

Anzeige in Liste (man. FR)

Checkbox. Wenn aktiv, erscheint dieser Empfänger in der Ansicht "Manueller Funkruf" der WebApp zur Auswahl.

Funkruf-Server (optional)

Angabe des Funkrufservers, über den die Meldung weitergegeben wird. Die Verteilung auf unterschiedliche Rufserver muss im Funkruf-Gateway berücksichtigt werden.

Sprache (optional)

Lookup-Liste zur Auswahl der Systemsprache, in der die Meldetexte an diesen Funkrufempfänger gesendet werden.
