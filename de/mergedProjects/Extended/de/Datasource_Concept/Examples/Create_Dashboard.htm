Dashboard mit manuellen Meldungen erstellen

Die folgenden Schritte beschreiben, wie ein neues Dashboard basierend auf dem Datasource-Model "Archived Alarms" erstellt werden kann, in dem nur manuelle Meldungen angezeigt werden.

neues Dashboard anlegen

Boardlet "Generische Tabelle" reinziehen

im Tab Datasources des Boardlets:

die Datenquelle "Archived Alarms" auswählen

Bereich "Node" auf "NodeList" setzen

Bereich "Time" auf "TimeFrame" setzen

im Bereich "Actions" alles deaktivieren

im Tab Filter des Boardlets:

Klick auf "Parameter hinzufügen"

Name "Manual" eingeben

aus Drop-down-Liste den Typ "Number" auswählen (eigentlich handelt es sich um eine Bool'sche Angabe, wird DB-seitig aber als Zahl gehändelt)

als Wert "1" eingeben

zurück zum Tab Datasources:

im Bereich "Felder" al_archive aufklappen

bei der Spalte "Manual?" Klick auf + "Filter hinzufügen"

aus der linken Drop-down-Liste "Equal" auswählen

aus der rechten Drop-down-Liste "Manual" auswählen

im Tab "Tabelleneinstellungen"

ganz unten die neu hinzugefügte Spalte "al_archive/manual_insert" auf "Sichtbar" setzen
