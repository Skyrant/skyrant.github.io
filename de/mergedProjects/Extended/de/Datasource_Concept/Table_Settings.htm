Tabellen-Einstellungen

Für jedes Boardlet, das eine Tabelle anzeigt, steht ein weiterer Reiter mit Einstellungen zur Verfügung. Hierüber können Spalten ausgeblendet, gelöscht und hinzugefügt werden. Außerdem können für jede Spalte Einstellungen getroffen werden.

Alternativer Spaltenname

Hier kann die Spaltenüberschrift definiert werden.

Sortierbar

Über den Schieberegler kann festgelegt werden, ob die Spalte sortiert werden kann.

Skalierbar

Über den Schieberegler kann festgelegt werden, ob die Größe der Spalte per Drag & Drop verändert werden kann.

Verschiebbar

Über den Schieberegler kann festgelegt werden, ob diese Spalte per Drag & Drop woanders hin gezogen werden kann oder nicht.

Komponente

Formatierung der Spalte. Folgende Eingaben sind möglich:

date-table-cell-component

stellt den Inhalt im lokalen Datumsformat (Datum, Uhrzeit) dar. Der Inhalt der Zelle muss vom Typ Date sein.

icon-table-cell-component

stellt den Inhalt als Icon dar. Der Inhalt der Zelle muss der Name des Icons als String sein.

limit-table-cell-component

stellt eine Zahl dar und färbt diese je nach Limits ein

 

max

Limit max

 

min

Limit min

 

format

Legato Formatstring

 

mode

Markierung bei Grenzwertüberschreitung (border, color, background)
bar-table-cell-component

stellt den Inhalt als Balken mit Maximalwert dar.
  max: key String

gibt den Maximalwert vor
  formatAs: 'time'

Formatierung als 00:00:00

custom-style-cell-component

ermöglicht die Formatierung des CSS-Stils der Zelle. Kann mit "common" für alle Themes verwendet werden oder mit "themes" und dem jeweiligen Theme-Namen für spezifische Themes.

 

 font-weight

Schriftgewicht

 

 font-style

Schrift-Stil, z.B. italic oder bold

 

 color

Schriftfarbe

 

 background-color

Hintergrundfarbe

 

Breite

Über den Schieberegler kann festgelegt werden, ob die Breite der Spalte fest vorgegeben werden soll. Falls aktiv, wird eine Zeile eingeblendet, in der die Breite in Pixeln eingegeben werden kann.

Sichtbar

Über den Schieberegler kann festgelegt werden, ob diese Spalte ausgeblendet werden soll oder ob sie angezeigt wird.
