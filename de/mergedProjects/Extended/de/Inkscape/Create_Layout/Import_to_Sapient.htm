SVG in Sapient importieren

Positionierung

Bevor der Pfad der neuen SVG in Sapient eingefügt werden kann, muss das Objekt positioniert werden. Die obere linke Ecke der SVG sollte ungefähr an der oberen linken Ecke der Vorlage beginnen, da es sonst Probleme mit dem Detailbereich in Sapient gibt.

Transformationsattribut entfernen

Beim Kombinieren von Objekten wird oftmals ein "transform"-Attribut generiert. Dieses muss gelöscht werden, da nur der Pfad in Sapient verwendet wird.  Wenn Informationen zur Größe und Position im Transformationsattribut enthalten sind, werden diese in Sapient nicht berücksichtigt. Das Attribut kann im XML-Editor gelöscht werden.

Pfad in Sapient einfügen

Wie beim Importieren kann nun in Sapient der neue Pfad mithilfe des Boardlets "Einstellungen zum Knotenlayout" eingefügt werden.
