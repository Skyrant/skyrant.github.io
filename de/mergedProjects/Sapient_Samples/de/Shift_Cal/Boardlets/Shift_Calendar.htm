Schichtkalender Boardlets

Allgemein

Steuerungen, welche die selben Schichtzeiten haben, werden einem Zeitschema zugeordnet. Das Schema wird jedoch nicht direkt den Steuerungen zugeordnet, sondern über den jeweiligen Knoten der Fertigungsstruktur. Jeder Knoten sollte einem Zeitschema zugeordnet sein.

Jedem Zeitschema kann für einen Produktionstag ein Schichtmodell zugewiesen werden. Ein Schichtmodell beinhaltet dabei die eigentlichen Schichtzeiten für die Schichten. Es ist außerdem möglich, ein Schichtmodell zu generieren, welches keine Zeiten enthält (Bsp.: keine Produktion). Jede einzelne Schicht kann außerdem Pausen haben.

Zur Pflege des Schichtkalenders werden zwei Boardlets benötigt: Eines zur Auswahl der verfügbaren Zeitschemata (Zeitschema), und eines zur Zuweisung der Schichtmodelle zum Zeitschema (Schichtmodellzuweisung).

Zeitschema-Auswahl

In diesem Boardlet werden alle Zeitschemata angezeigt, für die der angemeldete Benutzer Berechtigungen hat.

Über die Checkboxen vor den Zeilen wird ausgewählt, welche Zeitschemata im Boardlet "Schichtkalender - Schichtmodelle zuweisen" zur Verfügung stehen.

Schichtkalender - Schichtmodelle zuweisen

Dieses Boardlet dient zum Zuweisen von Schichtmodellen zu Zeitschemata zu einer bestimmten Zeit.

Diese Zuweisungen dienen als Basis für Netto-Zeitberechnungen von Störmeldungen, für die Berechnung der KPI-Werte und auch als Grundlage für das Modul Schaltfunktion.

Wird bei "Quelldatum" ein Datum ausgewählt, so können die Einstellungen von diesem Tag auf einen oder mehrere Ziel-Tage kopiert werden (Auswahl unter Destination date). Die vorhandenen Schichtmodell-Zuweisungen werden dann unten beim jeweiligen Zeitschema eingeblendet. Bei Bedarf kann die Vorauswahl auch angepasst werden.

Wird kein Quelldatum selektiert, so können einem Ziel-Datum für jedes gefundene Zeitschema Schichtmodelle zugewiesen werden. Zu den Schichten des gewählten Modells können dann noch Schichtgruppen oder Produktionsgruppen zugewiesen werden, sofern diese im LC2 parametriert sind.

Bei der Auswahl des Zieltages kann im Kalender gesehen werden, für welche Tage bereits Daten gepflegt sind.

Die Einfärbungen haben folgende Bedeutung:

grün: alle einem Knoten zugewiesenen Zeitschemata sind für diesen Tag mit einem Schichtmodell beplant.

gelb: mindestens ein, aber nicht alle Zeitschemata sind für diesen Tag mit einem Schichtmodell beplant.

rot: keines der einem Knoten zugewiesenen Zeitschemata ist für diesen Tag mit einem Schichtmodell beplant.

Einstellungen

Die häufigsten Einstellungen auf den Reitern Einstellungen, Layout und Filter sind unter Boardlet-Einstellungen beschrieben:

  • Reiter Einstellungen

  • Reiter Layout

  • Reiter Filter

Für ein Schichtkalender Boardlet stehen keine weiteren Einstellungen zur Verfügung.

Benötigte Parametrierung im LC2

Schichttypen, Pausentypen und Feiertage werden im LC2 unter Werksverwaltung - Arbeitszeit-Einstellungen - Grundeinstellungen parametriert.

Zeitschemata werden im LC2 unter Werksverwaltung - Arbeitszeit-Einstellungen parametriert.

Schichtmodelle, Schichtgruppen und Produktionsgruppen werden unter dem jeweiligen Zeitschema zugewiesen.

Unter Anwendungsstruktur - Fertigungsstruktur - Fertigungsknoten wird dem Knoten ein Zeitschema zugewiesen (Spalte "Zeitschema").
