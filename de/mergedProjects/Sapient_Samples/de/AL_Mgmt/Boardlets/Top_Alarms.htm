Top Meldungen Boardlet

Allgemein

Dieses Boardlet zeigt zu einem selektierten Knoten eine statistische Auswertung von Meldungshäufigkeit, Meldungsdauer und aggregierter Durchschnittsdauer. Die Anzahl der Ergebnisse kann über die Einstellungen eingeschränkt werden. Weil diese Anzahl jedoch pro Kriterium gilt, kann das Ergebnis natürlich mehr als die eingegebenen Zeilen enthalten (z.B. hat die häufigste Meldung oft nicht die längste Dauer).

:

Die angezeigte Dauer ist die Nettodauer der Meldung, also abzüglich Pausenzeiten. Ist für den selektierten Knoten kein Schichtmodell gepflegt, gibt es keine Nettodauer, der Wert ist dann 0.

 

Einstellungen

Die häufigsten Einstellungen auf den Reitern Einstellungen, Layout und Filter sind unter Boardlet-Einstellungen beschrieben:

  • Reiter Einstellungen

  • Reiter Layout

  • Reiter Filter

Für ein Top Meldungen Boardlet stehen außerdem folgende Einstellungen zur Verfügung.

Reiter Layout

Tabelle sortieren nach:

Drop-Down-Liste zur Auswahl der Spalte, nach der die Auswertung sortiert werden soll.

Reiter Filter

Meldeklassen:

Filter auf Meldeklassen. Es werden alle Meldungen der hier eingegebenen Klassen für die selektierten Knoten angezeigt. Die Meldeklassen können aus einer Dropdown-Liste gewählt werden. Leer = kein Filter, alle Meldeklassen werden angezeigt.

Meldearten:

Filter auf Meldearten. Es werden alle Meldungen der hier eingegebenen Arten für die selektierten Knoten angezeigt. Die Meldearten können aus einer Dropdown-Liste gewählt werden. Leer = kein Filter, alle Meldearten werden angezeigt.

Meldungstext:

Hier kann ein Meldungstext eingegeben werden, auf den gefiltert werden soll. Als Platzhalter können % für beliebig viele Zeichen oder _ für genau ein Zeichen verwendet werden. Der Filter berücksichtigt Groß- und Kleinschreibung. Mehrere Filterkriterien können durch Komma verbunden werden (ODER-Filter).

Anzahl der Top-Meldungen:

Eingabe, wieviele Ergebnisse angezeigt werden sollen. Diese Anzahl gilt jedoch pro Kriterium, daher kann das Ergebnis natürlich mehr als die eingegebenen Zeilen enthalten (z.B. hat die häufigste Meldung oft nicht die längste Dauer).

Benötigte Parametrierung im LC2
