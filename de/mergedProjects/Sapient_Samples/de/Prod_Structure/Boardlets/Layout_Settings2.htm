Einstellungen zum Knotenlayout

Zum weiterführenden Editieren des Layouts empfiehlt es sich, zusätzlich das Boardlet "Einstellungen zum Knotenlayout" zu öffnen. Dieses kann natürlich später wieder aus dem Dashboard entfernt werden.

Das Einstellungen zum Knotenlayout Boardlet ist in die verschiedenen Bereiche des Layout-Boardlet unterteilt und bietet die Möglichkeit, diverse Parameter wie zum Beispiel Höhe und Breite eines Bereichs direkt einzugeben.

:

Zum Übernehmen der Werte muss im Bearbeiten-Modus des Knotenlayouts auf "Speichern" geklickt werden!

Die einzelnen Parameterwerte sind dabei in sinnvolle Schrittweiten gestaffelt, um eine einfachere Bearbeitung zu ermöglichen. Befindet sich ein Wert außerhalb einer solchen Schrittweite, wird er rot umrandet. Ein Klick auf den Spinner rechts neben dem Wert setzt den Wert auf die nächsthöhere bzw. nächstniedrigere Schrittweite.

Outline

Diese Parameter gelten für die im Boardlet Knotenlayout aktuell selektierte Outline.

Offset X: / Offset Y:

Mit diesen Werten kann der Bereich im Boardlet verschoben werden.

Width: / Height:

Mit diesen Parametern kann die Breite und die Höhe des Bereichs geändert werden. Für den Fall, dass eine Outline unten im Boardlet im unsichtbaren Bereich verschwindet, kann hier beispielsweise die Größe deutlich reduziert werden und der Pfad anschließend wieder auf ein geeignetes Maß gezogen werden.

Path:

Kompletter Pfad des Bereichs (ein Array der obigen Parameter). Wird dieser Eintrag gelöscht, wird der Pfad auf den Defaultwert zurückgesetzt.

Der Pfad kann auch aus einem externen Zeichentool wie zum Beispiel Inkscape eingefügt werden. Dies ist vor allem bei komplexen Formen zu empfehlen.

   

Text

Defaultmäßig ist der Text mit dem Detail-Bereich verknüpft, d.h., wenn man den Text verschiebt, verschiebt man den Detail-Bereich ebenfalls. Wird der Detail-Bereich einmal alleine verschoben, so ist die Verbindung zum Text unterbrochen und der Text kann ebenfalls allein verschoben werden.

X: / Y:

X- und Y-Koordinate des Textes in Relation zum zugehörigen Bereich.

Rotate:

Ermöglicht es, den Text schräg darzustellen.

Size:

Größe des Texts in Relation zum Rahmen.

   

Details

X: / Y:

X- und Y-Koordinate des Detail-Bereichs.

Breite: / Höhe:

Mit diesen Parametern kann die Breite und die Höhe des Bereichs geändert werden. Für den Fall, dass ein Bereich unten im Boardlet im unsichtbaren Bereich verschwindet, kann hier beispielsweise die Größe deutlich reduziert werden und der Bereich anschließend wieder auf ein geeignetes Maß gezogen werden.
