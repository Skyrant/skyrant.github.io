Zuordnung von Rechten für mehrere Datenquellen

Es ist möglich, für mehrere Datenquellen gleichzeitig Rechte zu vergeben. Dafür gibt es zwei Möglichkeiten:

Selektieren Sie ein Element auf einer beliebigen Ebene im Navigationsbaum. Öffnen Sie mit der rechten Maustaste das Kontextmenü und selektieren Sie den Eintrag "Berechtigungen für alle Tabellen ab hier im Baum".

Oder öffnen Sie die Tabelle "Datenquellen" für Ihre Verbindung (z.B. unter System-Administration - Verbindungen - Legato - Datenquellen). Markieren Sie alle Einträge, für die Sie Berechtigungen vergeben wollen und öffnen Sie dann mit der rechten Maustaste das Kontextmenü. Selektieren Sie den Eintrag "Berechtigungen".

In beiden Fällen öffnet sich ein neuer Tab mit folgendem Dialog:

In der Kopfzeile werden alle selektierten Datenquellen aufgelistet; diese können bei Mouseover auch als Tooltipp angezeigt werden.

Darunter können Berechtigungen für Benutzergruppen zugewiesen werden.

Ein grün gefülltes Kästchen bedeutet, dass die jeweilige Gruppe das entsprechende Recht für eine oder mehrere der selektierten Datenquellen hat, aber nicht für alle. Für welche Datenquellen genau das Recht bereits vergeben ist, kann im Tooltipp nachgesehen werden.

Ein grünes Häkchen bedeutet, dass die jeweilige Gruppe das entsprechende Recht für alle Datenquellen bereits hat.

:

Mit dieser Methode können nur Rechte für Datenquellen vergeben werden, nicht für Targets (Knoten, Verbindungen, Zeitschemata, Dashboards)! Sie können hierüber also beispielsweise das Recht zum Anlegen von Verbindungen vergeben, aber nicht das Recht zum Editieren von Meldungsdefinitionen einer bestimmten Verbindung.

Zuordnung von Rechten für mehrere Targets

Für die sogenannten Targets (Knoten, Verbindungen, Zeitschemata, Dashboards) funktioniert die Zuordnung von Rechten im Grunde genommen ebenso wie für die Datenquellen, da Datenquellen auch ein Target sind.

Öffnen Sie die gewünschte Target-Tabelle, z.B. Fertigungsknoten unter Anwendungsstruktur - Fertigungsstruktur - Knoten. Gegebenenfalls ist es sinnvoll, die Tabelle inklusive aller Unterelemente anzuzeigen (über das Kontextmenü - Tabelle - Alle Elemente in der Tabelle öffnen in...). Markieren Sie dann die gewünschten Knoten und öffnen Sie das Kontextmenü über die rechte Maustaste. Wählen Sie den Eintrag "Berechtigungen..."

Es öffnet sich ein neuer Tab mit folgendem Dialog:

Es gibt hier mehr Rechte als für Datenquellen, ansonsten funktioniert die Zuordnung von Rechten analog zu den Datenquellen.

Ist die Checkbox "Änderungen auf Unterelemente anwenden" aktiv, so wird die Zuordnung auch für alle untergeordneten Elemente angewendet, auch wenn diese nicht explizit selektiert sind.

Einer Gruppe alle Rechte zuordnen

Um einer Gruppe alle Rechte zuzuordnen, müssen folgende Schritte ausgeführt werden:

Für jede benötigte Verbindung (üblicherweise ist "Legato" ausreichend) unter "System-Administration - Verbindungen": Tabelle der Datenquellen öffnen, alle Einträge markieren, Kontextmenü - Berechtigungen --> hier die gewünschte Berechtigung der Gruppe zuweisen.

Jede Target-Tabelle (diese sind: Knoten, Verbindungen, Zeitschemata und Dashboards) mit allen Einträgen öffnen (über das Kontext-Menü - Tabelle - Alle Elemente in der Tabelle öffnen in...). Alle Einträge markieren, Kontextmenü - Berechtigungen --> hier die gewünschte Berechtigung der Gruppe zuweisen.
