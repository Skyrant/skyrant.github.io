Benutzerverwaltung

Initial wird nur eine Person mit einem zugehörigen Benutzeraccount ausgeliefert (Person -1 SYS mit Benutzer -1 sys). Legen Sie im LC2 die weiteren benötigten Personen an und erstellen Sie Benutzeraccounts dazu. Definieren Sie dann ein Konzept für Benutzergruppen oder ordnen Sie die neuen Benutzer vorläufig der Gruppe SYS zu. Alle Einträge können bei Bedarf später geändert werden.

Zum genauen Vorgehen, lesen Sie bitte das LC2-Kapitel Benutzerverwaltung.

Vergeben Sie anschließend Berechtigungen für die Gruppen. Zum genauen Vorgehen, lesen Sie bitte das LC2-Kapitel Zuordnung von Rechten im LC2.
