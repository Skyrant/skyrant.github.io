Basis-Rechte

Anzeige von Dashboards im Webclient

VIEW-Rechte auf folgende Datenquellen:

U_PERMISSIONS (damit der Client die Berechtigungen nachschlagen kann)

U_DASHBOARDS

U_DASHBOARD_FOLDERS (weil auch das Dashboard auf oberster Ebene in einem Ordner liegt)

U_DASHBOARD_PROFILES

GP_R_BOARDLETS (zur Anzeige der Boardlet-Library)

GP_R_BOARDLET_CATEGORIES (zur Anzeige der Boardlet-Library)

GP_R_LOCALES (für die Spracheinstellungen)

T_SHIFT_TYPES (für den Zeitfilter)

L_NODES (für die Knotenauswahl) - sollen Daten nur für einen bestimmten Knoten angezeigt werden, ist es auch möglich, diesen über den Filter direkt einzugeben. Ohne Rechte auf L_NODES kann allerdings die Filterinfo nicht angezeigt werden

AL_NODE_STATES (um die Hintergrundfarbe im Layout gemäß Knotenstatus anzuzeigen)

U_BOOKMARKS

U_BOOKMARK_TYPES

das gewünschte Dashboard

alle Tabellen, deren Daten im Dashboard angezeigt werden
