Navigation

 Auto-Verbergen Stecknadel. Wird sie angeklickt, wird der Navigationsbaum ausgeblendet und nur beim Überfahren mit der Maus angezeigt.

 Schließt den Navigationsbaum. Über den Ribbon Ansichten kann er jederzeit wieder aktiviert werden.

 Ansicht autom. öffnen. Wenn dieser Button aktiviert ist, werden Tabellen, die im Navigationsbaum angeklickt werden, direkt in der Dokumentenansicht angezeigt. Eine bereits geöffnete Tabelle wird überschrieben.

 Flachen Baum anzeigen. Wenn dieser Button aktiviert ist, wird der Navigationsbaum als flache Liste dargestellt. D.h., Einträge von untergeordneten Tabellen werden im Navigationsbaum nicht dargestellt.

Gesperrte Elemente ausblenden. Wenn dieser Button aktiviert ist, werden im Navigationsbaum nur Elemente angezeigt, für die der angemeldete Benutzer Änderungsrechte besitzt. Über die Suche angewählte Elemente werden auch ohne Änderungsberechtigung angezeigt. Ebenso können auch alle Elemente über die Funktion "Im Navigationsbaum anzeigen" aufgerufen werden.

INFO:

Die Rechtestruktur im LC2 entspricht weitgehend der Navigationsstruktur. Da allerdings z.B. unter der Fertigungsstruktur auch Tabellen aus anderen Bereichen angezeigt werden (z.B. Berechnete Werte des Knotens), führt ein fehlendes Recht "Fertigungsstruktur bearbeiten" nicht zum Ausblenden des gesamten Menüpunkts. Die Unterelemente aus anderen Rechte-Bereichen werden natürlich trotzdem angezeigt.

 Suche anzeigen. Öffnet die Suche für den Navigationsbaum. Siehe auch eigenes Kapitel (Suche und Erweiterte Suche).

 Filter anzeigen. Öffnet ein Feld zur Eingabe eines Filterkriteriums. Dieser Filter wirkt sich nur auf sichtbare Einträge im Navigationsbaum aus (d.h., Unterelemente werden nur berücksichtigt, wenn sie aufgeklappt sind).
