Suche und Erweiterte Suche

Durch Klick auf das Fernglas  wird die Suche für den Navigationsbaum angezeigt. Es kann nach Schlagworten gesucht werden. Die Ergebnisse werden dann in einer Liste dargestellt, über die der Eintrag im Navigationsbaum angesteuert werden kann. Ist die Checkbox "Tabelle" aktiviert, kann auch direkt nach dem Namen der Datenbank-Tabelle gesucht werden.

Es werden maximal 25 Ergebnisse angezeigt.

 

Doppelklick auf einen Eintrag der Ergebnisliste markiert die Position des Eintrags im Navigationsbaum und springt dorthin, wenn möglich. Dieselbe Funktion wird durch Klick auf die Lupe  ausgeführt.

Alternativ kann durch Klick auf die Lupe mit Stern  die erweiterte Suche aufgerufen werden. Diese Funktion steht für alle Ergebnisse zur Verfügung, die Tabellen darstellen; nicht für Einträge.

Die erweiterte Suche wird in einem neuen Reiter im Dokumenten-Fenster geöffnet.

 

Unter "Spalte" kann über eine Drop-down-Liste ausgewählt werden, welche Spalte der Tabelle den Suchbegriff enthalten soll. In "Suchbegriff" wird nun der zu suchende Begriff eingegeben. Klick auf "Suche" öffnet die Tabelle mit den gefundenen Ergebnissen. Wird kein Suchbegriff eingegeben, wird die gesamte Tabelle geladen. Unter "Maximale Suchergebnisse" kann eingestellt werden, wieviele Ergebnisse höchstens geladen werden sollen.

 

Der gefundene Eintrag kann nun bei Bedarf bearbeitet werden. Außerdem besteht die Möglichkeit, den Eintrag direkt im Navigationsbaum aufzurufen (über Kontextmenü - Element im Navigationsbaum finden).
