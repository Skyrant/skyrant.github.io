Kontextmenü

Durch einen Klick mit der rechten Maustaste auf einen Eintrag im Navigationsbaum kann ein Kontextmenü geöffnet werden. Es gibt drei Arten von Kontextmenüs:

Kontextmenü für Tabellen

Kontextmenü für Zeilen / Tabelleneinträge

erweitertes Kontextmenü für Tabellen mit speziellen Funktionen (z.B. Import, Export, etc.)

 

Die gängigen Funktionen sind Folgende:

[Tabellenname] öffnen in

Öffnet eine Auswahl, ob die selektierte Tabelle im aktuellen oder einem neuen Reiter in den Dokumenten geöffnet werden soll.

[Tabellenname] hinzufügen…

Öffnet den Wizard, um ein Element zur selektierten Tabelle hinzuzufügen.

Bearbeiten…

Öffnet den Wizard, um das selektierte Element zu bearbeiten.

Element löschen

Löscht das selektierte Element. Wird das selektierte Element von anderen Elementen referenziert, so erfolgt eine entsprechende Meldung. Diese Referenzen müssen dann zuerst gelöscht werden. Dies kann über den folgenden Punkt "Referenzen analysieren" erfolgen.

Referenzen analysieren…

Nur für Datensätze verfügbar, nicht für Tabellen. Öffnet einen Dialog (siehe Referenzen analysieren). Prüft die Referenzen des Datensatzes und listet sie auf. Nach Rückfrage können alle Referenzen gelöscht werden (ggf. nötig, um Datensatz zu löschen).

Referenziert von

Zeigt Tabellen, die den selektierten Datensatz referenzieren.

Tabelle

Hier kann gewählt werden, ob die selektierte Tabelle im aktuellen oder einem neuen Reiter angezeigt werden soll und ob die Darstellung als Grid oder Pivot erfolgen soll.

Aktualisieren

Aktualisiert die Tabelle (z.B. um zwischenzeitliche Änderungen eines anderen Benutzers verfügbar zu machen).

Baum neu laden

Lädt den Navigationsbaum neu. Hierbei werden alle aufgeklappten Elemente geschlossen.

Benutzerrechte aktualisieren

Aktualisiert die Rechte des angemeldeten Benutzers.

Hilfe

Öffnet die Hilfe für das selektierte Element im Browser.
