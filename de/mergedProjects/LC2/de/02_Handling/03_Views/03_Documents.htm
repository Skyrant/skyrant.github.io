Dokumente

In dieser Ansicht werden die im Navigationsbaum ausgewählte Tabelle und ihr untergeordnete Tabellen angezeigt. Sie werden als Tabs angezeigt, über die navigiert werden kann. Gibt es untergeordnete Datensätze zum momentan selektierten Tabelleneintrag, so wird dies durch einen gelben Balken unten rechts an der Tabelle angezeigt. Diesen kann man nach oben bzw. links ziehen, um die darunterliegenden Daten ebenfalls anzuzeigen.

Die enthaltenen Datensätze können hier editiert werden. Über die Spaltenauswahl bzw. Einstellungen - Layout kann definiert werden, welche Spalten angezeigt werden sollen (freie Auswahl; nur Pflichtfelder; alle Felder). Beim Überfahren des Spaltennamens mit der Maus wird ein Tooltip angezeigt.

Tabellen, für die der angemeldete Benutzer keine Berechtigung hat, werden schreibgeschützt dargestellt. Welche Rechte benötigt werden, wird im Beschreibungs-Fenster angezeigt.

In dieser Ansicht können die Datensätze der Tabellen editiert werden. Zur Bedienung der Tabellen siehe Kapitel Bedienung der Tabellen / Ansichten.

Außerdem sind Such- und Filteroptionen für die selektierte Tabelle verfügbar.

Der Name der Tabelle ist als Reiter dargestellt. Ein Klick auf den gelben Stern speichert die Tabelle als Favoriten.
