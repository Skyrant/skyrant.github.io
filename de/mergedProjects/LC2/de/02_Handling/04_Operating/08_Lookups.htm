Nachschlagefelder / Lookup-Listen

Felder, die auf eine andere Tabelle referenzieren, sind mit einer Nachschlagefunktion hinterlegt. Diese wird durch drei Punkte oder einen Pfeil nach unten gekennzeichnet. Ein Klick auf die drei Punkte öffnet den Navigationsbaum oder die referenzierte Tabelle. Für bis zu 20 Einträge wird eine reine Lookup-Liste angezeigt, aus der der gewünschte Wert ausgewählt werden kann.

Durch das Kontextmenü in einer Lookup-Zelle kann die Nachschlagetabelle entweder im Navigationsbaum lokalisiert werden (Im Navigationsbaum finden) oder auch direkt geöffnet werden (Lookup-Tabelle öffnen); z.B. um Änderungen daran vorzunehmen. Beide Aktionen können auch kombiniert ausgeführt werden (Im Navigationsbaum finden und in neuem Tab öffnen). Auch der markierte Eintrag kann zur Ansicht oder im Bearbeitungsmodus geöffnet werden (Lookup-Element anzeigen / bearbeiten). Außerdem können die Nachschlageelemente direkt im Autofill-Fenster gesucht werden (Lookup-Elemente im Autofill-Fenster öffnen). Ist das Autofill-Fenster noch nicht aktiv, so wird es gestartet und in den Vordergrund gebracht.

Soll der Wert in einem Lookup-Feld von einer Eingabe auf NULL geändert werden, kann dies über "Wert löschen" im Kontextmenü erfolgen.
