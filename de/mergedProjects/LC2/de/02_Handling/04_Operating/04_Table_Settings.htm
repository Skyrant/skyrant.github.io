Tabellen-Einstellungen

Das Zahnrad-Symbol in der linken oberen Ecke der Tabelle bietet weitere Einstellungen für die Tabelle:

Spaltenauswahl

Hier können die darzustellenden Spalten ausgewählt werden. Standardmäßig werden alle Spalten angezeigt.

Tabelle öffnen als

Hier kann zwischen Grid- und Pivot-Darstellung gewechselt werden.

Layout

Bietet verschiedene Layout-Optionen.

Spaltenbreite anpassen

Setzt alle Spalten auf die optimale Breite.

Alle Spalten anzeigen

Zeigt alle Spalten der Tabelle an.

Nur Pflichtfelder anzeigen

Zeigt nur die Pflichtfelder der Tabelle an.

Erweiterte Eigenschaften ausblenden

Blendet die erweiterten Eigenschaften aus.

Layout zurücksetzen

Setzt das Layout der Tabelle zurück (z.B. Spaltensortierung, ausgeblendete Spalten etc.)

Anzeige

Bietet verschiedene Optionen für die Anzeige. Diese Optionen sind globale Einstellungen. Daher können sie nicht über "Layout zurücksetzen" in den Tabellen-Einstellungen zurückgesetzt werden, sondern nur über den Button "Tabellenlayout zurücksetzen (alle)" im Reiter "Start".

Balken anzeigen / ausblenden

Hierüber können die Navigationsbalken ein- bzw. ausgeblendet werden.

Balken anpassen

Hierüber können die Navigationsbalken individuell angepasst werden.

Zoom anzeigen

Hierüber kann der Zoom unten rechts in der Tabellen-Navigation ein- bzw. ausgeblendet werden.

Buttons zur Spaltennavigation anzeigen

Hierüber können die Buttons zum spaltenweisen Navigieren ein- bzw. ausgeblendet werden.

Buttons zur Zeilennavigation anzeigen

Hierüber können die Buttons zum zeilenweisen Navigieren ein- bzw. ausgeblendet werden.
