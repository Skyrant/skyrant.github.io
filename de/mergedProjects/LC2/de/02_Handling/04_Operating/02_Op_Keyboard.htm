Bedienung über Tastatur

In der Dokumentenansicht können die Tabellen editiert werden. Diverse Funktionen können dabei über die Tastatur aufgerufen werden.

Allgemein

Beschreibung

Wechsel in den Bearbeitungsmodus

Wechsel in den Anzeigemodus

Zeilen- / Spaltenweises Blättern

Erste Spalte

Strg+F

Suchen

Strg+H

Spalte ausblenden

Strg+W

Ansicht fixieren (Tabelle bleibt bis zur markierten Spalte immer sichtbar)

Strg+A

Alles markieren

Strg+D

Markiertes duplizieren (auch für Mehrfachauswahl möglich)

Strg+Einfg

Neue Zeile

Strg+Entf

Löschen

Taste

Beschreibung

Strg+Pos1

Strg+Ende

Letzte Zeile

Strg+Klick / Shift+Klick

Mehrfachselektion

Strg+Shift+Ende

Strg+B

Breite der selektierten Spalte anpassen

Strg+Alt+B

Breite aller Spalten anpassen

 

Bearbeitungsmodus

Taste

Beschreibung

Eingaben in selektierte Zelle 

Existierender Text wird überschrieben

Strg+Pos1

Cursor in erste Zeile

Strg+Ende

Cursor in letzte Zeile

Pos1

Cursor an Textanfang der selektierten Zelle

Ende

Cursor an Textende der selektierten Zelle

Esc (1x)

Änderungen an aktueller Zelle rückgängig

Esc (2x)

Änderungen in ganzer Zeile rückgängig

Enter

Änderungen übernehmen

Strg+C

Zelleninhalt kopieren

Strg+V

Zelleninhalt einfügen

Strg+Alt+C

Zeile kopieren

Strg+Alt+V

Zeile(n) aus Zwischenablage unten anfügen

Strg+Alt+R

vorhandene Zeilen mit Einträgen aus Zwischenablage überschreiben (nicht vorhandene Keys werden verworfen)

Strg+Alt+N

vorhandene Zeilen mit Einträgen aus Zwischenablage überschreiben (nicht vorhandene Keys werden unten angehängt)

Strg+N

Wert setzen für selektierte Variable (nur in Variablen und Variablenzuordnungen).

Strg+M

Wert setzen (multi) für mehrere selektierten Variablen (nur in Variablen und Variablenzuordnungen).
