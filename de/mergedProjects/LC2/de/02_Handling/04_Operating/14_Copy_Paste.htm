Kopieren & Einfügen (Copy & Paste)

Daten können mit der Tastenkombination Strg + Alt + C oder über das Kontextmenü aus dem LC2 in die Zwischenablage kopiert werden. Anschließend können sie z.B. in Excel eingefügt werden und dort bearbeitet werden. Für jedes Lookup-Feld der Tabelle wird eine zusätzliche Spalte mit dem entsprechenden Key exportiert. Bei der Bearbeitung z.B. in Excel sind diese Key-Spalten zu befüllen, damit das LC2 in der Lage ist, nach dem Einfügen der Daten die Zuordnung vorzunehmen. Fehlen hier Einträge, können die Daten dennoch eingefügt werden und die Zuordnung hinterher im LC2 manuell vorgenommen werden. Die fehlerhaften Zeilen sind in diesem Fall entsprechend gekennzeichnet.

Mit Strg + Alt + V können Daten aus der Zwischenablage wieder ins LC2 eingefügt werden. In diesem Fall werden die Daten ans Ende der Tabelle angehängt (Anhängen). Für die Option "Aktualisieren" lautet die Tastenkombination Strg + Alt + R, für "Zusammenführen" Strg + Alt + N.
