Filter

Eine Tabelle kann über jede Spalte gefiltert werden. Es ist zudem möglich, nach mehreren Spalten / Kriterien zu filtern.

Um den Filterdialog zu öffnen, klickt man mit der linken Maus auf das Filtersymbol in der Spalte, nach der gefiltert werden soll. In der erscheinenden Liste den gewünschten Eintrag als Filterkriterium auswählen.

Um einen detaillierteren Filter zu erstellen, kann der Filter-Editor über die Spalteneinstellungen aufgerufen werden. Sämtliche Worte im Filter-Editor bieten ein Kontextmenü. Beim Überfahren der Worte mit der Maus verwandelt der Mauszeiger sich in eine Hand. Die verschiedenen Farben kennzeichnen verschiedene Teile des Filters:

Rot = Logischer Operator

Blau = Spaltenname

Grün = Filteroperanden

Schwarz = Filterkriterium

Es können mehrere Filtergruppen erstellt werden, wobei für jede Gruppe ein anderer Operator verwendet werden kann. Außerdem können jedem Operator beliebig viele Kriterien zugewiesen werden. Die folgenden logischen Operatoren stehen zur Verfügung (rote Schrift):

Und

Und-Filter. Es werden nur Ergebnisse angezeigt, die allen Filterkriterien entsprechen.

Oder

Oder-Filter. Es werden Ergebnisse angezeigt, die einem der Filterkriterien entsprechen.

NichtUnd

Nicht-Und-Filter. Es werden nur Ergebnisse angezeigt, die nicht allen Filterkriterien entsprechen.

NichtOder

Nicht-Oder-Filter. Es werden nur Ergebnisse angezeigt, die keinem der Filterkriterien entsprechen.
