Einstellungen

Im Menüpunkt "Einstellungen" können benutzerspezifische Einstellungen vorgenommen werden.

Im Untermenü " Fenster" werden Einstellungen für das Hauptfenster getroffen.

Sprache

Legt fest, in welcher Sprache die Oberfläche des LC2, die Menüs und die Hilfe angezeigt werden sollen. Es kann zwischen Deutsch und Englisch umgeschaltet werden; die Einstellung System definiert die Sprache automatisch basierend auf der Sprache des Betriebssystems. Eine Änderung dieser Einstellung wird erst nach einem Neustart des LC2 wirksam. Diese Einstellung hier hat keinen Einfluss darauf, in welcher Sprache Tabelleninhalte im Navigationsbaum oder in Lookup-Fenstern angezeigt werden. Dies wird über den Sprachindex festgelegt. Als Datumsformate werden immer die lokal auf dem System eingestellten Formate verwendet, unabhängig von der eingestellten Oberflächensprache.

Favoritenleiste anzeigen

Legt fest, ob die Favoritenleiste angezeigt werden soll oder nicht.

Statusleiste anzeigen

Legt fest, ob am linken unteren Rand des Screens eine Statuszeile angezeigt werden soll.

Ribbon minimieren

Legt fest, ob der Ribbon minimiert werden soll oder nicht. Wenn diese Checkbox aktiviert ist, wird der Ribbon ausgeblendet und klappt auf, wenn man mit der Maus darüber fährt.

Layout speichern

Legt fest, ob benutzerspezifische Layout-Änderungen gespeichert werden sollen oder nicht. Wenn diese Checkbox aktiviert ist, so werden Änderungen des Benutzers wie beispielsweise die Spaltenreihenfolge bestimmter Tabellen etc. auch beim Schließen der Anwendung gespeichert und beim nächsten Öffnen wiederhergestellt.

Sprachindex

Der Sprachindex definiert, ob die Daten im Navigationsbaum und den Lookup-Feldern in der ersten oder zweiten Systemsprache angezeigt werden sollen. Sprachindex 0 = erste Systemsprache; Sprachindex 1 = zweite Systemsprache. Diese Einstellung ist unabhängig von der Spracheinstellung der Oberfläche.

ID an Anzeige-Namen anhängen

Legt fest, ob bei der Anzeige von Zeilen / Datensätzen im Navigationsbaum und Lookup-Feldern zusätzlich zum angezeigten Namen (üblicherweise NAME_0) auch die ID des Datensatzes angezeigt wird.

 

Im Untermenü "Navigationsbaum" können Einstellungen für den Navigationsbaum getroffen werden.

Maximal sichtbare Zeilen

Legt fest, wie viele Datenzeilen für eine Tabelle im Navigationsbaum angezeigt werden sollen. Weitere Zeilen können immer durch Klick auf "Doppelklick, um mehr Elemente zu laden..." geladen werden.

Ansicht autom. öffnen

Diese Checkbox legt fest, ob Tabellen bei Anklicken im Navigationsbaum in den Dokumenten angezeigt werden sollen oder ob dies über das Kontextmenü (rechte Maustaste) explizit ausgewählt werden soll. Wenn diese Checkbox aktiviert ist, wird die aktuell angezeigte Tabelle durch die neue überschrieben.

 

Im Untermenü "Logging" werden Einstellungen für das Loggen getroffen.

Log-Level

Hier kann festgelegt werden, bis zu welchem Level Meldungen geloggt werden sollen. Übergeordnete Level werden in jedem Fall geloggt, d.h., wenn "Warnung" gewählt ist, werden "Fehler" auch geloggt. Folgende Optionen stehen zur Verfügung (Auswahl über Drop-down-Liste):

  • Off: Es werden keine Meldungen geloggt.

  • Error: Es werden nur Fehlermeldungen geloggt.

  • Warning: Es werden Warnungen und Fehlermeldungen geloggt.

  • Info: Es werden Informationen, Warnungen und Fehlermeldungen geloggt.

  • Verbose: Es werden Hintergrund-Informationen, Informationen, Warnungen und Fehlermeldungen geloggt.
    Wird das Loglevel zur Laufzeit geändert, werden ab dem Änderungszeitraum entsprechend mehr oder weniger Einträge geloggt. Bereits im Log befindliche Einträge bleiben bestehen.

Max. Zeilen

Hier kann definiert werden, wie viele Zeilen maximal geloggt werden sollen.

 

Im Untermenü "Tabellen" werden Einstellungen für die Tabellen getroffen.

Layout speichern

Legt fest, ob benutzerspezifische Layout-Änderungen (z.B. angezeigte Spalten, Spaltenbreite, etc.) an Tabellen gespeichert werden sollen oder nicht.

Zoom anzeigen

Hier können die Zoom-Optionen unter den Tabellen ein- / ausgeblendet werden.

Balken anzeigen

Hier können die Navigationsbalken zum Bearbeiten der Tabellen ein- /ausgeblendet werden.

Buttons zur Spaltennavigation anzeigen

Hierüber können die Buttons zum spaltenweisen Navigieren ein- bzw. ausgeblendet werden.

Buttons zur Zeilennavigation anzeigen

Hierüber können die Buttons zum zeilenweisen Navigieren ein- bzw. ausgeblendet werden.

 

Im Untermenü "Verschiedenes" können verschiedene Einstellungen getroffen werden.

Tabellen cachen

Wenn diese Checkbox aktiviert ist, werden die Tabellen gecached. Dies kann möglicherweise Ladezeiten verkürzen, kann aber auch zu inkonsistenten Daten führen, wenn mehrere Benutzer gleichzeitig dieselbe Tabelle bearbeiten.
