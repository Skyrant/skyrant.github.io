Funktion

Über Schaltbefehle können zeitabhängig beliebige Variablen auf einen definierten Wert gesetzt (gesteuert) werden. Die Schaltzeit wird dabei immer in Bezug auf ein definiertes Ereignis, z.B. Schichtbeginn festgelegt.

:

Dafür muss bei der Variablen die Checkbox "Weitergabe an GW" = 1 sein.

Das Zeitmodell ist Voraussetzung für den Betrieb dieses Moduls. Aus dem Zeitmodell, der Verknüpfung von Zeitschema mit der Fertigungsstruktur und der Parametrierung des Schichtkalenders werden entsprechende schichtabhängige Ereignisse (Beginn, Ende etc.) generiert.

Soll ein Schaltbefehl sofort ausgeführt werden, so kann einfach der Wert der entsprechenden Variable gesetzt werden.

Feldbeschreibung

 

Feld

Beschreibung

ID

Eindeutige Identifikationsnummer. Wird vom System vergeben. Schreibgeschützt.

Knoten

Lookup-Liste zur Auswahl eines Knotens aus der Fertigungsstruktur, für den dieser Schaltbefehl gelten soll. Wird automatisch befüllt und kann geändert werden.

Variable

Lookup-Liste zur Auswahl einer Variablen. Je nach Variablentyp wird mit dem Schaltbefehl die Variable auf den Wert (num/str/Bool) der nachfolgenden Felder gesetzt.

Name

Frei wählbarer Name für den Schaltbefehl.

Beschreibung (optional)

Zusätzliche Beschreibung des Schaltbefehls.

Offset (optional)

Offset. Kann auch negativ sein, um eine Vorlaufzeit anzugeben.

Leerzeichen getrennte Liste von Dauern angegeben. Dauern können auch mit einem Minus-Zeichen davor als negativ gekennzeichnet werden. Sie können entweder

  • im Uhrzeit-Format hh:mm:ss.SSS oder

  • als Wert-und-Einheit angegeben werden. Folgende Einheiten sind möglich:

  •  y für Jahre

  • M für Monate

  • w für Wochen

  • d für Tage

  • h für Stunden

  • m für Minuten

  • s für Sekunden

  • ms für Millisekunden

 

Z.B. der Wert "5 1/2 Minuten vorher" kann u.a. als "-00:05:30", "-5m -30s", "-330s" oder "-6m 30s" angegeben werden.

Ereignis-Typ

Lookup-Liste zur Auswahl eines Ereignisses, mit dem der Schaltbefehl ausgeführt werden soll.
Zu den einzelnen Ereignissen siehe Liste unten.

Min. Lücke

(Bool)

Mindestdauer ohne Produktion/"Ereignis Produktions-Beginn/Ende: Mindest-Dauer ohne Produktion davor/danach":

Schichtfreie Zeitspanne vor Produktionsbeginn-Ereignissen [in min] (SysParam: -5).

Schichtfreie Zeitspanne nach Produktionsende-Ereignissen [in min] (SysParam:-6).

Sortiernummer (optional)

Mehrere Variable können mit Angabe einer Sortiernummer aufsteigend sortiert werden.

Wert (Bool / num / str)

Vorgabe des Variablenwerts, den die in der Spalte "Variable" definierte Variable mit dem Schaltbefehl annehmen soll. Der Wert ist entsprechend dem Typ der Variablen in dessen Wertfeld einzutragen.

Schichttyp (optional)

Lookup-Liste zur Auswahl eines Schichttyps. Schaltbefehle können mit einem Schichttyp verknüpft werden, um ihn nur bei einem Ereignis dieser Schicht zu aktivieren.

Pausentyp (optional)

Lookup-Liste zur Auswahl eines Pausentyps. Schaltbefehle können mit einem Pausentyp verknüpft werden, um ihn nur bei einem Ereignis dieses Pausentyps zu aktivieren.

Ohne Feiertage

Checkbox. 1 = Feiertage (Einträge in der Tabelle "Feiertage") werden ausgeschlossen (d.h., Schaltbefehle werden an Feiertagen nicht ausgeführt).

0 = Schaltbefehle werden jeden Tag ausgeführt.

Wochentage

Wochentage, an denen dieser Schaltbefehl ausgeführt werden soll.

Durch Komma getrennte Liste von Wochentagen oder Bereichen davon.

Einzelne Wochentage können als Zahl (1 für Montag bis 0 oder 7 für Sonntag) oder als Abkürzung des englischen Namens (Sun, Mon, Tue, Wed, Thu, Fri, Sat) angegeben werden.

Bereiche werden durch ein Minus-Zeichen (-) gekennzeichnet und enthalten beide Grenzen, also steht 'mon - 3' für Montag, Dienstag und Mittwoch.

Aktueller Wert

Hier wird der aktuelle Wert der Variablen angezeigt. Nur zur Information; schreibgeschützt.

Letztes Update

Hier wird der Zeitstempel der letzten Wertänderung der Variablen angezeigt. Nur zur Information; schreibgeschützt.

 

Verwendung

Eine typische Anwendung dieser Funktion ist z.B. das Einschalten von Hilfsaggregaten vor Schichtbeginn, Starten und Stoppen von Montagebändern mit Schichtbeginn /-ende und Pausenende /-beginn.

Ereignisse

Diese Ereignisse können in der Spalte "Event type" ausgewählt werden, um den Schaltbefehl zu triggern.

 

Ereignis

Beschreibung

Schichtbeginn

Ereignis zum Zeitpunkt des Schichtanfangs aus dem Zeitmodell.

Schichtende

Ereignis zum Zeitpunkt des Schichtendes aus dem Zeitmodell.

Pausenbeginn

Ereignis zum Zeitpunkt eines Pausenanfangs aus dem Zeitmodell.

Pausenende

Ereignis zum Zeitpunkt eines Pausenendes aus dem Zeitmodell.

Produktionsbeginn

Ereignis zum Beginn der ersten Schicht nach einer produktionsfreien Zeit.

Die Zeit wird mit dem Systemparameter -5 definiert.

Mindestlänge der produktionsfreien Zeit bevor ein Produktionsbeginn ausgelöst wird (Systemparameter -5). Wenn kein Schaltbefehl angegeben wird, wird der Wert vom Systemparameter verwendet.

Produktionsende

Ereignis zum Ende der letzten Schicht vor Beginn einer produktionsfreien Zeit.

Die Zeit wird mit dem Systemparameter -6 definiert.

Mindestlänge der produktionsfreien Zeit bevor ein Produktionsende ausgelöst wird (Systemparameter -6). Werte in dieser Spalte überschreiben festgelegte Systemparameterwert. Wenn kein Wert für den Schaltbefehl angegeben wird, wird der Wert vom Systemparameter verwendet.

Tagesbeginn

Ereignis um 00:00:00 Uhr.

Beginn erste Schicht

Ereignis zum Zeitpunkt des ersten Schichtbeginns eines Tages.

Ende letzte Schicht

Ereignis zum Zeitpunkt des Schichtendes der letzten Schicht eines Tages.

Tagesbeginn ohne Produktion

Ereignis um 00:00:00 Uhr, wenn für den beginnenden Tag keine Produktion geplant ist.

Tagesbeginn mit Produktion

Ereignis um 00:00:00 Uhr, wenn für den beginnenden Tag mindestens eine Produktionsschicht geplant ist.

manuell

Ereignis wird nicht automatisch eingetragen (Eintrag wird in der  WebApp Ansicht Schaltbefehle vom Benutzer manuell ausgelöst).

 

Informationen zur produktionsfreien Zeit:

Die Definition der produktionsfreie Zeit wird mit den Parametern -5 und -6 eingestellt.

 

Beispiel

30 Minuten vor Beginn der Produktion sollen Hilfsaggregate eingeschaltet werden, und zum Produktionsbeginn das Montageband anlaufen. Während der Pausen soll das Band angehalten werden, zum jeweiligen Schichtende die Stückzähler in den Steuerungen genullt werden.

Die Schaltbefehle werden mit den Ereignissen verknüpft und für die Schaltungen boolesche Variable gesetzt bzw. zurückgesetzt. Eine Variable Iststückzahl wird mit dem Ereignis Schichtende auf den Wert <0> gesetzt. In Gateway und OPC-Server-Konfiguration muss diese Variable als READWRITE konfiguriert sein, damit der Wert auch in den SPS genullt werden kann.
