Funktion

Zuordnung von Störmeldungen und Texten zu einem Knoten. Wird in der Regel durch den Import der SPS-Zuordnungslisten automatisch generiert.

Jede Meldung wird fest einem reellen Knoten der Fertigungsstruktur zugeordnet (entspricht standardmäßig dem "Standardknoten" der Verbindung). Auch einzelne Meldungen lassen sich gezielt mit Knoten verbinden.

Die zugeordneten Meldungen können hier nochmals geprüft und ggf. Meldungseigenschaften angepasst werden, z.B. Klasse oder Art.

Feld

Beschreibung

ID

Eindeutige Identifikationsnummer. Wird vom System vergeben. Schreibgeschützt.

Verbindung

Lookup-Liste zur Auswahl der Verbindung aus dem Datenaustausch, für die die Meldung parametriert ist.

Knoten

Lookup-Liste zur Auswahl des Knotens, für den die Meldung parametriert ist. Wird automatisch befüllt und kann geändert werden.

Meldungsnummer

Meldungsnummer. Muss innerhalb dieser Verbindung eindeutig sein (z.B. Code des Störoperanden im Steuerungssystem).

Operand (optional)

SPS-Operand (falls Operandenbezug vorhanden) der Meldung.

Symbol (optional)

SPS-Symbolik (falls Operandenbezug vorhanden) der Meldung.

Text

Meldetext bis max. 80 Zeichen Länge.

Meldeklasse

Lookup-Liste zur Auswahl einer vorhandenen Klasse zur Klassifizierung der Meldung.

Meldeart

Lookup-Liste zur Auswahl einer vorhandenen Melde-Art. Meldearten sind ein weiteres Merkmal von Meldungen und können entsprechend ausgewertet werden.

Default-Begründung (optional)

Lookup Liste, um eine Default-Begründung für diese Meldungsdefinition auszuwählen. Wird hier ein Eintrag ausgewählt, wird diese Begründung als Default für Meldungen dieser Definition in der WebApp verwendet.

: In der WebApp können nur Einträge ohne weitere Untereinträge als Filter verwendet werden, keine Parents, z.B. in der Pareto-Ansicht!

Weiterleitungsart (optional)

Lookup-Liste zur Auswahl einer Weiterleitungsart für den Funkruf (siehe Kapitel Funkruf).

Typische Weiterleitungsarten sind:

keine Weiterleitung

Erstwerterkennung (1. Meldung wird weitergeleitet, Folgemeldungen werden ausgeblendet)

generelle Weiterleitung (alle Meldungen werden weitergeleitet).

Überschreibt die indirekte (default) Weiterleitungsart der zugewiesenen Meldeklasse (Defaulteinstellung unter Meldungsmanagement | Meldeklassen – Spalte Weiterleitungsart).

Funkruf-Empfänger (optional)

Auswahlfeld zur Zuordnung eines im Meldungsmanagement definierten Funkruf-Empfängers (siehe Abschnitt Meldungsmanagement).

Funkruf-Priorität (optional)

Priorität der Meldung für den Funkruf (unabhängig von der Meldungs-Priorität).
