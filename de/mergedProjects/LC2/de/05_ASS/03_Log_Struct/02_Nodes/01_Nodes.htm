Funktion

In dieser Tabelle können neue Knoten erstellt werden. Die Knoten können hierarchisch strukturiert werden.

Feldbeschreibung

 

Feld

Beschreibung

ID

Eindeutige Identifikationsnummer. Wird vom System vergeben. Schreibgeschützt.

Parent (optional)

Lookup-Liste zur Auswahl eines übergeordneten Knotens.

Kurzname (optional)

Symbolische Kurzbezeichnung des Knotens.

Name

Knotenname.

Beschreibung (optional)

Zusätzliche Beschreibung des Knotens.

Hierarchiestufe

Lookup-Liste zur Auswahl der Hierarchiestufe des Knotens aus der Grundkonfiguration.

Zeitschema (optional)

Lookup-Liste zur Auswahl des Zeitschemas, das für diesen Knoten Verwendung findet, zur Zuordnung von Schicht- und Pausenzeiten.

Werden für untergeordnete Knoten andere Zeitschemata verwendet, so erscheint eine Abfrage, ob das neue Zeitschema auch für die untergeordneten Knoten geändert werden soll oder nicht. Falls nicht, wird es nur für den aktuell selektierten Knoten geändert.

Eine Änderung des Zeitschemas wird ab dem folgenden Produktionstag nach dem nächsten Lauf des zyklischen AE Jobs "Ereignisse generieren" wirksam (im Standard läuft dieser Job jede Stunde).

Knotentyp (optional)

Lookup-Liste zur Auswahl eines unter Grundkonfiguration definierten Knotentyps.

Stückzahl-Faktor

Faktor für Sollwertvorgaben vom Typ Sollstückzahl. Wird die Sollwertvorgabe vom übergeordneten Knoten vererbt, kann dieser für den eigenen Knoten mit dem Faktor multipliziert werden. Default ist 1.

Anwendung z.B. bei der Aufteilung eines Montagebandes auf zwei parallele Teilbänder mit je 0.5 * Sollstückzahl des Gesamtmontagebandes.

Siehe auch das Kapitel "Automatische Sollwertberechnung".

Stückzahl-Offset

Offset für die Sollwertverarbeitung. Default ist 0.

Verwendung, wenn eine feste Stückzahl z.B. für die zerstörerische Materialprüfung ausgeschleust werden soll.

Siehe auch das Kapitel "Automatische Sollwertberechnung".

Taktzeit-Faktor (optional)

Faktor für Sollwertvorgaben vom Typ Solltaktzeit. Wird die Sollwertvorgabe vom übergeordneten Knoten vererbt, kann dieser für den eigenen Knoten mit dem Faktor multipliziert werden. Default ist 1.

Anwendung z.B. bei der Aufteilung eines Montagebandes auf zwei parallele Teilbänder mit je 0.5 * Sollstückzahl des Gesamtmontagebandes.

Siehe auch das Kapitel "Automatische Sollwertberechnung".

Taktzeit-Offset (optional)

Offset für die Sollwertverarbeitung. Default ist 0. Siehe auch das Kapitel "Automatische Sollwertberechnung".

Sortiernummer (optional)

Sortierreihenfolge. Mehrere Unterelemente auf gleicher Knotenebene können mit Angabe einer Sortiernummer aufsteigend sortiert werden. Die Angabe dient der Strukturierung des Hierarchiebaums vor allem in der Anzeige des WebClients.

Ausgeblendet?

Checkbox. Soll der Knoten in der WebApp ausgeblendet werden? 0 = Nein (Standard), 1 = Ja.

Daten werden auch für ausgeblendete Knoten erfasst.

Ausgeblendete Knoten werden in der WebApp weder im Fertigungsbaum noch in den Filtern angezeigt.

Display-Optionen

Anzeige-Optionen für generische Layouts.

Funkruf inaktiv

Funkrufweiterleitung inaktiv setzen.

Erstwertmeldungs-Ebene

Ebene in Fertigungsbaum, wo die Erstwertmeldung für den Funkruf zu suchen ist.

Sende-Verzögerung

Verzögerung der Funkrufweiterleitung in Sekunden.

Rücksetzen-Verzögerung

Verzögerung des Zurücksetzens in Sekunden.

Empfänger

FK - Verweis auf einen Funkruf-Empfänger.

LGD Dateipfad

Referenz auf eine XML-Datei, die für diesen Knoten im Legato Graphic Designer Boardlet visualisiert werden soll, z.B. Node1.xml. Wenn für den Knoten keine Datei angegeben ist, wird die des nächsthöheren Knotens verwendet.

Datei muss im Verzeichnis \SapiEng_xxx\web\app\custom\detail-layout\ gespeichert sein

Erw. Eigenschaften

Die definierten erweiterten Eigenschaften werden hier als eigene Spalten angehängt und können mit Werten versehen werden.

Die Parametrierung von erweiterten Eigenschaften ist in einem separaten Kapitel beschrieben, zu finden hier: Erweiterte Eigenschaften.

 

Kontextmenü

Layout-Snippet erstellen (Meldungsstatus)

Erstellt ein Layout-Snippet mit den Databinding-Informationen des selektierten Knotens (Mehrfachauswahl möglich) und kopiert diese Info in die Zwischenablage. Es wird eine Zeile je selektiertem Knoten erstellt. Aus der Zwischenablage kann die Information in eine Visualisierung eingefügt werden, um Meldungsstatus für Knoten anzubinden. Für weitere Informationen, siehe bitte Kapitel Erweiterte Bedienung - Legato Graphic Designer.
