Platzhalter im Meldetext

Meldungen können neben einem statischen Meldetext auch einen dynamisch angepassten Meldetext enthalten. Der dynamische Teil des Textes wird durch einen oder mehrere Begleitwerte bestimmt. Dabei geben im vordefinierten (statischen) Meldetext eingefügte Formatanweisungen an, wie der jeweilige Begleitwert zu interpretieren ist. Die Interpretation erfolgt gemäß der in Legato importierten Zuli (siehe auch Meldungsdefinitionen: Import-Einstellungen…)

Für eine Meldung, die keinen Begleitwert enthält, kann der definierte Meldetext ohne weitere Verarbeitung verwendet werden.

Meldetext: FG00 Schütz-Fehler Versorgung 400V Bänder Pos 650-690 M_+LS01-KF403 (M3005.3)

 

Für eine Meldung mit Begleitwert muss der Meldetext analysiert werden und jede Formatanweisung mit dem entsprechenden Begleitwert interpretiert werden. Jedes Vorkommen einer Formatanweisung wird durch die Interpretation mit dem zugehörigen Begleitwert ersetzt.

Beispiel: Meldetext mit interpretierter Formatanweisung für den Begleitwert '11':

Meldetext: FG01 SWE M_+6100-M2 Fehlercode @1X%4d@ @1X%tSWE@ (M3016.5)

Begleitwert: 11

Formatierter Meldetext: FG01 SWE M_+6100-M2 Fehlercode 11 Übertemperatur (M3016.5)

 

Begleitwert Formatanweisungen (PDIAG Formaloperanden)

Die Legato Formatanweisungen sind eine Erweiterung der PDIAG-Formaloperanden. Es werden alle (bekannten) PDIAG Formate unterstützt.

Die verschiedenen Möglichkeiten werden im Folgenden kurz beschrieben. Für weitere Informationen zum Thema Begleitwerte, siehe bitte auch das entsprechende Kapitel im Dokument "Legato_Database".

Numerische Konvertierung

Der Begleitwert kann als Zahl formatiert werden. Das Schema für die numerische Konvertierung eines Begleitwerts kann wie folgt aussehen:

´@´BEGLEITWERTZAHL(´X´|´W´)´%´[STELLEN]FORMATZEICHEN´@´

 

BEGLEITWERTZAHL := Ganzzahl n, welche den n. Begleitwert bezeichnet, auf den sich das anschließendes Format bezieht

STELLEN := Ganzzahl i, welche die Anzahl Stellen des auszugebenden Nummernformates angibt.

FORMATZEICHEN := ´d´|´u´|´X´|´b´

d:    Dezimal mit Vorzeichen mit i Stellen

u:    Dezimal ohne Vorzeichen mit i Stellen

X:    Hexadezimal mit i Stellen

b:    Binär mit i Stellen

 

Beispiel: Der zweite Begleitwert wird hexadezimal mit 8 Stellen formatiert ausgegeben

Formatstring:   @2W%8X@

Begleitwerte:   5,31,"Störung"

Ergebnis:       0000001F

Textbibliothek nachschlagen

Für Zulis vom Typ PDiag xml und csv können die Textbibliotheken mit importiert werden. Für andere PDiag Zulis müssen die Textbibliotheken mit ihren Inhalten zum Teil manuell in den LC2 importiert werden (siehe "Meldungsmanagement" - "Textbibliotheken" - "Inhalt der Textbibliothek").

Der Begleitwert gibt einen Eintrag in einer Textbibliothek an, der an entsprechender Stelle eingesetzt werden soll. Das Schema sieht beispielsweise so aus:

´@´BEGLEITWERTZAHL(´X´|´W´)´%t#´BIBLIOTHEKNAME´@´

 

BEGLEITWERTZAHL := Ganzzahl n, welche den n. Begleitwert bezeichnet, auf den sich das anschließendes Format bezieht

BIBLIOTHEKNAME := Name der Bibliothek, in der der Eintrag mit dem Index des Begleitwertes nachgeschlagen werden muss

 

Im unten stehenden Beispiel wird der erste Begleitwert zum Nachschlagen in der Textbibliothek Roboter verwendet, aus der der 5. Eintrag verwendet wird.

Formatstring:   @1X%t#Roboter@

Begleitwerte:   5,31,"Störung"

Ergebnis:       Greifarm blockiert

 

Text ersetzen

Der Begleitwerttext kann ohne weitere Bearbeitung an gewünschter Stelle in den Meldetext eingesetzt werden.

´@´BEGLEITWERTZAHL[(´X´|´W´)´%t´]´@´

 

BEGLEITWERTZAHL := Ganzzahl n, welche den n. Begleitwert bezeichnet, auf den sich das anschließendes Format bezieht

 

Beispiel: Der Formatstring wird direkt mit dem dritten Begleitwert ersetzt:

Formatstring:   @3X%t @

Begleitwerte:   5,31,"Störung"

Ergebnis:       Störung

 

Beispiel: Zulässige abgekürzte Schreibweise für die Textersetzung:

Formatstring:   @3@

Begleitwerte:   5,31,"Störung"

Ergebnis:       Störung

 

Implizite Konvertierung

Begleitwerte können auch ohne Formatanweisung in den Meldetext eingefügt werden. Diese implizite Konvertierung findet immer dann statt, wenn zwar Begleitwerte übergeben wurden, aber keine Formatanweisungen im Meldetext gefunden werden. In diesem Fall wird die Begleitwertliste als einfache CSV-Liste an den Meldetext angehängt.

:

Ausnahme bildet ein einzelner Begleitwert mit dem Wert 0. Eine 0 ohne Formatanweisung wird ignoriert und nicht angefügt.
