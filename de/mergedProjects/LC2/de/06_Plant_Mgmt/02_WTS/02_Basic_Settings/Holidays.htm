Funktion

Liste der Feiertage. Diese werden mit genauem Datum (z.B. 09.05.2013) eingetragen. Über die Spalte "Jährl." wird angegeben, ob der Feiertag jedes Jahr am selben Datum wiederholt wird, z.B. 25.12., Weihnachten.

Feld

Beschreibung

ID

Eindeutige Identifikationsnummer. Wird vom System vergeben. Schreibgeschützt.

Name

Name des Feiertags.

Datum

Datum des Feiertags. Muss immer im Format yyyy-MM-DD eingegeben werden.

Jährlich

Checkbox. Gibt an, ob der Feiertag jährlich wiederholt wird (1) oder nicht (0).

Zeitzone

Falls gewünscht, Zeitzone (gemäß Java Zeitzonen, z.B. Europe/Berlin) für den Feiertag; z.B. wenn der Server der SE in einer anderen Zeitzone steht. Wenn leer, wird die Zeitzone des SE-Servers für Berechnungen (z.B. Schichtplanung, Rufweiterleitung etc.) verwendet.
