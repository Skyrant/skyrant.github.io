Funktion

Ein Knoten ist ein Element in der logischen Fertigungsstruktur von Legato. Die Darstellung der Knoten erfolgt in der Baumstruktur des Webclients und dient der Navigation. Für jeden Knoten kann eine beliebige Anzahl von Unterelementen (weitere Knoten) definiert werden. So entsteht die hierarchisch gegliederte logische Struktur der Fertigung. Durch die Definition von Knotentypen besteht die Möglichkeit, bezogen auf eine bestimmte Hierarchieebene zwischen verschiedenen Typen von Elementen auf dieser Ebene zu unterscheiden. Z.B. könnten für die Ebene "Anlagen" die Knotentypen Fertigungsanlage und Förderer unterschieden werden.

Checkbox. Wenn aktiv, kann für diesen Knotentyp ein Schichtprotokoll erfasst werden.

Verfügbarkeits-relevant (optional)

Checkbox. Wenn aktiv, wird dieser Knotentyp für statistische Berechnungen berücksichtigt, z.B. KPIs.
