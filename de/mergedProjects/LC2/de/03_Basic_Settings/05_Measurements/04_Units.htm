Funktion

Die Vorgabe von Einheiten dient der Darstellung von Prozesswerten im WebClient.

In der Übersicht des Clients können Prozesswerte mit einer zugehörigen Einheit dargestellt werden. Für jede verwendete Einheit muss in dieser Tabelle eine Bezeichnung und eine Kurzbezeichnung (wird im Client visualisiert, z.B. °C, min, s oder Stk.) angegeben werden.

Feldbeschreibung

 

Feld

Beschreibung

ID

Eindeutige Identifikationsnummer. Die ID kann frei vergeben werden, muss jedoch eindeutig sein.

Einheits-Symbol

Symbol der Einheit, das auch im WebClient angezeigt wird (z.B. °C, min, %)

Name

Name der Einheit (verwendet für die Auswahl bei der Parametrierung).

Parent

Lookup-Liste zur Auswahl eines übergeordneten Eintrags.

Faktor

Berechnungsfaktor, der das Verhältnis der Einheit zum Parent-Eintrag beschreibt. Beispiel für die Einheit mm (Millimeter): Parent ist m (Meter), Faktor ist 1000.
