Kontextmenü

Durch einen Klick mit der rechten Maustaste auf eine Person entweder im Navigationsbaum oder am Anfang der Tabellenzeile öffnet sich ein Kontextmenü. Darüber kann das Benuterkonto der Person erstellt oder geändert werden oder das Passwort geändert werden.

 

Konto: Passwort setzen

Gibt es zu dieser Person genau einen Benutzeraccount, so öffnet sich ein Dialog zum Ändern des Passworts. Der Benutzername ist voreingestellt und kann hier geändert werden. Das neue Passwort kann eingetragen werden und muss mit einem zweiten Eintrag bestätigt werden. Durch Klick auf OK wird das Passwort verschlüsselt beim jeweiligen Benutzer gespeichert.

Gibt es zu dieser Person mehrere Benutzeraccounts, so ermöglicht es eine Abfrage, zur Benutzer-Liste zu wechseln und dort den entsprechenden Eintrag zu selektieren.

Existiert noch kein Benutzeraccount zu dieser Person, so erscheint eine entsprechende Meldung. Es kann dann entweder ein neues Benutzerkonto angelegt werden oder der Vorgang abgebrochen werden.

 

Konto: Einstellungen bearbeiten

Gibt es zu dieser Person genau einen Benutzeraccount, so öffnet sich ein Dialog, in dem die entsprechenden Einstellungen editiert werden können.

Gibt es zu dieser Person mehrere Benutzeraccounts, so ermöglicht es eine Abfrage, zur Benutzer-Liste zu wechseln und dort den entsprechenden Eintrag zu selektieren.

Existiert noch kein Benutzeraccount zu dieser Person, so erscheint eine entsprechende Meldung. Es kann dann entweder ein neues Benutzerkonto angelegt werden oder der Vorgang abgebrochen werden.
