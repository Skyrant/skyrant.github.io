Funktion

Alle verfügbaren Boardlets mit Tags, die bei der Suche verwendet werden können. Die Tabelle dient nur zum Zuweisen von Schlüsselwörtern. Es ist nicht möglich, Einträge hinzuzufügen oder zu löschen. Diese Aktionen können nur im Developer oder System-Developer Mode ausgeführt werden.

Feld

Beschreibung

ID

Vom System vergebene, eindeutige ID. Schreibgeschützt.

Name

Name des Boardlets in der jeweiligen Oberflächensprache.

Beschreibung (optional)

Beschreibung des Boardlets in der jeweiligen Oberflächensprache.

Tags (optional)

Tags / Schlüsselwörter für die Suche nach dem Boardlet. Komma-separierte Liste. Leerzeichen werden bereinigt.

Kategorie

Kategorie, in der das Boardlet angezeigt wird.

Default-Titel (optional)

Default-Titel, mit dem das Boardlet in der Library angezeigt wird.

Veröffentlicht

Checkbox. 1 = veröffentlicht

0 = nicht veröffentlicht (und somit nicht sichtbar).

Default ist 0.

System-Updates

Checkbox. Sind für diese Spalte System-Updates erlaubt? 0 = nein, 1 = ja.

Wenn 0, ist diese Spalte nicht Teil des Installationsskripts.

Default ist 0.

 

Deaktivieren der Boardlets

In der Boardlet Library können Boardlets deaktiviert werden. Hierdurch ist es möglich, ungenutzte Module für den Anwendungsbereich auszublenden. Versehentliches und/oder möglicherweise falsches Verwenden dieser ausgeblendeten Module können somit vermieden werden.

Die Boardlet-Kategorien von auswählbaren Sapient-Modulen und Boardlets der entsprechenden Boardlet-Kategorien sind in der Boardlet-Bibliothek des Dashboard-Designers nicht sichtbar.

 

Es müssen hierzu die "veröffentlicht"-Einstellungen der Boardlets in der Tabelle gp_r_boardlets im LC2 auf false (Kontrollkästchen deaktiveren) gesetzt werden.

Gehen Sie wie folgt vor, um im LC2 Boardlets zu deaktivieren:

 

  1. Navigieren Sie im Navigationsbereich des LC2 auf Frontend > Boardlets, damit die Tabelle gp_r_boardlets geöffnet wird.

  2. Öffnen Sie die Spalte "Kategorien". Klicken Sie herzu auf die Spaltenüberschrift "Kategorien" und ziehen Sie diese per Drag-and-Drop in den oberen Bereich "Boardlets".

  3. Sie erhalten eine Übersicht der Kategorien der Boardlets mit z.B. "Kategorie: 09 - TPM".

  4. Falls die Spalte "Veröffentlicht" nicht sichtbar ist, scrollen Sie in der Tabelle nach hinten und ziehen Sie die Spalte "Veröffentlicht" in den sichtbaren Bereich der Tabelle.

  5. Öffnen Sie anschließend eine Kategorie der Boardlets (z.B. "Kategorie: 09 - TPM"), indem Sie in der Zeile auf den Pfeil klicken.

  6. In der Spalte "Veröffentlicht" wird durch ein Häckchen im Kästchen angezeigt, ob das jeweilige Boardlet in der Kategorie sichtbar ist oder nicht.

  7. Aktivieren und/oder deaktivieren Sie "Veröffentlicht", in dem Sie ein Häcken setzen oder dieses entfernen.

  8. Im Dashboard-Editor werden anschließend die verschiedenen Boardlets angezeigt oder eben nicht.
