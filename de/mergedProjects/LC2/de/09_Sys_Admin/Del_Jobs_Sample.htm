Lösch-Job Beispiel

Beschreibung

Im Folgenden wird beispielhaft beschrieben, wie ein Lösch-Job für bestimmte Meldeklassen mit einer kürzeren Haltedauer als im Standard-Job eingerichtet werden kann. Basierend darauf kann geschultes Personal weitere projekt-spezifische Skripte zum Löschen erstellen. Diese sollten immer in folgendem Ordner abgelegt werden:

"..\sapient-engine\src\dist\scripts\jobs\table_del"

:

Gibt es für eine Tabelle einen Standard-Löschjob, so können Ausnahmen nur für kürzere Haltedauern als dort definiert erstellt werden. Der Standard-Job sollte also immer die größte benötigte Haltedauer berücksichtigen.

Damit der Javascript-Job korrekt ausgeführt wird, muss ein neuer Eintrag in die Tabelle SYS_TABLE_DEL eingefügt werden. Dabei muss der korrekte Pfad zum Skript in der Spalte "Lösch-Funktion" eingetragen werden. Im Beispiel unten liegt die Datei unter <code>scripts/jobs</code>. In der Spalte "Parameter" werden dem Job die Meldeklassen 99 und 98 mit einer Haltedauer von 168 Stunden übergeben.

:

Projektspezifische Skripte können nicht auf Daten in den anderen Spalten der Tabelle SYS_TABLE_DEL zugreifen. Das heißt, Angaben zu Intervall, Haltedauer etc. müssen im Skript definiert werden, sie können nicht aus den entsprechenden Spalten gelesen werden.

Beispiel Insert-Statement

Mit folgendem Statement kann der Job in die Datenbank eingefügt werden. Natürlich ist auch die Parametrierung über den LC2 möglich.

<pre>

insert into SYS_TABLE_DEL (TABLE_NAME, COLUMN_NAME, PRIORITY, SE_JOB_SYMBOL, INTERVAL_MINS, HOLD_HOURS,

    DEL_JOB_FUNCTION_PATH, DEL_JOB_PARAMS, OPTIONS, DEL_PORTION_PERC,

    DESC_INTERNAL)

select td.TABLE_NAME, td.COLUMN_NAME, td.PRIORITY - 1, td.SE_JOB_SYMBOL, td.INTERVAL_MINS, 0,

    'jobs/table_del/table_del_AL_ARCHIVE_al_class/execute', '{"AL_CLASS": [99,98], "HOLD_HOURS": 168}', '2', 0,

    'Project-specific deletion of table AL_ARCHIVE with respect of column AL_CLASS'

  from SYS_TABLE_DEL td

 where td.TABLE_NAME = 'AL_ARCHIVE'

   and td.PRIORITY = 50;

</pre>

Beispiel JavaScript

Hier finden Sie ein Beispiel für die JavaScript-Datei. Diese kann bei Bedarf auch kopiert und entsprechend angepasst werden.

function /*int*/ execute(/*SapientCallContext*/ ctx, /*JSON map*/ params) {

    logger.info("[TableDel_AL_ARCHIVE] started to execute");

    logger.debug('[TableDel_AL_ARCHIVE] params: ' + JSON.stringify(params));

   

    var con = null;

    var qry = null;

    var affectedRows = 0;

    var tableName = "AL_ARCHIVE";

   

    try {

        // Get writable connection to Sapient DB

        con = new DbCon(ctx, DbCon.LEGATODB, true);

        var dbType = con.dbtype;

        logger.debug('[TableDel_AL_ARCHIVE] dbType: ' + dbType);

       

        var rowid = "";

        if ("postgresql" == dbType) {

            rowid = "ctid";

        } else if ("oracle" == dbType) {

            rowid = "rowid";

        }

       

        var sql = "select " + rowid + " from " + tableName + " where TIME_START < current_timestamp - interval '";

       

        if (params) {

            if (params["HOLD_HOURS"] != undefined) {

                sql += params["HOLD_HOURS"] + "' hour";

            }

            if (params["AL_CLASS"] != undefined) {

                sql += " and AL_CLASS in (" + params["AL_CLASS"] + ")";

            }

        } else {

            var msg = "[TableDel_AL_ARCHIVE] no parameters provided; please provide at least parameter HOLD_HOURS";

            logger.error(msg);

            throw(msg);

        }

       

        // only delete 100k records at once

        sql += getLimit(dbType, 100000);

        sql = "delete from " + tableName + " where " + rowid + " in (" + sql + ")"

        logger.debug('[TableDel_AL_ARCHIVE] executing SQL: ' + sql);

       

        qry = new DbQuery(con, sql, null);

        affectedRows = qry.execute();

        logger.info("[TableDel_AL_ARCHIVE] Successfully deleted " + affectedRows + " rows");

    } finally {

        if (qry != null) qry.close();

        if (con != null) con.close();

    }

   

    return affectedRows;

}

 

 

function /*String*/ getLimit(/*String*/ dbType, /*int*/ rows) {

    if ("postgresql" == dbType) {

        return " limit " + rows;

    } else if ("oracle" == dbType) {

        return " fetch first " + rows + " rows only";

    } else {

        // no limitation

        return "";

    }

}
