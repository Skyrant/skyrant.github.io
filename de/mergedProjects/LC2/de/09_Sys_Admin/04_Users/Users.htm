Funktion

In dieser Tabelle sind alle Legato-Benutzer aufgeführt. Die Einträge werden automatisch über das Kontextmenü "Benutzerkonto anlegen" bei "Stammdatenverwaltung - Personen" erstellt. Es können hier keine Einträge hinzugefügt werden.

Feldbeschreibung

 

Feld

Beschreibung

ID

Eindeutige ID.

Benutzer-Kennung

Benutzer-Kennung.

PW-Hash

Verschlüsseltes Passwort des Benutzers. Wird beim Anlegen automatisch verschlüsselt. Nur zur Information, schreibgeschützt.

ERP-Benutzer

Checkbox. 1 = Benutzer existiert auch im ERP. 0 = Benutzer existiert nicht im ERP.

ERP-ID (optional)

ID des Benutzers im ERP. Nur sinnvoll, wenn ERP-Benutzer = 1 ist.
Person

Lookup-Liste zur Auswahl einer Person, für die der Benutzer erstellt wurde. Nur zur Information, schreibgeschützt.
Ablaufdatum (optional)

Datum, an dem der Benutzer abläuft. Danach kann der Benutzer sich nicht mehr am System anmelden. Wird kein Datum eingetragen, ist der Benutzer unbegrenzt gültig.
Autorisierungs-Code (optional)

Autorisierungs-Code des Benutzers, der es ihm ermöglicht, Aktionen mit "Autorisierung erf." = 1 auszuführen.
ID SSO

ID des Benutzers im externen Single-Sign-On (SSO) System. Nur zur Information, schreibgeschützt.
Unsicheren Login erlauben?

Checkbox. Wenn 1, wird auch unsicherer Login erlaubt, wenn 0 nicht.
