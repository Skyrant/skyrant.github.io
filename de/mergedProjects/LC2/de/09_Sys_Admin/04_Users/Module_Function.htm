Funktion

Hier können Berechtigungen für Funktionen innerhalb des LC2 vergeben werden. Berechtigungen können von jedem Benutzer mit der entsprechenden Berechtigung vergeben werden. Die Tabelle selbst kann nur im System Developer Mode editiert werden.

Die Tabelle enthält 4 Basis-Funktionen für den LC2:

"LC2 benutzen":

Ermöglicht es dem entsprechenden Benutzer, den LC2 zu verwenden. Hat der Benutzer diese Berechtigung nicht, kann er sich nicht am LC2 anmelden - der LC2 wird geschlossen. Nachdem auch ältere LC2 Versionen diese Funktionen verwenden, wurde sicher gestellt, dass auch ältere LC2 Versionen sich nicht mit neueren Sapient-Systemen verbinden können, wenn die Berechtigung fehlt.

"LC2 Developer Mode benutzen":

Nach erfolgreicher Authentifizierung und dem Versuch, den LC2 im Developer Mode (Verbindungseinstellungen) zu öffnen, werden die entsprechenden Berechtigungen geprüft. Hat der Benutzer keine Berechtigung für diesen speziellen Mode, wird der Login beendet.

"LC2 System Developer Mode benutzen":

Nach erfolgreicher Authentifizierung und dem Versuch, den LC2 im System Developer Mode (Verbindungseinstellungen) zu öffnen, werden die entsprechenden Berechtigungen geprüft. Hat der Benutzer keine Berechtigung für diesen speziellen Mode, wird der Login beendet.

"SQL Fenster benutzen":

Der Ribbon-Button um den SQL-Screen zu öffnen wird abhängig von dieser Berechtigung angezeigt.

 

:

Um diese neuen Funktionen zu verwenden, müssen die Berechtigungen den Benutzern zugewiesen werden. Nachdem initial nach dem Update auf die aktuelle Version keine Berechtigungen existieren, kann sich nur der sapient_sys Benutzer am LC2 anmelden - dieser Benutzer kann den LC2 generell im Developer oder System Developer Mode benutzen. Er kann dann die "LC2 benutzen" Berechtigung für die gewünschten Benutzer vergeben.
