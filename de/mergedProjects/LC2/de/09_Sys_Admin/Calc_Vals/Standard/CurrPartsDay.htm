CurrPartsDay

Beschreibung

Für einen parametrierten Knoten wird der Kennwert Soll aktuell für den Knoten für den gesamten Produktionstag auf Basis einer Soll-Stückzahl berechnet. Dazu muss in der Fertigungsstruktur der Knoten mit einem Zeitschema verknüpft sein.

Mit Zeitschema und dem aktuellen Schicht- /Pausenmodell aus dem Schicht-Kalender kann das Verhältnis vergangene Schichtzeit zu gesamter Schichtzeit (jeweils als Netto-Zeiten) auf den ganzen Tag berechnet werden. Hierbei wird der Schichtfaktor mit berücksichtigt. Im gleichen Verhältnis stehen Soll aktuell und Soll-Stückzahl gesamt zueinander:

Die Zeiten werden in [sec] berechnet.

Parameterliste

ParamNo 0 – RESULT

Typ: Variable, Datentyp Numerisch

InOut: Out

Multi: 0

Beschreibung: Sollstückzahl des Knotens zum aktuellen Zeitpunkt

ParamNo 1 – NODE

Typ: Knoten

InOut: In

Multi: 0

Beschreibung: Knoten, für den die Soll-Stückzahl berechnet werden soll

ParamNo 2 – PARTS

Typ: Variable, Datentyp Numerisch

InOut: In

Multi: 0

Beschreibung: Soll-Stückzahl des Knotens für den ganzen Produktionstag
