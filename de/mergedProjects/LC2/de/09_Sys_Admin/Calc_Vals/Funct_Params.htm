System Administration | Berechnete Werte | Funktionen | Parameter

Funktion

Unter der jeweiligen Funktion werden die Ein- und Ausgabeparameter angezeigt.

Feldbeschreibung

 

Feld

Beschreibung

ID

Vom System vergebene, eindeutige ID. Schreibgeschützt.

Funktion

Funktion, auf die sich die Parameter beziehen. Nur zur Information; schreibgeschützt.

Param-Nr.

Interne Nummer des Parameters innerhalb der Funktion.

Param-Name

Name des Parameters.

Beschreibung (optional)

Zusätzliche Beschreibung des Parameters.

Parameter-Typ

Parameter-Typ. Es gibt folgende Typen:

·         NODE: Knoten-ID, ohne Trigger-Event

·         VALUE: Variablen-ID

·         AL_NODE: Knoten-ID, Trigger auf Meldungen

·         SHIFT_NODE: Knoten-ID, Trigger auf Schichten

·         AL_SH_NODE: Knoten-ID, Trigger auf Schichten & Meldungen

In/Out

Dieses Feld gibt an, ob es sich um einen In-, Out- oder In/Out-Parameter handelt.

Multi

Checkbox. 0 = diesen Parameter kann es nur einmal geben, z.B. Ergebnis. 1 = diesen Parameter kann es in beliebiger Anzahl geben, z.B. Summanden.

Datentyp

Datentyp des Werts (numerisch, Bool, String).
