Funktion

Unter der jeweiligen Berechnung werden die Ein- und Ausgabeparameter der aufgerufenen Funktion angezeigt und können beschaltet werden. Der jeweilige Parametertyp kann in der Spalte "Parametertyp" nachgeschlagen werden.

Bei manchen Funktionen gibt es Parameter, die beliebig oft vorkommen können, z.B. bei SUM. Diese sind in der Spalte "Multi" mit 1 gekennzeichnet. Solche Parameter können sooft eingefügt werden, wie sie benötigt werden. Dies kann am besten durch "Zeile duplizieren" erfolgen.

Parameter vom Typ "Knoten-ID" sind mit einem Knoten der Fertigungsstruktur zu beschalten.

Parameter vom Typ "Variablen-ID" werden mit Variablen aus dem Datenaustausch beschaltet. In-Parameter werden in die Funktion zur Durchführung der Berechnung übernommen. Alternativ können dafür auch Konstanten in den Feldern Wert (Bool, numerisch, String) übergeben werden. Out-Parameter werden von der Funktion ausgegeben.

Der oder die Parameter, die die Berechnung anstoßen sollen, müssen als Trigger markiert werden. Zusätzlich muss im Reiter "Trigger" ein ereignisgesteuerter Trigger eingerichtet werden.

Feldbeschreibung

 

Feld

Beschreibung

ID

Vom System vergebene, eindeutige ID. Schreibgeschützt.

Berechnung

Berechnung, auf die sich die Parameter beziehen. Nur zur Information; schreibgeschützt.

Param-Name

Name des Parameters.

Array-Index

Bei Parametern, die mehrfach existieren können (Spalte "Multi" = 1), muss hier die Reihenfolge der Abarbeitung eingetragen werden, beginnend mit 0.

Trigger

Checkbox. Wenn aktiv, dient dieser Parameter als Trigger.

Flag ist nur für die Ausführungs-Trigger "eventgesteuert" oder Schichtereignissen relevant und wird durch die gewählten Variable(n) bzw. Schichtereignisse / Meldungen des Knotens angestoßen.

Variable (optional)

Lookup-Liste zur Auswahl einer Variablen aus dem Datenaustausch. Damit werden Parameter des Typs "Variablen-ID" beschaltet.

Knoten (optional)

Lookup-Liste zur Auswahl eines Knotens aus der Fertigungsstruktur. Damit werden Parameter des Typs "Knoten-ID" beschaltet.

Wert (Bool) (optional)

Konstanter Parameter-Wert Bool. Kann nur 0 oder 1 sein.

Wert (num.) (optional)

Konstanter Parameter-Wert numerisch.

Wert (String) (optional)

Konstanter Parameter-Wert String.

Aktueller Wert

Hier wird der aktuelle Wert des Parameters angezeigt. Nur zur Information; schreibgeschützt.

Letztes Update

Hier wird der Zeitstempel der letzten Wertänderung des Parameters angezeigt. Nur zur Information; schreibgeschützt.

Parameter-Typ

Parameter-Typ. Nur zur Information; schreibgeschützt. Es gibt folgende Typen:

·         NODE: Knoten-ID, ohne Trigger-Event

·         VALUE: Variablen-ID

·         AL_NODE: Knoten-ID, Trigger auf Meldungen

·         SHIFT_NODE: Knoten-ID, Trigger auf Schichten

·         AL_SH_NODE: Knoten-ID, Trigger auf Schichten & Meldungen

In/Out

Dieses Feld gibt an, ob es sich um einen In-, Out- oder In/Out-Parameter handelt. Nur zur Information; schreibgeschützt.

Multi

Checkbox. Wenn aktiv, kann dieser Parameter mehrmals vorkommen (z.B. Summanden). Wenn nicht aktiv, kann dieser Parameter nur einmal existieren (z.B. Ergebnis). Nur zur Information; schreibgeschützt.

Datentyp

Datentyp des Parameters. Nur zur Information; schreibgeschützt.
