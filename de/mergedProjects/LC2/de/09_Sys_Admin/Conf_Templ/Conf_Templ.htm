Funktion

In dieser Tabelle werden die verfügbaren Projektierungsvorlagen referenziert. Die Vorlagen selbst sind als JavaScript-Dateien im entsprechenden SE-Verzeichnis abgelegt.

Feldbeschreibung

 

Feld

Beschreibung

ID

Eindeutige Identifikationsnummer.
Symbol Eindeutiges Symbol der Vorlage.
Name Name der Vorlage.
Beschreibung (optional) Beschreibung der Vorlage.
Eigenschaften

Eigenschaften der Vorlage, z.B. Link zur zugehörigen Skript-Datei im Format: {"script":"create_node"}. Die Skripten müssen im SE-Verzeichnis C:\SapiEng\src\dist\scripts\configurationtemplates abgelegt sein.
Parameter Parameter der Vorlage, entsprechen den zu befüllenden Spalten der Zieltabelle. Für jeden der hier eingetragenen Parameter wird in der Tabelle "Instanzen" eine Spalte angezeigt, die mit dem jeweiligen Wert befüllt werden kann / muss (bei "required": true). Beispiel siehe unten.

 

Parameterdefinition

Zu jedem Parameter können folgende Zusatzinformationen erfasst werden:

caption: Spaltentitel, der für den Parameter in der Tabelle "Vorlagen-Instanzen" angezeigt wird.

dataType: Datentyp des Parameters (z.B. String, Int32, Int64, Double, Boolean). Die Tabellenfelder zeigen die Parameterwerte dementsprechend an und validieren die Eingabe gegebenenfalls (für Boolean-Werte wird z.B. eine Checkbox angezeigt, für lange Strings ein Dropdown-Eingabefeld).

maxLength: (nur für Strings). Längenbegrenzung des Stringwertes.

required: true = Eingabe für diesen Parameter ist erforderlich; false = Eingabe ist optional.

lookupTable: Zieltabelle, sofern der Parameter eine andere Tabelle referenziert. In diesem Fall ist keine direkte Eingabe eines Wertes möglich, sondern es öffnet sich die entsprechende Lookup-Tabelle zur Auswahl eines Eintrags.

 

Beispiel-Eintrag:

{

  "parent_node": {"caption": "Parent node", "dataType": "Int32", "lookupTable": "l_nodes"},

  "parent_con": {"caption": "Parent connection", "dataType": "Int32", "lookupTable": "p_connections"},

  "stringparam1": {"caption": "String param 1", "dataType": "String", "maxLength": 16, "required": true},

  "stringparam2": {"caption": "String param 2", "dataType": "String"},

  "stringparam3": {"caption": "String param 3"},

   "intparam1": {"caption": "Int param 1", "dataType": "Int64"},

   "doubleparam1": {"caption": "Double param 1", "dataType": "Double"},

   "boolparam1": {"caption": "Bool param 1", "dataType": "Boolean"}

}
