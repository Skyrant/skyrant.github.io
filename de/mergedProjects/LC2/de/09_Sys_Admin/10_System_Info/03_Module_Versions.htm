System Administration | Systeminfo | Modulversionen

Funktion

Liste der Modulversionen als Erweiterung aus MES. Als Module gelten Legato Bausteine wie beispielsweise KVP, TPM, KPIs, Zeitmodelle, etc. Die Tabelle dient nur zur Information und ist schreibgeschützt.

Feldbeschreibung

 

Feld

Beschreibung

ID

Vom System vergebene, eindeutige ID.

Modul-ID

Verweis auf das Modul aus SYS_MODULES.

Name

Name der Modulversion; zusammengesetzt aus Modul-Symbol aus SYS_MODULES und Beschreibung der Version im Format Symbol: Beschreibung.

Version

Bezeichnung der Version des Moduls, z.B. 1.3.1.14852

Installationszeit

Zeitstempel, wann das  Modul installiert wurde.

Aktuell

Checkbox. Wenn aktiv, ist diese Version die aktuelle Version, sonst nicht.

Untertyp

Untertyp des Moduls für Versionsinformationen (Basetype 37).
