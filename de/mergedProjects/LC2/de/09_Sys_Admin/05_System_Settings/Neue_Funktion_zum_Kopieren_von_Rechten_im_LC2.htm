Funktion zum Kopieren von Rechten im LC2

Mit dieser Funktion ist es möglich, alle Gruppenberechtigungen von einer Gruppe in eine andere zu kopieren.

 

Die Prozedur im LC2 besteht aus zwei Parametern:

 

  • "SOURCE_GRP_ID" (JsonValueType.INT, mandatory, "U-GROUPS.ID"),

  • "DEST_GRP_ID" (JsonValueType.INT, mandatory, "U-GROUPS.ID")

 

Die Prozedur hat den Pfad:

de/gefasoft/sapient/db/userrights/Userrights/copy

 

Ablauf der Prozedur

  1. Im ersten Schritt werden die parametrisierten Gruppen überprüft (wenn keine vorhanden ist, wird der Vorgang abgebrochen).

  2. Anschließend werden alle Einträge mit user_group = "DEST_GRP_ID" aus der Tabelle U_PERMISSIONS gelöscht.

  3. Im nächsten Schritt werden alle Einträge aus der Tabelle U_PERMISSIONS mit user_group = "SOURCE_GRP_ID" in die user_group "DEST_GRP_ID" kopiert.

Die Prozedur gibt entweder ERROR oder die Anzahl der eingefügten Zeilen bei erfolgreichem Ablauf zurück.

 

Bedienung im LC2

Die Funktion kann im LC2 über das Kontextmenü "Berechtigungen von Gruppe kopieren..." verwendet werden.

 

  1. Navigieren Sie zur Tabelle User Groups (System Administration > Users > User Groups) und wählen Sie die gewünschte Benutzergruppe mit einem Mausklick aus.

  2. Wählen Sie im Kontextmenü "Berechtigungen von Gruppe kopieren..." aus.


  3. Wählen Sie im folgenden Dialogfenster eine Quellgruppe aus.


  4. Abschließend fragt der LC2 nach, ob Sie den Vorgang wirklich vornehmen möchten.
