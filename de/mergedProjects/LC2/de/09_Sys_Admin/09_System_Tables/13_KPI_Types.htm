Funktion

Diese Tabelle zeigt alle in Legato verfügbaren KPI-Typen. Diese Tabelle dient rein Informationszwecken und ist schreibgeschützt.

Feldbeschreibung

 

Feld        

Beschreibung

ID

Eindeutige ID. Wird vom System vergeben.

Name

Name des KPI-Typen.

Beschreibung

Beschreibung des KPI-Typen und ggf. die Formel der Berechnung.

Quelle

Verweis auf die Quelle des KPI-Typen, z.B. Meldungen oder MDE-Zustände.

Klasse

Verweis auf die KPI-Klasse, der dieser Typ zugeordnet ist.

Summierbar?

Checkbox. 1 = der KPI-Typ kann summiert werden, 0 = nicht summierbar.

Einheit

Verweis auf die Einheit des KPI-Typen.

Schicht-KPI

Checkbox. 1 = Schicht-KPI,  0 = kein Schicht-KPI.

Online-KPI

Checkbox. 1 = Online-KPI, 0 = kein Online-KPI.

Format

Darstellungsformat von Online-KPIs (Variablen).

Online Delta

Delta für Online-KPIs. Der automatische Default ist abhängig von der Einheit des KPIs: Stück = 0,5, Zeit = 5 Sek., Prozent = 1. Der KPI wird immer bei Änderungen > oder = Delta berechnet.

Symbol

Eindeutiges Symbol. Wird zur Referenzierung in JavaScript verwendet.

Es dürfen nur die Zeichen [a-z] und [0-9] verwendet werden.
