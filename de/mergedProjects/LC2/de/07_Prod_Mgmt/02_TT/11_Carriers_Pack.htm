Funktion

Tatsächlich vorhandene Transportmittel. Hier werden alle wiederverwendbaren und identifizierbaren Transportmittel abgebildet.

Feldbeschreibung

 

Feld

Beschreibung

ID

Eindeutige Identifikationsnummer. Wird vom System vergeben. Schreibgeschützt.

Identifier

Identifier / Name des Transportmittels gemäß der ID Format-Beschreibung der Transportmittel-Definition.

Transportmittel Def.

Lookup-Liste zur Auswahl einer Transportmittel-Definition. Wird automatisch befüllt und kann geändert werden.

Parent-Transportmittel (optional)

Lookup-Liste zur Auswahl eines übergeordneten Transportmittels.

Transportmittel-Typ (optional)

Lookup-Liste zur Auswahl eines Transportmittel-Typs (z.B. gesperrt, freigegeben).

Transportmittel-Zustand (optional)

Lookup-Liste zur Auswahl eines Zustands für das Transportmittel (z.B. im Umlauf, ausgemustert, etc.).

Zustandsdetails (optional)

Eingabefeld für weitere Details zum Zustand des Transportmittels.

Umläufe (optional)

Anzahl der Umläufe (nur relevant für Transportmittel-Definitionen mit Circuit Use = 1).

Im Einsatz (optional)

Checkbox. Wenn aktiv, ist das Transportmittel gerade in Verwendung und belegt.

Wenn nicht aktiv, ist das Transportmittel frei.
