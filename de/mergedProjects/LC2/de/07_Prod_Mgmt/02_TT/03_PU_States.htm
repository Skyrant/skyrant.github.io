Funktion

Zur Dokumentation aller Statuswechsel während der Produktion werden alle Ereignisse mit Zeitstempel archiviert.

Die möglichen Status werden hier parametriert.

Feldbeschreibung

 

Feld

Beschreibung

ID

Eindeutige Identifikationsnummer. Wird vom System vergeben. Schreibgeschützt.

Symbol

Eindeutiges Symbol des Status.

Name

Frei wählbarer Name des Status, z.B. gesperrt oder freigegeben.

Beschreibung (optional)

Zusätzliche Beschreibung des Status.

Haltedauer

Je nach Zustand der PU variiert die Haltedauer der PU.

Abhängig vom Status wird die Haltedauer der Statusdefinition verwendet. Die Haltedauer gibt an, wann die PU gelöscht wird.

 

ACHTUNG

In der Tabelle gibt es einen speziellen Eintrag. Dieser (Daten-)Eintrag wird verwendet, wenn die PU keinen Status hat.

 

Feld

Beschreibung

ID

 -6

Symbol

 NONE

Name

 Kein Status

 

Der Datensatz darf nicht gelöscht und das Symbol nicht geändert werden!

 

Informationen zur Syntax

Für Zeitspannen wird mehrmals an verschiedenen Stellen (z.B. bei Job Triggern) eine Syntax verwendet:

 

 

 

Jahr

year(s)

y

Monat

 month(s) M

Woche

 week(s) w

Tag

 day(s) d

Stunde

 hour(s) h

Minute

 minute(s) m

Sekunde

 second(s) s

Millisekunde

 millisecond(s) ms

 

Alternativ ist auch möglich:

[+-] hh:mm:ss [.SSS]

 

 
Stunden hh
Minuten mm
Sekunden ss
Millisekunden SSS
