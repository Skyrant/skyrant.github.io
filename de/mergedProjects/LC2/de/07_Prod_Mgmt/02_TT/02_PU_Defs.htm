Funktion

Hier werden die benötigten Definitionen von Produktionseinheiten (PU) angelegt (z.B. "Stück" oder "Los").

Feldbeschreibung

 

Feld

Beschreibung

ID

Eindeutige Identifikationsnummer. Wird vom System vergeben. Schreibgeschützt.

Symbol (optional)

Eindeutiges Symbol für die PU.

Name

Name der PU.

Beschreibung (optional)

Zusätzliche Beschreibung der PU.

ID Format (optional)

Lookup-Liste zur Auswahl einer Formatdefinition für den Identifier.

Einheit (optional)

Lookup-Liste zur Auswahl einer Einheit für die PU.

feste Menge

Gibt an, was die festgelegte Menge der PU ist:

0 = Menge ist im Auftrag änderbar.

1 = 1 Stück (Losgröße 1) in der Einzelfertigung.

n = beliebige Menge

Default-Menge (optional)

Vorgeschlagene Menge für die PU (kann durch den Auftrag geändert werden).
