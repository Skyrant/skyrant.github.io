Trigger

In PostgreSQL können Tabellenänderungen durch Trigger mitgeloggt werden. Dazu wurde folgender Trigger erstellt:

CREATE TRIGGER taudi_tablename

AFTER INSERT OR UPDATE OR DELETE ON tablename

FOR EACH ROW EXECUTE PROCEDURE if_modified_func();

 

Aufruf aus J-LIW

Der Aufruf funktioniert mit J-LIW erst ab Version 2.0.000RC3

LgtConfig

-outDir=C:/SVN/legatodb/sapient/trunk/code/postgresql/trigger

 

-triggerBody=if_modified_func()

 

-isDBMS=postgresql;

 

Der Aufruf kann auch mit der Option tables eingeschränkt werden:

Beispiel: -tables=P_*

LgtConfig

-outDir=C:/SVN/legatodb/sapient/trunk/code/oracle/trigger

 

-triggerBody='null'

 

 -tables=P_*

 

-isDBMS=oracle;
