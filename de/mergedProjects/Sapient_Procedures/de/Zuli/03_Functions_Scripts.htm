Aufgerufene Funktionen in Skripten

In jedem Skript muss zwingend die Funktion

this.processZuli = function() {

.

}

 

vorhanden sein.

Diese Funktion hat keine Parameter.

Alle selbst hinzugefügten Funktionen müssen innerhalb von

function MeldungsImport() { ... }

 

mit der Syntax

this.functionName = function() { ...}

 

angelegt werden. Diese Vorgehensweise hat den Vorteil, dass solche "functions" unter dem identischen Namen in allen Importskripten angelegt werden können und auf Namensüberschneidungen nicht geachtet werden muss.
