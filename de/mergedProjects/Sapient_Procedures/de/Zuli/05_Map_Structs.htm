Beschreibung des Aufbaus der Maps

Aufbau der Map beim Aufruf von saveLine(Map).

Folgende Felder müssen mit Werten versorgt werden:

storeLine.ext_code

 Ext_code, Nummer

storeLine.def_node

 ID des Standardknotens aus LC2, Nummer

storeLine.al_class

 ID der Default-Meldeklasse aus Import-Optionen oder aus Import-Datei / Import-Skript

storeLine.al_type

 ID der Default-Meldeart aus Import-Optionen oder aus Import-Datei / Import-Skript

storeLine.operand

 Operand, String

storeLine.sym[0]

 Symbolik_0, String

storeLine.text[0]

 Text_0, String

storeLine.sym[1]

 Symbolik_0, String

storeLine.text[1]

 Text_0, String

storeLine.sym[2]

 Symbolik_0, String

storeLine.text[2]

 Text_0, String

storeLine.sym[3]

 Symbolik_0, String

storeLine.text[3]

 Text_0, String

storeLine.sym[4]

 Symbolik_0, String

storeLine.text[4]

 Text_0, String

storeLine.sub_sym

 Unterknoten des Standardknotens, String

 

Aus Standard-Importskripten werden nur die Felder storeLine.sym[0] und storeLine.text[0] beschrieben.

 

Aufbau der Map beim Aufruf von saveListLine(Map).

Folgende Felder müssen mit Werten versorgt werden:

storeLine.number

Nummer, Nummer

storeLine.text

Text, String
