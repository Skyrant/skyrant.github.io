Aufruf von Prozeduren

Prozeduren können ganz einfach über den Browser aufgerufen und ausgeführt werden. Dazu wird der Pfad zur Sapient Engine in die Adresszeile eingegeben, gefolgt von /call/ (sic! der Slash am Ende ist obligatorisch, sonst klappt der Aufruf nicht). Die notwendigen Benutzerrechte zum Ausführen von Prozeduren vorausgesetzt, erscheint folgender Dialog:

In "ID" kann eine optionale Aufruf-ID eingetragen werden, die dann im Rückgabewert angegeben wird.

In "Method" ist die Aufruf-URL der gewünschten Methode einzutragen, z.B. /de/gefasoft/sapient/db/alarms/Alarms/alarm_coming

:

Wenn Sie die Aufruf-URL aus der Hilfe kopieren, kommt es abhängig vom Browser manchmal zu Format-Fehlern und die Prozedur kann nicht ausgeführt werden. In diesem Fall fügen Sie die URL in eine Text-Datei ein (.txt) und kopieren Sie sie dort erneut heraus. Durch das Einfügen in eine .txt-Datei werden alle eventuell mitkopierten Formatierungen gelöscht.

In "Params" werden die gewünschten Parameter übergeben, z.B. {"CONNECTION": 10, "EXT_CODE": 1, "OPERAND": "abcdef", "TIME_STAMP": "2017-04-23T18:25:43.511Z"}

:

Dezimalwerte müssen mit Punkt (.) als Trennzeichen übergeben werden!
