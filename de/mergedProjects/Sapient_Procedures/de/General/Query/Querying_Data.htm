Daten abfragen

Die JSON-RPC-Methode zum Abfragen von Daten ist

de/gefasoft/sapient/data/Query/select

Die Methode enthält im Wesentlichen eine Abfrage-Definition als Parameter und gibt einen Array von Datensätzen zurück.

Einfaches Beispiel

{"schema": "legato",

"query":

{"resources": [

{ "resource": "se_dbpool_stat" }

]

}

}
