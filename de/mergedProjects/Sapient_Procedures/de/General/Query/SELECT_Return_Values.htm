Rückgabewert des SELECTs

Der zurückgegebene Wert ist ein JSON-Objekt mit den folgenden zwei Eigenschaften:

queryTime (String Zeitstempel) - Der Zeitpunkt, zu dem die Abfrage ausgeführt wurde.

data (JSON-Array von JSON-Objekten) - Die Datensätze. Jedes Zeilen-Objekt enthält eine JSON-Eigenschaft mit dem Namen und Wert von jedem abgefragten Feld.

more (Boolean) - Nur präsent, wenn die Abfrage einen Grenzwert für die Anzahl zurückgelieferter Zeilen definiert hat. In diesem Fall gibt der Wert an, ob es mehr Zeilen gegeben hätte (true) oder nicht (false).

Das vollständige JSON-RPC Rückgabeobjekt könnte folgendermaßen aussehen:

{

    "jsonrpc": "2.0",

    "id": 4,

    "result": {

        "queryTime": "2018-07-10T18:41:12.308Z",

       "data": [

          {

             "id": 17,

             "name": "Halle 2",

             "parent": 4

          }

       ]

    }

}
