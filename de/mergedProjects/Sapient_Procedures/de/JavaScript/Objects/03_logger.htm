logger

Dient zur Ausgabe von Meldungen in das Log der Sapient Engine. Jeder Script-Job erhält einen Logger mit Namen "JsJob-<Job-Name>. Dieser bietet folgende Methoden:

trace(text, exception)

Gibt eine Meldung der Ebene "trace" aus. Wird für detaillierte Ablauf-Verfolgung benutzt und üblicherweise nicht in Logdateien erfasst. Die Angabe eines Ausnahme-Objekts ist optional, bei solchen Meldungen aber unüblich.

debug(text, exception)

Gibt eine Meldung der Ebene "debug" aus. Wird für allgemeine Ablauf-Verfolgung benutzt. Die Angabe eines Ausnahme-Objekts ist optional, bei solchen Meldungen aber unüblich.

info(text, exception)

Gibt eine Meldung der Ebene "info" aus. Wird im Allgemeinen für Informationen über die Durchführung des Jobs genutzt. Die meisten Jobs geben eine Info-Meldung beim Start und eine am Ende des Job-Durchlaufs aus. Die Angabe eines Ausnahme-Objekts ist optional, bei solchen Meldungen aber unüblich.

warn(text, exception)

Gibt eine Meldung der Ebene "warn" aus. Wird für Fehlermeldungen benutzt, bei denen zwar eine bestimmte Aktion des Jobs misslungen ist, der Job aber weiter ablaufen kann. Die Angabe eines Ausnahme-Objekts ist optional.

error(text, exception)

Gibt eine Meldung der Ebene "error" aus. Wird für Fehlermeldungen benutzt, bei denen der Ablauf des Jobs abgebrochen wird. Die Angabe eines Ausnahme-Objekts ist optional, bei dieser Funktion aber durchaus üblich.

fatal(text, exception)

Gibt eine Meldung der Ebene "fatal" aus. Sollte in Skript-Jobs nicht benutzt werden, da Meldungen dieser Ebene nur ausgegeben werden sollten, wenn die Funktion der gesamten Instanz der Application Engine gestört ist. Auch hier kann zusätzlich ein Ausnahme-Objekt angegeben werden.

 

Logger hat die folgenden Eigenschaften:

traceEnabled(Boolean)

True, falls dieses Level geloggt wird.

debugEnabled(Boolean)

True, falls dieses Level geloggt wird.

infoEnabled(Boolean)

True, falls dieses Level geloggt wird.

warnEnabled(Boolean)

True, falls dieses Level geloggt wird.

errorEnabled(Boolean)

True, falls dieses Level geloggt wird.

fatalEnabled(Boolean)

True, falls dieses Level geloggt wird.
