SapientUser

Benutzer, den die Sapient Engine verwendet.

 

Objekte vom Typ SapientUser haben folgende Eigenschaften:

fullName

Vollständiger Name des Benutzers.

id

ID des Benutzers.

isSys

Boolean, ob der Benutzer Systemrechte hat.

name

Benutzername.

persId

ID der Person, zu der der Benutzer angelegt wurde.

roles

Array mit den Gruppen, denen der Benutzer angehört.

 

SapientUser haben die folgenden Funktionen:

allowedActions(dbcon, permProxyId):

Array mit den Aktionen, die der Benutzer ausführen darf.

hasRole(roleId):

Boolean, ob der Benutzer die übergebene Rolle hat oder nicht.

isAllowed(dbcon, permProxyId, actionId):

Boolean, ob der Benutzer die übergebene Aktion auf die übergebene Datenquelle ausführen darf oder nicht.

loadPref(dbcon, prefName):

Javascript-Objekt mit den Benutzereinstellungen.
