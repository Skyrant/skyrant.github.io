BitSet

Objekte dieses Typs erlauben die bequeme und effiziente Verwaltung eines Arrays boolescher Werte. Jedes der Bits kann gesetzt, zurückgesetzt oder abgefragt werden, außerdem können komplette BitSets miteinander logisch verknüpft werden und auf Überschneidung geprüft werden ("Mengen-Operationen").

Objekte vom Typ BitSet werden mittels new BitSet() angelegt. In neuen BitSet-Objekten ist ursprünglich kein Bit gesetzt.

BitSets haben folgende Eigenschaften:

cardinality (Zahl, readonly):

Die Anzahl gesetzter Bits in diesem BitSet.

empty (Boolean, readonly):

Ob dieses BitSet leer ist (d.h., kein einziges gesetztes Bit enthält).

first (Zahl, readonly):

Der Index des ersten gesetzten Bits. Falls kein einziges Bit gesetzt ist: -1.

last (Zahl, readonly):

Der Index des letzten gesetzten Bits. Falls kein einziges Bit gesetzt ist: -1.

 

Sie haben folgende Methoden:

and(bitset)

Löscht in diesem BitSet alle Bits, die nicht auch im Übergebenen gesetzt sind.

andNot(bitset)

Löscht in diesem BitSet alle Bits, die im Übergebenen gesetzt sind.

clear()

Löscht alle Bits in diesem BitSet.

clearAt(index)

Löscht das Bit an dem angegebenen Index (0..n).

clearRange(from, to)

Löscht alle Bits im angegebenen Bereich (inklusive).

clone()

Erzeugt ein neues BitSet, in dem genau die gleichen Bits gesetzt sind.

equals(bitset)

Prüft, ob in diesem und dem übergebenen BitSet die genau gleichen Bits gesetzt sind (Boolean).

flipAt(index)

Schaltet das angegebene Bit um.

flipRange(from, to)

Schaltet alle Bits im angegebenen Bereich um (inklusive).

getAt(index)

Liefert den Wert des angegebenen Bits (Boolean).

getRange(from, to)

Liefert ein neues BitSet, das nur die Bits aus dem angegebenen Bereich (inklusive) dieses BitSets enthält.

intersects(bitset)

Prüft, ob es mindestens ein Bit gibt, das in beiden BitSets gesetzt ist.

nextClear(index)

Liefert den Index des nächsthöheren nicht gesetzten Bits an oder ab dem angegebenen Index oder -1, falls es kein solches gibt.

nextSet(index)

Liefert den Index des nächsthöheren gesetzten Bits an oder ab dem angegebenen Index gesetzten Bits oder -1, falls es kein solches gibt. Um über alle gesetzten Bits zu iterieren, kann man z.B. folgende Schleife benutzen:

for (var i = bitset.first; i >= 0; i = bitset.nextSet(i + 1)) { ... }

or(bitset)

Setzt in diesem BitSet alle Bits, die im Übergebenen gesetzt sind.

setAt(index)

Setzt das Bit am angegeben Index.

setRange(from, to)

Setzt alle Bits im angegebenen Bereich (inklusive).

xor(bitset)

Schaltet in diesem BitSet alle Bits um, die im Übergebenen gesetzt sind.
