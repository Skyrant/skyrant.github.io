Anforderungen an ein benutzerspezifisches JavaScript

Wenn es eine Funktion namens "setup" oder "cleanup" gibt, wird diese aufgerufen, wenn das Script geladen bzw. bevor es "entladen" wird. Als Parameter wird außerdem "SapientContext" übergeben. Dabei ist zu beachten, dass der SapientContext eines Jobs beispielsweise ein anderer sein kann als der beim Laden/Entladen des Scripts.
