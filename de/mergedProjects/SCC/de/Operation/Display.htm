Anzeige des Cloud Connectors

Die Anzeige des Cloud Connectors besteht aus drei Teilen:

Das Projektfenster links oben. Der Aufbau des Projekts wird in einer Baumstruktur dargestellt.

Das Multidokumentfenster rechts oben. In diesem MDI-Fenster werden Fenster zum Beobachten von Variablenwerten, Statuszustände und Editierfenster für Scripten ausgegeben.

Das Infofenster im unteren Bereich des Hauptfensters.

Projektfenster

Im Projektfenster wird das aktuell geladene Projekt als Baum dargestellt. Der Baum besteht aus folgenden Abschnitten, der je nach gewähltem Projekt-Typ (Windows-Projekt *.lpr oder UNIX-Projekt *.lpx) unterschiedliche Knoten enthält:

 

Element

Verwendbarkeit

Funktion

Basic

Kommandoschnittstelle zwischen Cloud Connector und Redis. Kann Redis direkt ansprechen und Daten dorthin senden bzw. von dort holen. Außerdem können über Spools Aufgaben auf mehrere Cloud Connectors verteilt werden.

Database

Enthält die Elemente der ODBC-Datenbankverbindungen.

Event

Beinhaltet die Elemente, welche die Reaktion auf interne Ereignisse wie z.B. Timer festlegen.

SAP

                 

Enthält die Elemente der SAP-Schnittstelle.

OPC

Stellt alle Elemente dar, die eine Verbindung zu einem OPC-Server herstellen.

OPC_UA

Stellt alle Elemente dar, die eine Verbindung zu einem OPC-UA-Server herstellen.

PubSub

Stellt alle Elemente dar, die eine Verbindung über Publish-Subscribe Protokolle herstellen.

Sapient

Enthält die Elemente der Sapient Schnittstelle.

Script

Enthält alle Scripten, die durch Auftreten eines Ereignisses (Event oder Wertänderung) gestartet werden sollen.

SendRecv

Stellt alle Elemente dar, die eine direkte Verbindung zu einem Kommunikationspartner (SPS oder Rechner, ohne OPC) haben.

Value

Enthält interne Variablen, die während der Projektlaufzeit gültig sind und beliebig gelesen und beschrieben werden können.

 

Die meisten Einträge im Projektfenster verfügen über ein Kontextmenü, über das die für den Eintrag gültigen Befehle aufgerufen werden können. Der Aufruf des Kontextmenüs erfolgt (wie bei Windows üblich) mit der rechten Maustaste.

Multidokumentfenster

Im Multidokumentfenster werden sowohl Editierfenster, als auch Status- und Beobachtungsfenster angezeigt.

Infofenster

Das Infofenster bietet die Möglichkeit, den Ablauf eines Legato Projektes mitzuverfolgen.
