Allgemein

Siehe Logging (WIN64 und UNIX).

Editor-Log

Öffnen Sie das Menü "Logging" - "Einstellungen für Editor". Es öffnet sich folgender Dialog:

Hier kann ein Name und ein Verzeichnis für die Logdatei festgelegt werden. Durch Klick auf ... in der Zeile "Logmaske" kann festgelegt werden, welche Klassen geloggt werden sollen.

Service-Log

Öffnen Sie das Menü "Datei" - "Services verwalten". Selektieren Sie dort den gewünschten Service und klicken Sie auf "Bearbeiten". Nun kann hier ebenfalls ein Verzeichnis für die Logdateien eingegeben werden sowie die Logmaske festgelegt werden.

Verwaltung Logfilezipper Service

Hier kann der Logfilezipper Service verwaltet werden. Weitere Details finden Sie unter Logfilezipper.
