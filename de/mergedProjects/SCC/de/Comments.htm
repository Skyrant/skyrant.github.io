Kommentare

Dem Script können beliebig viele Kommentare beigefügt werden. Kommentare können über mehrere Zeilen gehen und auch mitten in einer Anweisungsfolge stehen. Eine Verschachtelung von Kommentaren ist möglich.

Zeilenkommentare werden mit '//' eingeleitet. Der Rest der Zeile wird als Kommentar angesehen.

Kommentarblöcke werden mit '/*' eingeleitet und mit '*/' beendet.

 

Beispiel:

//Scriptdatei für MyScript

/*

KOMMENTAR

KOMMENTAR

KOMMENTAR

*/

string   s //Variablendeklaration

if s = 'OK' then

  //TRUE

  ..

else

  //FALSE

  ..

endif
