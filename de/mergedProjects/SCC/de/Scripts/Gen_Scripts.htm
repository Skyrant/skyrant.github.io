Erstellen von Scripten

Beim Anlegen eines Scriptelements wird der "Ergebnistyp" des Scripts, der Name des Scripts sowie die "Namen" und "Typen" der "Parameter" (soweit erforderlich) festgelegt. Nach Anklicken des Scriptelements mit der rechten Maustaste oder der Menütaste der Windows-Tastatur kann im Kontextmenü der Script-"Editor" aufgerufen werden.

Zum Einfügen von Elementen wie Variablen oder anderen Scripten in das Script empfiehlt es sich, über das Kontextmenü des gewünschten Elements - Knotennamen kopieren den gesamten Pfad in die Zwischenablage zu holen. Von dort kann der Knotenpfad dann an der gewünschten Stelle im Script eingefügt werden.

Beim Arbeiten im Textfenster stellt das Kontextmenü (rechte Maustaste) folgende Funktionen zur Verfügung:

Rückgängig:

Macht die letzte Aktion rückgängig.

Ausschneiden:

Schneidet den markierten Text aus und speichert ihn in die Zwischenablage.

Kopieren:

Kopiert den markierten Text und speichert ihn in die Zwischenablage.

Einfügen:

Kopiert den Text aus der Zwischenablage ein.

Löschen:

Löscht den markierten Text.

Suchen:

Es wird der Windows-Standarddialog für Textsuche geöffnet.

Ersetzen:

Es wird der Windows-Standarddialog für Suchen & Ersetzen geöffnet.

Weitersuchen:

Der zuletzt ausgeführte Suchen- oder Ersetzen-Befehl wird wiederholt.

Hilfe:

Befindet sich der Cursor in einem Wort, wird die Hilfe geöffnet und dieses Wort als Suchbegriff eingegeben.

Verfügbare Parameter:

Hier erhält man eine Auswahlliste sämtlicher Parameter, die dem Script übergeben werden.

Kompilieren (F7):

Es wird ein Übersetzungslauf gestartet. Syntaktische Fehler werden in der Statuszeile angezeigt. Bei erfolgreicher Übersetzung erscheint "Script OK".

:

Beim Editieren von Scripten zur Laufzeit sollte man die nötige Vorsicht walten lassen. Enthält ein Script einen Fehler, so kann es nicht ausgeführt werden. Dies führt zu Fehlermeldungen.
