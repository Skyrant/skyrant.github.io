Allgemeines

Die Scripten sind ein flexibles Werkzeug  zur Programmsteuerung. Die Bearbeitung eines Scripts in Ihrem Projekt erfolgt ereignisgesteuert.

Typische Anwendungen für Verwendung von Scripten sind:

Mathematische Verknüpfung mehrerer Variablen

Auslösen von programm- oder zeitgesteuerten Funktionen im Prozess

Einfache Steuerungsfunktionen

Starten von beliebigen Windows-Applikationen

Zugriff auf das Logbuch

 

Die an Pascal angelehnte Syntax verfügt über Sprachelemente wie if ... then ... else, case und while. Als Operanden sind in den Scripten "OPC-Variable" (und damit alle Prozesswerte), "Datenbankvariable" und "interne Variable" verfügbar.

Die Bearbeitung eines Scripts durch das Ablaufprogramm kann durch Ereignisse wie Applikation öffnen/schließen, Timer oder Variablenwertänderungen angestoßen werden. Dadurch werden z.B. bei Änderung eines Prozesswertes Berechnungen sofort durchgeführt.

Die Verwendbarkeit von Scriptbefehlen ist abhängig von der Betriebssystem-Umgebung, in der sich das Cloud Connector Projekt im Ablauf befindet. Der Hinweis dazu findet sich direkt bei der Befehlsbeschreibung als
