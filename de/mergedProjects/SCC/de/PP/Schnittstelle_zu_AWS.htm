Schnittstelle zu Amazon Web Services (LINUX)

Überblick

Im Cloud Connector Editor des UNIX Cloud Connectors wird im Projektbaum ein Knoten AWS angelegt.

Anlegen der SQS-Verbindung

Selektieren Sie das Grundelement "AWS" im Projektbaum und wählen Sie im Kontextmenü den Befehl "Neues Item".

Wählen Sie SQS connections aus und legen Sie mit "OK" ein neues Element an.

Der "Name" ist frei wählbar. Das SQS-Element dient nur zur übersichtlichen Strukturierung und Abgrenzung zu eventuell weiteren AWS-Schnittstellen. Unter der SQS-Verbindung können jetzt Recv (Empfangen von Nachrichten) oder/und Send Verbindungen (Senden von Nachrichten) angelegt werden.

Anlegen der SQS Recv-Verbindung

Über eine Rec-Verbindung können Nachrichten empfangen werden.

Für eine Recv-Verbindung sind folgende Angaben erforderlich:

  • Name (Name des Knotens)

  • Queuename (Name der SQS Queue)

  • Queuetyp (Standard oder FIFO)

  • Max. Nachrichten pro Receive

  • Receive Timer (maximale Dauer des Recv-Aufrufs)

  • Nachricht löschen (Haken gesetzt: Nachricht wird aus Queue gelöscht, ...)

 

Unter einer Recv-Verbindung können die Items OnValue und OnStatus angelegt werden.

OnValue dient zum Empfang von Nachrichten. Damit kann dann ein entsprechendes Skript zur Verarbeitung der empfangenen Nachricht getriggert werden. @Value hat das Format String.

 

OnStatus wird zurzeit nicht verwendet.

 

Anlegen der SQS Send-Verbindung

Über eine Send-Verbindung können Nachrichten versendet werden.

Für eine Send-Verbindung sind folgende Angaben erforderlich:

• Name (Name des Knotens)

• Queuename (Name der SQS Queue)

• Queuetyp (Standard oder FIFO)

Unter einer Send Verbindung kann das Item OnStatus angelegt werden.

Dieses Item beinhaltet in @Value die ID, die beim Send übergeben wurde, wenn das Senden mittels Skriptbefehl SendMsg erfolgreich war. Falls das Senden nicht erfolgreich war, wird ebenfalls die ID übergeben und zusätzlich das 64. Bit (0x8000000000000000) gesetzt.

Um eine Nachricht senden zu können, steht der Befehl SendMsg zur Verfügung.
