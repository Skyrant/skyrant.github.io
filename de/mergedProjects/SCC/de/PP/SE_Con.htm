Nur Legato-Sapient: Datenbankverbindung

Bei Legato-Sapient wird die Datenbank-Verbindung über die Sapient Engine erkannt. Die Sapient Engine muss gestartet sein. Dann wird im Menüpunkt "Sapient" die Verbindung zur Sapient Engine eingetragen.

Der "Name" ist frei wählbar, sollte aber eindeutig sein.

Unter "Protokoll / Inhalt" muss aus einer Drop-Down-Liste das gewünschte Protokoll / der gewünschte Inhalt gewählt werden.

Unter "Ergebnistyp" kann aus einer Drop-Down-Liste die Art des Ergebnisses (CSV oder JSON) selektiert werden.

Unter "URI" geben Sie den Verbindungsstring zur Sapient Engine ein.

Unter "Port" muss der lokale Port eingegeben werden.

Schließlich muss noch der Benutzer mit Passwort eingegeben werden, mit dem der Cloud Connector sich mit der Sapient Engine verbinden soll.
