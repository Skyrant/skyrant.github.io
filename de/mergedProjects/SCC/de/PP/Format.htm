Variablen-Format

Elemente, die als Variable im Projekt angesprochen werden können, besitzen eines der folgenden Formate:

Format

Beschreibung

BOOL

Boolescher Wert; Zustand 1 (wahr/true) oder 0 (unwahr/false)

INT

Integerwert; Zahl mit Vorzeichen, Größe 4 Byte

DOUBLE

8-Byte IEEE Gleitkommazahl

STRING

Zeichenkette beliebiger Länge

BLOCK

Beliebige Aneinanderreihung von Zahlen, Strings und Bool-Werten. Die maximale Länge eines Blockes ist 65.496 Byte.
