Steuerung von externen Prozessen

Mit diesen Funktionen können externe Prozesse wie z.B. Notepad, Browser etc. gesteuert werden.

Für die Funktionen kann in einem Script eine Variable vom Typ process deklariert werden. Alle weiteren Funktionen werden dann mit dem Namen der Variablen, einem Punkt und dem Namen der Funktion gemeinsam angesprochen.

Close

Deklaration: bool process.close(int terminationtype)

Die Funktion versucht, den Prozess zu schließen, und liefert true bei Erfolg. Durch terminationtype wird festgelegt, auf welche Art versucht wird, den Prozess zu schließen:

0

Es wird versucht, nur den ursprünglich gestarteten Prozess zu schließen. Manche Programme starten aber einen neuen Prozess und der Startprozess wird sofort geschlossen. Dann funktioniert diese Variante nicht.

1

Es wird versucht, den ursprünglich gestarteten Prozess sowie alle von diesem abhängigen Prozesse zu schließen.

Beispiel:

static process pro

static bool created

string cmdline

cmdline := 'C:\\Program Files (x86)\\Mozilla Firefox\\firefox.exe

            https://github.com'

if not created then

  created := pro.create(cmdline)

else

  created := not pro.close(1)

endif

 

Closebyname

Deklaration: bool process.closebyname(string programname)

Die Funktion versucht, den Prozess zu schließen, und liefert true bei Erfolg. Es werden alle Prozesse geschlossen, die programname entsprechen.

Beispiel:

static process pro

static bool created

string cmdline

cmdline := 'C:\\Program Files (x86)\\Mozilla Firefox\\firefox.exe

            https://github.com'

if not created then

  created := pro.create(cmdline)

else

  created := not pro.closebyname('firefox.exe')

endif

 

Create

Deklaration: bool process.create(string cmdline)

In cmdline wird der Pfad zur ausführbaren Datei inklusive der optionalen Parameter übergeben. Die Funktion versucht, den Prozess zu erzeugen, und liefert true bei Erfolg.

Beispiel:

static process pro

static bool created

string cmdline

cmdline := 'C:\\Program Files (x86)\\Mozilla Firefox\\firefox.exe

            https://github.com'

if not created then

  created := pro.create(cmdline)

else

  created := not pro.close(1)

endif
