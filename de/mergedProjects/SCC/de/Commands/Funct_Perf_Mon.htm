Funktionen des Performancemonitors

GetPerfVals

Deklaration: string getperfvals(string Name, string Format, bool Kommentar)

Die Funktion liefert die Werte einer Zeile aus der Performancetabelle. Die Zeile wird über den Namen adressiert. Eine Zeile enthält auch Zeitstempel. Das Format für die Ausgabe eines Zeitstempels wird als Parameter angegeben. Ist Format ein Leerstring, werden keine Zeitstempel ausgegeben. Die Werte werden als String geliefert, jeweils getrennt durch ein Semikolon. Die Ausgabe erfolgt auf Wunsch mit Kommentar.

Die Reihenfolge entspricht den Spalten der Tabelle. Timerwerte werden im ms mit 5 Nachkommastellen ausgegeben.

Beispiel:

string s

   

s := GetPerfVals('Script', 'DD.MM.YYYY HH:mm:ss:ttt', false)

// liefert die Werte mit Zeitstempel

s := GetPerfVals('Script', '', false)

// liefert die Werte ohne Zeitstempel

s := GetPerfVals('Script', 'DD.MM.YYYY HH:mm:ss:ttt', true)

// liefert die Werte mit Zeitstempel und mit Kommentar

s := GetPerfVals('Script', '', false)

// liefert die Werte ohne Zeitstempel und ohne Kommentar

 

ResetAllPerf

Deklaration: resetallperf

Alle Zeilen in der Performancetabelle werden auf den Nullzustand zurückgesetzt.

Beispiel:

ResetAllPerf

 

ResetOnePerf

Deklaration: resetoneperf(string Name)

Eine Zeile in der Performancetabelle wird auf den Nullzustand zurückgesetzt. Die Zeile wird über den Namen adressiert.

Beispiel:

ResetOnePerf('Script')
