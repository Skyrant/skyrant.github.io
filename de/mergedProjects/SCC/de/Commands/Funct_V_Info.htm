Funktionen zur Versionsabfrage

GetOSVersion

Deklaration: string getosversion

Die Funktion liefert die Version des Betriebssystems als String.

Beispiel:

string s

s := GetOsVersion

 

GetVersion

Deklaration: string getversion

Die Funktion liefert die Version des Cloud Connectors als String.

Beispiel:

string s

s := GetVersion

 

GetVersion (OPC-Verbindung)

Deklaration: string getversion

Die Funktion liefert die Version des OPC-Treibers als String.

Beispiel:

string   s

s := OPC.Plc1.GetVersion

 

GetVersion (SQL-Verbindung)

Deklaration: string getversion

Die Funktion liefert die Version des ODBC-Treibers als String.

Beispiel:

bool     b

string   s

Database.Legato.Connect

if b then

  s := Database.Legato.GetVersion

endif
