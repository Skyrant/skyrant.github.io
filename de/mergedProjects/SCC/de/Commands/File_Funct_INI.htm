Dateifunktionen (INI-Dateien)

Mit diesen Funktionen können Konfigurationsdateien im Stil der INI-Dateien geschrieben und gelesen werden.

Für die Dateifunktionen kann in einem Script eine Variable vom Typ INIFILE deklariert werden. Alle weiteren Funktionen werden dann mit dem Namen der Variablen, einem Punkt und dem Namen der Funktion gemeinsam angesprochen.

Beispiel:

inifile  f

f.open('c:\\files\\config.ini')

 

Close

Deklaration: inifile.close

Die Datei f wird geschlossen.

Beispiel:

inifile  f

f.open('c:\\files\\config.ini')

f.write('Section', 'Entry', 'Value')

f.save

f.close

 

Open

Deklaration: bool inifile.open(string datei)

Öffnet die Datei. Die Funktion liefert "false", wenn ein ungültiger Pfad übergeben wurde.

Beispiel:

inifile  f

f.open('c:\\files\\config.ini')

f.write('Section', 'Entry', 'Value')

f.save

f.close

 

Read

Deklaration: string inifile.read(string Section, string Entry)

Liest aus der Sektion Section den Eintrag Entry.

Beispiel:

inifile  f

string   s

f.open('c:\\files\\config.ini')

s := f.read('SIZE', 'XPOS')

f.close

 

Save

Deklaration: inifile.save

Sichert die Datei. Wurde die INI-Datei nach dem Öffnen mit "Write" beschrieben, werden die Änderungen erst nach dem Aufruf von "Save" auf die Festplatte geschrieben. Wird die Datei ohne "Save"-Aufruf mit "Close" geschlossen, gehen die Änderungen verloren.

Beispiel:

inifile  f

f.open('c:\\files\\config.ini')

f.write('Section', 'Entry', 'Value')

f.save

f.close

 

Write

Deklaration: inifile.write(string section,
                           string entry,
                           string value)

Schreibt in die Sektion section den Eintrag entry mit dem Wert value. Ist die Sektion oder der Eintrag nicht vorhanden, werden sie neu angelegt.

Beispiel:

inifile  f

f.open('c:\\files\\config.ini')

f.write('SIZE', 'XPOS', '624')

f.write('SIZE', 'YPOS', '100')

f.save

f.close
