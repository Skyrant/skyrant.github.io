SQL-Funktionen mit Feldern

AsString

Deklaration: string AsString

Liefert den Inhalt des Feldes als String

Beispiel:

bool     ok

long     rows

string   starttime, value

ok        := Database.Legato.P_OUT_VALUES.Select('', 2)

ok        := Database.Legato.P_OUT_VALUES.First

starttime := Database.Legato.P_OUT_VALUES.TIME_STARTED.AsString

value     := Database.Legato.P_OUT_VALUES.VALUE_NUM.AsString

debug(starttime + ': ' + value)
