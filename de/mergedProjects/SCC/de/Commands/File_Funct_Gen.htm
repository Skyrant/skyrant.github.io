Dateifunktionen (allgemein)

CopyFile

Deklaration: bool CopyFile(string source, string dest)

Kopiert die Datei source nach dest. Die Funktion liefert "true", wenn der Kopiervorgang ausgeführt werden konnte. Dabei ist zu beachten, dass im String vor '\'-Zeichen (z.B. bei Pfadangaben) das Escape-Zeichen '\' gesetzt werden muss.

Beispiel:

bool     ok

ok := CopyFile('c:\\files\\file1.txt', 'c:\\files\\file2.txt')

 

CreateDir

Deklaration: bool CreateDir(string path)

Erzeugt das durch path festgelegte Verzeichnis. Konnte das Verzeichnis angelegt werden oder war es vor dem Aufruf von CreateDir betreits vorhanden, wird true zurückgegeben.

Beispiel:

bool ok

string path

path := 'c:\\projects'

ok := CreateDir(path)

if ok then

  debug('Directory '+path+' created')

endif

 

DeleteFile

Deklaration: bool DeleteFile(string datei)

Die datei wird gelöscht. Die Funktion liefert "true", wenn der Löschvorgang ausgeführt werden konnte.

Beispiel:

bool     ok

ok := DeleteFile('c:\\files\\file1.txt')

 

DirExist

Deklaration: bool DirExist(string path)

Prüft, ob das durch path festgelegte Verzeichnis vorhanden ist.

Beispiel:

bool ok

string path

path := 'c:\\projects'

ok := DirExist(path)

if ok then

  debug('Directory '+path+' exists')

endif

 

FileExist

Deklaration: bool FileExist(string path)

Prüft, ob die durch path festgelegte Datei vorhanden ist.

Beispiel:

bool ok

string path

path := 'c:\\projects\\recipe.txt'

ok := FileExist(path)

if ok then

  debug('File '+path+' exists')

endif

 

MoveFile

Deklaration: bool MoveFile(string source, string dest)

Die Datei source wird in dest umbenannt. dest darf zuvor nicht vorhanden sein. Die Funktion liefert "true", wenn der Vorgang ausgeführt werden konnte.

Beispiel:

bool    ok

ok := MoveFile('c:\\files\\file1.txt', 'c:\\files\\file2.txt')
