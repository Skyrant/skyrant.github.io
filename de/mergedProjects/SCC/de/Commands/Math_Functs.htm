Mathematische Funktionen

Exp

Deklaration: double Exp(double x)

Berechnet die Exponentialfunktion ex.

Beispiel:

double   d

d := exp(1)

debug(DToStr(d))

 

Ln

Deklaration: double Ln(double x)

Berechnet den natürlichen Logarithmus des Arguments x.

Beispiel:

double   d

d := ln(2.718281)

debug(DToStr(d))

 

Log

Deklaration: double Log(double x)

Berechnet den Logarithmus des Arguments x zur Basis 10.

Beispiel:

double   d

d := log(100)

debug(DToStr(d))

 

Pi

Deklaration: double Pi

Liefert die Zahl Pi als Double-Wert.

Beispiel:

double   d

d := 2 * pi

debug(DToStr(d))

 

Pow

Deklaration: double Pow(double x, double y)

Pow liefert das Ergebnis der Berechnung xy.

Beispiel:

double   d

d := KGToD(OPC.GOPS.PLC1.Length)

d := pow(d, 3)

debug(DToStr(d))

 

Round

Deklaration: long Round(double x, bool up)

Rundet den Double-Wert x und liefert als Ergebnis einen Long-Wert. Ist up = true wird aufgerundet, sonst abgerundet.

Beispiel:

double   d

long      up, down

d   := 123.45

up   := round(d, true)

down := round(d, false)

debug('Rounded up   : ' + DToStr(up))

debug('Rounded down : ' + DToStr(down))

 

Sqrt

Deklaration: double Sqrt(double x)

Berechnet die Quadratwurzel des Arguments x.

Beispiel:

double   d, r

string   s

d := 100.0

r := sqrt(d)

s := printf('The squareroot of %s is %s',

            DToStr(d), DToStr(r))

debug(s)
