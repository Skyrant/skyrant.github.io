Installation Linux ITC / OTC

Die Linux ITC und OTC Binaries, sowie die dazu benötigten Libraries (ausschließlich der System Libraries) werden unter

https://github.com/Gefasoft/scc-rts-ix/releases

abgelegt.

 

Beispiel Release 3.17.0:

 

Je Release ist jeweils eine Zip-Datei für die Linux-Derivate

• RedHat

• SUSE

• Ubuntu

verfügbar.

Diese Zip-Datei kann in einem beliebigen Verzeichnis auf dem Linux-Zielsystem entpackt werden.

Für die ITC und OTC Binaries müssen dann die Ausführungsrechte entsprechend gesetzt werden. Außerdem muss der LD_LIBRARY_PATH das o.g. Verzeichnis enthalten, damit die Libraries gefunden werden können.

Ob alle Referenzen eines Binaries in Bezug auf Libraries richtig aufgelöst werden können, kann in der Konsole durch den Befehl:

ldd <Binary>

geprüft werden.

 

Beispiel:

ldd ./ITCRuntimeStd

 

:

Der Aufruf eines Binaries ohne Parameter zeigt an, ob das Binary ausgeführt werden kann. Ist dies der Fall, werden darüber hinaus die Version und alle verfügbaren Parameter angezeigt.

 

Usage: ITCRuntimeStd

  required parameters:

    -p <processcfgfile> process configuration file

  alternatively:

    -f <projectfile> scc project file

    -m <logmask> mask for logmessages (64 bit value)

    -l <logdir>  directory for logfiles

    -n <logname> name for logfile

  optional parameters:

    -d <logdest> destination for logmessages (0x01 -> file  0x02 -> stdout  0x03 -> file and stdout) default:0x01

    -r <runmode> runmode (1 -> normal  2 -> containerized) default:1

    -i <listen port manager> listen port mdx manager

    -w <listen port monitor> listen port prometheus monitor

 

ITC und OTC Prozesse können sowohl durch die Angabe aller notwendigen Kommandozeilenparameter, als auch durch die Übergabe einer Parameterdatei (INI Dateiformat), welche diese enthält, gestartet werden.

 

Beispiel für Start ITC (Übergabe Kommandozeilenparameter):

/legato/bin/ITCRuntimeStd -f /legato/prj/test.lpx -l /legato/log/ -m 0xFFFFFFFFFFFFFFFF -n itc_test

 

Beispiel für Start ITC (Übergabe mittels Parameterdatei):

/legato/bin/ITCRuntimeStd -p /legato/cfg/itc_test.cfg

 

Beispiel für eine Parameterdatei (itc_test.cfg):

[Service]

Signature=6C52A5EE-5505-49CF-B1E2-D872A7654E1C

Name=itc_test

 

[Logging]

Dest=0x02

Mask=0xFFFFFFFFFFFFFFFF

Path=/legato/log

 

[Start]

Project=/legato/prj/test.lpx
