Allgemein

Für die Installation der Legato Application Engine wird mindestens Java 1.7 Standard Edition benötigt. In dieser Anleitung empfehlen wir die Verwendung von Java 1.8 Standard Edition, die Voraussetzung für die Legato Installation ist.

Installation von JDK 1.8

Die folgenden Schritte zeigen, wie Java JDK 1.8 in Ubuntu installiert wird.

Schritt 1 - Terminal öffnen

Das Terminal kann im Dashboard oder im Zubehör-Ordner gefunden werden. Es kann auch durch Klicken von Strg+Alt+T geöffnet werden.

Schritt 2 - Quellen aktualisieren

Geben Sie "sudo apt-get update" ein und drücken Sie Enter.

Schritt 3 - Prüfen, ob Java bereits installiert ist

Geben Sie "java –version" ein und drücken Sie Enter. Wenn kein Java oder Java 8 läuft, fahren Sie fort, Java 8 zu installieren.

Schritt 4 - Java JDK 1.8 installieren

Geben Sie "sudo apt-get install openjdk-8-jre" ein und drücken Sie Enter. OpenJDK 8 wird installiert.

Schritt 5 - Zu verwendende Java Version wählen

Wenn mehrere Versionen von Java installiert sind, wählen Sie die gewünschte Java Version aus, indem Sie "sudo update-alternatives - -config java" eingeben und Enter drücken. Eine Liste aller installierter Java Versionen wird angezeigt. Verwenden Sie die Nummerntasten, um die gewünschte Version zu wählen. Klicken Sie Enter, um mit der gewählten Version fortzufahren.
