Einleitung

Um eine Verbindung zur Datenbank herstellen zu können, benötigt das Legato-Gateway unter anderem den Datenbankclient der gewünschten Zieldatenbank. Für eine Oracle-Datenbank ist dies der Oracle Client. Eine Anleitung, wie der Oracle Client zu installieren ist, ist Inhalt dieses Dokuments.

Voraussetzungen

Es muss bereits ein Oracle-Server installiert sein und das Legato Datenbankmodell sollte bereits eingerichtet sein. Für weitere Details zu dieser Installation, siehe bitte die entsprechende Dokumentation Installation Oracle Server.

:

In der Oracle Client Version 11.2.0.3. kommt es hin und wieder zu Problemen, weil der OLE Treiber nicht korrekt installiert wird. Daher wird die Installation der Version 11.2.0.1 empfohlen.

Bis auf Schritt 2, der nur bei der Installation der neuen Version erscheint, erfolgt die Installation für beide Versionen gleich.

:

Es muss immer die 32-Bit Version des Oracle Clients installiert werden, auch wenn das Betriebssystem eine 64-Bit Version ist.
