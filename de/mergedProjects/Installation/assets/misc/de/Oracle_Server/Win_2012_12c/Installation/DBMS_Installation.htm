Durchführung der DBMS-Installation

Nach den Vorbereitungen kann nun die Installation durchgeführt werden. Dazu wechselt man zunächst in das Verzeichnis "database" und führt dort das Programm "setup.exe" aus.

Damit wird das Java-Installationsprogramm gestartet und es erscheint die folgende Maske:

Schritt 1: In dieser Maske kann die E-Mail-Adresse zu einem gültigen Metalink-Zugang eingetragen werden, um über Updates zu diesem Server automatisch benachrichtigt zu werden.

In diesem Beispiel tragen wir hier nichts ein und deaktivieren die Auswahlbox für die Security Updates.

Da man keine gültige E-Mail-Adresse angegeben hat, erfolgt die folgende Warnmeldung, die man jedoch ignorieren kann, indem man auf "Yes" klickt.

 

Schritt 2: Die Maske mit dem automatischen Download von Software-Paketen kann unausgefüllt bleiben, sofern man keinen gültigen Metalink-Zugang hat, indem man den Radiobutton "Skip software updates" aktiviert.

Schritt 3: Bei dieser Maske wählen wir die Option, dass nur die Datenbanksoftware installiert werden soll.

Schritt 4: Hier bitte die Option "Single Instance Database" auswählen.

:

Die Installation von Legato auf Oracle Real Application Clusters (RAC) wird nicht unterstützt!

 

Schritt 5: Bitte wählen Sie die gewünschten Sprachen der Datenbanksoftware.

 

Schritt 6: Bei der Auswahl der Edition wählen wir die Enterprise Edition.

Schritt 7: Im Gegensatz zu früheren Oracle-Versionen benötigt Oracle 12c einen eigenen Windows-Account (Windows-User), um zu laufen. Dieser wird jetzt neu angelegt, falls notwendig

Das gewählte Passwort muss der Windows System Policy entsprechen, sonst erscheint der folgende Warnhinweis:

Falls ein passender User schon vorher angelegt worden ist, kann er natürlich hier genutzt werden.

Schritt 8: Auswahl der Verzeichnisse – Bitte wählen Sie hier sehr hohe Verzeichnisse, damit die Verzeichnisstruktur nicht so tief wird, zum Beispiel das Verzeichnis "C:\oracle". (Ein vollständiger Pfadname in Windows – inklusive Dateiname – darf nicht länger als 255 Zeichen sein.)

Im Schritt 9 werden alle Voraussetzungen von der Installationssoftware überprüft.

Sofern alles in Ordnung ist, erfolgt dann eine Zusammenfassung im Schritt 10. Diese Zusammenfassung kann man sich lokal abspeichern.

Um fortzufahren, dann bitte den "Install-Button" klicken.

 

Im Schritt 11 wird die Oracle Software-Installation durchgeführt.

Nach der Installation erscheint dann erwartungsgemäß die obige Meldung. Damit ist die DBMS Installation erfolgreich durchgeführt. Das Fenster kann mit dem "Close"-Button geschlossen werden.

Ergebnisse der Installation können in dem Logfile "installActions.log" nachgelesen werden.

Eine spätere eventuelle Deinstallation erfolgt ebenfalls über das Installationsprogramm. Dafür sollte der Ordner mit den Installationsdateien auf dem System vorgehalten werden.
