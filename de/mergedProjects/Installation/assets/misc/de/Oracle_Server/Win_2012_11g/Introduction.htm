Einleitung

Über dieses Dokument

In dieser Dokumentation wird in kurzen Texterläuterungen und anhand von Screenshots beschrieben, wie man einen Oracle Server (ab Generation 11g) auf einem Windows Serversystem installiert.

Die Beschreibung richtet sich an Datenbank-Administratoren. Grundlegende Kenntnisse im Umgang mit Datenbank-Servern sowie Kenntnis der Systemumgebung wird vorausgesetzt.

Im Dokument werden folgende Symbole verwendet:

:

Stellt einen Sicherheitshinweis dar. Die folgenden Informationen warnen vor möglichem Sachschaden, z.B. Datenverlust oder mangelnder Funktionalität bei unsachgemäßer Handhabung. Beachten Sie diese Hinweise unbedingt!

Allgemeiner Überblick

Als Oracle DBMS-Version wird eine Enterprise Edition der Oracle Server verwendet, die für den Betrieb von Legato empfohlen wird. Zu den genauen Systemvoraussetzungen siehe bitte das Dokument "Legato_System_Installation" im Kapitel "Datenbank".

Als Betriebssystem dient ein Windows Server 2012. Die Installationsanleitung gilt analog für Windows Server 2008 und Windows Server 2008 R2.
