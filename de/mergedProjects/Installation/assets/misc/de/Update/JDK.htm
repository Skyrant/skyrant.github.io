JDK

In seltenen Fällen ist auf den Web- und AppEngine-Servern auch ein Update der JDK Version gefordert. Entsprechende Anforderungen sind ebenfalls in den Release-Notes vermerkt. Falls dies notwendig ist, wie folgt vorgehen:

Setup der geforderten JDK-Version ausführen und die Default-Einstellungen für die Installation übernehmen.

Aus der Programmliste im Ordner "Apache Tomcat XXX TomcatX" den "Configure Tomcat" starten (als Administrator ausführen).

Im Register "Java" den Pfad auf den neuen JDK umstellen (z.B. von: "C:\Program Files\Java\jre6\bin\server\jvm.dll" auf "C:\Program Files\Java\jre7\bin\server\jvm.dll")
