Überblick

Für den Betrieb der Legato-Webapplikation muss ein Web-Application-Server zur Verfügung stehen, der die J2EE-Architektur oder höher unterstützt. Beispiele sind Glassfish, WebLogic Server (BEA), JBoss (Red Hat), WebSphere Application Server und WebSphere Application Server Community Edition (IBM), JRun (Adobe), Apache Geronimo oder Apache Tomcat (Apache Software Foundation).

Zu diesem Web-Application-Server muss die eigentliche Legato-Webapplikation (Lieferung in Form eines War-Files) installiert und konfiguriert werden.
