Installation Legato Configuration Center (LC2)

Grafische Installation

Die Installation des LC2 erfolgt über das Installationsprogramm SetupLC2.exe. Das Setup prüft, ob alle o.g. benötigten Komponenten vorhanden sind und installiert gegebenenfalls fehlende Komponenten.

Das Legato Configuration Center (LC2) kann auf allen Rechnern installiert werden, die zur Parametrierung und Pflege des Leitsystems verwendet werden sollen – eine spezielle Freischaltung oder Lizenz ist nicht erforderlich.

Es ist darauf zu achten, dass der Windows-Dienst "Windows Installer" auf dem PC gestartet ist. Andernfalls schlägt die Installation fehl, es erscheint eine entsprechende Fehlermeldung. Üblicherweise ist der Windows Installer jedoch korrekt konfiguriert, es sind dann keine Anpassungen notwendig.

Installation im Silent Mode

Es ist möglich, das Setup auch im sogenannten Silent Mode im Hintergrund laufen zu lassen, ohne grafische Dialoge.

Dabei wird das Setup mit optionalen Parametern über die Kommandozeile ausgeführt.

:

Die Kommandozeile muss als Administrator gestartet werden, sonst erscheint ein Dialog, der eine Bestätigung erfordert, um die Installation zu starten.

Es gibt folgende Möglichkeiten:

1. Installation im Silent Mode mit Default-Einstellungen über die Kommandozeile erfolgt durch den Parameter /s.

Beispiel:

<setup.exe> /s

 

2. Silent Installation mit Angabe des Installationspfades über die Kommandozeile erfolgt durch die Variable TARGETDIR.

Beispiel:

<setup.exe> /s TARGETDIR=C:\Gefasoft

 

3. Silent Installation mit Log-Ausgabe erfolgt durch den Parameter /l.

Beispiel:

<setup.exe> /s /l=C:\LC2\install.log

 

:

Hier ist als Logging-Level verbose eingestellt. Es werden dadurch während der Installation viele Informationen geloggt und das Logfile ist recht groß (120 MB).

4. Silent Installation mit reduziertem Log erfolgt mit dem Parameter LOGDIR. Der Pfad zur Log-Datei ist beliebig einstellbar.

Beispiel:

<setup.exe> /s LOGDIR=C:\LC2\install.txt

 

5. Silent Installation ohne Sqlcompact erfolgt durch die Variable SQLCOMPACT=FALSE.

Beispiel:

<setup.exe> /s SQLCOMPACT=FALSE

 

6. Deinstallation im Silent Mode mit Default-Einstellungen über die Kommandozeile erfolgt durch den Parameter /s MODIFY=FALSE REMOVE=TRUE UNINSTALL=YES.

Beispiel:

<setup.exe> /s MODIFY=FALSE REMOVE=TRUE UNINSTALL=YES

 

Optional kann hier noch der Parameter /l für das Logging mit angegeben werden.

7. Deinstallation im Silent Mode mit reduziertem Logging:

<setup.exe> /s MODIFY=FALSE REMOVE=TRUE UNINSTALL=YES LOGDIR=C:\LC2\uninstall.txt

 

:

Die Installation/Deinstallation im Silent Mode dauert länger als die normale Installation. Dauer der Installation: ca. 20 Minuten. Dauer der Deinstallation: ca. 10 Minuten.
