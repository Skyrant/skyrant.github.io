Installation of CentOS 6.5

Step 1 - Boot from CentOS 6.5 ISO Image

Boot your system with CentOS with the ISO image which you downloaded. You will get the following screen.

Press ‘ENTER’ on Install or upgrade an existing system from the above screen.

Step 2 – Choose Skip to skip the media test and start the installation

You will now be presented with a media Check.

Here if you want to check your media for any type of missing files in your media click ‘OK’, for this tutorial we will skip this step and select ‘Skip’.

Step 3 – Installation Welcome Screen

Here you can see the installation welcome screen.

Press ‘Next’ to begin the process.

Step 4 – Select Installation Language

Choose your language and click ‘Next’.

Step 5 – Choose Keyboard Layout

Select your keyboard layout and then click ‘Next’.

 

Step 6 – Select Storage Devices

If you are installing CentOS 6.5 to your local Hard Drive, select "Basic storage Devices". If you have a storage area network (SANs) then select the bottom option. We will choose here "Basic storage Devices" and click ‘Next’.

Step 7 – Storage Device Warning

Click "Yes, discard any data" to continue, if there is no data on your Hard Drive where you are trying to install CentOS.

Step 8 – Set Hostname

Type your hostname and click ‘Next’.

Step 9 – Set Time Zone

Choose your time zone and click ‘Next’.

Step 10 – Set password for root user

Set the password for ‘root’ user which is actually your admin user same as ‘Administrator’ in Windows and click ‘Next’.

Step 11 – Select installation type

You will now be set with a bunch of options for your disk partitioning, as this is a new fresh install. We will use the top option "Use All Space". Choose the relevant option for your setup and tick the "Review and modify partition layout" option.

Step 12 – Review Partitions

Review the Partitions and modify it if required. Click ‘Next’ to continue.

 

Step 13 – Writing storage configurations to Hard Disk

Click ‘Write Changes to Disk’ if all of your desired partitions are created correctly.

Step 14 – Configure Boot Loader

With default options click ‘Next’.

Step 15 – Customize Package Selection

We will select ‘Minimal’ for this setup. This installs a minimal setup which is ideal for servers. Selecting this option will keep the install as lean as possible reducing potential security holes. If you are installing for a desktop you might want to choose one of the other options.

Step 16 – Starting Installation

Installation process window will appear.

Step 17 – Installation Completed

Installation process has been completed successfully.

Click ‘Reboot’ to finish the installation.
