Overview

Introduction

This tutorial will cover installing and basic configuration of Tomcat 7 version 7.0.42 on Red Hat Enterprise Linux Server Version 6.4 64 bit Operating System.

In this tutorial, we will install Tomcat 7, the new JDK 7, configure Tomcat as a service, create a start/stop script, and configure Tomcat to run under a non-root user.

Prerequisites

The released versions for use of Legato can be found in the system documentation "Legato_System_Installation_en.pdf".

:

Apache Tomcat 7.0 requires a Java Standard Edition version 6 or later. In this tutorial we recommend to install version 7 which is essential for Legato installation.

Download and install Java SE version 7 from:

http://www.oracle.com/technetwork/java/javase/downloads/index.html.

:

If you are running a 32-bit version of RHEL, you MUST use a 32-bit distribution of Apache Tomcat, as well as a 32-bit JDK. So you will need jdk-7u45-linux-i586.tar.gz for a 32-bit version of RHEL. If you are using a 64-bit system, you can technically run either a 32- or 64-bit distribution, as long as the bit architecture of the JDK you have installed matches your Tomcat distribution. So you will need the jdk-7u5-linux-x64.tar.gz for a 64-bit version of RHEL.

 

Click "Download" and proceed with downloading the appropriate version from the list.
