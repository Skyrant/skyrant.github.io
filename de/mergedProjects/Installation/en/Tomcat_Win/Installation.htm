Installation of Apache Tomcat 7.0 using the Windows EXE Distribution

Step 1 – Download Tomcat 7.0

 

Using the Windows EXE distribution is by far the easiest way to get Tomcat up and running on your Windows machine.  As this is a quick-start guide, it is recommended that you use this option. Browse http://tomcat.apache.org/ in your internet browser. Click on Tomcat 7.0 from the left sidebar to locate the download version of Apache Tomcat 7.0.

Download the latest release of Apache Tomcat that is found reliable and stable. In this tutorial we will be using the Tomcat version 7.0.42. The most convenient installation package is 32-bit/64-bit Windows Service Installer.

:

It is recommended to restart your Windows after installing Java and configuring the Environment Variables to have changes take place.

Step 2 – Start Setup

Download the installation file from the website and run the setup. The installation wizard will begin. Please click "Next" to proceed with the setup.

Step 3 – License Agreement

Apache Tomcat License Agreement will be presented to you which you need to accept before installation takes place. Read through the agreement carefully and click on "I Agree", if you completely agree with the license.

Step 4 – Choose Components

You get to have multiple types of installations. The average Tomcat user needs normal features available under the "Normal" installation package. Click on additional components if you want them on your computer and click "Next".

Step 5 – Basic Configurations

Apache Tomcat setup will prompt you to configure a few basic configurations of Apache Tomcat server like default HTTP Connector port, and your administrator login. Please keep the default settings for the HTTP Connector and Tomcat Administrator Login and click "Next".

Step 6 – JVM Path Selection

The next step asks you to locate the Java Virtual Machine. JVM is a prerequisite for installing Apache Tomcat Server. Browse to the location where you have installed the Java Virtual Machine. Click "Next" if the pre-selected location is correct.

Step 7 – Installation Location

Choose the install location for Apache Tomcat. Browse through the folder of your choice and click on "Install" to make the installation.

Step 8 – Installation Complete

The final step of installation wizard is going to ask you if you wish to execute Apache Tomcat or you wish to see the Readme file. Check "Run Apache Tomcat" to run Apache Tomcat server and click "Finish".

Step 9 – Confirm Tomcat Installation

Go to the web browser of your choice and write "localhost:8080″ as you have specified "8080″ to be the port number where Apache Tomcat is supposed to work and you’re in the Apache Tomcat Server.

Tomcat as a Windows Service

Running the Windows installer automatically installs Tomcat as a Windows Service.  Windows Services, similar to daemons on Unix based systems, are processes that are owned by the system, and work in the background without user intervention.  Running Tomcat as a service allows you to start Tomcat automatically on login, among other things.  

In order to configure the Tomcat service check the Apache feather icon in the systray (or when absent, run Monitor Tomcat from the start menu). Double-click the feather icon and navigate to the "General" tab, and choose "Automatic" for the "Startup type".

Additional Files

Configure Environment Variable

Step 1 - Locate the Tomcat Installation Directory

If you already know the installation path for the Tomcat, go to Step 2 below. Otherwise, find the installation path by following these instructions:

1. If you didn't change the installation path for the Tomcat during installation, it will be in a directory under C:\Program Files\Apache Software Foundation. Using Explorer, open the directory C:\Program Files\Apache Software Foundation.

2. Inside that path will be one or more subdirectories such as C:\Program Files\Apache Software Foundation\Tomcat 7.0. 

Step 2 - Set the CATALINA_HOME Variable

Once you have identified the Tomcat installation path:

1. Right-click the My Computer icon on your desktop and select Properties.

2. Click the Advanced tab.

3. Click the Environment Variables button.

4. Under System Variables, click New.

5. Enter the variable name as CATALINA_HOME.

6. Enter the variable value as the installation path for the Tomcat for example, C:\Program Files\Apache Software Foundation\Tomcat 7.0.

7. Click OK.

8. Click Apply Changes.

Step 3 – Set Tomcat in System PATH Variable

1. Right-click the My Computer icon on your desktop and select Properties.

2. Click the Advanced tab.

3. Click the Environment Variables button.

4. Under System Variables, click Path.

5. In the Edit windows, modify Path by adding the location of the Tomcat bin folder to the value for Path for example, C:\Program Files\Apache Software Foundation\Tomcat 7.0\bin.

 

:

It is recommended to restart your Windows after configuring the Environment Variables to have changes take place.
