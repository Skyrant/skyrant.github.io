Tomcat Parameters

General

:

The following parameters and their values are just only recommendation for a single user and for single instance of Web application running under Tomcat. It is strongly recommended to set the most value that you can afford for parameters such as Initial memory size, Maximum memory size and Thread size depending on your operating setup.

Tomcat7w Monitor Application

Tomcat7w is a GUI application for monitoring and configuring Tomcat services.

In order to configure the Tomcat memory settings check the Apache feather icon in the systray (or when absent, run Monitor Tomcat from the start menu). Double-click the feather icon and navigate to the "Java" tab, and set the values for Initial memory pool, Maximum memory pool and Thread stack size.

For the use of Legato, the following parameter values are recommended:

Initial memory pool (PermGen size): 512 MB

Maximum memory pool (the maximum amount of memory that can be allocated to the Java VM heap): 4096 MB
