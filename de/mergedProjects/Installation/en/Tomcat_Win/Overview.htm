Overview

Introduction

This tutorial is a guide to installing and configuring Apache Tomcat 7 version 7.0.42 for use as a standalone Web server on Windows Server 2008 R2 Operating System. There are several ways to set up Tomcat for running on Windows. Using the Windows EXE distribution is by far the easiest way to get Tomcat up and running on your Windows machine.  As this is a quick-start guide, it is recommended that you use this option.

Prerequisites

The released versions for use of Legato can be found in the system documentation "Legato_System_Installation_en.pdf".

:

Apache Tomcat 7.0 requires a Java Standard Edition version 6 or later. In this tutorial we recommend to install version 7 which is essential for Legato installation.

Download and install Java SE version 7 from:

http://www.oracle.com/technetwork/java/javase/downloads/index.html.

:

If you are running a 32-bit version of Windows, you MUST use a 32-bit distribution of Apache Tomcat, as well as a 32-bit JDK. If you are using a 64-bit system, you can technically run either a 32- or 64-bit distribution, as long as the bit architecture of the JDK you have installed matches your Tomcat distribution.  However, the Tomcat installer automatically requires that 64-bit operating systems use the 64-bit version of Tomcat, so you'll have to manually install the lower-bit version.  

Click "Download" to start downloading Java SE 7 from the Oracle website.
