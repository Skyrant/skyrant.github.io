Das Produkt ist für nachfolgende Betriebssysteme, unterteilt nach den jeweiligen Produktbestandteilen, freigegeben:

 

Datenbank

Das aktuelle Release von Legato ist für den Betrieb unter Oracle 12.1 und Oracle 12.2 (alias 18) freigegeben. Bezüglich der von Oracle freigegebenen Betriebssysteme zum Betrieb des Datenbank-Managementsystems (DBMS) verweisen wir auf Oracle: http://www.oracle.com/index.html.

 

Web Applikation

Das User-Frontend von Legato ist durchgängig webbasiert in Form von Java-Servlets implementiert. Dieses ist daher für alle gängigen Betriebssysteme freigegeben, die den Betrieb eines Webservers mit Servlet-Engine (mind. Java 1.8) ermöglichen. Zu den aktuell freigegebenen Betriebssystemen zählen:

Windows XP / 7 (32Bit) / 7 (64Bit) / 10 (32Bit) / 10 (64Bit)

Windows Server 2008R2 / 2012

Linux (alle gängigen Distributionen von Suse, Red Hat, Ubuntu etc.)

AIX 5, 6, 7

HP-UX 11i

 

Application Engine

Auch die Application Engine ist durchgängig in Java implementiert und benötigt daher mind. Java 1.8 Unterstützung. Zu den aktuell freigegebenen Betriebssystemen zählen:

Windows XP / 7 (32Bit) / 7 (64Bit) / 10 (32Bit) / 10 (64Bit)

Windows Server 2008R2/2012

Linux (alle gängigen Distributionen von Suse, Red Hat, Ubuntu etc.)

AIX 5, 6, 7

HP-UX 11i

 

Windows Gateway (bis Legato 4.7.2.4; danach Ablöse durch Sapient Cloud Connector 3.0.0)

Wird im Projekt eine Kopplung über OPC gefordert, kann ausschließlich die Windows-Variante des Gateways eingesetzt werden. Dieses ist freigegeben für:

Windows Server 2008 R2 / 2012 R2 / 2016

 

Sapient Cloud Connector Windows (ab Version 3.0.0)

Wenn das Projekt eine Verbindung über OPC benötigt, kann nur die Windows Variante des Sapient Cloud Connectors genutzt werden. Dieser ist freigegeben für:

Windows Server 2012 / 2012R2 / 2016

Windows Server 2012 EN / 2012R2 / 2016 EN

 

Unix Gateway (bis Legato 4.7.2.4; danach Ablöse durch Sapient Cloud Connector 3.0.0)

Der Code des Gateways basiert auf GNU C++. Ein Kompilieren des Codes auf dem Zielsystem muss möglich sein. Daher ist sicherzustellen, dass der GNU-Compiler auf dem System auch real verfügbar ist. Folgende Betriebssysteme sind freigegeben:

Linux (alle gängigen Distributionen von Suse, Red Hat, Ubuntu, etc.)

AIX 6, 7

 

Sapient Cloud Connector Unix (ab Version 3.0.0)

Der Code des Cloud Connectors basiert auf GNU C++. Ein Kompilieren des Codes auf dem Zielsystem muss möglich sein. Daher ist sicherzustellen, dass der GNU-Compiler auf dem System auch real verfügbar ist. Folgende Betriebssysteme sind freigegeben:

Linux (alle gängigen Distributionen von Suse, Red Hat, Ubuntu, etc.)

 

Legato Configuration Center

Zum Administrieren der Legato Installation ist ein Windows-basiertes System erforderlich. Folgende Systeme sind freigegeben:

Windows 7 (64Bit) / 8 (64Bit) / 10 (64Bit)

Windows Server 2008R2 / 2012 / 2012R2 / 2016

 

Web-Client

Als User-Frontend ist ein Rechner mit Web-Browser erforderlich. Folgende Browser sind derzeit für den Betrieb freigegeben:

Mozilla Firefox 55.0

Mozilla Firefox ESR 45.9.0

Internet-Explorer 11

Microsoft Edge 38.14393.1066.0

 

Bei Verwendung des IE11 ist darauf zu achten, dass die Checkbox "Intranetsites im Kompatibilitätsmodus öffnen" in den Einstellungen nicht aktiviert ist, wenn der Legato Web-Client im Intranet betrieben werden soll.
