In dieser Konfigurationsdatei sind allgemeine Einstellungen (Abschnitt [Common]) und Datenbankeinstellungen (Abschnitt [Database]) enthalten.

Die Einträge haben folgende Bedeutung

Abschnitt [Common]

Port (Listenport des Servers für eingehende TCP Verbindungen der Clients)

MediaClientsConfigPath (Konfigurationspfad der Clients)

WavFilesPath (Pfad der WAV Dateien, in dem sich die WAV Dateien auf den Windows-Clients befinden)

 

Abschnitt [Database]

Connect (Name der Legato Datenbank)

User (Datenbankbenutzer)

Password (verschlüsseltes Datenbankpasswort)

ProjectId (Kennung des Medienservers für den Aufruf von P_SYNC_SYSMODULE)

SQLMsgStmt (SQL Statement zum Selektieren der Meldungen)

 

Konfiguration der Meldearten (AlTypes.cfg)

[Config]

AlTypesCnt=2

 

[AlType1]

Type=11

BitNo=1

Melody=1

Expression="\D*\d+(.*)?"

 

[AlType2]

Type=12

BitNo=0

Melody=2

Expression="\D*\d+(.*)?"

 

In dieser Konfigurationsdatei befindet sich die Konfiguration der Meldearten.

Die Einträge haben folgende Bedeutung

Abschnitt [Config]

AlTypesCnt (Anzahl der Meldearten)

 

Abschnitte [AlType1, AlType2, …]

Type (Meldeart)

BitNo (BitNummer, die in der Statusvariablen gesetzt wird, die Zählung beginnt mit 0)

Expression (Regular Expression um die Taktnummer aus der Symbolik zu ermitteln)

Melody (0 = keine Melodie abspielen (default), 1 = langsame Melodie abspielen, 2 = schnelle Melodie abspielen)

StatusVarsConnectionId (VerbindungsId unter der das Konfigurationsprogramm MediaConfig die Statusvariablen anlegt)

SectionsCnt (Anzahl der Bandabschnitte der Montagelinie)

Section1Name, Section2Name … (Namen der Bandabschnitte der Montagelinie)

Der Dateiname einer Wavdatei setzt sich folgendermaßen zusammen:

M<NR><1/2>.wav

|  |   |     |

|  |   |     |-------- Extension (.wav)

|  |   |

|  |   +-------------- 1 für Melodie langsam, 2 für Melodie schnell

|  |

|  +------------------ dreistellige Nummer der Melodie rechtsbündig (ggf. links

|                      mit Nullen aufgefüllt)

+--------------------- fixes Prefix M

In dieser Konfigurationsdatei befindet sich die Zuordnung der Melodien zu Gruppen der Montagelinien. Die Einträge in dieser Datei werden vom Konfigurationsprogramm MediaConfig generiert.

Abschnitt [Config]
