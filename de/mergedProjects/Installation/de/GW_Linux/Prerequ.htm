Betriebssystem

Allgemein

Das Gateway kann sowohl als 32 Bit als auch als 64 Bit Applikation compiliert werden.

Das 32-Bit-Gateway ist sowohl unter 32 Bit als auch unter 64 Bit Linux Betriebssystemen lauffähig. Das 64-Bit-Gateway ist nur unter 64 Bit Linux Betriebssystemen lauffähig.

Für TCP-Native und ISO on TCP Kommunikation (RFC 1006) ist nur das Gateway und kein Telegrammhandler erforderlich. Dabei ist das Gateway der aktive Kommunikationspartner, passive Verbindungen seitens des Gateways werden nicht unterstützt.

:

Bibliotheken, mit denen das Gateway gelinked wird, müssen in der zugehörigen 32 bzw. 64 Bit Variante vorhanden sein!

Programme

Für das Compilieren des Gateways sind folgende Programme erforderlich:

g++ (GNU C++ Compiler)

make (GNU Make Tool), Version >= 3.80

gunzip

Bibliotheken

32 Bit

Wenn das Gateway als 32 Bit Applikation compiliert wird, müssen folgende Bibliotheken (in der 32 Bit Version) auf dem Gatewayrechner vorhanden sein (Beispiel Ubuntu 14.04):

linux-gate.so.1

libclntsh.so.11.1

librfccm.so

libACE.so.6.3.0

libstdc++.so.6

libm.so.6

libgcc_s.so.1

libpthread.so.0

libc.so.6

libnnz11.so

libdl.so.2

libnsl.so.1

librt.so.1

/lib/ld-linux.so.2

libaio.so.1

64 Bit

Wenn das Gateway als 64 Bit Applikation compiliert wird müssen folgende Bibliotheken (in der 64 Bit Version) auf dem Gatewayrechner vorhanden sein (Beispiel Ubuntu 14.04):

linux-vdso.so.1

libclntsh.so.11.2

librfccm.so

libACE.so.6.3.0

libstdc++.so.6

libm.so.6

libgcc_s.so.1

libpthread.so.0

libc.so.6

libnnz11.so

libdl.so.2

libnsl.so.1

libaio.so.1

librt.so.1

/lib64/ld-linux-x86-64.so.2

Oracle Client

Der Oracle Client muss in der Version >= 11.2.0.3.0 komplett mit allen Headerdateien,  Bibliotheken und OCI (Oracle Call Interface) installiert sein.

Die Umgebungsvariable ORACLE_HOME muss den Pfad des Oracle Clients enthalten, die Umgebungsvariable LD_LIBRARY_PATH muss den Pfad der Bibliotheken des Oracle Clients enthalten. Die Umgebungsvariable NLS_LANG ist entsprechend der gewünschten Ländereinstellung zu setzen.
