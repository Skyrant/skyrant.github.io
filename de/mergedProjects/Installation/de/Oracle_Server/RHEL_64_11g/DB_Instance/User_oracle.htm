User "oracle" einrichten

Für den Produktivbetrieb der Datenbank editieren Sie bitte die Login-Skripte .bash_profile und .bashrc des Betriebssystem-Users oracle und hängen Sie unten die folgenden Zeilen an:

 

# Oracle-Umgebungsvariablen setzen:

ORACLE_SID="LEGATO"; export ORACLE_SID

# Abfrage unterdrücken:

ORAENV_ASK="NO"; export ORAENV_ASK

. /usr/local/bin/oraenv

 

Die Zeile

. /usr/local/bin/oraenv

 

ist gültig für die KornShell, Bourne Shell und Bash (Bourne-again Shell). Wenn Sie die C-Shell verwenden, dann schreiben Sie stattdessen

. /usr/local/bin/coraenv

 

Dadurch erhält der Betriebssystem-User oracle bei jedem Einloggen automatisch die richtigen Umgebungsvariablen für den Betrieb der Datenbank zugewiesen.
