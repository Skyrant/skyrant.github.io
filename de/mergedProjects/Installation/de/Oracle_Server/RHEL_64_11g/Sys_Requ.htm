Systemvoraussetzung

Hardwarevoraussetzungen

ein 64-Bit-Prozessor (erkennbar z. B. an der Antwort "x86_64" auf den Befehl "uname -m")

64-Bit-Linux-Betriebssystem mit ausreichend hoher Kernel-Version (vgl. http://docs.oracle.com/cd/E11882_01/install.112/e24326/toc.htm#BHCGGJAB)

ausreichend dimensionierte Hardware (vgl. http://docs.oracle.com/cd/E11882_01/install.112/e24326/toc.htm#i1011296). Absolute Minimalvoraussetzungen:

1 GB RAM (2 GB RAM dringend empfohlen)

1,5 bis 16 GB Swap-Space (auf der Festplatte), abhängig vom RAM-Ausbau

6 GB Festplattenplatz für Software und Daten

Softwarevoraussetzungen

Betriebssystem

der Server ist bis auf die Installation des Betriebssystems und einiger üblicher Utilities im Wesentlichen "blank"; das Betriebssystem enthält nur den User root und einen interaktiven Standard-User (z. B. "gefasoft"); insbesondere hat es vorher keine Installation eines Oracle-RDBMS gegeben

Benutzerzugang zum Server mit dem User root (entweder direkt/interaktiv oder über VNC/X-Window System o. ä.)

Oracle-Spezifika

bestimmte Package-, Compiler- und sonstige Software-Anforderungen (Abschnitte 4.3 bis 4.5 in der Anleitung, siehe abhttp://docs.oracle.com/cd/E11882_01/install.112/e24326/toc.htm#BHCGJCEA), die hier nicht weiter ausgeführt werden

die Installationsdateien für Oracle 11gR2 64 Bit
