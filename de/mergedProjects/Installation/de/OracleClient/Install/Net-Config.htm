Net-Konfigurationsassistent

Nachdem die Oracle-Client Software am PC installiert wurde, kann ein Assistent zur Einrichtung der Verbindung zur Datenbank über das Windows Startmenü aufgerufen werden. Standardmäßig ist dieser Assistent zu finden unter

Startmenü\Programme\Oracle – OraClient11g_home1\Konfigurations- und Migrations-Tools\Net-Konfigurationsassistent

 

Der Assistent führt durch den Verbindungsaufbau zur Datenbank.

Schritt 1 - Konfiguration

Wählen Sie "Konfiguration von lokalem Net-Service Name" aus und klicken Sie auf "Weiter".

Schritt 2 - Konfiguration

Wählen Sie "Hinzufügen" aus.

Schritt 3 - Dienstname

Geben Sie als "Service-Name" (Dienstname) den Oracle-Datenbanknamen auf dem Server an. In diesem Beispiel ist das "Legato".

Schritt 4 - Netzwerkprotokoll

Wählen Sie als Netzwerkprotokoll "TCP" aus.

Schritt 5 – Hostname

Im Feld "Host name" ist entweder der Rechnername oder die IP-Adresse des Datenbankservers anzugeben. Als Port muss der Port eingestellt werden, der auch bei der Datenbank eingestellt ist. Im Beispiel ist dies der Standard-Port 1521. Klicken Sie "Weiter".

Schritte 6-8 – Verbindung testen

Sie haben nun die Möglichkeit, einen Verbindungstest zur Oracle-Datenbank durchzuführen. Wählen Sie dazu "Ja, ein Test soll durchgeführt werden" aus und klicken Sie dann auf "Weiter".

Da noch kein Benutzername und Passwort zum Zugriff auf die Datenbank eingegeben wurden, erscheint zunächst eine Fehlermeldung:

Ein Klick auf den Button "Anmeldung ändern" öffnet einen Dialog, in dem Benutzername und Kennwort für die Datenbank eingegeben werden können.

Ein erneuter Verbindungstest sollte erfolgreich sein.

Schritt 9 – Net-Service Name

Abschließend muss noch ein Name für den konfigurierten Net-Service eingegeben werden. In diesem Beispiel verwenden wir den Namen "Workshop".

Danach kann die Konfiguration abgeschlossen und der Assistent beendet werden.
