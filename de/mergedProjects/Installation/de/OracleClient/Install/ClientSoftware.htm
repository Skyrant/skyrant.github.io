Installation der Client-Software

Die Oracle Client-Software ist im Lieferumfang der Datenbanklizenz von Oracle enthalten bzw. kann über die Homepage von Oracle per Download geholt werden. Es ist darauf zu achten, dass die Version des Clients zur Version der aktuell eingesetzten Datenbank kompatibel ist.

Starten Sie die Setup.exe des Oracle Clients. Über eine DOS-Eingabeaufforderung wird automatisch der Oracle-Universal-Installer gestartet. Danach öffnet sich das Oracle Client Installationsprogramm.

Schritt 1 - Installationsart

Wählen Sie als Installationsart "Benutzerdefiniert".

Schritt 2 – Softwareupdates

Überspringen Sie die Softwareupdates. Diese Maske erscheint nur bei der Installation von Version 11.2.0.3.

Schritt 3 – Produktsprachen

Wählen Sie die gewünschten Produktsprachen. In unserem Beispiel sind dies Deutsch und Englisch.

Schritt 4 – Installationsverzeichnis angeben

Ändern Sie gegebenenfalls das Oracle Base-Verzeichnis auf C:\oracle\

Prüfen Sie, dass das Softwareverzeichnis auf …\client_1 endet.

Schritt 5 – Produktkomponenten

Wählen Sie die benötigten Produktkomponenten aus.

Die folgenden Komponenten werden benötigt:

Oracle Database Utilities

Oracle Java Client

SQL *Plus

Oracle JDBC/THIN Interfaces

Oracle Call Interface (OCI)

Oracle Net

Oracle Connection Manager

Oracle ODBC Driver

Oracle SQL Developer

Oracle Objects for OLE

Oracle Provider for OLE DB

Schritt 6 – Installation ausführen und abschließen

Prüfen Sie die Zusammenfassung und führen Sie dann die Installation aus.
