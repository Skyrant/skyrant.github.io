Parameter Informationen

Die Beschreibung der Parameter erfolgt nach einem festen Schema. Für jeden Parameter werden folgende Informationen ausgegeben:

  • <Bezeichnung>

  • Typ: <Systemparameter>

  • ID: <Zahl>

  • Default: <Bool oder numerisch oder String>

  • Format: <Bool oder numerisch oder String>

  • Beschreibung: <Text>

 

  • Bezeichnung

Die Bezeichnung beschreibt den Parameter. Sie dient nicht dazu, den Parameter im System wiederzufinden, sondern soll kurz und prägnant auf die Funktion des Parameters hinweisen. Identifiziert wird der Parameter über das Feld ID oder Name.

  • Typ

Die Information gibt Hinweis darauf, an welcher Stelle der Parameter im System zu ändern ist.

Die Systemparameter werden LC2 unter Systemparameter verwaltet. Um einen Systemparameter in der Parameterauflistung wiederzufinden, bedient man sich der ID.

  • ID

Dient zur Identifikation eines Systemparameters.

  • Default

Beinhaltet den Default-Wert des Parameters. Dieser kommt zum Tragen, falls der Parameter nicht gesetzt ist bzw. der Wert des Feldes vom System als null erkannt wird. Die Angabe kann je nach Parameter ein boolescher Wert, eine Zahl (numerisch) oder ein beliebiger Text (String) sein. Der Text kann auch eine Zeichen-separierte Liste enthalten. Die Beschreibung kann Hinweise auf den möglichen Inhalt geben.

  • Format

Gibt das Format des Wertes an. Zur Auswahl stehen ein boolescher Wert (0 = false oder 1 = true), eine Zahl (numerisch) oder ein Text (String). Der Text kann auch eine Zeichen-separierte Liste enthalten.

  • Beschreibung

Dieses Feld kann einen beliebigen Text enthalten. Der Text dient zur näheren Beschreibung des Parameters oder auch zur Angabe von möglichen Werten, die der Parameter annehmen kann.
