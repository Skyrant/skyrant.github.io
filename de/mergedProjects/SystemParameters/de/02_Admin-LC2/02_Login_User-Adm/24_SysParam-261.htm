Web-App secure login

Typ:

Systemparameter

Name:

Web-App secure login

ID:

-261

Default:

0

Format:

Bool

Beschreibung:

Legt fest, ob die Anmeldefunktion für den LC2 und die WebApp per http (false / 0) oder https (true / 1) aufgerufen werden soll. Ist dieser Parameter auf 1 gesetzt, muss die Sapient Engine entsprechend konfiguriert sein.
