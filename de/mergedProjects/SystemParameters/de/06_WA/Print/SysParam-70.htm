Auflösung der Druckvorschau

Typ:

Systemparameter

Name:

Auflösung der Druckvorschau

ID:

-70

Default:

0

Format:

String

Beschreibung:

Gibt die Auflösung der Druckvorschau und des Drucks im Querformat DIN A4 an. Muss mit einem Wert im Format HöhexBreite belegt werden. Empfohlen wird eine Auflösung von 150 PPI. Folgende Werte können eingegeben werden:

72 PPI

595x842

96 PPI

794x1123

150 PPI

1240x1754

300 PPI

2480x3508

600 PPI

4960x7016

720 PPI

5953x8419

1200 PPI

9921x14032
